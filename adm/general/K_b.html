Base-Ionization Equilibria

Equilibria involving weak bases are treated similarly to those for weak acids.

B(aq) + H2O(l) ⇌ BH+ (aq) + OH− (aq)

The corresponding equilibrium constant is

K c = [ B H + ] [ OH − ] [ B ] [ H 2 O ]

Because the concentration of H2O is nearly constant, you can rearrange this equation as you did for acid ionization, and write Kb = [H2O]Kc

K b = [ B H + ] [ OH − ] [ B ]

Kb always refers to the reaction of a base with water to form the conjugate acid and the hydroxide ion.

Ammonia, for example, ionizes in water as follows:

NH3 (aq) + H2O(l) ⇌ NH4+ (aq) + OH− (aq)

Kb = [NH4+][OH−]/[NH3][H2O]

Relating the Ionization Constants for an Acid and Its Conjugate Base

Examing the values of Ka of an acid the the corresponding value Kb of the conjugate base of the acid we can note that as the strengths of the acids decline (Ka becomes smaller), the strengths of their conjugate bases increase (the values of Kb increase), and in particular the following relation holds:

Ka x Kb = Kw

Consider the specific case of the ionization of a weak acid, say HCN, and the interaction of its conjugate base, CN−, with H2O.

Weak acid:      HCN(aq) + H2O(l) ⇌ H3O + (aq) + CN− (aq)   Ka = 4.0 × 10−10
Conjugate base:  CN− (aq) + H2O(l) ⇌ HCN(aq) + OH− (aq)   Kb = 2.5 × 10−5

2H2O(l) + H3O+ (aq) + OH− (aq)   Kw = 1.0 × 10−14

Ka x Kb = [H3O+][CN−]/[HCN] x [HCN] [OH−]/[CN−] = Kw

This realtion is useful because Kb can be calculated from Ka.

If we take the logarithm of the realtion we obtain also

pKw = pKa + pKb

Let's see some example of reaction between acidds and bases.

  1. Can HF react with CL− as follows?

    HF + Cl− ⟶ HCl + F−

    This reaction doesn't occur since HF is a weak acid, it cannot form a stronger acid as HCl from the base Cl−.

  2. Can Formic acid react with ammonia as follows?

    HCOOH + NH3 ⟶ HCOO− + NH4+

    The reaction can occur in the sense indicated as Ka(HCOOH) > Ka(NH4+) and ammonia is a stronger base than formiate ion, Kb(NH3+) > Kb(HCOO−).

  3. Can Hydrogen sulfide, H2S form HNO2 form a solution of nitrite ion NO2−, as follows?

    H2S + NO2− ⟶ HNO2 + HS−

    H2S is a weaker base than HNO2 thus it cannot protonate it conjugate base NO2−; and on the other hand NO2− is too weak as base to deprotonate H2S to form HS−. The reaction can occur in the opposite direction.
