Bilanciamento Reazioni

  1. Bilanciare la seguente reazione di ossidoriduzione in forma molecolare (rappresentando compiutamente le semireazioni coinvolte): CH3CH2OH + K2Cr2O7 + H2SO4 -> K2SO2 + Cr2(SO4)3 + CH3CHO + H2 O
  2. Bilanciare la seguente reazione di ossidoriduzione, che procede in ambiente acido, rappresentando compiutamente le semireazioni coinvolte: K2Cr2O72- + H2C2O4 --> Cr3+ + CO2

  3. Bilanciare la seguente reazione di ossidoriduzione: ICl3 + H2O --> I2 + HIO3 + HCl

  4. Bilanciare la seguente reazione, che procede in ambiente basico, con il metodo delle semireazini: ClO2 + H2O2 --> ClO2- + O2
  5. Porre i corretti coefficienti stechiometrici alla seguente reazione di ossido-riduzione scritta in forma molecolare: MnO + PbO2 + HNO3 -- > HMnO4 + Pb(NO3)2 + H2O
