Equilibrium Expressions Involving Pressures

So far we have been describing equilibria involving gases in terms of concentrations. For a chemical equilibrium

aA + bB ⇌ cC + dD

Representing the concentration of all species in mol ⋅ l−1 the equilibrium constnat K is represented by the symbol Kc adn written as:

Kc = [C]c[D]d/[A]a ⋅ [B]b

For gaseous species it is possible to replace the concentration with the partial pressure. In this case the equilibrium constnat K is represented by the symbol Kp adn written as:

Kc = pcpd/pa ⋅ pb

We note that each partial pressure is raised to a power corresponding to the stoichiometric coefficient. The relationship between the pressure and the concentration of a gas can be seen from the ideal gas equation:

n/V = p/RT = M   or   p = RTn/V = MRT

Where M represents the molar concentration.

K p = ( n c V ) c ⋅ ( n d V ) d ( R T ) c ⋅ ( R T ) d ( n A V ) a ⋅ ( n B V ) b ( R T ) a ⋅ ( R T ) b

Recalling that

( n c V ) c ⋅ ( n d V ) d ( n A V ) a ⋅ ( n B V ) b = K c

we have

Kp = Kc(RT)Δn

where Δn = (c + d) − (a + b).

It is clear that when Δn = 0, we have that Kc = Kp. This condition is verified when the moles of reactants is equal to the moles of products as in the following case:

H2 + Cl2 ⇌ 2HCl

If we consider a gas phase for which Δn ≠ 0:

N2 + 3H2 ⇌ 2NH3

in terms of the equilibrium partial pressures of the gases,

Kp = pNH32/(pN2)(pH23)

Kc does not not correspond to Kp.
