Lunghezza di un arco di curva

Un problema molto naturale di calcolo infinitesimale è quello di definire cosa si intenda per lunghezza di un arco di curva, e calcolaral. Un modo altrettanto naturale di impostare il problema è quello di considerare una spezzata inscritta sull'arco di curva e calcolarne la lunghezza. Approsimando sempre meglio, la curva con spezzate via via più fini (cioè con un numero crescente di segmenti, di lunghezza sempre più piccola), si dovrebbe ottenere, come limtie, la lunghezza della curva stessa.
Consideriamo per semplicità un arco di curva piana e regolare r: [a,b] ⟶ ℝ2. Per mettere in pratica questa idea, suddividiamo l'intervallo [a,b] in n intervalli uguali mediante punti t0 = a, t1, t2, ..., tn = b e consideriamo i punti corrispondenti sulla curva:

(x(ti), y(ti)), i = 0,1,2,..,n

La lunghezza della spezzata che unisce questi punti è

∑ i = 1 n ( x ( t i ) − x ( t i − 1 ) ) d s = x ′ ( t ) 2 + y ′ ( t ) 2 d t 2 + ( y ( t i ) − y ( t i − 1 ) ) 2

Applicando alle funzioni x(t), y(t) il teorema di Lagrange su ognuno degli intervallini [ti − 1, ti], si trova che esistono punti ξi, ηi ∈ [ti − 1, ti] tali che l'espressione appena scritta è uguale a

∑ i = 1 n [ x ′ ( ξ i ) ( t i − t i − 1 ) ] 2 + [ y ′ ( η i ) ( t i − t i − 1 ) ] 2 = ∑ i = 1 n ( t i − t i − 1 ) x ′ ( ξ i ) 2 + y ′ ( η i ) 2

Poiché per n ⟶ ∞ l'ampiezza degli intervallini tende a zero, sarà ξi ≃ ηi e il limite dell'espressione appena scritta coincide col limite delle somme di Cauchy della funzione x ′ ( t ) 2 + y ′ ( t ) 2 . La lunghezza della curva sarà dunque uguale all'integrale

L = ∫ a b x ′ ( t ) 2 + y ′ ( t ) 2 d t

Si pone anche

che si chiama lunghezza d'arco elementare. Il suo significato intuitivo è quello di spazio percorso dal punto mobile nell'intervallo di tempo "infinitesimo" dt. Si può anche vedere ds al modo seguente: nell'intervallo di tempo [t, t + dt] il punto si sposta di un incremento (dx,dy) dato approsimativamente da dx = x'(t) dt, dy = y'(t)dt (abbiamo approssimato l'incremento della funzione con il suo differenziale, come è lecito fare, a meno di infinitesimiu di ordine superiore). Lo spazio percorso è quindi in prima approssimazione

( d x ) 2 + ( d x ) 2 = ( x ′ ( t ) )Rcos θ 2 + ( y ′ ( t ) ) 2 d t

cioè, ds. (Abbiamo applicato una versione infinitesimale del Teorema di Pitagora, analogamente a quanto già visto a proposito del calcolo della lunghezza del grafico di una funzione).
Si noti che la quantità sqrt[x'(t) + y'(t)] non è altro che il modulo del vettore velocità. Si può anche scrivere, quindi, per una generica curva regonale in ℝm,

ds = |r'(t)| dt   ∫ba f(x) dt

La formula della lunghezza è quindi una riformulazione integrale della relazione "spazio = velocità x tempo".

Esempio 2.5.1. Per un arco di circonferenza

x = Rcos θ θ ∈ [θ1, θ2]
y = Rsin θ

è r'(θ) = (−R sin θ, R cos θ), |r'(θ)| = R, e ds = Rdθ; L = (θ2 − θ1) ■

Esempio 2.5.3. Per l'arco di elica cilindrica

x = R cos t t ∈ [0,2π]
y = R sin t
z = pt

si ha:

r'(t) = (R sin t, R cos t, p)

d s = x ′ ( t ) 2 + y ′ ( t ) 2 + z ′ ( t ) 2 d t = R 2 + p 2 d t L = ∫ 0 2 π d s = 2 π R 2 + p 2

Esempio 2.5.4. Per una curva piana in forma polare

ρ = f (θ)

ovvero r(θ) = (f(θ) cos θ, f (θ) sin θ) si ha

f' (θ) cos θ − f (θ) sin θ, f' (θ) sin θ + f (θ) cos θ)

dunque

d s = f ( θ ) 2 + f ′ ( θ ) 2 d θ

Ad es. la lunghezza dell'arco di spirale di Archimede ρ = θ per θ ∈ [0, 2π] è

Parametro arco

La lunghezza dell'arco r(τ) per τ da t0 a t è una funzione di t:

s ( t ) = ∫ t 0 t | r ′ ( τ ) | d τ

Se si è in grtado di calcolare esplicitamente tale funzione, e poi di invertirla, esprimendo t come funzione di s, è possiible riparametrizzare la curva in funzione del parametro arco s.
Ad es., per l'elica cilindrica dell'esempio precedente è

s ( t ) = ∫ t 0 t R 2 + p 2 d τ = t R 2 + p 2

da cui t = s/sqrt(R2 + p2). Si può allora riparamettrizzare la curva come
