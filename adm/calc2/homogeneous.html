Euler's theorem on Homogeneous Functions and Total Derivative

A function is which every term is of the same degree is known as a homogeneous function of that degree.
For instance, let

f(x,y) = a0xn + a1xn−1y + a2xn−2y2 + ... + an−1 xyn − 1 + anyn

be a function of x and y. We see that every term of this function has the same degree; i.e. n. Hence, it is a homogeneous function of x and y of degree n. We can rewrite it as

f(x,y) = xn[a0 + a1 (y/x) + a2(y/x)2 + ... + an−1 (y/x)n − 1 + anyn + an(y/x)n] = xnF(y/x)

Thus, every homogeneous function of x and y of degree n can be expressed in the form

xnF(y/x)

whther F may have any vlaue, e.g. x5 sin y/x, x4 tan−1 y/x; x3 log x/; are homogeneous functions of degree 5,4,3 respectively.

A test for Homogeneity of a function of x and y

Let f(x,y) be any function of x and y. Then f(x,y) is said to be a homogeneous function of x and y of degree n, if

f(tx,ty) = tnf(x,y)

Example. The function

f(x,y) = xyex/y

is a homogenous function of degree 2, since

f(t ⋅ x, t ⋅ y) = (tx) (ty) etx/ty = t2xyey/x = t2 ⋅ f(x,y)

Euler's theorem on homogeneous functions

Statement: If u is a homogeneous function of x and y of degree n, then

x ∂ u ∂ x + y ∂ u ∂ y = n u  (1)

Proof. Since u is homogeneous function x and y of degree n, hence we can write

u = xnF (y/x)

Note: Euler's theorem can be extended to a homogeneous function of any number of variables. Thus, if f(x1, ..., xn) is a homogeneous function of n variables, then

x 1 ∂ f ∂ x 1 + x 2 ∂ f ∂ x 2 + … + x n ∂ f ∂ x n = n f

It is of interest to see the connection between homogenous functions and extensive thermodynamic properties.
