Entropy function

Inverting the fundamental relation U(S,V,N) with respect to S, we get the entropy S(U,V,N) as a function of the independent variables, U, V, N. In fact for an infinitesimal change

dU = TdS - pdV

hence

dS = dU/T + (p/T)dV

which states that S = S(U,V), for a closed system.

At constant volume, the equation U(S,V) is the inverse of the equation of state S(U,V). Thus, these functions have a symmetry with respect the bisector of the first quadrant in the (S,U) plane.
