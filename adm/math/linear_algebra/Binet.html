Elementary properties of the determinant

3.3.1 Theorem. Let A an n x n matrix

  1. If A has a row or column of zeros then det A = 0;

  2. If two row (columns) are interchanged the determinant will change sign;

  3. If A has two equal rows or columns, then det A = 0;

  4. The determinant is a linear function of each row (or column), i.e.

    d1

    det ( a 1 + b 1 a 2 … a n ) = det ( a 1 a 2 … a n ) + ( b 1 a 2 … a n )

    d2

    det ( λ a 1 a 2 … a n ) = λ det ( a 1 a 2 … a n )

    Where ai are the row vectors in the matrix.

  5. If a linear combination of rows (columns) is added to a row, the determinant is unchaged.

  6. If the rows (columns) of A are linearly independent vectors, det A = 0;

  7. det (λA) = λn det A.

  8. If A is triangular (in particular, diagonal) det A = a11 ⋅ a22 ⋅⋅⋅ ann.

  9. Suppose that A is upper triangular and calculate det A by Laplace expansion along the first column. Then det A = a11A11. But also the matrix whose determinant is A11 is upper triangular, thus by calculating A11 in the same way recursevely, we have the claim.

Proof.

  1. It suffices to calculate det A using Laplace expansion along the row made up by zeros;

  2. We use induction. For n = 2 the conclusion may be checked directly (this is the induction base). We assume the property to hold for (n − 1) x (n − 1) matrices with n ≥ 2 (induction hypothesis) and prove it for n x n matrices. If we perform a Laplace expansion along the k row of the n x n matrix, which has not been moved, then the n cofactors will be determinants of (n − 1) x (n − 1) matrices whose sign has changed with respect to the original matrix. Thus the sign of the determinant of the n x n matrix has changed sign.

  3. By exchanging two equal rows the determinant must change sign by b); However the matrix is the same, hence the determinant is unchanged; The only possibility is that det A = − det A = 0;

  4. We calculate the determinant by Laplace expansion along the 1st row, by letting

    a1 = (a11, a12, ..., a1n), b = (b11, b12, ..., b1n)

    det ( a 1 + b 1 a 2 … a n ) = ( a 11 + b 11 ) A 11 + … + ( a 1 n + b 1 n ) A 1 n = ( a 11 A 11 + … + a 1 n A 1 n ) + ( b 11 A 11 + b 1 n A 1 n ) = det ( a 1 a 2 … a n ) + det ( b 1 a 2 … a n )

    Analogously the other property is verified.

  5. We indicate the rows of the A with the vectors a1, a2, ..., an we have using point d)

    det ( a 1 + λ 2 a 2 + λ 3 a 3 + … + λ n a n a 2 … a n ) = det ( a 1 a 2 … a n ) + λ 2 det ( a 2 a 2 … a n ) + … + λ n det ( a n a 2 … a n ) = det ( a 1 a 2 … a n )

  6. If the rows are linearly independent vectors, one of them is a linear combination of the others, e.g. say the first row is a linear combination of the other rows, then:

    A = ( λ a 2 + λ 3 a 3 + … + λ n a n a 2 … a n )

    Then by point d)

    det A = λ 2 det ( a 2 a 2 … a n ) + … + λ n det ( a n a 2 … a n ) = 0

    By point c), since each of the matrices above has two rows that are equal.

  7. We apply n times property d2)

    8. det ( λ A ) = det ( λ 1 a 1 λ 2 a 2 … λ n a n ) = λ 1 det ( a 1 λ 2 a 2 … λ n a n ) = λ 2 ( a 1 a 2 … λ n a n ) = … = λ n det ( a 1 a 2 … a n ) = λ n det A

Example. We use the previous theorem to simplify the calculation of the determinat. Let's calculate the determinant of the following matrix

A = ( 2 1 2 1 3 1 4 − 1 − 2 )

If we subtract from the first row the double of the second, we obtain a row with two zeros. By point e) of the theorem this does not alter the determinant.

det ( 0 − 5 0 1 3 1 4 − 1 − 2 ) = 5 | 1 1 4 − 2 | = 5 ( − 2 − 4 ) = − 30

Binet's Theorem

Binet's theorem. If A and B are matrices of the same order then

det AB = det A ⋅ B

Sarrus' rule

Sarrus' rule allows to calculate the determinant of a matrix of order 3 x 3. Let's calculate the determinant of the following

M = ( a 11 a 12 a 13 a 21 a 2 a 33 a 31 a 32 a 33 )

by Laplace expansion along the first row

det M = a 11 | a 22 a 23 a 32 a 33 | − a 12 | a 21 a 23 a 31 a 33 | + a 13 | a 21 a 22 a 31 a 32 |

by pairing positive and negative terms together

det M = a11 a22a33 + a12 a23 a31 + a13 a21 a32 − a13 a22 a31 − a11 a23 a32 − a12 a21 a33
