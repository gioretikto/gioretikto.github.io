Dimension of a vector space

A fundamental fact concerning vector spaces is that if a vector space V has a finite basis, meaning a basis consisting of a finite number of vectors, then all bases of V contain precisely the same number of vectors. We must show that any basis is a maximal linearly independent set and a minimal set of generators and these are true also in the infinite-dimensional case.

  3. X is a linearly independent spanning set.

So a space is if every linearly independent set can be enlarged to a larger linearly independent set. Infinite-dimensional spaces are important, but here we are only concerned with the finite-dimensional ones.

    Theorem 1.5.12 If V has a basis, then any two bases have the same number of elements.

    Proof. Suppose that X and Y are both maximal.
