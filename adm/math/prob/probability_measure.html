Probability Measure

Definition 2.2.1. (Measurable space) A pair (Ω, ℱ), where Ω is a set and ℱ is a σ-algebra on Ω, is called a measurable space. □

Definition 2.2.2. A probability measure ℙ on the measurable space (Ω, ℱ) is a function ℙ: ℱ → [0, 1] satisfying

  1. ℙ(0) = 0, ℙ(Ω) = 1;

  2. if A1, A2, ..., is a collection of disjoint members of ℱ, in that Ai ∩ Aj = ∅ for all pairs i, j satisfying i ≠ j, then

    P ( ⋃ i = 1 ∞ ) = ∑ i = 1 ∞ P ( A i )

    The triple (Ω, ℱ, ℙ), comprising a set Ω, a σ-ﬁeld ℱ of subsets of Ω, and a probability measure ℙ on (Ω, ℱ), is called a probability space. □

    We can associate a probability space (Ω, ℱ, ℙ) with any experiment, and all questions associated with the experiment can be reformulated in terms of this space. It may seem natural to ask for the numerical value of the probability ℙ(A) of some event A. The answer to such a question must be contained in the description of the experiment in question. For example, the assertion that a fair coin is tossed once is equivalent to saying that heads and tails have an equal probability of occurring; actually, this is the deﬁnition of fairness.

    Example 2.2.2. A coin, possibly biased, is tossed once. We can take Ω = {H, T} and ℱ = {∅, H, T, Ω}, and a possible probability measure ℙ : ℱ → [0, 1] is given by

    ℙ(∅) = 0,  ℙ(H) = p,  ℙ(T) = 1 − p, ℙ(Ω) = 1,

    where p is a ﬁxed real number in the interval [0, 1]. If p = 1/2 , then we say that the coin is fair, or unbiased. ■

    Example 2.2.3. A die is thrown once. We can take Ω = {1, 2, 3, 4, 5, 6}, ℱ = {0, 1}Ω, and the probability measure ℙ given by

    ℙ(A) = Σi ∈ A pi   for any A ⊆ Ω

    where p1, p2 , ... , p6 are speciﬁed numbers from the interval [0, 1] having unit sum. The probability that i turns up is pi. The die is fair if pi = 1/6 for each i, in which case

    ℙ(A) = |A|/6  for any A ⊆ Ω

    where |A| denotes the cardinality of A.
    For example suppose we want to answer "what is the probability that the number rolled is odd?". Then A = {1,3,5}, and ℙ(A) = p1 + p3 + p5 = 1/2.  ■

    The triple (Ω, ℱ, ℙ) denotes a typical probability space. We now give some of its simple but important properties.

    Lemma 2.2.4.

    1. ℙ(Ac) = 1 − ℙ(A),

    2. if B ⊇ A then ℙ(B) = ℙ(A) + ℙ(B\A) ≥ ℙ(A),

    3. For events A1, ..., An, we have the inclusion-exclusion principle, which for n = 2 has the form

      ℙ(A1 ∪ A2) = ℙ(A1) + ℙ(A2) − ℙ(A1 ∩ A2),

      for n = 3

      ℙ(A1 ∪ A2 ∪ A3) = ℙ(A1) + ℙ(A2) + ℙ(A3) − ℙ(A1 ∩ A2) − ℙ(A2 ∩ A3) + ℙ(A2 ∩ A3 ∩ A3),

      more generally,

      P ( ⋃ i = 1 n A i ) = ∑ i = 1 n P ( A i ) − ∑ i < j P ( A i ∩ A j ) + ∑ i < j < k P ( A i ∩ A j ∩ A k ) + ⋯ + ( − 1 ) n − 1 ∑ i < … < n P ( ⋂ i = 1 n A i )

    Proof.

    1. A ∪ Ac = Ω and A ∩ Ac) = ∅, so ℙ(A ∪ Ac) = ℙ(A) + ℙ(Ac) = .

    2. B = A ∪ (B \ A). This is the union of disjoint sets and therefore

      ℙ(B) = ℙ(A) + ℙ(B \ A)

    3. A ∪ B = A ∪ (B \ A), wich is a disjoint union. Therefore, by (b)

      ℙ(A ∪ B) = ℙ(A) + ℙ(B \ A) = ℙ(A) + ℙ(B \ (A ∪ B)) = ℙ(A) + ℙ(B) − ℙ(A ∩ B)

    4. (d) The proof is by induction on n.
