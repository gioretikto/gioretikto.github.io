Monomials, Binomials and Multinomials

Two of the simplest types of polynomials are monomials, which consist of just a single term (like 4x2 or −5x2y3z), and binomials, which consist of two terms added together (like 5x2 + 3x or 7xy2 − 5x3y). If we take a power of a binomial like (x + y)n then it is often useful to know what the coefficient of each term in the resulting polynomial is once everything is expanded out. For example, it is straightforward to check that

(x + y)2 = x2 + 2xy + y2,   and   (x + y)3 = x3 + 3x2y + 3xy2 + y3

More generally, we can notice that when we compute

( x + y ) n = ( x + y ) ( x + y ) ⋯ ( x + y ) ⏟ n  times

by repeatedly applying distributivity of multiplication over addition, we get 2n terms in total—we can imagine constructing these terms by choosing either x or y in each copy of (x + y), so we can make a choice between 2 objects n times. The binomial coefficient has an important combinatorial meaning: it represents the number of ways of choosing k elements from a set of n elements, without considering the order of the objects in each subset. More generally, we can use a similar technique to come up with an explicit formula for the coefficients of powers of any polynomials—not just binomials. In particular, when expanding out an expression like

( x 1 + x 2 + … + x n ) n = ( x 1 + x 2 + … + x n ) ⋯ ( x 1 + x 2 + … + x n ) ⏟ p  times
