Periodic Functions

Deﬁnition 10.1.1 A map f : ℝ → ℝ is periodic of periodT > 0 if

f (x + T) = f(x) ,  ∀x ∈ ℝ □

From the definition it follows that if f(x) = f(x + T) then it holds also f(x + T) = f((x + T) + T) = f(x + 2T). So If f is periodic of period T it is also periodic of period kT , with k ∈ ℕ \ {0}, including that f might be periodic of period T /k, k ∈ ℕ \ {0}. The minimum period of f is the smallest T (if existent) for which f is periodic. Moreover, f is known when we know it on an interval [x0, x0 + T) (or (x0, x0 + T]) of length T. Usually one chooses the interval [0, T) or [−T/2 , T/2) . Note that if f is constant, it is periodic of period T , for any T > 0.
Any map deﬁned on a bounded interval [a, b) can be prolonged to a periodic function of period T = b − a, by setting

f(x + kT) = f (x),  k ∈ ℤ , ∀x ∈ [a, b) .

Such prolongation is not necessarily continuous at the points x = a + kT , even if the original function is.

Examples 10.1.2

  1. The functions f(x) = cos x, g(x) = sin x are periodic of period 2kπ with k ∈ ℕ \ {0}. Both have minimum period 2π.

  2. The functions sin 3x, cos 3x are instead 2π/3-periodic; in genreal sin nx, cos nx are 2π/n periodic functions.

  3. f(x) = cos x/4 is periodic of period 8π, 16π, . . .; its minimum period is 8π.

  4. The maps f(x) = cos ωx, g(x) = sin ωx, ω ≠ 0, are periodic with minimum period T0 = 2π/ω.

  5. The mantissa map f(x) = M(x) is periodic of minimum period T0 = 1. 2

Deﬁnition 10.1.3 A map f periodic of period 2π is piecewise continuous (continua a tratti) if it is continuous on [0, 2π] except for at most a ﬁnite number of points x0, x1, ..., xn. At such points there can be a removable singularity or a singularity of the ﬁrst kind, so the left and right limits

f ( x 0 + ) = lim x → x 0 + f ( x ) and f ( x 0 − ) = lim x → x 0 − f ( x )

exist and are ﬁnite.
If, in addition,

f ( x 0 ) = 1 2 ( f ( x 0 + ) + f ( x 0 − ) )

at each discontinuity x0 , f is called regularised. □

It is obvious from the interpretation of a definite integral as the area under a curve, that if a periodic function is integrated over a period interval, this interval can be replaced by any other of the same length without changing the value of the integral.

We have the following lemma.

10.1.4 Proposition. Let f a T-periodic, integrable function over [0, T]. For every a ∈ ℝ, we have:

∫ 0 T f ( x ) d x = ∫ a a + T f ( x ) d x

Proof. By the additivity with respect to the Interval of Integration, we can write the integral

∫ 0 a + T f ( x ) d x = ∫ 0 T f ( x ) d x + ∫ T a + T f ( x ) d x = ∫ 0 a f ( x ) d x + ∫ a a + T f ( x ) d x

In the second integral on the left, we make the substitution x = u + T and see that dx = du; the limits of integration become x = T ⟶ u = 0 and x = a + T ⟶ u = a. So one has

∫ 0 T f ( x ) d x + ∫ 0 a f ( u + T ) d x = ∫ 0 a f ( x ) d x + ∫ a a + T f ( x ) d x

Since f(u + T) = f(u), we have

∫ 0 T f ( x ) d x + ∫ 0 a f ( u ) d x = ∫ 0 a f ( x ) d x + ∫ a a + T f ( x ) d x

or

∫ 0 T f ( x ) d x = ∫ a a + T f ( x ) d x

 □

10.1.5 Proposition. Let f be periodic of period T1 > 0; and take T2 > 0. Then g(x) = f(T1x/T2).

Proof. For any x ∈ ℝ,

g ( x + T 2 ) = f ( T 1 T 2 ( x + T 2 ) ) = f ( T 1 T 2 x + T 1 ) = f ( T 1 T 2 x ) = g ( x ) □
