Separable Differential Equations

A class of differential equations that is easy to solve formally is that of variables separable type as the following

y' = a(t) b(y) 11.2.1

The term ‘separable’ refers to the fact that the right-hand side of Equation 11.2.1 can be separated into a function of t times a function of y. With a(t) continuous in I ⊂ ℝ and b(y) continuous in J ⊂ ℝ.
Examples of separable differential equations include

y' = (x2 − 3)(3y + 2)
y' = 6x2 + 4x
y' = xy + 3x − 2y − 6   (can be factored as (x+3)(y−2))

We notice first that y = 0 is a solution of the equation. So ew shall find non-null solutions of the equation. Secondly if the number ȳ is a solution of the equation b(y) = 0, the constant function y = ȳ is a solution of 11.2.1. Indeed the right-hand side is zero as b(ȳ) = 0 and the left-hand side is zero because the derivative of a constant function is zero. Suppose instead that b(y) ≠ 0, 11.2.1 can be written as

y' / b(y) = a(t)

By using Leibniz notation dy/dt for the derivative of y allows to obtain easily the analytical solution of such equations: treating dt and dy as independent entities we are able to "separate the variables"

dy/b(y) = a(t) dt

By taking the integrals of both sides, we obtain

∫ d y b ( y ) = ∫ a ( t ) d t

By calcululating the primitive we solve the problem. If B(y) is a primitive of 1/b(y) and A(t) is a primitive of a(t) , the general integral of 12.2.1 is assigned implicitly by

B(y) = A(t) + c

with c a constant. If one is able to find the inverse function of B, , we are able to express y explicitly as

y = B−1 (A(t) + c)

that is an expression of the form

y = F(t, c)

(in which the arbitraty constant c can appear as well in a non-addive form).

In practical terms, given a separable equation, is not said that we are able to calculate the primitives B(y) , A(t) and even in the case we are is not said we are able to express y in the form y = F(t, c). This is possible in particular simple cases.

Example 12.2.2. For the equation

dy/dt = ty2

separating the variables, yields

dy/y2 = t dt

and integrating

−1/y = ½ t2 + c

That is

y(t) = −1/(c + t2/2)   c arbitraty constant.  ■

Cauchy Problem

Consider the Cauchy problem

If a(t) is a continuous function in a neighbour of t0 and the function b(y) is continuously differentiable in a neighbour of y0, then the Cauchy problem has one and only one solution, defined at least in a neighbour of t0.

Notice that in this case b(y) = 1/y is continuously differentiable for y ≠ 0, while a(t) = 1 is continous everywhere. Thus the Cauchy problem for this equation has only one solution given that the initial condition is not of the form

y(t0) = 0

For in this case the equation yt' = 1 cannot be solved in such a point t0 as would give 0 = 1, and the Cauchy problem does not admit solutions.
