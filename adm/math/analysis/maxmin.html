Finding the maximum/minimum values of a function

Suppose we want to find maxima and minima of the function f: [a,b] ⟶ ℝ. If f is differentiable we can proceed as follows

  1. Calculate the values f(a) and f(b).

  2. Calculate f'(x) and solve the equation f'(x) = 0 to find the stationary points.

  3. If there are no stationary points then f(a) and f(b) are extreme values. Conversely if x0 ∈ (a,b) is a stationary point, that is f'(x0) = 0.

    • If in a left neighborhood of x0 we have f'(x) ≥ 0 and in a right neighborhood of x0 we have f'(x) ≤ 0, then x0 is a point of local maximum.

    • If in a left neighborhood of x0 we have f'(x) ≤ 0 and in a right neighborhood of x0 we have f'(x) ≥ 0, then x0 is a point of local minimum.

    x0 maximum point
    x < x0 x > x0
    f'(x) ≥ 0 f'(x) ≤ 0
    f ↗ f ↘
    x0 minimum point
    x < x0 x > x0
    f'(x) ≥ 0 f'(x) ≤ 0
    f↘ f↗

  4. Once found the extrema we evaluate the function at these points and compare their values with those of f(a) and f(b)

Example 5.10.1. Let f(x) = xe−x2 with x ∈ [0,2]. Find the global maxima and minima of f.

  1. f(0) = 0, f(2) = 2e−4.

  2. f'(x) = e−x2 + xe−x2(−2x) = e−x2(1 − 2x2)

    f'(x) = 0  ⇐⇒  1 − 2x2 = 0  ⇒  x = ± 1/√2.

    Since only +1/√2 ∈ [0,2], this is the only stationary point.

  3. Since there's only a stationary point we can compare the value of f 1/√2 with those at the boundaries:

    f(1/√2) = 1/√2 ⋅ e−1/2 = 1/1/√(2e) is greater than both f(0) and f(2) = 2/e4. We conclude that

    f(0) = 0   is a global minimum
    f(1/√2) is a global maximum

  4. If domain of f would have been ℝ, we would have had to study the sign of f' in a neighborhood of x0 = 1/√2.

    f'(x) ≥ 0  i.e. 2x2 − 1 ≥0  for  −1/√2 ≤ x ≤ 1/√2

    Thus

    x = 1/√2 is global maximum;   x = −1/√2 is a global minimum. ■

Monotone sequences

We can used the tools of differential calculus to study the monotony of a sequence, which could be difficult to analyzer algebrically.

Example 5.10.2. Let an = log n/n. We know that an ≥ 0 and an ⟶ 0 (by comparing the limit of of log n and n). WE want to understand if the sequence is decreasing, which by definition means that

an+1 ≤ an  i.e.  log(n+1)/(n+1) ≤ log n/n

As n increases both the numerator and denominator increase, and it is not easy to come to a conclusion. So let

f(x) = log x/x  for  x > 0

(now the variable x is not an integer but a real number). The first derivative is

f'(x) = [1 − log x]/x2  for  x ≥ e

it follows that f is decreasing for x ≥ e and consequently the sequence an is decreasing for n ≥ 3 (the first integer > e).

A possible application is the study of the series

∑ n = 2 ∞ = ( − 1 ) n log ⁡ n n

It is an alternating series and since an = log n/n is positive, infinitesimal, and as proved monotonically decreasing, by Leibniz Criterion. The monotoniciy is paramount to prove the convergence of the series. ■
