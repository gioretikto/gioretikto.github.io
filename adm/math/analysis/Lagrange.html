Mean value theorem

Theorem 5.6.1 (Mean value theorem). Let f: [a , b] → ℝ differentiable on the open interval (a, b), where a < b. Then there exists some ξ ∈ (a, b) such that

f ′ ( ξ ) = f ( b ) − f ( a ) b − a

Proof. The geometrical meaning of the theorem is illustrated by the following figure which indicates that the tanget to the point (ξ f(ξ)) is parallel to the secant to a and b.

The equation of the secant line that passes through the points (a,f(a)) and (b,f(b)) is :

y = f ( a ) + f ( b ) − f ( a ) b − a ( x − a )

Consider the auxiliary map ω(x) defined as the difference between f(x) and y. Then

ω ( x ) = f ( x ) − [ f ( a ) + f ( b ) − f ( a ) b − a ( x − a ) ]

Because f is continuous on [a, b], it follows that ω is also continuous on [a, b]. Furthermore, because f is differentiable on (a, b) then ω also differentiable on (a,b). Applying Rolle's theorem to the function ω, there exists a number ξ such that ω'(ξ) = 0. So there exists a number ξ in (a,b) such that

f'(ξ) − [f(b) − f(a)]/(b − a) = 0  ⇒  f'(ξ) = f(b) − f(a)/(b − a)  □

Note that Rolle's Theorem, is a special case of the Mean Value Theorem, where f(b) = f(a).

Applications

Many of the most basic results in calculus follow from the Mean Value Theorem. A function f defined on (a,b) is increasing if f(x1) ≤ f(x2) whenever a < x1 < x2 < b; and f is decreasing on (a,b) if f(x1) ≥ f(x2) whenever a < x1 < x2 < b. (So f is decreasing on (a,b) ⇔ −f is increasing on (a,b)). The function f being strictly increasing means that the ≤ above can be replaced with <, and the function f being strictly decreasing means that the ≥ above can be replaced with >.

Lemma 5.6.2 (Monotonicity test). Suppose that f is differentiable on (a, b), then ∀x ∈ (a,b):

  1. If f' ≥ 0  ⇐⇒  f is increasing;

  2. If f' ≤ 0  ⇐⇒  f is is decreasing;

  3. If f' = 0  ⇐⇒  f is constant.

Proof. Let for example f'(x) ≥ 0, ∀x ∈ (a,b); We shall prove that f is increasing on (a,b). We take two points x1, x2 ∈ (a,b) such that x1 < x2 and show that f(x1) ≤ f(x2). Indeed, by the mean value theorem, there exists a c ∈ (x1, x2) such that

f ′ ( c ) = f ( x 2 ) − f ( x 1 ) x 2 − x 1

By hypothesis f'(c) ≥ 0 and x2 − x1 > 0, hence f(x1) − f(x2) ≥ 0, which proves the claim. The same reasoning proves point b) and c).  □

Remark. Point c) of the previous lemma can be applied only on intervals and not on more general sets as the next example show.
