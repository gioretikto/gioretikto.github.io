Lenght of an arc

Consider a regular plane curve r: [a,b] ⟶ ℝ2. We subdivide the interval [a,b] in n identical intervals, with extreme points t0 = a, t1, t2, ..., tn = b and consider the corresponding points on the curve

(x(ti), y(ti)), i = 0,1,2,..,n

The length of the segment joining these points is

∑ i = 1 n ( x ( t i ) − x ( t i − 1 ) ) 2 + ( y ( t i ) − y ( t i − 1 ) ) 2

By applying the Lagrange theorem to the function x(t), y(t) on each intevals [ti − 1, ti], we find that there exist the points ξi, ηi ∈ [ti − 1, ti] such that the expression above is equal to

∑ i = 1 n [ x ′ ( ξ i ) ( t i − t i − 1 ) ] 2 + [ y ′ ( η i ) ( t i − t i − 1 ) ] 2 = ∑ i = 1 n ( t i − t i − 1 ) x ′ ( ξ i ) 2 + y ′ ( η i ) 2

For n ⟶ ∞ the amplitude of each interval tends to zero, hence ξi ≃ ηi and the limit of the expression above is identical to the limit of Cauchy sums of the fujnction x ′ ( t ) 2 + y ′ ( t ) 2 . The length of the curve is hence equal to the integral

L = ∫ a b x ′ ( t ) 2 + y ′ ( t ) 2 d t
