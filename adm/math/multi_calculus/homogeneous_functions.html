Homogenous Functions. Euler's Theorem

Definition. (Homogenous Functions). Given a real function of several varibles f: A: ⊆ ℝn ⟶ ℝ and a real number k ∈ ℝ, f is said to be a homogeneous function of degree, k if, for any (x1, ..., xn) ∈ A and for any real number t > 0, it holds that

f(t ⋅ x1, ..., t ⋅ xn) = tk ⋅ f(x1, ..., xn)■  5.1.1

Example. The function

f(x,y) = xyex/y

is a homogenous function of degree 2, since

f(t ⋅ x, t ⋅ y) = (tx) (ty) etx/ty = t2xyey/x = t2 ⋅ f(x,y)

Consider F(tx, ty, tz, ...) = tk F(x, y, z, ...). If we define

tx = x',   ty = y',  tz = z', ... 5.1.2

Equation 5.1.1 becomes

F(x', y', z', ...) = tk F(x, y, z, ...)

By differentiating with respect to t

∂ F ∂ x ′ ∂ x ′ ∂ t + ∂ F ∂ y ′ ∂ y ′ ∂ t + ∂ F ∂ z ′ ∂ z ′ ∂ t + … = k t k − 1 F ( x , y , z , … )

Upon using equation 5.1.2

∂ x ′ ∂ t ′ = x , ∂ y ′ ∂ t ′ = y , ∂ z ′ ∂ t ′ = z , …

This is Euler's theorem on homogeneous functions. If we now set t = 1 in the previous equation, (i.e. for a homogeneous functions of degree 1), the result is

x ∂ F ∂ x + y ∂ F ∂ y + z ∂ F ∂ z … = k t k − 1 F ( x , y , z , … )

It is of interest to see the connection between homogenous functions and extensive thermodynamic properties.
