Vector Functions

Real-valued functions are also called scalar-valued or just scalar functions; A real function f: U ⊆ ℝn → ℝ assings a unique real number f(x) = f(x1, ..., xn) to each element x = (x1, ..., xn) ∈ U. A vector function is a synonym for a vector-valued function. We use the notation f: U ⊆ ℝn → ℝm to describe a vector function. A vector function assign a unique vector that we write as

f(x) = (f1(x), ..., fm(x))  with  fi:U ⊆ ℝn → ℝ

There are many examples of physical objects represente by vector functions:

  • parametric surfaces;

  • coordinate transformations;

  • vector fields.

The real-valued functions fi are said components of f. We say that

limx ⟶ x0 f(x) = ℓ

if

|f(x) − ℓ| ⟶ 0  for  |x − x0|

Likewise to the definition of limit for curves we can calculate the limit componentwise

lim x → x 0 ( f 1 ( x ) , … , f m ( x ) ) = ( lim x → x 0 f 1 ( x ) , … , lim x → x 0 f m ( x ) )

It can be proved that f is continuous if and only if the all its components fi are. Similarly for differentiability, we say that f is differentiable in x0 if all the components fi are. This means that

f i ( x 0 + h ) − f i ( x 0 ) = ∑ j = i n ∂ f i ∂ x j ( x 0 ) h j + o ( | h | ) f o r h → 0 for i = 1 , 2 , … , m

We can write the relation above more consiely representing f the points of ℝm as colum vectors of their components.

f = ( f 1 f 2 … f m ) x 0 = ( x 01 f 02 … x 0 n ) h = ( h 1 h 2 … h n )

and by introducting the Jacobian matrix of f, which has as rows the gradients of the components of f at x0:

J f ( x 0 ) = ( ∂ f 1 ∂ x 1 ∂ f 1 ∂ x 2 … ∂ f 1 ∂ x n ∂ f 2 ∂ x 1 … … ∂ f 1 ∂ x n … ∂ f m ∂ x 1 … … ∂ f m ∂ x n ) ( x 0 )

With the Jacobian matrix, we can express the differentiability condition of f as

f(x0 + h) − f(x0) = Jf(x0) ⋅ h + o(|h|)   h ⟶0

with ⋅ the matrix product (rows by columns) between the Jacobian matrix and the increment vector, and the symbol o(|h|) represents a vector that divide by |h| tends to zero for h ⟶ 0.
The differential of f (calculated in x0) is a linear function

df(x0): ℝn ⟶ ℝm
df(x0): h ⟼ Jf(x0) ⋅ h
