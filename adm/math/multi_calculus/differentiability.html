Differentiability

In una dimensione la derivabilità di una funzione garantisce l'esistenza della retta tangente, nel caso di una funzione a più dimensioni la sola derivabilità, intesa come esistenza delle derivate parziali, non implica nè la continuità nè l'esistenza del piano tangente. Per trovare la proprietà che garantisce l'esistenza del piano tangente, riconduciamoci al caso unidimensionale. La retta tangete è espressa tramite la relazione:

f(x + h) − f(x) = f'(x) h + o(h)   h → 0

Questa relazione esprime il fatto che l'incremento della funzione coincide con l'incremento valutato sulla retta tangente, chiamato differenziale. I due incrementi differiscono per un infinitesimo di ordine superiore all'incremento h.
In più variabili il concetto analogo è quello di differenziabilità: l'incremento di f è uguale all'incremento calcolato lungo il piano tangente, più un infinitesimo di ordine superiore rispetto al valore dell'incremento delle variabili indipendenti. Ossia:

f ( x 0 + h , y 0 + k ) − f ( x 0 , y 0 ) = ∂ f ∂ x ( x 0 , y 0 ) h + ∂ f ∂ y ( x 0 , y 0 ) k + o ( h 2 + k 2 ) , ( h , k ) → ( 0 , 0 )

dove il primo membro rappresenta l'incremento della funzione; al secondo membro i primi due addendi sono gli incrementi valutati lungo il piano tangente; l'ultimo addendo è l'errore che si commette valutando l'incremento di f lungo il piano tangente. Il simbolo di o piccolo significa che:

lim ( h , k ) → ( 0 , 0 ) o ( h 2 + k 2 ) h 2 + k 2 = 0

Se la (8.2) è verificata diremo che f è differenziabile at (x0,y0).
Più in generale per una funzione di n variabili si ha, usando la definizione di gradiente:

f(x + h) = f(x) + ∇f ⋅ h + o(|h|), as h → 0

Definition. Let A be an open set in ℝn. One says a function f : A → ℝ is differentiable at x ∈ A if it admits partial derivatives at x (i.e., the gradient ∇f(x) is defined) and

lim h → 0 f ( x + h ) − f ( x ) − ∇ f ( x ) ⋅ h h = 0

A function f is said differentiable on A if it is differentiable at any x ∈ A. Given x, the linear map, called the differential of f, at the point x, defined on ℝn by df(x): h → ∇f(x) ⋅ h.
Note that in (3.12), h is a variable in ℝn , |h| is its norm and ∇f(x) ⋅ h is the inner product of h and the gradient of f at x.
