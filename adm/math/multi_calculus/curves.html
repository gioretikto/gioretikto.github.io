Curves

Example 2.1.1. (of Curves) Curves in the plane can be described in at least three different ways:

  • as the graph of a function y = g(x); e.g. a straight line is represented as y = mx + q or x = my + q;

  • as the vanishing locus of a two variable function, f(x,y) = 0; for a line the first degree polynomial ax + by + c = 0;

  • in parametric form as the image of a map f: ℝ → ℝ2 having the form f(t) = (αt + β, γt + δ). ■

Definition 2.1.2 A continuous function γ: I ⊆ ℝ → ℝm is called a curve. The set Γ = γ(I) ⊆ ℝm is said trace of the curve. We says that a subset S of ℝm is parametrized by γ provided there is a subset I ⊆ ℝ such that γ: I ⟶ ℝm is a curve for which γ(I) = S.  □

Definition 2.1.3 Let f and g be continuous functions of a variable t on an interval I ⊆ ℝ. The the equations x = f(t) and y = g(t) are called parametric equations, with parameter t. The set of points (x,y) = (f(t,g(t) trace out a parametric curve as t varies over the interval I. Here, the domain set I is called the parametric set. □

A parametrization is not unique, there can by many different pairs of functions that can exactly describe a given geometrical locus.

If the trace of the curve lies on a plane, one speaks about a plane curve. The most common curves are those in the plane (m = 2)

γ(t) = (x(t), y(t)) = x(t) i + y(t) j

and curves in space (m = 3)

γ(t) = x(t), y(t) = x(t) i + y(t) j + z(t) k

It's important to stress the difference occurring between a curve, which is a function of one real variable, and its trace (sostegno), a set in Euclidean space ℝm. A curve provides a way to parametrise its trace by associating to the parameter t ∈ I one (and only one) point of the trace. Still, Γ may be the trace of many curves, because it may be parametrised in distinct ways. For instance the plane curves γ(t) = (t, t), t ∈ [0, 1], and δ(t) = (t2, t2), t ∈ [0, 1] have the segment AB of end points A = (0, 0) and B = (1, 1) as common trace; γ and δ are two parametrisations of the segment. The mid-point of AB, for example, corresponds to the value t = 1/2 for γ, t = 1/2 for δ.

A curve γ is said simple if γ is a one-to-one map, i.e., if distinct parameter’s values determine distinct points of the trace.
If the interval I = [a, b] is closed and bounded, as in the previous examples, the curve γ will be named arc. We call end points of the arc the points P0 = γ(a), P1 = γ(b); precisely, P0 is the initial point and P1 the end point, and γ joins P0 and P1. An arc is closed if its end points coincide: γ(a) = γ(b); although a closed arc cannot be simple, one speaks anyway of a simple closed arc (or Jordan arc) when the point γ(a) = γ(b) is the unique point on the trace where injectivity fails. Example 1. Consideriamo la parametrizzazione:

r(t) = 2 cos t i + 2 sin t j, t ∈ [a, b].

Se [a, b] = [0, 2π] è facile convincersi che r parametrizza la circonferenza γ di raggio 2 centrata nell’origine del piano xy. Si tratta perciò di una curva chiusa e piana, la cui parametrizzazione risulta semplice (ogni punto di γ eccetto (2, 0) = r(0) = r(2π) corrisponde ad esattamente un valore del parametro). Se invece [a, b] = [0, 3π] otteniamo ancora una parametrizzazione di γ, che questa volta viene “percorsa” una volta e mezza. Perciò la parametrizzazione non è semplice, né chiusa. Infine si consideri la paramettrizzazione:

r(t) = 2 cos t j + 2 sin t i, t ∈ [0, 2π].

Si tratta di un'altra parametrizzazione di r, stavolta percorsa in senso orario.  ■

Risulta chiaro dall’esempio precedente che un sostegno può essere parametrizzato in infiniti modi diversi.

La parametrizzazione di una curva induce un'orientazione del proprio sostegno, cioè il verso in cui γ(t) percorre il sotegno all'aumentare di t. L'orientazione del sostegno viene indicata con frecce. Per cambiare orientazione occorre cambiare paramettrizzazione. Ad esempio la circonferenza unitaria percorsa in senso orario ha la parametrizzazione:

r(t) = cos t i − sin t j, t ∈ [0, 2π]

Il sostegno di una curva piana, semplice e chiusa divide il piano in due insiemi aperti e connessi, di cui uno è limitato e si chiama interno della curva, l'altro è illimitato e si chiama esterno della curva. Ciò porta alla seguente definizione:

Definizione 5.2- Una curva γ piana, semplice e chiusa si dice curva di Jordan. Una curva di Jordan si dice orientata positivamente (negativamente) se γ(t) percorre il sostegno in senso antiorario (orario), ovvero tenendo l'interno della curva a sinistra.
