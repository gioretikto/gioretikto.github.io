The law of Cosine

Trigonometry can be used to find lenghts and angles in non right-angled triangles.

The law of cosines tells us how to find one side, say c of any triangle if we know the other two sides a and b and the angle θ between them, or to find the angles of a triangle when all three sides are known.

Thorem (Law of Cosine). Given ΔABC, let a,b, and c denote, the lenghts of sides BC, AC, and AB, respectively. Then

c2 = a2 + b2 − 2ab cos C

If one of the angles of a triangle, say, ∠C is a right angle, then cos 90° = 0, and the Law of Cosines reduces to the Pythagorean Theorem, c2 = a2 + b2. Thus the Pythagorean Theorem is a special case of the Law of Cosines.

Proof. Since a plane triangle can have only one obtuse angle, we can be sure that at least one of ∠A or ∠B is an acute angle, say ∠B < 90°. Draw altitude AP from A to BC and write h = AP. Note that there are three possibilities:

  1. Either ∠C < 90° and P lies on segment BC, as is Fig. 1.a;

  2. ∠C > 90° and P lies on an extension of side BC, as in Fig. 1.b;

  3. ∠C = 90°, as we have already seen that in this case the law holds and we've the Pythagorean Theorem.

Suppose first ∠C < 90°, and write x = PC. Since ΔABC is a right triangle, we see that cos(C) = PC/AC = x/b, and so x = b cos (C). Also, two application of the Pythagorean Theorem yied b2 = x2 + h2 and c2 = h2 + (a − x)2. This gives

c2 = (b2 − x2) + (a2 − x2)2 = b2 − x2 + a2 − 2ax + x2 = a2 + b2 − 2ax

Since x = b cos (C), we obtain the desired result.
When ∠C < 90°, we refer to Fig. 2b. Here too we write x = PC, but in this case, we have cos (C) = −x/b and x = −b cos (C). Two application of the Pythagorean Theorem yied b2 = x2 + h2 and c2 = h2 + (a + x)2 and we have

c2 = (b2 − x2) + (a2 − c2)2 = b2 − x2 + a2 − 2ax + x2 = a2 + b2 − 2ax

Since x = − b cos(C) in this case, the proof is complete.  □
