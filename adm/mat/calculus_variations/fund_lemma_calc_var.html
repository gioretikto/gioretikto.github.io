Fundamental lemma of the calculus of variations

In the proof of the Euler-Lagrange equation, the final step invokes a lemma known as the fundamental lemma of the calculus of variations (FLCV).

Lemma 1.1.1 (FLCV). Let M(x) be continuous on [a, b], and suppose that for all η(x) ∈ C1[a, b] such that η(a) = η(b) = 0 we have

∫ a b M ( x ) η ( x ) = 0

Then M(x) = 0 for all a ≤ x ≤ b.

Proof. Suppose, for a contradiction, that A.3.1 holds while M(x0) ≠ 0, for some x0 ∈ (a,b). Withtout loss of generality we may assume M(x0) > 0. By continuity, M(x) > 0 in a neighborhood [x0 − ε, x0 + ε] ⊂ (a,b). We can imagine constructing a function η as a nonnegative bell-shaped function such that η(x0) > 0, and η(x0) = 0 outside (x0 − ε, x0 + ε). The product M(x)η(x) is nonnegative everywhere and positive near x0. Hence ∫ba M(x)η(x) dx > 0, a contradiction.  □
