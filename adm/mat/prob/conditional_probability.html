Conditional Probability; Dependent And Independent Events

So far, we have calculate probabilities of various events on the assumption that no information was available about the random experimento other than the sample space Ω. If, however, it is known that an event B has occured, the the probability of the event A has to be calculated with the condition that the event B has already occured. In defining this (conditional) probability of A given that B has occured, we take B to be the new sample space, and relative to this, we define the probability of A. The usual notation for 'event A occurs given that event B has occured", is A|B. Note that in the classical interpretation of probability, the definition for the conditional probability of A, given that B has occured, will be

P(A|B) = n(A ∩ B) / n(B)   where n(B) ≠ 0 2.3.1

Example 2.3.2. Consider the experiment of tossing two fair coins. Then, the sample space Ω is

Ω = {HH, HT, TH, TT}

Consider the two events A and B defines by

  • A: Both coins show the same face.

  • B: At least one coin shows T.

Then, we know that the favourable cases for A are HH and TT so that n(A) = 2. In this case,

P(A) = 2/4 = 1/2

Suppose that the event B has already occured i.e. we are given the information "at least one coin shows T". This information changes the favourable number of cases for A. In fact, the favourable case for A, given the condition that B has occured, is just one point, viz TT. Note that the sample space has also changed. Since B has already occured, the sample space for A is the event B. Note that B contains only the outcomes TH, HT, TT so that n(B) = 3. Thus, the (conditional) probability of A (given that B has occured) is obtained as

P(A|B) = n(A ∩ B) / n(B) = 1/3

Example 2.3.3. What is the probability of rolling a prime number in one roll of a die? Let Ω = {1, 2, 3, 4, 5, 6}, so n(Ω) = 6; and let A = {2, 3, 5}, so n(A) = 3. Therefore,

P(A) = n(A) / n(Ω) = 3/6 = 1/2

Next, what is the probability of rolling a prime number in one roll of a die, given that an odd number has turned up? Let B = {1, 3, 5} be the new sample space of odd numbers. Then n(B) = 3. We are now interested in only that part of A (rolling a prime number) that is also in B, in other words, A ∩ B. Therefore, because E ∩ B = {3, 5}, the probability of A given B is

P(A|B) = n(A ∩ B) / n(B) = 2/3 ■

The above two examples are in the framework of classical interpretation of probability, namely, the number of outcomes of the random experiment are finite and equally likely. To derive a general definition for conditional probability of the event A, given that B has occured, we rewrite 2.3.1 as

P ( A | B ) = n ( A ∩ B ) n ( B ) = n ( A ∩ B ) / n ( Ω ) n ( B ) / n ( Ω ) = P ( A ∩ B ) P ( B ) where P ( B ) ≠ 0

These observations lead to the following formal definition of conditional proability of an event given that another event has happened.

Definition 2.3.4. Let (Ω, ℱ, ℙ) be a probability space, and the events A,B ∈ ℱ. The conditional probability of A given B, denoted ℙ(A|B), is defined as

ℙ(A|B) = ℙ(A ∩ B)/ℙ(B),   ℙ(B) ≠ 0 □

Example 2.3.5. A family has two children. What is the probability that both are boys, given that at least one is a boy? The older and younger child may each be male or female, so there are four possible combinations of sexes, which we assume to be equally likely. Hence we can represent the sample space in the obvious way as

Ω = {GG, GB, BG, BB}

where P(GG) = P(BB) = P(GB) = P(BG) = 1/4. From the definition of conditional probability,

P(BB|one boy at least) = P(BB | GB ∪ BG ∪ BB) = P(BB ∩ (GB ∪ BG ∪ BB))/P(GB ∪ BG ∪ BB) = P(BB) / P(GB ∪ BG ∪ BB) = 1/3.

A popular but incorrect answer to the question is 1/2. This is the correct answer to another question: for a family with two children, what is the probability that both are boys given that the younger is a boy? In this case,

P(BB|younger is a boy) = P(BB | GB ∪ BB) = P(BB ∩ (GB ∪ BB))/P(GB ∪ BB) = P(BB) / P(GB ∪ BB) = 1/2. ■

Example 2.3.6. A die is tossed. Find the probability that a 4 came up if it is known that an even number turned up.

Solution. Let E be the event of rolling a 4, and let F be the event of rolling an even number. Therefore, E = {4} and F = {2, 4, 6}, from which we obtain E ∩ F = {4}. Using the previous definition, we obtain

P(E|F) = P(E ∩ F)/P(F) = (1/6) / (3/6) = 1/3

2.2.3 Example. Suppose the probability that a student will enroll in a mathematics course is 0.45, the probability that he or she will enroll in a science course is 0.38, and the probability that he or she will enroll in both courses is 0.26. Find the probability that a student will enroll in a mathematics course, given that he or she is also enrolled in a science course. Also, find the probability that a student will enroll in a science course given that he or she is enrolled in mathematics.

Let M be the event Will enroll in mathematics, and let S be the event Will enroll in science. Therefore, using Definition 10.3, we obtain

P(M|S) = P(M ∩ S)/P(S) = 0.26 / 0.38 = 26/38 = 13/19

and

P(M|S) = P(M ∩ S)/P(M) = 0.26 / 0.45 = 26/45 = 26/45
