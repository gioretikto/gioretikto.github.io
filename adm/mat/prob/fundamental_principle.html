Fundamental Principle of Counting

Fundamental Principle: If an operation can be done in m different ways and second operation can be done in n different ways both operations can be done in succession in mn ways, e.g. if a person has 6 shirts and 4 trousers, he can have altogethere 6 x 4 = 24 pairs of dresses.

Suppose a boy has two jeans and four T-shirts. He wants to know in how many different ways he can dress up, by pairing a jeans with a T-Shirt.

We indicate the two jeans as J1, J2, and the four T-Shirts as T1, T2, T3, T4. All the possible pairs are represented by the cartesian product of the two pairs

J X T = {(J1T1), (J1T2), (J1T3), (J1T4), (J2T1), (J2T2), (J2T3), (J2T4)}

To represent the situation we can use the following diagram

T1 T2 T3 T4
J1 ∘ ∘ ∘ ∘
J2 ∘ ∘ ∘ ∘

Thus in total we have: 2 ⋅ 4 groups.

More generally to calculate how man groups can be formed by assigning the first place to an element of a set A with n elements, the second to an element of a set B with m elements and the third to an element of a set C with k elements etc;, we calculate the product

n ⋅ m ⋅ k ⋅ ...

For a two-stage process with n1 possible outcomes for stage 1 and n2 possible outcomes for stage 2, the number of possible outcomes for the two stages taken together is n1n2. More generally if there are k stages, with n1 possible outcomes for stage i then the number of possible outcomes for all k stages taken together is the product n1n2n3 ⋅⋅⋅ nk.
