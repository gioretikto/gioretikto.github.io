Random Variables

Consider a random experiment with sample space Ω. Let w be a sample point in Ω. We are interested in assigning a real number to each w ∈ Ω. A random variable, X(w), is a single-valued real function that assigns a real number, called the value of X(w), to each sample point w ∈ Ω That is, it is a mapping of the sample space onto the real line. Generally a random variable is represented by a single letter X instead of the function X(w). Therefore, in the remainder of the book we use X to denote a random variable. The sample space Ω is called the domain of the random variable X. Also, the collection of all numbers that are values of X is called the range of the random variable X. Figure 2.1 illustrates the concepts of domain and range of X.

A random variable that assumes either a finite number of values or a countable infinite number of possible values is termed as a discrete random variable. For example, consider an experiment to measure the number of customers who arrive in a shop during a time interval of 3 minutes. The possible outcome may vary from 0 customers to n customers. The outcomes (0, 1, 2, ..., n) are the values of the discrete radom variable.

By contrast, a continuous random variable is one for which the number of possible outcomes cannot be counted and therefore, probabilities cannot be attached to specific outcomes.

Example. A coin-tossing experiment has two sample points: Heads and Tails. Thus, we may define the random variable X associated with the experiment as follows:

X (heads) = 1
X(tails) = 0 ■

Distribution Functions

Associated with both types of random variables is the concept of cumulative distribution function, FX(x).

The cumulative distribution function (CDF) of a real-valued random variable X is the function given by:

FX(x) = P⁡ (X ≤ x)

where the right-hand side represents the probability that the random variable X takes on a value less than or equal to x. If X is a real-valued discrete random variable that takes on specific values x1,x2, ..., the CDF is given by

F X ( x ) = ∑ x i ≤ x p X ( x i )

where pX(xi) is the probability mass function i.e. the probability X takes on the value xi. Thus one would write

pX(xi) = P(X = xi)

When the random variable only takes on a handful of possible values, we can write out the probabilities in a table.

0 1 2 3 4 5
pX(x) .2546 .4243 .2496 .0642 .0071 .0002

The probability density function (PDF), fX(x), for a continuous random variable X is the analogue of the probability mass function of a discrete random variable. The PDF is a nonnegative function defined for all real x ∈ (-∞, &infin). A random variable X has density fX, where fX is a non-negative Lebesgue-integrable function, if:

P [ a ≤ X ≤ b ] = ∫ a b f X ( x ) d x

Thus, the area under the curve defined by the PDF is the probability that the random variable lies between the values limiting the area. Intuitively, one can think of fX(x) dx as being the probability of X falling within the infinitesimal interval [x, x + dx].

For any real number a, the probability that the random variable X is less than a is given by cumulative distribution function, FX(a)

P ( X ≤ a ) = F X ( a ) = ∫ − ∞ a f X ( t ) d t

Hence either fX(x) or FX can be used to calculate the probability that X falls into the interval [a,b] by means of

P(a ≤ X ≤ b) = FX(b) - FX(a) = ∫ba fX(x) dx

Because integration and differentiation are inverse procedures, the PDF is the derivative of the cumulative distribution function

f X ( x ) = d F X ( x ) d x ≥ 0

The properties of fX(x) are as follows:

  1. fX(x) ≥ 0; that is, it is a nonnegative function, as stated earlier.

  2. Since X must assume some value, ∫∞-∞ fX(x) dx = 1;

  3. P[a ≤ X ≤ b] = ∫ba fX(x) dx, which means that P[X = a] = ∫aa fX(x) dx = 0; Thus, the probability that a continuous random variable will assume any fixed value is zero.
