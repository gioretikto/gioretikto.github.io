Expectation

Consider some experiment, such as the tossing of a coin or the rolling of a die, that has n possible outcomes, each with probability pj, where j = 1, 2, ... , n. If the experiment is repeated indefinitely, we can use the relative-frequency definition of probability

pj = limn ⟶∞  nj/n j = 1,2,...,n

where n, is the number of times that the event j occurs and n is the total number of repetitions of the experiment.

Definition. Let X be a random variable which takes on values in the finite set {x1, x2, ..., xn}, with probabilities pj for each j = 1, 2,.., n. Then the expectation (or expected value or mean) of X, denoted by ⟨X⟩ is defined by

⟨ X ⟩ = ∑ j = 1 n x j p j

We have "weighted" each possible value xj of the random variable by the probability pj that this value is taken on. These weighted values are then added to provide a weighted sum

Example 1. Suppose we are given the following data:

x px
5 0.20
7 0.25
9 0.55

Calculate the average value of x.

Solution.

⟨x⟩ = 5 ⋅ 0.2 +7 ⋅ 0.25 + 9 ⋅ 0.55 ■

The definitions of variance and standard deviation of a random variable is correspondent to that of variance of a sample of measurements, with mathematical expectation substituting the average. Thus

σ x = ⟨ ( x − ⟨ x ⟩ ) 2 = ∑ j = 1 n ( x j − ⟨ x ⟩ ) ) 2 p j

We denote the square root of the variance by σx, which is called the standard deviation.
