Limit points

On the real number line we have a notion of “closeness”. For example each point in the sequence

0.1, .01, .001, .0001, .00001, ...

is closer to 0 than the previous one.
Indeed, in some sense, 0 is a limit point of this sequence. So the interval (0, 1] is not closed, as it does not contain the limit point 0. A metric space is a topological space that includes the notion of a metric or distance between its elements but in a general topological space we do not have a metric, so we must proceed differently. We shall define the notion of limit point without resorting to distances. Even with our new definition of limit point, the point 0 will still be a limit point of (0, 1]. The introduction of the notion of limit point will lead us to a much better understanding of the notion of closed set.

Definition 1.3.1. (Neighbourhood) In a topological space (X,τ), a neighbourhood of a point x ∈ X is a subset U ⊆ X such that x ∈ O ⊆ U for some open set O ∈ τ.

Throughout this notes, we use the lette U to denote a neighbourhood from the German word "Umgebung".

Definition 1.3.2. Let A be a subset of a topological space (X,τ). A point x ∈ X is said to be a limit point (or accumulation point or cluster point) of A (that is not necessarily in A) if every open set, U, containing x contains a point of A different from x. Symbolically we can say that x is a limit point of A if for every open neighborhood Ux of x, we have

Ux ∩ (A \{x}) ≠ ∅.

Example 1.3.3. Consider the topological space (X,τ) where the set X = {a, b, c, d, e} , the topology τ = {X, Ø, {a}, {c, d}, {a, c, d}, {b, c, d, e}}, and A = {a, b, c}. Then b, d, and e are limit points of A but a and c are not limit points of A.

  • The set {a} is open and contains no other point of A. So a is not a limit point of A.

  • The set {c, d} is an open set containing c but no other point of A. So c is not a limit point of A.

  • The only open sets containing b are X and {b, c, d, e} and both contain another element of A, namely c. So b is a limit point of A.

  • The point d is a limit point of A , even though it is not in A. This is so since every open set containing d contains a point of A. Similarly e is a limit point of A even though it is not in A.  ■

Example 1.3.4. Consider the subset A = [a, b) of ℝ. Then it is easily verified that every element in [a, b) is a limit point of A. The point b is also a limit point of A.  ■

The next proposition provides a useful way of testing whether a set is closed or not.

Proposition 1.3.5. A subset A of a space X is closed if and only if it contains all of its limit points (of A). In other words, iff the complement X\A of A contains no limit points of A.

Proof. Assume that A is closed in (X, τ). Suppose that x is a limit point of A which belongs to X \ A. Then X \ A is an open set containing the limit point x of A. Therefore X \ A contains an element of A. This is clearly false and so we have a contradiction to our supposition. Therefore every limit point of A must belong to A.
Conversely, assume that A contains all of its limit points. For each z ∈ X \ A, our assumption implies that there exists an open set Uz ∋ z such that Uz ∩ A = Ø; This is because if z were a limit point we'd have Uz ∩ (A \{z}) ≠ ∅; negating this condition knowing that A \ {z} = A we have Uz ∩ A = Ø. Now if two subsets are disjoint, then each one is contained in the complement of the other that is, Uz ⊆ X \ A. Therefore X \ A = ⋃z ∈ X\A Uz. So X \ A is a union of open sets and hence is open. Consequently its complement, A, is closed.  □

Example 1.3.6. As applications of Proposition 1.3.5 we have the following:

  • the set [a, b) is not closed in ℝ, since b is a limit point and b ∉ [a, b);

  • the set [a, b] is closed in ℝ, since all the limit points of [a, b] (namely all the elements of [a, b]) are in [a, b];

  • (a, b) is not a closed subset of ℝ, since it does not contain the limit point a;

  • ([a, ∞) is a closed subset of ℝ. ■

The set of all limit points of a set A is called the derived set of the set A and denoted by A'.

Nota bene. In general a limit point for A, needs not to be a point in A.

Proposition 1.3.6. For any set A ⊂ X, the set A ∪ A' is closed.

Proof. We shall show that the set X\(A ∪ A') is open. Let x be an arbitrary point from X\(A ∪ A'). Then x is neither in A nor a limit point of A. If x is not a limit point of A the negation of the limit point property (def.1.3.2) Ux ∩ (A \{x}) ≠ ∅ must be true, i.e. there exists a neighborhood Ux of x that does not intersect neither A nor A'. Hence Ux is disjoint from A and A', thus we conclude that Ux ⊆ X\(A ∪ A'). Given the arbitrary of x we can find other open sets for all points in X\(A ∪ A'), whose union must be the open open set X\(A ∪ A'). This proves A ∪ A' is closed, as desired.  □

Definition 1.3.7. Let the set A be a subset of a topological space (X,τ). Then the set A ∪ A' consisting of A and all its limit points is called the closure of A and is denoted by Ā,.

Every closed set containing A must also contain the set A'. So A ∪ A' = Ā is the smallest closed set containing A. This implies that Ā is the intersection of all closed sets containing A.

Example 1.3.8. Let X = {a, b, c, d, e} and τ = {X, Ø, {a}, {c, d}, {a, c, d}, {b, c, d, e}}. Show that {b̅} = {b, e} , {a, c} = X , and {b, d} = {b, c, d, e}.

Solution. To find the closure of a particular set, we shall find all the closed sets containing that set and then select the smallest. We therefore begin by writing down all of the closed sets – these are simply the complements of all the open sets.

The closed sets are Ø, X, {b, c, d, e}, {a, b, e}, {b, e} and {a}. So the smallest closed set containing {b} is {b, e}; that is, {b̅} = {b, e}. Similarly {a, c} = X, and {b, d} = {b, c, d, e}. ■

Proposition 1.3.9. Let ℚ be the subset of ℝ consisting of all the rational numbers, then ℚ̅ = ℝ.

Proof. Suppose ℚ̅ ≠ ℝ. Then there exists an x ∈ ℝ \ ℚ̅. As ℝ \ ℚ̅ is open in ℝ, there exist a, b with a < b such that x ∈ (a, b) ⊆ ℝ \ ℚ̅. But in every interval (a, b) there is a rational number q; that is, q ∈ (a, b). So q ∈ ℝ \ ℚ̅ which implies q ∈ ℝ \ ℚ̅. This is a contradiction, as q ∈ ℚ. Hence ℚ = ℝ.  □

Definition 1.3.10. Let A be a subset of a topological space (X,τ). Then A is said to be dense in X or everywhere dense in X if Ā = X.  ■

In view of Proposition 1.3.9 ℚ is a dense subset of ℝ. In Example 1.3.8 {a,c} is dense in X.

Example 1.3.11. Let (X,τ) be a discrete space. Then every subset of X is closed (since its complement is open). Therefore the only dense subset of X is X itself, since each subset of X is its own closure.

Definition 1.3.11. Let A be a subset of a topological space (X, τ). Then A is dense in X if and only if every non-empty open subset of X intersects A non-trivially (that is, if U ∈ τ and U = Ø then A ∩ U ≠ Ø.)

Proof. Assume, firstly that every non-empty open set intersects A non-trivially. If A = X , then clearly A is dense in X. If A ≠ X, let x ∈ X \ A. If U ∈ τ and x ∈ U then U ∩ A ≠ Ø . So x is a limit point of A. As x is an arbitrary point in X \ A , every point of X \ A is a limit point of A. So A' ⊇ X \ A and then, by Definition 3.1.9, Ā = A' ∪ A = X; that is, A is dense in X.
Conversely, assume A is dense in X. Let U be any non-empty open subset of X. Suppose U ∩ A = Ø. Then if x ∈ U, x ∉ A and x is not a limit point of A, since U is an open set containing x which does not contain any element of A. This is a contradiction since, as A is dense in X , every element of X \ A is a limit point of A. So our supposition is false and U ∩ A ≠ Ø, as required.
