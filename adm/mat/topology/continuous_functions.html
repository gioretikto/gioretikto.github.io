Continuous functions

Continuous functions represent key tools for comparing topological spaces and for identifying their topological properties.
To a topologist, a map is a continuous function from one space to another.

From calculus, the term continuous function should remind us of an unbroken grapg, usually presenting an equation in x and y such as the rule y = x2, which transform real numbers x into their squares, x2. The square of x = 2 is 4, but notice also that if x is close to 2, say, x = 2.9 or x = 3.002143, then x2 is also close to 9. Thus all points (x- values) is a small neighborhood remain relatively close together after the function has been applied; hence we say that a continuous function preserves closeness.

If we consider the constant function f(x) = 0, which takes as input any point of the number line, agives the value 0 as output. The entire output set of f consists of a single point. which is very different from the real line, thus not every continuous map preserves the essential nature of a space. However, every two points a and b which are close in ℝ are also close in {0} are close, since f(a) = f(b) = 0 and the distance from a to b is 0.

Definition 3.1.1 (Continuous Functions). A function f : (X,τX) → (Y,τY) is said to be continuous if and only if for every open set U in Y, f−1(U) is open in X. In other words, a function f is continuous if f−1 maps open sets into open sets.  □

To check continuity, you need to check all opens in the codomain, and see if thery are mapped by f−1 to open sets.

Continuity expressed in terms of open sets is more general and applied to functions defined on topological spaces; Definition 3.1.1 agrees with that introduced in calculus for metric spaces, however it does not require to introduce a metric.

Lemma 3.1.2. Let f : S → T be a function between two topological spaces. Then f is continuous if, and only if, f−1 (U) is closed whenever U is a closed subset of T.

Proof First let us assume that f is continuous, i.e., f−1 (V) is open whenever V is an open subset of T. If U ⊂ T is closed, then T − U is open, so f−1 (T − U) is open. However, f−1 (T − U) = S − f−1 (U), so the complement of f−1 (U) is open, i.e., f−1 (U) is closed.

For the converse, supposing f is continuous we have

f−1(U) is open for all open U  ⇐⇒  f−1(Uc) is open for all closed U  ⇐⇒  [f−1(Uc)]c is closed for all closed U.  □

Proposition 3.1.3. (Intermediate Value Theorem) Let (X,τ) and (Y,τ') be topological spaces and f : (X, τ) → (Y, τ') surjective and continuous. If (X, τ) is connected, then (Y, τ') is connected.

Proof. Suppose (Y,τ') is not connected. Then it has a clopen subset U such that U ≠ Ø and U ≠ Y . Then f−1 (U) is an open set, since f is continuous, and also a closed set, by Lemma 3.1.2; that is, f−1 (U) is a clopen subset of X. Now f−1 (U) ≠ Ø as f is surjective and U ≠ Ø. Since f is surjective f(X) = Y, so f(X) can't be some proper subset of Y such as U and we can conclude that f−1 (U) ≠ X, as well. Thus (X, τ) is not connected. This is a contradiction. Therefore (Y, τ') is connected.  □

Definition 3.1.4 A topological space (X,τ) is said to be path-connected (or pathwise connected) if for each pair of (distinct) points a and b of X there exists a continuous mapping f : [0, 1] → (X, τ) , such that f (0) = a and f (1) = b. The mapping f is said to be a path joining a to b.
