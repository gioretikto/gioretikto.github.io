Connectedness

If we are aske ho many pieces make up the space X = ℝ − {0}?’ We are able to distinguish two parts in X , namely X− = {x < 0} and X+ = {x > 0}, that in our mind match the word ‘piece’ of the question.

Definition 5.1.1. A topological space (X,τ) is said to be connected if the only subsets that are both open and closed are X and Ø.
A non-connected topological space is called disconnected.  □

Examples 5.1.2.

  1. Evidently, every indescrete toplogical space is connected;

  2. A discrete topological space X is never connected unless |X| = 1 (in which case the discrete and indiscrete spaces coincide). ■

Lemma 5.1.3 On a topological space X the following properties are equivalent:

  1. X is disconnected;

  2. X is the disjoint union of two open, proper subsets;

  3. X is the disjoint union of two closed, proper subsets.

Proof

  1. (1) ⇒ (2), Let A ⊂ X be open, closed and non-empty. If A ≠ X the complement C = X − A is open, closed and non-empty and X is the disjoint union of A and C.

  2. (2) ⇒ (1). Suppose A1 ∪ A2 = X with A1, A2 open, non-empty and disjoint. Then A1 = X − A2 is closed, too.

  3. (3) ⇒ (1). If C1 ∪ C2 = X , where C1, C2 are closed, non-empty and disjoint, C1 = X − C2 is open. □

Thus, X is connected if and only if it cannot be partitioned into two open sets.
An interval which makes these statements true is that an interval is connected—it is all in one piece. The domain (−∞, −1) ∪ (1, ∞) is not connected: It separates into two disjoint open sets (−∞, −1) and (1, ∞).

Example 5.1.3. The space X = {x ∈ ℝ | x ≠ 0} is disconnected. The non-empty subsets

X− = X ∩ ]−∞, 0[,   X+ = X ∩ ]0, +∞[

are open in the subspace topology, disjoint and their union is X.  ■

Example 5.1.4. Let X = {a, b, c, d, e} and τ = {X, Ø, {b}, {c, d}, {b, c, d}, {a, c, d, e}}, then (X, τ) is not connected as {a, c, d, e} is a clopen subset.  ■

Example 5.1.5. If (X,τ) is any discrete space with more than one element, then (X,τ) is not connected as each singleton set is clopen.  ■

Proposition 5.1.6. Let (X,τ) be any topological space; (X,τ) is not connected (or disconnected) if and only if it has proper non-empty disjoint open subsets A and B (i.e. A ∩ B = ∅) such that A ∪ B = X.

Proof. If X = A ∪ B with A and B open and disjoint, then X − A = B and so B is the complement of an open set and hence is also closed. Similarly, A is clopen.  □

So a topological space (X,τ) is said to be connected if it is impossible to partition it into the union of two non-empty disjoint open sets A and B.

Proposition 5.1.7. The space X = {x ∈ ℝ | x ≠ 0} is disconnected. The non-empty subsets

X− = X ∩ (−∞,0) ∩ (a),   X+ = X ∩ (0, ∞) ∪ (a)

are open in the subspace topology, disjoint and their union is X.  □

Theorem 5.1.8 The interval [0, 1] is connected in the Euclidean topology.

Proof. Take C and D non-empty closed subsets of [0, 1] such that C ∪ D = [0, 1]; we claim C ∩ D ≠ ∅. Suppose, to fix ideas, 0 ∈ C, and write d ∈ [0, 1] for the infimum of D. It will be enough to prove d ∈ C ∩ D. d := inf(D) is ≥ 0 and d is in D, as D is closed, d must belong in D, so if d = 0 we are done. But then [0, d) is a subset of C, so d must also be in C, as C is closed, so d ∈ C. □

Path-connectedness

Definition 5.1.9 A topological space X is path(wise)-connected if, given any two points x, y ∈ X, there is a continuous mapping α : [0, 1] → X such that α(0) = x and α(1) = y. Such an α is called a path from x to y.  □

Lemma 5.1.10 Any path connected space is connected.

Proof. Let X be a path connected space and A, B ⊂ X open, non-empty and such that A ∪ B = X: we want to prove A ∩ B ≠ ∅. Choose x ∈ A, y ∈ B and a path α : [0, 1] → X such that α(0) = x and α(1) = y. The open sets α−1 (A), α−1 (B) are non-empty, their union is [0, 1] and so there exists t ∈ α−1 (A) ∩ α−1 (B). The point α(t) belongs to A ∩ B, which is thus not empty.  □

We can prove that a punctured disk is U = {(x,y) \in ℝ2: 0 < x2 + y2 < 1}, is connected by proving that U is path-connected, since any path-connected space is automatically connected.
