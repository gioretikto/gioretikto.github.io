Spazi vettoriali normati

Definizione P.5- Dato X uno spazio vettoriale, si chiama norma ||⋅|| su X un'applicazione

||⋅|| : X → ℝ

con le seguenti proprietà:

  1. ||x|| = 0 se e solo se x =0;

  2. (omonegeità): ||λx|| = ||λ|| ||x|| per ogni x ∈ X e λ ∈ ℝ

  3. (disuguaglianza triangolare): ||x + y|| ≤ ||x|| + ||y|| per ogni x,y ∈ X

La coppia (X, ||⋅||) si chiama spazio normato.

La coppia (X, ||⋅||) si chiama spazio normato.

Una norma vuole rappresentare una misura della lunghezza di un vettore x ∈ X, per cui le proprietà 1, 2, 3 dovrebbero apparire come una naturale richiesta.

Uno spazio normato è anche metrico.

La proprietà (3) è nota come disuguaglianza triangolare.
