Homogeneous systems

We have seen that every system of linear equations has either no solution, a unique solution, or infinitely many solutions. However, there is one type of system that always has at least one solution. A system of the type Ax = 0 is called homogeneous.

Un sistema si dice omogeneo se (b1, b2, ..., bm) = 0, quindi se

Ax = 0

By Cramer's theorem if det A ≠ 0, the only solution is x = 0. Conversely, if det A = 0, then teorema R.0 the columns of A, A1, ..., An are linerly dependent vectors, that is there exist the numbers x1, ..., xn not all null such that

x1 A1 + x2 A2 + ...+ xn An = 0

ossia il sistema Ax = 0 in questo caso ammette almeno una soluzione x non banale (ossia una soluzione x ≠ 0). Si noti che se Ax = 0 si ha per ogni λ ∈ ℝ,

A(λx) = 0

perciò in questo caso le soluzioni del sistema omogeneo sono infinite. Riassumendo vale il seguente teorema.

Teorema 6.0. Il sistema omogeneo di n equazioni in n incognite

Ax = 0

ha solo la soluzione banale (x = 0) se e solo se det A ≠ 0; ha almeno una soluzione non banale se e solo se det A = 0.

Structure of the solutions of a linear system

Proposition 4.1.1. Let Ax = b a consistent linear system. All the solutions of the system Ax = b can be found adding up to the solutions of the associated homogeneous system Ax = 0 an arbitrary solution of Ax = b (said particular).

Proof. Let y,y' two solutions of the Ax = b system. Then we have Ay = b and Ay' = b, thus A(y − y') = 0. Thus the difference between two solutions of Ax = b is always solution of the homogeneous system. Further, a paricular solution y0 of the homogeneous system summed to an arbitraty solution of the comple system gives yet another solution of the complete system since A(y0 + y) = Ay + Ay0 = b.  □
