Rank of a matrix

Definition 5.1.1 (Rank). The rank of any matrix A, written as r(A) is the order of the largest non-vanishing minor (determinant of the square submatrix) in A.  □

Example 5.1.2. Let

A = ( 0 8 1 − 1 2 − 2 − 1 1 1 3 0 0 )

First of all we notice that A can have at most rank 3, as the square submatrices that can be extracted from A can have at most order three. All the possible submtraices of order three of A are

B 1 = ( 0 8 1 2 − 2 − 1 1 3 0 ) B 2 = ( 0 8 1 2 − 2 1 1 3 0 ) B 3 = ( 0 1 − 1 2 − 1 − 1 1 0 0 ) B 4 = ( 8 1 − 1 − 2 − 1 1 3 0 0 )

and it can be readily verified that det B1 = det B2 = det B3 = det B4 = 0. Thus A has not non-null minors of order 3, and its rank must be equal or less than 2. Since there exists in A e.g. det (2 -2; 1 3) = 8 then r(A) = 2. ■

The notion of rank of a matrix is strictly connected to the linear indpendence of its rows (columns). The following results holds

Theorem 5.1.2. Let S = {v1,v2,...,vn} be a set of n-dimensional vectors, then the set S is linearly independent if and only if the determinant of the matrix having the vectors as rows, (columns) is non-zero.

Proof. The vectors v1,v2,...,vn are linearly independent if and only if the vector equation b1 v1 + b2v2 + ... + bnvn = 0 has only the trivial solution b1 = b2 = bn = 0. This vector equation is equivalent to a homogenous system of equations, Ab = 0, and by Cramer's theorem, the trivial solution is the only one iff det A ≠ 0.

For the converse suppose that det A ≠ 0. Then the system Ab = 0, by Cramer's theorem has only the trivial solution b = 0.  □

Example. Consider the three vectors that follow.

v1 = (1,0,2)
v2 = (3,4,−1)
v3 = (4,4,1).

Taking the components of the three vectors to be rows of a matrix we obtain the following matrix,

A = ( 1 0 2 3 4 -1 4 4 1 )

We now calculate the determinant of A

det(A) = 1 ⋅ (4 + 4) −0 ⋅(3 + 4) + 2 ⋅ (12 − 16) = 8 − 0 + (−8) = 0

Since det A = 0 we conclude that the three given vectors are linearly dependent. This can also be seen very easily as follows,

(4,4,1) = (1,0,2) + (3,4,1)
v3 = v1 + v2
⇒ v1 + v2 − v3 = 0 ■

Definition 5.1.2 (Rank). The maximum number of linearly independent vectors (those forming either the rows or the columns) is callde the rank of the matrix. We use the notation

rank(A)

to denote the rank of the matrix A. □

Because multiplication by a nonzero scalar does not change the linear independence of vectors, for the scalar a ≠ 0, we have

rank(aA) = rank(A)
