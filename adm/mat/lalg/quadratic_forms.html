Quadratic forms

Quadratic forms, which can be thought of as a generalization quadratic functions of one input variable (e.g., q(x) = 3x2 + 2x − 7) to multiple variables, in the same way that linear forms generalize linear functions from one input variable to multiple variables.

Definition 2.A.1 (Quadratic Forms). Suppose V is a vector space over ℝ. Then a function q : V → ℝ is called a quadratic form if there exists a bilinear form f : V × V → ℝ such that q(v) = f(v, v) for all v ∈ V. □

For example, the function q : ℝ2 → ℝ defined by q(x, y) = 3x2 + 2xy + 5y2 is a quadratic form, since if we let v = (x, y) and

A = ( 3 1 1 5 ) then q ( v ) = v T A v

A quadratic form over ℝn è un polinomio omogeneo di secondo grado in n variabili:

q(h) = q (h1, h2, ..., hn) = ∑ni,j=1 aij hihj

where the aij are real coefficients, known as coefficients of the quadratic form, che individuano una matrice quadrata n x n. Sia A tale matrice, questa è una matrice simmetrica detta associata alla forma quadratica q. La f.q può scriversi in forma matriciale:

q(h) = hT A h

indicando con h un vettore colonna, with hT il suo trasposto (row vector), and with hT A h il prodotto righe per colonne. la f.q. può scriversi mediante una matrice A simmetrica, in altre parole non è restrittivo supporre aij = aji.

Un altro esempio di forma quadratica

q(h1, h2) = fxx(x0,y0) + 2fxy(x0,y0) + fyy(x0,y0) h22    A = Hf(x0,y0)

questo è il differenziale secondo di una funzione di due (o n) variabili e la matrice Hessiana è la matrice dei coefficienti.

Siamo interessati al segno che una f.q. può assumere al variare di h. I seguenti semplici esempi in ℝ2 illustrano le possibilità di comportamento:

  1. h12 + h22   è positiva per ogni (h1, h2) ≠ (0,0);

  2. −h12 + h22   è positiva per ogni (h1, h2) = (0,1) e negativa per (h1, h2) = (1,0);

  3. −h12 − h22   è positiva per ogni (h1, h2) = (0,1) e negativa per (h1, h2) = (1,0);

  4. h12   è semppre positiva tranne che nei vettori del tipo (h1,0);

  5. h12   è semppre negativa tranne che nei vettori del tipo (h1,0);

These examples suggest the following characterization of the q.f. or the matrix associated to it.

Definition. A quadratic form (or the correspondent symmetric matrix) q(h), h ∈ ℝn is said

  1. definita positiva (negativa) se per ogni h ≠ 0, q(h) > 0 (< 0);

  2. semidefinita positiva (negativa) se per ogni h ≠ 0, q(h) ≥ 0 (≤ 0);

  3. indefinita se esistono h1, h2 tali che q(h1) > 0 e q(h2) < 0

Le condizioni a),b) and c) sono mutualmente esclusive.
