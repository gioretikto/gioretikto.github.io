Multilinear Transformations

We introduce now multilinear transformations, which are functions that act on multiple vectors, each in a linear way. Recall from Section 1.2.3 that a linear transformation is a function that sends vectors from one vector space to another in a linear way, and that a multilinear form is a function that sends a collection of vectors to a scalar in a manner that treats each input vector linearly.
