Gauss Elimination Method

Gauss elimination is a procedure that transforms a linear system of equations into upper-triangular form, the solution of which is found by back substitution.

Definition 4.4.1. Two linear systems (not necessarily square) of the same order are said equivalent if they have exactly the same solutions. □

Definition 4.4.2. Given two equations

a1x1 + ... + anxn = a  b1x1 + ... + bnxn = b

it said linear combination of the two equation of coefficients h,k ∈ ℝ the equation

h(a1x1 + ... + anxn) + k(b1x1 + ... + bnxn) = ha + kb

The coefficients of this new equation are haj + kbj, for j = 1,...,n. □

To maintain the identity of a set of linear equations so that they are equivalent the possible manipulations are as follows

  • shuffling or swapping the equations (or rows);

  • multiplying any equation by a constant

  • replacing any equation of a linear system by a linear combination of equations of the sane linear system the solutions of the system are unchanged (as the Next Lemma shows).

Lemma 4.4.3. Let Ax = b be the following linear system

a1x1 + ... + anxn = a
b1x1 + ... + bnxn = b (4.4.1)

Let Ãx = b̃ the linear system obtained by substituting Eq. 4.4.1 with

h(a1x1 + ... + anxn) + k(b1x1 + ... + bnxn) = ha + kb  (4.4.2)

with h,k ∈ ℝ two real numbers such that k ≠ 0. Then the systems Ax = b and Ãx = b̃ are equivalent.

Proof. Let (v1, ..., vn) a solution of Ax = b. This means that the numbers v1, ..., vn satisfy all the equations of the system; In particular

{ a 1 v 1 + … + a n v n = a b 1 v 1 + … + b n v n = b 4.4.3

Then (v1, ..., vn) is also a solution of Eq. 4.4.2 − thus of Ãx = b̃ as well.

Conversely, let (v1, ..., vn) a solution of Ãx = b̃. Then the first row of 4.4.3 is still verified and substituting it in Eq. 4.4.2 we find

k(b1x1 + ... + bnxn) = kb

and since k ≠ 0, we see that v1, ..., vn satisfy Eq. 4.4.1 as well, thus are a solution of Ax = b.  □

Algorithm

The elements aii of the coefficient matrix corresponding to the given linear system of equations are called the pivot elemnets.

  • Step 1: Write the augmented matrix for the system of linear equations;

  • Step 2: Use elementary row operation on the augmented matrix [A|b] to transform A into upper triangular form. If a zero is located on the diagonal, switch the rows until a nonzero is in that place. If you are unable to do so, the sysmte has either infinite or no solutions.

  • Step 3: Use back substitution to find the solution of the problem.

2x + y − z = 8
−3x − y + 2z = −11
−2x + y + 2z = −3
