Rouché-Capelli Theorem

Let A the coefficient matrix m x n of a linear system of m linear equations in n unknowns. We can write the linear system as

Ax = b

Or fixing two bases in the vector space ℝn and ℝm as a linear transformation

L(x) = b

with x ∈ ℝn and ∈ ℝm. Solving the system consists in determining which vectors x ∈ ℝn have as image through L the vector ∈ ℝm.

The Rouché-Capelli Theorem connects the behavior of a linear system with the ranks of its coefficient matrix A and that of the augmented matrix [A|b], giving a necessary and sufficient condition for the linear system to be consistent.

(Rouché–Capelli Theorem) A system of m linear equations in n unknowns is consistent if and only if the rank of the augmented matrix [A|b] is equal to the coefficient matrix A.
