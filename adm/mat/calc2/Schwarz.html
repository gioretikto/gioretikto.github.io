Derivate successive. Teorema di Schwarz

Quando una funzione f(x,y) è derivabilie in un aperto A ⊆ ℝn, le sue derivate parziali possono essere a loro volta derivabili. È possibile calcolare la derivata parziale rispetto a x o y. Ad esempio se fx(x, y) è derivabile la si può derivare ulteriormente rispetto le variabili x o y ottenendo le derivate parziali seconde indicate con i simboli equivalenti:

f x x ( x , y ) , ∂ 2 f ∂ x 2 ( x , y ) , ∂ ∂ x ∂ f ∂ x ( x , y ) f x y ( x , y ) , ∂ 2 f ∂ y ∂ x ( x , y ) , ∂ ∂ y ∂ f ∂ x ( x , y )

Le derivate fxy e fyx vengono dette derivate parziali seconde miste.

5.0 (Schwarz Theorem). Sia f: A ⊆ ℝ → ℝn con A aperto. Se due derivate seconde miste esistono e sono continue in un punto x0, allora in tale punto esse coincidono. Se le derivate miste esistono e sono continue in A allora esse coincidono in tutto A.

Proof. Dimostriamo il teorema per una funzione f(x,y) ma questa avrà valore generale per funzioni ad n variabili. Si vuole dunque verificare che:

∂ 2 f ∂ y ∂ x ( x 0 , y 0 ) = ∂ 2 f ∂ x ∂ y ( x 0 , y 0 )

Ipotizziamo che tali derivate esistano in un intorno Ur(x0,y0) di x0 = (x0, y0) e siano anche continue in x0. Introduciamo la funzione ausiliaria

g(h,k) = f(x0 + h, y0 + k) − f(x0 + h, y0) − f(x0, y0 + k) + f(x0, y0)

Per h, fissato consideriamo la funzione di una variabile:

u(t) = f(x0 + h, t) − f(x0, t)

Si noti che

g(h,k) = u(y0 + k) − u(y0)

Sotto le ipotesi iniziali la funzione u risulta derivabile nell'itervallo [y0, y0+k]. Applicando il Teorema di Lagrange ad u su questo intervallo, otteniamo:

g(h,k) = u(y0 + k) − u(y0) = ku' (y0 + δ1k)

con δ1 ∈ (0,1) e dipendente da h e k. Sostituendo u' abbiamo:

g ( h , k ) = k [ ∂ f ∂ y ( x 0 + h , y 0 + δ 1 k ) − ∂ f ∂ y ( x 0 + h , y 0 + δ 1 k ) ]

Applichiamo una seconda volta il Teorema del valor medio alla funzione racchiusta tra parentesi quadre:

z ( s ) = ∂ f ∂ y ( s , y 0 + δ 1 k )

si ottiene

g ( h , k ) = h k [ ∂ f ∂ x ∂ f ∂ y ( x 0 + δ 2 h , y 0 + δ 1 k ) ]

con δ2 ∈ (0,1) e dipendente da h e k. Ora si ripeta il procedimento scambiando il ruolo delle variabili, ossia

g(h,k) = w(x0 + h) − w(x0)
w(t) = f(t, y0 + k) + f(t, y0)

con passaggi analoghi si ottiene:

g ( h , k ) = h w ′ ( x 0 + δ 3 h ) = h [ ∂ f ∂ x ( x 0 + δ 3 h , y 0 + k ) − ∂ f ∂ x ( x 0 + δ 3 h , y 0 ) ] = h k [ ∂ ∂ y ∂ f ∂ x ( x 0 + δ 3 h , y 0 + δ 4 k ) ]

con δ3,δ4 ∈ (0,1), dipendenti da h e k. Uguagliando le due diverse relazioni ottenute per la funzione g(h,k), otteniamo:

∂ 2 f ∂ x ∂ x ( x 0 + δ 2 h , y 0 + δ 1 k ) = ∂ 2 f ∂ y ∂ x ( x 0 + δ 3 h , y 0 + δ 4 k )

Facendo tendere h e k a zero, e per la continuità delle derivate miste si ha la tesi del teorema.  □

Esempio 1. Sia f(x,y) = y3 cos x funzioni sicuramente con derivate seconde continue; calcoliamo

∂ f ∂ x = − y 3 sin ⁡ x ; ∂ f ∂ x = 3 y 2 cos ⁡ x ∂ 2 f ∂ x 2 = ∂ ∂ x ( − y 3 sin ⁡ x ) = − y 3 cos ⁡ x ; ∂ 2 f ∂ y ∂ x = ∂ ∂ y ( − y 3 sin ⁡ x ) = − 3 y 2 sin ⁡ x ∂ 2 f ∂ x 2 = ∂ ∂ x ( 3 y 2 sin ⁡ x ) = − 3 y 2 sin ⁡ x ; ∂ 2 f ∂ y = ∂ ∂ y ( 3 y 2 cos ⁡ x ) = 6 y cos ⁡ x

Si noti che le derivate miste fxy e fyx sono uguali come previsto dal Teorema di Schwarz.
