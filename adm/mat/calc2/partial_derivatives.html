Partial derivatives

We are interesente in giving the definition of derivative for a function f: ℝn ⟶ ℝ. We shall treat the case n = 2, which can be generalized to higher dimensions.
For the case n = 2, f(x,y) has two variable that can be incremented. The strategy is to increment one variable say, x, while keeping the other constant, that is we set y = y0 and increment x, i.e. we differentiate the single variable function x ↦ f(x, y0) respect to x.
Analogously if we want to differentiate with respect to y we keep constant the value of x by setting x = x0 and increment y, i.e. we differentiate the single-variable function y ↦ f(x0, y). These two operations are expressed by the following two formulas, whenever such limit exists and is finite:

∂ f ∂ x ( x 0 , y 0 ) ≡ lim h → 0 f ( x 0 + h , y 0 ) − f ( x 0 , y 0 ) h

and

∂ f ∂ y ( x 0 , y 0 ) ≡ lim k → 0 f ( x 0 , y + k ) − f ( x 0 , y 0 ) k

the first is said partial derivative with respect to x, and the second partial variable with respect to y. They are indicated with the following symbols.

∂ f ∂ x ( x 0 , y 0 ) , ∂ x f ( x 0 , y 0 ) , D x f ( x 0 , y 0 ) , D 1 f ( x 0 , y 0 ) ∂ f ∂ y ( x 0 , y 0 ) , ∂ y f ( x 0 , y 0 ) , D y f ( x 0 , y 0 ) , D 2 f ( x 0 , y 0 )

Example 1. Let f = √x2 + y2 and calculate the partial derivatives at x0 = (2,3).

∂ f ∂ x ( 2 , 3 ) = x x 2 + y 2 | x = 2 = 2 13 ∂ f ∂ y ( 2 , 3 ) = y x 2 + y 2 | y = 3 = 3 13

Remark. To be able to calculate the partial derivatives of f at a point (x0, y0) is necessary that the incremented point belongs to the domain of f as well. This is certainly true if the domain of f is open, by property d) of theorem 2.1.3. For this reason we shall only deal with functions defined on open domains. ■

Gradient operator

Il vettore che ha per componenti le derivate parziali di f at x0 si dice gradient of f, and is indicated with one of the following symbols:

∇f(x0,y0),   Df(x0,y0),   grad f(x0,y0)

more explicitely as

∇ f ( x 0 ) = ∂ f ∂ x ( x 0 ) i + ∂ f ∂ y ( x 0 ) j + ∂ f ∂ z ( x 0 ) k

where we employed the vector operator, ∇, which is called “del” or “nabla”, like all mathematical operators, is an action waiting to happen. Just as √ tells you to take the square root of anything that appears under its roof, ∇ → is an instruction to take derivatives in three directions. Specifically,

∇ → = ∂ ∂ x ı ^ + ∂ ∂ y ȷ ^ + ∂ ∂ z k ^

where î, ĵ, and k̂ are the unit vectors in the direction of the Cartesian coordinates x, y, and z.

If U(r) = U(x,y,z) is a scalar field, ie a scalar function of position r = [x,y,z] in 3d imensions, then note immediately that ∇U is a vector field.

Saying that a function admits (first) partial derivatives at x0 ∈ ℝn is the same as saying that f has a gradient. L'esistenza del gradiente non implica la differenziabilità, ma solo la derivabilità. A stronger condition is requiring that at x ∈ ℝn the function be differentiable.

Theorem. Per ogni coppia di funzioni derivabili f,g: ℝn ⟶ ℝ valgono le seguenti proprietà del gradiente:

∇(αf + βg) = α∇f + β∇g,   ∇(fg) = g∇f + f∇g   ∇(f/g) = (g∇f − f∇g)/g2

Analoghe proprietà valgono per i differenziali, ad esempio

d(αf + βg) = α df + β dg, ecc
