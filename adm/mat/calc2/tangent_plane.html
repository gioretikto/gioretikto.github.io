Tangent plane

In una variavile definire la derivata di una funzione permette di definire la retta tangente. Per una funzione di due variabili il problema analogo è definire il piano tangente alla superficie del grafico della funzione. Ci proponiamo ora di determinare tale piano

Lasciandoci guidare dall'intuizione geometrica. Se sezioniamo il gradico di z = f(x,y) con il piano vericale y = y0, troveremo una curva in questo stesso piano, descritta dall'equazione z = f(x,y0). La retta r1 tangente a tale curva in x0 starà anche sul piano tangente che cerchiamo; ripetendo la stessa considerazione lungo il piano vericale x = x0 troviamo una seconda retta r2 passante per il piano (x0,y0, f(x0,y0)).

L'equazione della retta r1 nel piano y = y0 is

z = f ( x 0 , y 0 ) + ∂ f ∂ x ( x 0 , y 0 ) ( x − x 0 )

la stessa retta, considerata nello spazio (x,y,z) anziché nel piano y = y0, è individuata dal sistema:

{ z = f ( x 0 , y 0 ) + ∂ f ∂ x ( x 0 , y 0 ) ( x − x 0 ) y = y 0

Analogamente la retta r2 è individuata dal sistema:

{ z = f ( x 0 , y 0 ) + ∂ f ∂ y ( x 0 , y 0 ) ( y − y 0 ) x = x 0

Il piano che contiene entrambe le rette è:

z = f ( x 0 , y 0 ) + ∂ f ∂ x ( x 0 , y 0 ) ( x − x 0 ) + ∂ f ∂ y ( x 0 , y 0 ) ( y − y 0 )

che si verifica immediatamente dal fatto che ogni punto (x,y,z) che soddisfa (4.3) o (4.4) soddisfa anceh (4.5). Possiamo affermare che la (4.5) è l'equazione del piano tangente al grafico di z = f(x,y) nel punto (x0,y0, f(x0,y0)).

Example 1. Calculate the tangent plane to the graph of z = x2 + y2 at (1,1).

∂f/∂x = 2x   ∂f/∂y = 2y

∂f/∂x (1,1) = 2   ∂f/∂y (1,1) = 2   f(1,1) = 2

The equation is

z = 2 + 2(x − 1) + 2y(y − 1)
