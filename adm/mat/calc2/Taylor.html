Formula di Taylor al secondo ordine

Si è visto come la differenziabilità di una funzione permetta di approssimare il grafico della funzione con quello del piano tangente, in forma vettoriale

f(x) = f(x0) + ∇f(x0) ⋅ (x − x0) + o(||x − x0||),   x ⟶ x0

questo non è altro che uno sviluppo di Taylor al primo ordine, con resto secondo Peano.
Se la funzione è di classe C2 è possibile migliorare tale approssimazione con la formula di Taylor al secondo ordine, analogamente a quanto visto per le funzioni ad una variabile.

Sia f: ℝ2 → ℝ definita in un intorno di x0 e sia (h, k) un versore; il punto (x0 + ht, y0 + kt) appartiene dunque a tale intorno almeno per t abbastanza piccolo. Definiamo ora, la funzione di una variabile g nel seguente modo:

g(t) = f(x0 + th, y0+tk)

definita almeno nell'intorno t ∈ [0, ε]. Lo sviluppo di MacLaurin al secondo ordine per la funzione g(t) è

g(t) = g(0) + tg'(0) + 1/2 t2g''(0) + o(t2)  t ⟶ 0  (4.8.1)

Derivando tale funzione, applicando la regola della catena:

g ′ ( t ) = f x ( x 0 + t h , y 0 + t k ) h + f y ( x 0 + t h , y 0 + t k ) k g ″ ( t ) = f x x ( x 0 + t h , y 0 + t k ) h 2 + 2 f x y ( x 0 + t h , y 0 + t k ) h k + f y y ( x 0 + t h , y 0 + t k ) k 2

sostituendo nella 4.8.1:

f ( x 0 + t h , y 0 + t k ) = f ( x 0 , y 0 ) + f x ( x 0 , y 0 ) t h + f y ( x 0 , y 0 ) t k + 1 2 [ f x x ( x 0 , y 0 ) t 2 h 2 + 2 f x y ( x 0 , y 0 ) t 2 h k + f y y ( x 0 , y 0 ) t 2 k 2 ] + o ( t 2 )

per t → 0.

Ponendo (dx, dy) = (th, tk) abbiamo |(dx, dy)| = t.

f ( x 0 + d x , y 0 + d y ) = f ( x 0 , y 0 ) + f x ( x 0 , y 0 ) d x + f y ( x 0 , y 0 ) d y + 1 2 [ f x x ( x 0 , y 0 ) d x 2 + 2 f x y ( x 0 , y 0 ) d x d y + f y y ( x 0 , y 0 ) d y 2 ] + o ( d x 2 + d y 2 )

questa è la formula di Taylor al secondo ordine, con resto secondo Peano. Studiamo i termini che compaiono a secondo membro; la prima sommatoria non è altro che il differenziale di f, che equivale al prodotto scalare del gradiente per l'inciremento di x. La seconda sommatoria è rappresentata dal differenziale secondo di f in x0.

Definizone 3.2 Se f ∈ C2(A) si dice differenziale secondo di f in x0 ∈ A, la funzione

d 2 f ( x 0 ) = ∑ i = 1 n ∑ j = 1 n ∂ 2 f ∂ x i ∂ x j ( x 0 ) d x j d x i

I coefficienti che compiano nel differenziale secondo possono essere disposti in una matrice n  x  n detta matrice Hessiana di f in x0:

H f = [ ∂ 2 f ∂ x 1 2 ∂ 2 f ∂ x 1 ∂ x 2 ⋯ ∂ 2 f ∂ x 1 ∂ x n ∂ 2 f ∂ x 2 ∂ x 1 ∂ 2 f ∂ x 2 2 ⋯ ∂ 2 f ∂ x 2 ∂ x n ⋮ ⋮ ⋱ ⋮ ∂ 2 f ∂ x n ∂ x 1 ∂ 2 f ∂ x n ∂ x 2 ⋯ ∂ 2 f ∂ x n 2 ]

In particolare se f è funzione di due variabili si ha la matrice Hessiana:

H f ( x 0 ) = [ f x x ( x 0 , y 0 ) f x y ( x 0 , y 0 ) f y x ( x 0 , y 0 ) f y y ( x 0 , y 0 ) ]

Dal Teorema di Schwarz sappiamo che se f ∈ C2(A) la matrice hessiana risulta simmetrica in ogni punto di A.

Usando il simbolismo della matrice Hessiana si può esprimere il teorema di Taylor nel seguente modo:

f(x0 + dx) = f(x0) + ∇f(x0) ⋅ (dx) + 1/2 ⋅ dxT Hf(x0) dx + o(||dx||2),   x ⟶ x0

indicating with dx a column vector and with dxT its transpone a row vector.
