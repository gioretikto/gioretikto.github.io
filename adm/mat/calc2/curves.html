Curves

Consideriamo un punto materiale che si muove nello spazio tridimensionale lungo una certa traiettoria. Le sue coordinate (x,y,z) saranno funzione del tempo t, perciò la sua posizione è specificata da una fuzione vettoriale del tipo:

r: ℝ ⟶ ℝ3

con r = (x,y,z) e

{ x = x ( t ) y = y ( t ) t ∈ [ t 1 , t 2 ] z = z ( t )

Si può usare anche la notazione vettoriale:

r(t) = x(t) i + y(t) j + z(t) k

o anche

r(t) = (x(t),y(t),z(t))

Se il punto si muovesse nel piano, per rappresentare il moto basterebbe una funzione

r: ℝ ⟶ ℝ2

con r = (x,y) e

{ x = x ( t ) y = y ( t ) t ∈ [ t 1 , t 2 ]

Se rappresentiamo nello spazio la linea percorsa dal punto in tutto l'intervallo di tempo [t1, t2], abbiamo una rappresentazione parziale di questa funzione: infatti, questa linea ci dice in quali punti passa l'oggetto, ma non in quale istante si trova in ciascun punto. Questo perché la traiettoria non è il grafico della funzione (ossia l'insieme {t, r(t)} che dovrebbe essere un oggetto nel spaizo ℝ4 ma l'immagine della funzione (cioè l'insieme dei suoi valori). Spesso si prescinde dall'interpretazione cinematica, e si usano espressioni come "la circonferenza, l'ellisse, ecc." per indicare le funzioni, e non solo le loro immagini.

Esempio 2.1.1. La funzione r: ℝ ⟶ ℝ2

{ x = cos ⁡ ( 2 π t T ) y = sin ⁡ ( 2 π t T ) t ∈ [ 0 , T ]

rappresenta un punto che si muove di moto circolare uniforme, sulla circonferenza unitaria di centro l'origine, nel piano, compiendo nell'intervallo di tempo [0, T] un giro completo. Si osservi che la funzione non si limita a descrivere la circonferenza, ma specifica il modo in cui il punto mobile la percorre. ■

Example 2.1.2. (of Curves) Curves in the plane can be described in at least three different ways:

  • as the graph of a function y = g(x); e.g. a straight line is represented as y = mx + q or x = my + q;

  • as the vanishing locus of a two variable function, f(x,y) = 0; for a line the first degree polynomial ax + by + c = 0;

  • in parametric form as the image of a map f: ℝ → ℝ2 having the form f(t) = (αt + β, γt + δ). ■
