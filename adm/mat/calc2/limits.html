Limits

The definition of limit for functions of one real variable is easily extended to several variables. To do that we consider a subset A in ℝn and a function f : A → ℝ, so f is a real function defined on A ⊆ ℝn. Let x0 be a limit point of A.
We say f(x) tends to ℓ ∈ ℝ ∪ {±∞} as x → x0 , written

lim x → x 0 f ( x ) = ℓ

if for every sequence {xk} of points of ℝn such that xk ⟶ x0 for k ⟶ ∞ (with xk ≠ x0, ∀ k) we have that

lim k → ∞ f ( x k ) = ℓ

Geometrically this means that as the distance between x and x0 decreases the value of f(x) gets closer to that of ℓ.

Using the ε-δ definition of limit we can state the same concept: for every ε > 0 there exists a δ such that 0 < |x − x0| < δ implies |f(x) − ℓ| < ε.

Corollary 1.0 (permanence of the sign) Let Ω ⊆ ℝn, f: Ω → ℝ and x0 ∈ ℝn an accumulation point in Ω. If there exists limx → x0 f(x) = ℓ ∈ R, ℓ ≠ 0, there there exists a neighborhood U ⊆ ℝn of x0 such that f on U ∩ (Ω \ {x0} ) has the same sign of ℓ.

Proof. By the ε-δ definition of limit, by setting ε = ℓ/2

ℓ − |ℓ| / 2 < f(x,y) < ℓ + |ℓ| / 2

If ℓ > 0 from the first inequality

0 < ℓ/2 < f(x,y)

If ℓ < 0 from the second inequality

f(x,y) < ℓ − ℓ/2 = ℓ/2 < 0. □

limx ⟶ x0   f(x) = f(x0)
