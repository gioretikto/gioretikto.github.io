Jacobian matrix and determinant

In vector calculus, the Jacobian matrix, is the matrix of all first-order partial derivatives of a vector-valued function. Specifically, suppose f: ℝn → ℝm is a function which takes as input the vector x ∈ ℝn and produces as output the vector f(x) ∈ ℝm. Then the Jacobian matrix of f denoted by J whose (i,j)th entry is Jij = ∂fi/∂xj an m x n matrix, is defined as follows:

J = [ ∂ f ∂ x 1 ⋯ ∂ f ∂ x n ] = [ ∇ T f 1 ⋮ ∇ T f m ] = [ ∂ f 1 ∂ x 1 ⋯ ∂ f 1 ∂ x n ⋮ ⋱ ⋮ ∂ f m ∂ x 1 ⋯ ∂ f m ∂ x n ]

This matrix, whose entries are functions of x, is also denoted by Df, Jf and ∂(f1,...,fm)/∂(x1,...,xn).

A transformation from an ℝn space to a ℝm is a correlation that maps a point from the first space (original) to a point in the second space (transformed) where each point in the original and transformed spaces is identified by n independent coordinates. The Jacobian represents the coordinate transformation we are looking for.

Example (polar-Cartesian transformation) The transformation from polar coordinates (r, φ) to Cartesian coordinates (x, y), is given by the function f: ℝ+ × [0, 2π) → ℝ2 with components:

x = r cos φ
y = r sin φ

J F ( r , φ ) = [ ∂ x ∂ r ∂ x ∂ φ ∂ y ∂ r ∂ y ∂ φ ] = [ cos ⁡ φ − r sin ⁡ φ sin ⁡ φ r cos ⁡ φ ] = r cos 2 ⁡ θ + r sin 2 ⁡ θ = r ( cos 2 ⁡ θ + sin 2 ⁡ θ ) = r .

The Jacobian determinant is equal to r. This can be used to transform integrals between the two coordinate systems.
