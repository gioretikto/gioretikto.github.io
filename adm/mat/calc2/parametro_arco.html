Parametro arco o ascissa curvilinea

La lunghezza dell'arco r(τ) per τ da t0 a t è una funzione di t:

s ( t ) = ∫ t 0 t | r ′ ( τ ) | d τ

Se si è in grtado di calcolare esplicitamente tale funzione, e poi di invertirla, esprimendo t come funzione di s, è possiible riparametrizzare la curva in funzione del s, detto parametro arco o ascissa curvilinea.

Esempio 2.6.2. Per la circonferenza

{ x = R cos ⁡ θ y = R sin ⁡ θ θ ∈ [ 0 , 2 π ]

si ha

s ( θ ) = ∫ 0 θ R d θ = R θ

da cui θ = s/R. Si può riparametrizzare la curva come:

{ x = R cos ⁡ ( s R ) y = R sin ⁡ ( s R ) s ∈ [ 0 , 2 π R ]

dove s è il parametro arco.

Esempio Ad es., per l'elica cilindrica dell'esempio precedente è

s ( t ) = ∫ 0 t R 2 + p 2 d τ = t R 2 + p 2

da cui t = s/sqrt(R2 + p2). Si può allora riparamettrizzare la curva come

{ x = R cos ⁡ s R 2 + p 2 y = R cos ⁡ s R 2 + p 2 s ∈ [ 0 , 2 π R 2 + p 2 ] z = p s R 2 + p 2

ed s viene allora detto parametro arco.

Ciò che ha di speciale questa parametrizzazione è che il parametro s è esattamente lo spazio percorso dal punto mobile quando il parametro passa da 0 a s.
Se r = r(s) è una cruva parametrizzata mediante il parametro arco, il vettore derivato r'(s) coincide col versore tangente, uT. Infatti, v(s) = |r'(s)| = 1, come si vede dalle relazioni

d r d s = d r d t ⋅ d t d s = 1 v ( t ) d r d t | d r d s | = 1 v ( t ) d r d t = v ( t ) v ( t ) = 1

Il vettore r'(s) contiene informazioni puramente geometriche: infatti, nonostante sia una "velocità", si tratta di un vettore adimensionale.
Se r(t) è una curva parametrizzata rispetto a un parametro t qualunque (non necessariamente il parametro arco), dalla (4.3) leggiamo la relazione:

ds/dt = |r'(t)| = v(t)

che si riscrive anche, simbolicamente, nella forma

ds = |r'(t)|dt = v(t)dt

Il simbolo ds si chiama lunghezza d'arco elementare. Il suo significato intuitivo è quello di spazio percorso dal punto mobile nell'intervallo di tempo "infinitesimo" dt: nell'intervallo di tempo [t, t + dt] il punto si sposta di un incremento dr dato da dr = r'dt, a meno di infinitesimi di ordine superiore.
