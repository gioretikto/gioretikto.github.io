Systems of Linear equations

Le matrici intervengono nello studio dei sistemi lineari. Per sistema lineare s'intende un sistema di equazioni algebriche di primo grado.

Let x1, x2, ..., xn, be n unknown variables. A linear equation (or of first degree) in the unknown variables xi with coefficients in 𝕂 is an equation of the form

a1x1 + ... + anxn = b   (4.1.1)

where a1, a2, ..., an and b are given scalars in 𝕂.

A solution of a linear equation is a vector (v1, v2, ..., vn) ∈ 𝕂n, whose components satisfy Eq. 4.1.1.

A system of linear equations is a finite set of linear equations, each with the same variables. A solution of a system of linear equations is a vector that is simultaneously a solution of each equation in the system. The solution set of a system of linear equations is the set of all solutions of the system. We will refer to the process of finding the solution set of a system of linear equations as “solving the system.”

Consider a system of n equations in n unknown variables.

{ a 11 x 1 + a 12 x 2 + … + a 1 n x n = b 1 a 21 x 1 + a 22 x 2 + … + a 2 n x n = b 2 ⋮ a n 1 x 1 + a n 2 x 2 + … + a n n x n = b n (4.1)

The coefficient matrix A

A = ( a 11 a 12 … a 1 n a 21 a 22 … a 2 n ⋮ a n 1 a n 2 … a n n )

The system can be written as

A1 x1 + A2 x2 + ...+ An xn = b

with Ai the column vectors of the matrix A.

Una soluzione del sistema (4.1) è un elemento di (x1, x2, ..., xn) ∈ 𝕂n che è soluzione simultanea delle n equazioni. Il sistema si dice compatibile (incompatibile) se possiede almeno una soluzione (se non ne possiede).
By letting

x = ( x 1 x 2 ⋮ x n ) b = ( b 1 b 2 ⋮ b n )

il sistema (4.1) può scriversi, come:

Ax = b

A system of linear equations is called consistent if it has at least one solution. A system with no solutions is called inconsistent. We will prove later that these same three possibilities hold for any system of linear equations over the real numbers. A system of linear equations with real coefficients has either

  1. a unique solution (a consistent system) or

  2. infinitely many solutions (a consistent system) or

  3. no solutions (an inconsistent system).

Soluzione di un sistema normale - Teorema di Cramer

One of the most popular methods for solving simple systems of linear equations (as 2 x 2 or 3 x 3) is Cramer's rule.

Per risolvere il sistema ricorriamo all'analogia con l'equazione scalare ax = b. Se a ≠ 0, l'unica soluzione è x = a/b. L'analogo della condizione a ≠ 0 è ora det A ≠ 0. Infatti, per il teorema sulla matrice inversa, se det A ≠ 0 esiste A−1; moltiplicando a sinistra A−1 entrambi i membri si trova

A−1(Ax) = A−1 ⋅ b

Utilizzando, la proprietà associativa per il prodotto tra matrici, si ricava:

(A ⋅ A−1)x = A−1 ⋅ b

ed infine poichè A ⋅ A−1 = In and In x = x, si ottiene la formula seguente, per il vettore soluzione del sistema:

x = A−1 ⋅ b

Si osserva che, sviluppando i calcoli indicati dalla, le componenti xi of x si ottengono dalla formula seguente:

xi = |Bi| / |A|   (4.2)

The matrices Bi in Cramer’s rule are obtained by replacing the ith column of A with the vector b.

B i = ( a 11 a 12 … b 1 … a 1 n a 21 a 22 … b 2 … a 2 n ⋮ ⋮ ⋮ ⋮ a n 1 a n 2 … b n … a n n )

Recalling the expression for the inverse matrix, we have

x i = ( A − 1 b ) i = 1 det  A ∑ j = 1 n A j i b j = 1 det  A ⋅ det  B i

Each xi in this relation, is obtained by multiplying each row of A−1 by a the column vector b.

xi = (1/det A) (A1ib1 + A2ib2 + ... + Anibn)

but (A1ib1 + A2ib2 + ... + Anibn) is the Laplace expansion of the matrix Bi, so we have

xi = det Bi/ det A

Cramer's theorem. Consider the system of n equations in n unknown variables

Ax = b

con A matrice (n,n), b ∈ ℝn assegnato e x ∈ ℝn incognito. Se det A ≠ 0, il sistema è determinato, ossia ha una e una sola soluzione, assegnata dalla (4.2).

Sistemi che ammettono le stesse soluzioni si dicono equivalenti. Due sistemi lineari sono sicuramente equivalenti se uno è ottenibile dall'altro operando sulle equazioni con un numero finito di trasformazioni, come:

  1. scambio di due equazioni

  2. moltiplicazione di una equazione per un numero reale λ ≠ 0.

  3. sostituzione di una equazione con la somma di essa e di un'altra.
