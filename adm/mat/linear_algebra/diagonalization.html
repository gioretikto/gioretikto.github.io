Diagonalization

Linear transformations from ℝn to ℝn are represented by n x n matrices. Given such a transformation, the use of all different bases of ℝn give a class of similarity of matrices. Structurally the simplest n x n matrix has the form

A = [ c 1 0 ⋯ 0 0 c 2 ⋯ 0 ⋮ ⋮ ⋱ ⋮ 0 0 ⋯ c n ]

known as diagonal matrix. This means that the transformation L: ℝ3 ⟶ ℝ3 can be written as

{ y 1 = c 1 x 1 y 2 = c 2 x 2 y 3 = c 3 x 3

which is extremely simple, as every variable yi depends only on a variable xi rather than all three variables. We say that in this case the transformation as diagonal form.

We want to examine how the representation of L changes when the basis of ℝn is changed.
Let e1, e2, ..., en the canonical basis of ℝn. If we take another basis of ℝn, ẽ1, ẽ2, ..., ẽn, the matrix

S = (ẽ1|ẽ2|...|ẽn)

is made by the columns vectors ẽ1, ẽ2, ..., ẽn and represent in the canonical basis the transformation between the two bases. Indeed as it can be readily seen

S ei = ẽ   i = 1, 2, ..., n

Since the vectors ẽ are independent, det S ≠ 0, hence S is invertible, e we can also write

ei = S−1 ẽ   i = 1, 2, ..., n

Thus if a vector of ℝn has with respect to the canonic basis, the coordinated x = (x1, x2, ..., xn), with respect to the other basis has coordinates x̃ = (x1, x2, ..., xn) given by

x̃ = S−1x

Thenf if y = Ax in the canonical basis, it results

ỹ = S−1y = S−1Ax = S−1ASx̃

Thus the transformation L in the new basis is represented by the matrix

Ã = S−1AS

where A,Ã are said equivalent, or similar.

Definition. If a square matrix A is similar to a diagonal matrix Λ we say that A is diagonazible, i.e. if

A = S−1ΛS

for an invertible matrix S. Where A,ΛS are matrices with elements if a field K  □

In the case of a diagonazible matrix, there exists a basis in Kn in which the transformation with respect to the canonical basis represented by the matrix A, can be represented with a diagonal matrix Λ; hence the transformation is more simple being represented by a diagonal matrix. We want to understand under which conditions given a square matrix A, there here exist an invertible matrix S such that S−1ΛS is diagonal? This problem is connected with another problem which is the computation of eigenvalues and eigenvectors.

Notice that rela numbers are a subfield of complex number, and a matrix with real coefficeints can be seen as a matrix with complex coefficents hence related to a transformation from ℂn to itself. However it can happen that a matrix is diagonazible over ℂ but not over ℝ.
