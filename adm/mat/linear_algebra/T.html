Transpose of a Matrix

Definition 2.3.1. (Transpose of a matrix). Let A = (aij) ∈ Mm,n (𝕂). The Transpose of a matrix A is the matrix AT ∈ Mm,n (𝕂) whose (i, j)-entry is aji. That is, AT is obtained from A by reflecting its entries across its main diagonal.

For example

A = ( 2 5 4 3 0 6 3 2 ) A T = ( 2 0 5 6 4 3 3 2 )

It results then (AT)T = A.

Definition 2.3.2 (Matrix of cofactors). If we replace every element in a matrix by its corresponding cofactor then we get the matrix of cofactors usually denoted by C

C = [ A 11 A 12 ⋯ A 1 n A 21 A 22 ⋯ A 2 n ⋮ ⋮ ⋱ ⋮ A n 1 A n 2 ⋯ A n n ]

Definition 2.3.3. The adjoint ("Hermitian transpose" or "Hermitian conjugate") of a complex matrix is its conjugate transpose ĀT and is denoted A† (or AH or A*). A matrix is called self-adjoint or Hermitian if A†, that is, if āij = aji.

To calculate the adjoint of a matrix we transpose the matrix, and take the complex conjugate of the entries. For example, if

A = ( 1 3 − i 2 i 2 + 3 − i 0 ) then A † = ( 1 2 − 3 i 3 + 1 i − 2 i 0 )

We see that A is not Hermitian. For a vector

A ≡ ( A 1 A 2 ) A † ≡ ( A 1 ∗ A 2 ∗ )

An example of Hermitian matrix is the following:

B = ( 2 i − i 3 ) then B † = ( 2 i − i 3 )

Note that if A is a real-valued matrix, then its adjoint is just its transposition because the complex conjugate operation does not change real entries. It is clear that if A is a matrix of order n x m then A† is a matrix of order m x n.

Definition. We say that a matrix A ∈ Mn is symmetric if it is equal to its transpose, AT = A. It is said antisymmetric, if A = − AT.

The following is Hermitian (but not symmetric),

B = ( 2 1 + 3 i 1 − 3 i − 4 )

We see that the adjoint of a real square matrix is equal to the transpose of the matrix, A* = AT.
