Image and Kernel of a linear transformation

If L is a linear transformation from ℝn into ℝm, the image of T is the set of vectors in ℝm which are the transformed of some vector of ℝn and is indicated by

Im(T)

If y ∈ Im(T) it must be y = L(x) for some x ∈ ℝn. The basis properties of the image are expressed by the following theorem.

Thereom 5.2.1.

  1. The set Im(T) is a vector subspace of ℝm;

  2. Fixed two bases of ℝn and ℝm, let A an m x n matrix, which represents L then

    dim Im(L) = rank (A)

Proof.

  1. From the linearity of L: if y1 = L(x1) and y2 = L(x2) then we have

    αy1 + βy2 = αL(x1) + βy2 = L(x2) = L(αy1 + βy2)

    α, β ∈ ℝ. Since α y1 + βy2 ∈ Im(T) being the transformed of a vector αx1 + βx2 by the subspace criterion then Im(T) is a subspace of ℝm.

  2. We known from the matrix representation theorem that the columns of A represent the scalar components of T(ui), with repsect to the basis v1, v2, ..., vm of Vm. The vector space Im(T) is thus generated by the vectors T(u1), T(u2), ... , T(un). If r among the columns of A are independent then the correspondent vectors T(ui) are independent and viceversa, that is r = dim Im (T). □
