Inverse trigonometric functions

Essendo le funzioni trigonometriche priodiche, per trattare le loro funzioni inverse bisognerà restringersi a intervalli in cui queste siano strettamente monotone, quindi invertibili. La funzione y = sin x è strttamente crescente nell'intervallo [-π/2,π/2]. La funzione inversa funzione arcoseno e indicata y = arcsin x; è definita in [−1,1] e ivi strettamente crescente.

Under suitable restrictions, each of the six trigonometric functions is one-to-one and so has an invere function. For instance the sine function is one-to-one on the interval [−π/2, π/2]. On this interval, you can define the inverse of the restricted sine function as

y = arcsin x; (read as “y is the angle whose sine is x”) also written as sin−1 (x)   if and only if   sin y = x
where   domain; −1 ≤ x ≤ 1   range: −π/2 ≤ y ≤ π/2

Under suitable restriction, each of the six trigonometric function is one-to-one and so has an inverse function. Analogamente, un intervallo di monotonia del coseno è [0, π], nel quale dunque esso è invertibile.

Per cos(x) l'intervallo di monotonia è [0,π], nel quale la funzione è possibile definire la funzione inversa arcoseno

Per la tangente l'intervallo di monotonia è (-π/2,π/2) ed ivi invertibile. La funzione inversa si chiama arcotangente
