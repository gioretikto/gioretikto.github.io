The Extreme Value theorem

We now show that if a function defined on a closed and bounded set is continuous, then it is necessarily bounded. So if f is continuous on a closed bounded interval I then the image of I under f is also closed and bounded interval. For this is crucial that the interval I is closed: for example, the function f(x) = 1/x is continuous on the interval (0,1), but takes arbitrarly large values as x ⟶ 0. The function g(x) = x is bounded on (0,1), but it does not have a maximum or minimum on the interval (0,1) − it approaches 0 and 1 arbitrarly closely, but never attaints either, as its value lie strictly between 0 and 1.
A continuous function defined on a bounded and closed interval is not just bounded but it attains a maximum and a minimum as well.
The following theorem is attributed to the great German mathematician Karl Weierstrass (1815–1897).

Theorem 4.3.2 (The Extreme Value Theorem). Let f: [a, b] → ℝ, a continuous function. Then f is bounded and attains it bounds, so that the range of f is again a closed bounded interval, and f has absolute maximumum and minimum points, i.e. there are points c,d ∈ [a, b] such that

f(d) ≤ f(x) ≤ f(c)   ∀x ∈ [a,b]

Proof. Observe first that whatever the function f(x) in I = [a, b], there exists a sequence {xn} in I such that

lim n → ∞ f ( x n ) = sup  f ( I )

Indeed it suffices to consider ∀n ∈ ℕ, xn ∈ I such that

  • f(xn) > n  if  sup f(I) = +∞

  • or f(xn) < M − 1/n  if  sup f(I) = M < +∞.

Now, we prove the theorem by contradiction. If the set f(I]) were unbounded above, then there would exist a sequence {xn} in [a, b] such that f(xn) ⟶ ∞. Since {xn} ∈ [a, b], it is a bounded sequence, by Bolzano–Weierstrass theorem we know there exists a convergent subsequence {xnk} ⟶ c. Since {xnk} ∈ [a, b] and since f is continuous on [a,b], namely at c, it must be the case that f(xnk) ⟶ f(c). But {f(xnk)} is a subsequence of {f(xn)} by Theorem 2.3.2 we know that they must converge to the same limit

f ( c ) = lim k → ∞ f ( x n k ) = lim n → ∞ f ( x n ) = sup  f ( [ a , b ] )

If f(xn) ⟶ ∞, it would be f(xnk) ⟶ ∞ as well. This is a contradiction, since f(xnk) ⟶ f(x) ≠ ∞. So we cannot have sup f([a,b]) = +∞ but M = sup {f(x) | x ∈ a,b]} and we have found a maximum point c for f in I. Similar arguments show that f attains a minimum value.  □
