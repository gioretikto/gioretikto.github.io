Properties of ℝ

The next theorem could be extracted from the construction of ℝ from ℚ with very little extra effort (see Theorem 3.1.4). However, we prefer to derive it from Theorem 1.8.1 since this provides a good illustration of what one can do with the least-upper-bound property.

Theorem 1.9.1

  1. (Archimedean property) If x ∈ ℝ, y ∈ ℝ, and x > 0, then there is a positive integer n such that

    nx > y.

  2. If x ∈ ℝ, y ∈ ℝ, and x < y, then there exists a p ∈ ℚ such that x < p < y.

Part (a) is usually referred to as the archimedean property of ℝ. Part (b) may be stated by saying that ℚ is dense in ℝ: Between any two real numbers there is a rational one.

Proof (a) Let A be the set of all nx, where n runs through the positive integers. If (a) were false, then y would be an upper bound of A. But then A has a least upper bound in ℝ. Put α = sup A. Since x > 0, α − x < α, and α − x is not an upper bound of A. Hence α − x < mx for some positive integer m. But then α < (m + l )x ∈ A, which is impossible, since α is an upper bound of A.
(b) Since x < y, we have y − x > 0, and (a) furnishes a positive integer n such that

n (y − x) > 1

Apply (a) again, to obtain positive integers m1 and m2 such that m1 > nx, m2 > −nx. Then

−m2 < nx < m1

Hence there is an integer m (withm −m2 ≤ m ≤ m1) such that

m − 1 ≤ nx < m

If we combine these inequalities, we obtain

nx < m ≤ 1 + nx < ny

Since n > 0, it follows that

x < m/n < y

This proves b) with, p = m/n. □

We shall now the existence of nth roots of positive reals. This proof will show how the difficulty of the irrationality of &sqrt;2 can be handled in ℝ.

Theorem 1.9.2 For every real x > 0 and every integer n > 0 there is one and only one real y such that yn = x.

Proof. That there is at most one such y is clear, since 0 < y1 < y2 implies y1n < y2n.
Let E be the set of all positive real numbers t such that tn < x. If t = x /(1 + x) then 0 < t < 1. Hence tn < t < x. Thus t ∈ E, and E is not empty.
Hence Theorem 1.8.1 implies the existence of

y = sup E

To prove that yn = x we will show that each of the inequalities yn < x and yn > x leads to a contradiction.

Thus (y + h)n < x, and y + h ∈ E. Since y + h > y, this contradicts the fact that y is an upper bound of E.
Assume yn > x. Put

k = (yn −x)/nyn−1

Then 0 < k < y. If t ≥ y − k, we conclude that

yn − tn ≤ yn − (y −k)n − (y − k)n < knyn−1 = yn − x

Thus tn > x, and t ≠ E. It follows that y − k is an upper bound of E. But y − k < y, which contradicts the fact that y is the least upper bound of E. Hence yn = x and the proof is complete.  □

Corollary 1.9.3 If a and b are positive real numbers and n is a positive integer, then

(ab)1/n = a1/nb1/n

Proof. Put α = a1/n, β = b1/n. Then

ab = αn βn = (αβ)n

since multiplication is commutative (see axiom M3). The uniqueness assertion of the previous theorem shows therefore that

(ab)1/n = αβ = a1/nb1/n. □

1.9.4 Decimals We conclude this section by pointing out the relation between real numbers and decimals.

Let x > 0 be real. Let n0 be the largest integer such that n0 ≤ x. (Note that the existence of n0 depends on the archimedean property of ℝ) Having chosen n0, n1, ..., nk−1 let nk be the largest integer such that

n0 + n1/10 + ... + nk/10k ≤ x

Let E be the set of these numbers

n0 + n1/10 + ... + nk/10k (k = 0,1,2,...) 1.9.5

Then x = sup E. The decimal expansion of x is

n0.n1n2n3... 1.9.6

Conversely, for any infinite decimal 1.9.6 the set E of numbers 1.9.5 is bounded above, and 1.9.6 is the decimal expansion of sup E.
