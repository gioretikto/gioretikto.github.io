The real number system

We have constructed integers from the natural numbers and rational numbers from the integers. Both constructions were primarily algebraic. This was possible since the rational numbers are the algebraic completion of the set of natural numbers. The same approach, however cannot be used in a construction the real numbres, since there are real numbers of no simple algebraic relation to rational numbers.

We now state the existence theorem which is the core of this chapter.

Theorem 1.8.1 There eixsts an ordered field ℝ which has the least-upper-bound property.
Moreover, ℝ contains ℚ as a subfield.

The second statement means that ℚ ⊂ ℝ and that the operations of addition and multiplication in ℝ, when applied to members of ℚ, coincide with the usual operations on rational numbers; also, the positive rational numbers are positive elements of ℝ.
The members of ℝ are called real numbers. The proof of this theorem actually construct ℝ from ℚ.

The real number system ℝ exists and satisfies the properties of a complete ordered field; the properties are not assumed as axioms, but are proved. The are several techniques for constructing the real number system from the rational number system. We use the method of Dedekind's cut (Julius Wihelm Richard Dedekind (1831-1916)), because it uses a minimum of new ideas and is fairly brief. Dedekind answered the long-standing question "What are the real numbers?", by defining them as suitable sets of rational numbers, called Dedekind cuts of ℚ, which are nonempty proper subset α of ℚ that do not contain a greatest element.

Definition 1.8.2 The members of ℝ will be certain subsets of ℚ, α ⊂ ℚ, called Dedekind cuts with the following properties:

  1. α ≠ ∅, α ≠ ℚ;

  2. If p ∈ α, q ∈ ℚ and q < p, then q ∈ α;

  3. There is a rational number r such that p < r for all p ∈ α.  □

The letters, p,q,r,... will always denote rational numbers, and α, β, γ,... will denote cuts.

Remarks: by property 2), whenever a rational is in one of these cuts, all other rationals that are smaller (i.e. lie to the left) are also included. 3) implies that α has no largest number; therefore, the cut for a particular real number will include all the rationals up to that number but not including it.

You should think of a cut α as the set of all rational numbers to the left of some point in the real line (i.e., it is an open half-line of rational numbers - see Fig. above).
For example, the set {x ∈ ℚ: x2 < 2} u { x ∈ ℚ: x < 0} is a cut. (Take care to note that √2 does not exist as a rational number so we are using a circuitous method to specify this set.) Since we have not constructed the real line yet, we cannot define this cut in that simple way; we have to make the construction more indirect. But if you consider the four properties of a cut, they describe a set that looks like a "rational left half-line".
Following directly from the definition of a cut are these two properties:

  1. If α is a cut and s ∉ α then any rational t > s is also not in α;

  2. If r ∈ α and s ∉ α, then it must be that r < s.

Now, ℝ can be defined as the set of all Dedekind cuts, intuitively identifying α with a real number sup A. Hence, ℝ is a certain subset of the power set of ℚ. Two real numbers α and β are equal if and only if both cuts are the same subset of ℚ.

Theorem 1.8.4. The ordered set ℝ has the least-upper-bound property.

Proof. Let A be a nonempty subset of ℝ, and assume β ∈ ℝ is an upper bound of A. Define γ to be the set that contains every rational number in every cut in A (i.e. γ is the union of all the cuts in A). We shall prove that γ ∈ ℝ and that γ = sup A. Then the theorem will be proved.
Since A is not empty, there exists a cut α0 in A, which is not empty. Since α0 ⊂ γ, γ is not empty. Next, γ ⊂ β (since α ⊂ β for every α ∈ A), and therefore γ ≠ ℚ. Thus γ satisfies property (I). If q is any rational number in γ then q lies in one of the cuts that form γ, and so every rational number smaller than q is in that cut and hance in γ; this proves (II). If γ contains a largest rational number then so does one of the cuts that form γ and that's a contradiction. Thus γ ∈ ℝ. it is clear that α ≤ γ for every α ∈ A. Suppose δ < γ. Then there is an s ∈ γ and that s ∉ δ. Since s ∈ γ, s ∈ α for some α ∈ A. Hence δ < α, and δ is not an upper bound of A. This gives the desired result: γ = sup A.  □

Now we introduce operations + and ⋅ that will turn the set of all cuts into a field.

Definition 1.8.5. If α and β are cuts, then we define

We saw in the previous proposition that the replacement of the rational numbers r by the corresponding "rational cuts" r* ∈ ℝ preserves sums, products, and order. This fact may be expressed by saying that the ordered field ℚ is isomorphic to the ordered field ℚ* whose elements are the rational cuts. Of course, r* is by no means the same as r, but the properties we are concerned with (arithmetic and order) are the same in the two fields. It is this identification of ℚ with ℚ* which allows us to regard ℚ as a subfield of ℝ.
The second part of the existence theorem (1.8.1) is to be understood in terms of this identification. Note that the same phenomenon occurs when the real numbers are regarded as a subfield of the complex field, and it also occurs at a much more elementary level, when the integers are identified with a certain subset of ℚ. It is a fact, which we will not prove here, that any two ordered fields with the least-upper-bound property are isomorphic. The first part of Theorem 1.8.1 therefore characterizes the real field ℝ completely.
