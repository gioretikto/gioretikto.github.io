Derivata seconda

La derivata seconda di una funzioen ha vari significati geometrici, che ci permetteranno di studiare meglio le proprietà del grafico di una funzione.

Significato geometrico della derivata seconda

Se la derivata prima ha come signigicato gemetrico quello di pendenza del grafico, la derivata seconda rappresenta la velocità di variazione di tale pendenza e pertanto costituisce una misura del grado di scostamento del grafico dall'andamento rettilineo.
Per precisare meglio il concetto, consideriamo la situazione illustrata in Fig.

La funzione f in figura, soddisfa le condizioni f(0) = f''(0) = 0, f''(0) = 0, f''(0) ≥ 0.
Consideriamo ora la famiglia di semicirconferenze con centro sull'asse y tangente al grafico di f nell'origine.
L'equazione della famiglia è

y = R − R 2 − x 2

come facilmente si verfica, e, per ogni R, si ha y'(0) = y(0) = 0.
Tra queste semicirconferenze selezioniamo quella che non soltanto ha la stessa tangente in x = 0 ma anche la stessa velocità di variazione della pendenza in x = 0.
In altri termini, scegliamo R in modo che

y'(0) = f''(0) = 0  (5.14.1)

Si ha,

y ′ ( x ) = x R 2 − x 2 , y ″ ( x ) = R 2 ( R 2 − x 2 ) 3 / 2

e quindi y''(0) = 1/R. Se dunque vogliamo che sia soddisfatta la (5.14.1) occorre scegliere R in modo che

f''(0) = 1/R (5.14.2)

La (5.14.2) epsrime il significato geometrico della derivata seconda: nella situazione descritta, f''(0) rappresenta il reciproco del raggio della circonferenza che "meglio approssima" f in x = 0; 1/R prende il nome di curvatura (del grafico) di f in x = 0 e R è il raggio di curvatura. Ripetendo un'analoga costruzione in un punto generico x del grafico di f (senza supporre a priori f(x) = f'(x) = 0) si trova che il cerchio che meglio approssima il grafico nel punto di ascissa x ha raggio R(x) dato da:

1 R ( x ) = | f ″ ( x ) | ( 1 + f ′ ( x ) 2 ) 3 / 2

Esempio 5.14.3. Per una parabola y = ax2, si ha y'(x) = 2ax and y''(x) = 2a e quindi

1 R ( x ) = 2 | a | ( 1 + 4 a 2 x 2 ) 3 / 2

Si noti che la curvatuva è massima per x = 0, cioè nel vertice per x ⟶ ∞ è 0, ed infatti entrambe le code della parabola diventano piatte per x tendente a infinito.  ■
