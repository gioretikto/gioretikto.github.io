Continuità

Si rifletta sull'esempio xx e seguente del paragrafo precedente. In entrambi i casi il limite al finito di una certa funzione esiste ed è finito. Nel primo caso, tale limite coincide col valore della funzione nel punto considerato, nel secondo caso no. Il buon senso suggerisce che la prima situazione è quella "normale", mentre la seconda è un po' "patologica". Questa idea è chiarita dalla prossima definizione, che introduce un nuovo concetto fondamentale.

Definizione 4.2.1 (Continuità) Se f: I ⟶ ℝ (I intervallo) e x0 ∈ I, si dice che f è continua in x0 se esiste

lim x → x 0 f ( x ) = f ( x 0 )

Si dice che f è continua in I se è continua in ciascun punto di I. Una funzione non continua in un punto x0 si dice discontinua in x0. □

Gli esempi precedenti si possono leggere dicendo che la funzioen sin x è continua in x = 0, mentre la funzione f(x) dell'esempio successivo è discontinua in x = 0. Si consideri anche il prossimo.

Esempio 4.2.1

lim x → 0 + x | x | = ± 1

Questo esempio è interessante per due motivi. Anzitutto, in questo caso non sarebbe neppure possibile calcolare f(0), quindi è chiaro a priori che il limite non si potrà calcolare semplicemente valutando la fuzione nel punto 0. Inoltre, in questo caso i limiti destro e sinistro esistono finiti e sono diversi tra loro. Questo fatto merita una definizione:

Definizione 4.2.2 Si dice che x0 è un punto di discontinuità a salto per f(x) quando i limiti destro e sinistro in x0 esistono finiti, ma diversi tra loro. Il salto è costituito dalla differenza dei limiti e precisamente:

salto in  x 0 = lim x → x 0 + f ( x ) − lim x → x 0 − f ( x )

Se uno dei due limiti destro o sinistro per x ⟶ x0 coincide con f(x0), si dice che f è continua da destra o da sinistra, rispettivamente. □

Nell'esempio precedente, la funzione x/|x| ha un punto di discontinuità a salto in x = 0, con salto 2.
Le funzioni che presentano discontinuità a salto si prestano bene a modellizzare fenomeni che registrano bruschi cambiamenti.
Le funzioni continue, d'atro canto, devono la loro importanza al fatto che limx ⟶ x0 f(x) = f(x0) può essere interpretato dicendo che "se x è vicino a x0" allora "f(x) è vicino a f(x0)", ossia

piccole variazioni di x ⟼ f ⟶ piccole variazione di f(x)

Per coglierne la rilevanza, pensiamo al seguente problema: noto x0 e nota f, mediante una calcolatrice tascabile, calcolare f(x0).
Se x0 è un numero irrazionale, la calcolatrice approssima x0 con x0 + Δx (Δx è l'errore commesso nell'approssimazione) e calcola di conseguenza f(x0 + Δx), approssimando ulteriormente il risultato finale.
Se f è continua in x0 e |Δx| « 1 (cioè l'errore assoluto |Δx| è molto piccolo) è ragionevole considerare il risultato ottenuto come una buona approssimazione del risultato finale. Ma se f è discontinua, in corrispondenza di uno spostamento, anche piccolissimo, a destra o a sinistra di x0, si trova un valore f(x0 + Δx) molto diverso da f(x0).

Esempio 4.2.2 Stabilire se la seguente funzione è continua in x = 2.

f ( x ) = { x 2 + 1 if  x ≥ 2 7 − x if  x < 2

Dalla definizione di continuità deve valere:

lim x → 2 f ( 2 ) = f ( 2 )

Controlliamo per prima cosa che la funzione sia definita nel punto x = 2 e calcolimoche il valore. Abbiamo che f(2) = 22 + 1 = 5. Calcolando i limiti destro e sinistro della funzione:

lim x → 2 + f ( x ) = lim x → 2 + x 2 + 1 = 2 2 + 1 = 5 lim x → 2 − f ( x ) = lim x → 2 − 7 − x = 7 − 2 = 5

Quindi la funzione è continua nel punto x = 2.   ■
