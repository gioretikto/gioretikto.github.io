Exercises on series

Test the following series

  1. If a chessboard were to have wheat placed upon each square such that one grain were placed on the first square, two on the second, four on the third, and so on (doubling the number of grains on each subsequent square), how many grains of wheat would be on the chessboard at the finish?

  2. ∑ n = 0 ∞ n + 5 n 3 − n 2 + 5
  3. ∑ n = 1 ∞ ln ⁡ n n
  4. ∑ n = 1 ∞ [ ( − 1 5 ) n + 5 n ]
  5. ∑ n = 1 ∞ n + e − n 3 n 2 + 5
  6. ∑ n = 1 ∞ ln ⁡ n − n 3 n 4 − 1
  7. ∑ n = 1 ∞ n 100 e − n
  8. ∑ n = 1 ∞ n 1 / n n !
  9. ∑ n = 1 ∞ cos ⁡ n n 3
  10. ∑ n = 1 ∞ n ! n n
  11. ∑ n = 1 ∞ ( − 1 ) n n

    1, 2, 4, 8, 16, 32, 64, 128, 256 ...

    form a geometric sequence, which grows at an increasing rate. Its sum can be obtained as

  9. We apply the ratio test

    lim n → + ∞ a n + 1 a n = lim n → + ∞ ( n + 1 ) ! ( n + 1 ) n + 1 n ! n n = lim n → + ∞ ( n + 1 ) ! ( n + 1 ) n + 1 ⋅ n n n !
