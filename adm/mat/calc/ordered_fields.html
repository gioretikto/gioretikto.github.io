Ordered fields

Definition 1.7.1. An ordered field is a field F which is also an ordered set, such that

  1. x + y < x + z if x, y, z ∈ F and y < z;

  2. xy > 0 if x ∈ F, y ∈ F, x > 0, and y > 0.

If x > 0, we call x positive; if x < 0, x is negative. □

The two postulates above serve to bind the order structure to the field structure of F.
ℚ is an ordered field, but there are many other ordered fields as well, as ℝ. All the familiar rules for working with inequalities apply in every ordered field: Multiplication by positive [negative] quantities preserves [reverses] in­ equalities, no square is negative, etc. The following proposition lists some of these.

Proposition 1.7.2. The following statements are true in every ordered field.

  1. If x > 0 then −x < 0, and vice versa;

  2. If x > 0 and y < z then xy < xz;

  3. If x < 0 and y < z then xy > xz;

  4. If x ≠ 0 then x2 > O. In particular, 1 > O;

  5. If 0 < x < y then 0 < 1/y < 1/x.

Proof

  1. If x > 0 then 0 = −x + x > −x + 0, so that −x < 0. If x < 0 then 0 = −x + x < −x + 0, so that −x > 0. This proves (a).

  2. Since z > y, we have z − y > y − y = 0, hence x(z − y) > 0, and therefore

    xz = x(z − y) + xy > 0 + xy = xy.

  3. By (a), (b), and Proposition 1.16(c),

    −[x(z − y)] (−x)(z − y) > 0,

    so that x(z − y) < 0, hence xz < xy.
    (d) If x > 0, part (ii) of Definition 1.7.1 gives x2 > O. If x < 0, then −x > 0, hence (−x)2 > O. But x2 = (−x)2, by Proposition 1.16(d).
    Since 1 = 12, 1 > O.
    (e) If y > 0 and v ≤ 0, then yv ≤ O. But y ⋅ (1/y) = 1 > O. Hence 1/y > O. Likewise, 1/x > O. If we multiply both sides of the inequality x < y by the positive quantity (l/x)(1/y), we obtain 1/y < l/x. □
