Logical Arguments

Proofs play a major role in mathematics, and deductive reasoning is the foundation on which proofs rest. In mathematics, as in law, a logical argument is a claim that from certain premises (statements that are assumed to be true) one can infer a certain conclusion (statement) is true. Logic is concerned with the connections between statements and with what deductions can be made, assuming that the premises are true. Notice that statements are said to be true or false, while arguments are said to be valid or invalid. We make the following deﬁnitions regarding arguments.

Definition (Valid and Invalid Arguments) An argument P1, ..., Pn .·. C with premises P1, ..., Pn and conclusion C is valid if and only if P1 ∧ P2 ∧ ... ∧ Pn ⇒ C is a tautology.

If an argument is not valid, it is called invalid.

The "argument" is an implication; For the implication to always be true, it is sufficient (and necessary) that the conclusion is true if the premise is true. The premise can be false as well and then the conclusion doesn't matter (See the definition implication). Thus, an argument (by definition of implication) is valid if and only if it is not possible for its premises to all be true while its conclusion is false. Stated another way, an argument is valid if and only if it is not possible for the conclusion to be false unless at least one of the premises is false. Validity concerns the relationship between the premises and the conclusion, and not the truth values of the premises and conclusions.

When an argument contains three or more different simple statements, it becomes tedious to determine the validity of the argument by making a truth table. A more convenient way to prove an argument is valid is to deduce its conclusion from its premises by a sequence of elementary arguments which are known to be valid. Hence, a formal proof of a given argument is a sequence of statements such that each statement is either: (1) a premise of the argument, or (2) a statement which follows from preceding statements by an elementary valid argument, or (3) a statement which is logically equivalent to a preceding statement in the sequence, or (4) the last statement in the sequence—the conclusion of the argument. The proper way to write a formal proof is to list the premises and the sequence of statements deduced from them along with a justification for each statement. The justification of each deduced statement specifies the preceding statement or statements from which it was deduced and the rule of inference which was used. Thus far, we have proved by the use of truth tables that four elementary arguments are valid—namely, the rule of disjunction, the rule of conjunction, the rule of conjunctive simplification, and the rule of disjunctive syllogism.

Example. (Determining the Validity of an Argument) Write the following argument in symbolic form. Determine if the argument is valid or invalid and state the name of the argument form.

I took a nap, or I watched television.
I did not watch television.
————————————————
Therefore, I took a nap.

Solution. Let N denote the statement “I took a nap.” And let T denote the statement “I watched television.” Then the symbolic form of the argument is

This argument has the form of the disjunctive syllogism and it is a valid argument. ■
