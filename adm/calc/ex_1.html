Exercises on R and C fields

  1. Prove that |𝑥−𝑎| < 𝜖 is is equivalent to a − 𝜖 < x < 𝜖 + a;

  2. If r is rational (r ≠ 0) and x is irrational, prove that r + x and rx are irrational.

  3. Prove that there is no rational number whose square is 12.

  4. Let E be a nonempty subset of an ordered set; suppose α is a lower bound of E and β is an upper bound of E. Prove that α ≤ β.

  5. Let A be a nonempty set of real numbers which is bounded below. Let −A be the set of all numbers −x ∈ A. Prove that

    inf A = −sup (−A)

  6. Fix b > 1.

    1. If m,n,p,q are integers, n > 0, q > 0, and r = m/n = p/q, prove that

      (bm)1/n = (bp)1/q

      Hence it makes sense to define br = (bm)1/n.

    2. Prove that br + s = brbs if r and s are rational.

    3. If x is real, define B(x) to be the set of all numbers bt, where t is rational t ≤x. Prove that

      br = sup B(r)

      where r is rational. Hence it makes sene to define

      bx = sup B(x)

      for every real x.

    4. Prove that bx + y = bxby

Solutions

  1. From the definition of abs. value we have

    | x − a | = { x − a , if  x − a ≥ 0 − ( x − a ) , if  x − a < 0 .

    If 𝑥−𝑎 ≥ 0, we have 𝑥−𝑎 < 𝜖. If 𝑥−𝑎<0, then −(𝑥−𝑎)<𝜖, which after multiplication by −1 becomes 𝑥−𝑎 > −𝜖. Thus, in both cases we have

    −𝜖 < 𝑥−𝑎 < 𝜖
    a − 𝜖 < x < 𝜖 + a   (having added a)

  2. Given that r is rational, we can write r = a/b for some integers a and b. We are also given that x is irrational. From here, we proceed with a proof by contradiction. We first assume that r + x is rational, and then we use this fact in some way to show that x is rational, contradicting one of the facts we were given. This will prove that r + x is instead irrational.
    So if r + x is rational, we can write r + x = c/d for some relatively prime integers c and d. But then

    x = c/d − r = c/d − a/b = (bc − ad)/bd

    and thus xis rational, which is a contradiction. Therefore, r + x is irrational.
    Next, we prove that rx is irrational using a similar contradiction proof. Assume that rx is rational. Then wer can write rx = c/d for some integers c and d. But then

    x = c/rd = c/ad/b = bc/ad

    and this x is rational, which is a contradiction. Therefore, rx is irrational. ■

  3. Let x be a rational number such that x2 = 12. Then we can write x = a/b, and furthermore, assume a and b to be relatively prime (which means there is no prime number dividing both a and b), so that the fraction a/b is written in lowest terms. With a little algebraic manipulation

    12 = x2 = a2/b2  ⇒  12b2 = a2

    Now, the prime factorization of 12 is 22 · 3. We’ll concentrate on 3 and see how it divides both sides of the equation 12b2 = a2. Notice that 3 divides the left side since it has a factor of 12. Therefore, 3 must divide the right side of the equation, a2. From here, the crucial step is realizing that if 3 divides a2, then it must also divide a. This is because if we factor a2 into its prime factors, saying that 3 divides a2 is equivalent to saying that 3 is one of those prime factors, but a square of an integer must have an even number of each factor (it can’t have just 1), so that means 32 must divide a2, and 3 must divide a.
    Since we have shown that 32 divides the right side, 32 must divide the left side, but there is only one factor of 3 in 12, so that means 3 divides b2. Using the same logic as before, this means that 3 must divide b. Therefore, we have shown that 3 divides both a and b, but this contradicts the fact that we already chose a and b to be relatively prime (so that a/b would be expressed in lowest terms). Since our initial assumption leads to a contradiction, we have instead that there is no rational number whose square is 12. ■

  4. Since E is nonempty, there exists e ∈ E. Then by defiition of lower and upper bounds we have α ≤ e ≤ β, and hence by property ii) in the definition of an ordering, we have α < β unless α = e= β. □

    5. We need to prove that −sup (−A) is the greatest lower bound of A. Fro brevity let α = −sup (−A). We need to show that α ≤ a for all a ∈ A and α ≥ β if β is any lower bound of A.
    Now let β any lower bound of A. This means β ≤ a for all a ∈ A. Hence −a ≤ β for all a ∈ A, which says y ≤ β for y ∈ −A. This means −β is an upper bound of −A. Hence −β ≥ sup (−A) by definition of sup, i.e., β ≤ −sup(−A), and so −sup(−A) is the greatest lowe bound of A.
