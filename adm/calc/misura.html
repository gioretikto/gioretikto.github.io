La misura di Lebesgue

The length of an interval is used frequently in differentiation and integration. For any bounded interval I (open, closed, half-open) with endpoints a and b (a < b), the length of I is defined bt l(I) = b - a. The length of any unbounded interval is defined to be infinite, that is, l(I) = ∞ if I is of the form (a, &infinf;), (-∞b), or (-&infinf;,∞). There are situations in which it would be useful to measure the size of sets other than intervals. An extension to unions of intervals is obvious, but is much less obvious for arbitrary sets. For instance, what is the size of the set of irrational number in [0,2]? Is it possible to extend this concept of length (or size) of an interval to arbitrary sets? Lebesgue measure is one of several approaches to solving this problem. The purpose of this chapter is to define and study the basic properties of Lebesgue measure. Given a set E of real numbers, μ(E) will represent its Lebesgue measure. Before defining this concept, let's consider the properties that it should have.

  1. If I is an interval, then μ(I) should be l(I);

  2. If A is a subset of B, then μ(I) should be less than or equal to μ(I);

  3. Given A ⊆ R and x0 ∈ R, define A + x0 = {x + x0 : x ∈ A};

  4. If A and B are disjoint sets, then μ(A ⋃ B) should be μ(A) + μ(B). In fact, if {Ai} is a sequence of disjoint sets, then μ(⋃∞i=1 Ai) should be Σ∞i=1 (Ai).

Unfortunately, it is not possible to define a measure that satisfies all of these properties. The difficulty lies in property (4). Since this property is essential to guarantee the linearity of the Lebesgue integral, it is necessary to restrict the collection os sets and consider only those for which all of the properties are valid. In other words, some sets will not have a Lebesgue measure.

Definizione. Un n-intervallo iper-intervallo in ℝn è il prodotto cartesiano di n intervalli unidimensionali, detti lati o componenti dell'intervallo n-dimensionale, contenuti in ℝ:

I ≡ I1 x I2 x ... x Ik,  dove   Ik ≡ [ak, bk], ak ≤ bk, k = 1, ..., n.

Chiamiamo pluriintervallo è l'unione di una famiglia finita o numerabile di n-intervalli

P = ⋃k Ik,  Ik ⊂ ℝn

dove s'intende che l'indice k descriva un insieme finito o numerabile di indici.
