Risoluzione approssimata di equazioni: il metodo di Newton

Supponiamo di voler risolvere l'equazione

f(x) = 0

che equivale a cercare le intersezioni del grafico di f con l'asse x, e supponiamo di aver accertato (mediante confronto grafico, ad esempio) l'esistenza di un'unica soluzione x = α, all'interno di un intervallo [a,b]. Si vuole dare una valutazione approssimata di α. Il metodo di Newton, che ora illustriamo, serve proprio a questo scopo. L'idea è quella di costruire una successione convergente ad α; questa successione, come si vedrà, è definita in modo ricorsiva, ossia:

  • si assegna esplicitamente il primo termine x0;

  • si assegna esplicitamente il secondo la legge con cui calcolare xn +1 a partire da xn, qualunque sia n;

In questo modo, il termine xn può essere costruito a partire da x0 con n iterazioni del medesimo algoritmo. Il metodo, perciò, si presta molto bene al calcolo automatico.
Illustriamo il metodo con riferimento alla fig.1. Partiamo da x0 = a e "linearizziamo" l'equazione f(x) sostituendo a f la retta tangente al suo grafico nel punto (a,f(a)). Tale retta ha equazione

y(x) = f(x0) + f′(x)(x − x0)

Invece di risolvere f(x) = 0 risolviamo

f(x0) + f′(x)(x − x0) = 0

ottenendo

x1 = x0 − f(x0)/f′(x)  [prima approssimazione di α]

Procediamo ora con x1 al posto di x0 = α, sostituendo f con la retta tangente in (x1,f(x1)). Anziché f(x) = 0 si risolve l'equazione

f(x1) + f′(x1) (x − x1) = 0

ottenendo

x2 = x1 − f(x0)/f′(x)  [seconda approssimazione di α]

Proseguendo con x2 come pr x1 e continuando si perviene alla legge di ricorrenza

{ x 0 = a x n + 1 = x n − f ( x n ) f ′ ( x n )  (5.21.2)

Nella situazione descritta in fig.1, si vede che xn → α.
Si tratta ora di dimostrare rigorosamente che, sotto opportune ipotesi, la successione xn effettivamente converge alla soluzione cercata. Vale in proposito il seguente.

Teorema 5.21.3 Sia f: [a,b] → ℝ due volte derivabile in [a,b], e supponiamo che:

  1. f(a) ⋅ f(b) < 0;

  2. f′(x), f′′(x) hanno segno costante in [a,b];

  3. f(a) ⋅ f′′(a) > 0.

Definiamo per ricorrenza la successione (5.21.2). Allora esiste uno e un solo punto c ∈ (a,b) tale che f(c) = 0, e la successione xn converge a c per difetto.
Se invecve di valere la 3 vale a

3'. f(b) ⋅ f′′(b) > 0, allora la successione

{ x 0 = b x n + 1 = x n − f ( x n ) f ′ ( x n )

converge a c per eccesso.

Osservazione Notiamo che, sotto le ipotesi 1 e 2, è senz'altro verificata la 3 o la 3'. Infatti f′′ ha lo stesso segno in a e b per la 2, mentre f ha segni opposti in a e b, per la 1, perciò o vale la 3 o vale la 3'. Questo significa che il metodo è applicabile ogni volta che le ipotesi 1 e 2 sono verificate: si tratta solo, a quel punto, di scegliere opportunamente se porre x0 = a oppure b. A sua volta, la condizione 2, apparentemente restrittiva, sarà in generale soddisfatta pur di scegliere un intervallo [a,b] abbastanza piccolo.

Dim. Poiché f è continua in [a,b] e vale la 2, per il T. degli zeri esiste almeno un c ∈ (a,b) tale che f(c) = 0. Inoltre, poiché f′(x) ha segno costante in (a,b), la f è strettamente monotona, quindi tale punto c è unico (una funzione strettamente monotona non può annullarsi in due punti distinti). Questo prova, esistenza e unicità del punto c in cui f si annulla. Preveremo che la successione xn è monotona. Da questo seguirà (T. di monotonia) che la successione è convergente. Mostriamo intanto che, se xn converge, certamente converge a c.
Infatti, se xn → β ∈ [a,b], passando al limite nell'uguaglianza

x n + 1 = x n − f ( x n ) f ′ ( x n )

si ha

β = β − f ( β ) f ′ ( β )

(abbiamo sfruttato il fatto che f è continua, e anche f′ è continua, in quanto f′ è derivabile, perché esiste per ipotesi f′′). Dall'ultima uguaglianza segue f(β) = 0, per cui β = c (unico punto in cui la funzione si annulla).
Proviamo dunque che xn è monotona, sotto le ipotesi (1), (2), (3). Per fissare le idee, supponiamo che sia f(a) < 0. allora per (1) è f(b) > 0. Poiché per la (2) f′(x) ha segno costante in [a,b], dovrà essere f′(x) > 0 in tutto [a,b] (se valesse l'altra disguaglianza, f sarebbe decrescente, e non potrebbe essere f(a) < 0 < f(b)). Poiché f(a) < 0 e vale la (3), f′′)(x) < 0 in tutto [a,b].
Siamo quindi nella situazione della seguente figura:

Consideriamo ora la funzione

g ( x ) = x − f ( x ) f ′ ( x )

Notiamo che è:

xn + 1 = g(xn);
g(c) = c

(perché f(c) = 0); inoltre

g ′ ( x ) = f ″ ( x ) f ( x ) 2 − f ( x ) f ′ ( x ) 2 f ′ ( x ) 2 = f ( x ) f ″ ( x ) f ′ ( x ) 2 > 0 ⟺ f ( x ) < 0

(perché f′(x) < 0 in tutto [a,b]), quindi g′(x) > 0 per x ∈ [a,c] (vedi grafico di f), ossia: in [a,c] la funzione g è strettamente crescente. Tenendo presenti queste cose, proviamo ora che:

∀n è xn < xn+1 < c  (5.21.4)

Infatti: per n = 0 si ha:

x1 = a − f(a)/f'(a) > a

(perché f(a) < 0 e f′(a > 0), ossia x1 > x0. Inoltre a < c, e poiché g è crescente, questo implica

g(a) < g(c), ossia
x1 < c

Quindi abbiamo provato che

x0 < x1 < c

Applicando ora la g alle disuguaglianze precedenti (g è strettamente crescente in [a,c], e i tre punti stanno in questo intervallo, quindi g conserva le disuguaglianze), si ha

g(x0) < g(x1) < g(c), ossia
x1 < x2 < c

Applicando ancora la g, troveremo x2 < x3 < c, e così via. Perciò la (5.21.4) è vera, e in particolare xn è monotona. Questo conclude la dimostrazione.  □
