Intergration of Rational functions by Partial Fractions

Among the problems we confront in a calculus course are

∫ 1 x ( x + 1 ) d x

and

∑ n = 1 ∞ 1 n ( n + 1 )

The key to both problems is the fact that we can write

1 x ( x + 1 ) = 1 x − 1 x + 1

Which becomes simple to integrate as

∫ 1 x ( x + 1 ) d x = ∫ 1 x − 1 x + 1 d x = ln ⁡ | x | − ln ⁡ | x + 1 | + c

And analogously for the Mengoli's series.

You add fractions by bringing them to a common denominator:

x x 2 + 1 + 2 x + 1 = 3 x 2 + x + 2 ( x 2 + 1 ) ( x 2 + 1 )

But how do you reverse this? That is, how do you find from 3 x 2 + x + 2 ( x 2 + 1 ) ( x 2 + 1 ) the partial fraction decomposition x x 2 + 1 + 2 x + 1 ?

Any proper rational fraction (which means deg(P) < deg(Q))

R ( x ) = P ( x ) Q ( x ) = a n x n + a n − 1 x n − 1 + … + a 1 x + a 0 b m x m + b m − 1 x m − 1 + … + b 1 x + b 0 ( n < m )

with coprime polynomials in the numerator and denominator can be uniquely decomposed into a sum of partial fractions.
