Valore assoluto. Disuguaglianza triangolare

Si dice valore assoluto del numero reale x (o modulo di x) il numero non negativo così definito:

| x | = { x , if  x ≥ 0 − x , if  x < 0. {\displaystyle |x|={\begin{cases}x,&{\text{if }}x\geq 0\\-x,&{\text{if }}x<0.\end{cases}}}

Dalla definizione di valore assoluto segue immediatamente che:

∀ x ≥ 0:   |a| ≤ x  ⇔  −x ≤ a ≤ x (2.5.1)

Dalla (2.5.1) segue l'importante disuguaglianza triangolare:

∀x,y ∈ ℝ:  |x + y| ≤ |x| + |y| (2.5.2)

Infatti, basta scrivere le relazioni:

−|x| ≤ x ≤ |x|  e  −|y| ≤ y ≤ |y|

quindi sommare membro a membro, ottenendo

−(|x| + |y|) ≤ x + y ≤ |x| + |y|

da cui, per la (2.5.1), segue la (2.5.2).
La disuguaglianza triangolare è spesso usata nella forma seguente:

|a − b| ≤ |a − c| + |b − c|   ∀a,b,c ∈ ℝ

Basta porre nella () x = a − c e y = c − b. Se invece, sempre nella (2.5.2), si pone x = a − b e y = b si ottiene

|a| ≤ |a − b| + |b|   cioè  |a| − |b| ≤ |a − b|

Analogamente, scambiando a con b otteniamo: |b| − |a| ≤ |a − b| e dunque, ricordando la (2.5.1)

||a| − |b|| ≤ |a − b|

L'espressione |a − b| rappresenta geometricamente la distanza (nel senso geometrio elementare) dei due punti a e b sulla retta.

La (2.5.1) può facilmente estendersi al caso di n addendi:

| ∑ k = 1 n x k | ≤ ∑ k = 1 n | x k |

Valgono anche le seguenti proprietà immediate:

|ab| = |a||b|   |a/b| = |a|/|b|   |−a| = |a|
