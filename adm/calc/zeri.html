Proprietà globali delle funzioni continue o monotone su un intervallo

Funzioni continue su un intervallo

Le proprietà delle funzioni continue incontrate sinora si basano sostanzialmente sul concetto di continuità in un punto. Come vedremo, ora, invece, è soprattutto il fatto che una funzione sia continua su un intervallo ad avere delle conseguenze interessanti. La proprietà di continuità su un intervallo ha una semplice interpretazione geometrica: il grafico della funzione, su quell'intervallo, si può tracciare "senza staccare la penna dal foglio". Il concetto di funzione continua si presta quindi a tradurre in maniera rigorosa l'idea goemetrica di linea continua, e quella fisica di grandezza che varia con continuità. Questo è vero, però, pur di utilizzare funzioni definite su un intervallo di ℝ. Come vedremo, infatti, le proprietà notevoli delle funzioni continue su un intervallo dipendono in maniera cruciale dall'assioma di continuità di ℝ, e non varrebbero, ad es., se l'ambiente di lavoro fosse il campo ℚ dei razionali.
La prima proprietà di cui ci occupiamo, che chiameremo teorema degli zeri, riguarda la risoluzione di un'equazione del tipo

f(x) = 0  (4.7.1)

Quando f è un polinomio di grado ≤ 4 esistono formule che forniscono le soluzioni della (4.7.1) mediante radicali. Se però f è un polinomio di grado > 4 o una funzione più complicata, salvo casi particolarmente fortunati, non esistono formule per le soluzioni. Geometricamente, risolvere la (4.7.1) significa determinare le ascisse dei punti di intersezione tra il grafico di y = f(x) e l'asse delle ascisse. Naturalmente possono esserci infinite soluzioni, un numero finito di soluzioni, nessuna soluzione. Ogni soluzione si chiama zero di f.

Il teorema degli zeri dà alcune semplici condizioni sotto le quali esiste uno zero di f e anche un modo per calcolarlo.

Teorema 4.7.2 (Degli zeri) Sia

  1. f continua in [a,b];

  2. f(a) ⋅ f(b) < 0.

Allora esiste c ∈ (a,b) tale che f(c) = 0.
Se f è anche strettamente monotona, lo zero è unico.

Dim. Costruiamo una successione che tende a uno zero di f. Poniamo c1 = (a+b)/2, punto medio dell'intervallo [a,b].
Se f(c1) = 0 siamo fortunati e il teorema è dimostrato.
Se f(c1) ≠ 0 guardiamo il segno di f(a) ⋅ f(b) e procediamo come segue:

f(a) ⋅ f(c1) < 0
↘		 dove a2 = a1, b2 = c2
consideriamo l'intervallo [a1, b1]
↗
f(a) ⋅ f(c1) > 0		 dove a2 = c2, b2 = b1

Continuando in questo modo troviamo una sequenza di intervalli [an, bn] con le seguenti proprietà:

  1. an ≤ an+1 e bn ≥ bn+1   ({an} cresce {bn} decresce)

  2. an − bn = (b − a)/2n (ciascun intervallo è lungo la metà del precedente)

  3. f(an) ⋅ f(bn) < 0 (per come sono stati scelti an e bn a ogni passo).

Per la 1), possiamo allora dedurre che le successioni {an} e {bn} hanno limite finito, in base al T. di esistenza del limite delle succesioni monotone; cioè:

an → ℓ1   e bn → ℓ2,  n → +∞

(Si noti che le successioni sono anche limitate perché contenute in [a,b].)
Dalla 2) deduciamo che bn − an = (b − a)/2n → 0 se n → +∞ e perciò ℓ1 = ℓ2 = ℓ.
Per la continuità di f, abbiamo allora che

f(an) ⋅ f(bn) → f(ℓ)2

mentre dalla 3) e dal T. della permanenza del segno deduciamo f(ℓ)2 ≤ 0. Deve perciò essere f(ℓ) = 0 e così ℓ è lo zero cercato. □

Arrestando il procedimento indicato nella dimostrazione dopo n passi, si può assumere an o bn come valore approssimato dello zero ℓ di f, rispettivamente, per difetto o per eccesso. L'errore commesso è non superiore a (b − a)/2n.
Si osservi che, se in [a,b] vi sono più zeri della funzione f, il procedimento illustrato non indica quale di essi venga determinato.

Dimostriamo le seguenti varianti del T. degli zeri su intervalli illimitati e su intervalli aperti

Teorema 4.7.3. Sia f: ℝ → ℝ continua e supponiamo che

lim x → + ∞ f ( x ) ⋅ lim x → − ∞ f ( x ) < 0

(supponendo anche che i due limiti esistano, finiti o infiniti). Allora esiste c ∈ ℝ tale che f(c) = 0.

Dim. Si tratta di dimostrare che esista un intervallo chiuso [a,b] nel dom f ed applicare il T. degli zeri.
Si hanno i seguenti casi:

  • entrambi i limiti siano infiniti. Supponiamo che lim x →−∞ f(x) = −∞. Allora se ∀M > 0, ∃k > 0 : ∀x > k ⇒ f(x) < −M. Scegliamo un M, any M e sia a = k + 1. Allora f(a) < -M < 0.
    Now we do the same for the other limit: lim x →+∞ f(x) = +∞ if ∀M >0 se ∀ M' > 0 esiste un k > 0, tale che ∀x ∈ I si ha che x > k ⇒ f(x) > M'. Scegliamo un M', any M' e sia b = k + 1. Allora f(b) > M > 0. Abbiamo trovato così un intervallo [a,b] per cui f(a) ⋅ f(b) < 0, quindi per il T. degli zeri, ∃c ∈ [a,b] : f(c) = 0;

  • Si hanno due asintoti orizzontali y = ℓ e y = ℓ'. Supponiamo che lim x →−∞ f(x) = ℓ con ℓ < 0. Allora si può determinare un numero N > 0 : ∀|x| verificante la condizione |x| > N, si abbia |f(x) - ℓ| < ε. Sia a = N + 1 allora ℓ − ε < f(a) < ℓ + ε < 0. Considerando il secondo limite lim x →+∞ f(x) = ℓ' con ℓ' > 0. Allora si può determinare un numero N > 0 : ∀x verificante la condizione x > N, si abbia |f(x) - 'ℓ| < ε. Sia b = N + 1, allora 0 < ℓ' − ε < f(b) < ℓ' + ε. Abbiamo trovato così un intervallo [a,b] per cui f(a) ⋅ f(b) < 0, quindi ∃c ∈ [a,b] : f(c) = 0;

In maniera analoga si dimostrano gli altri casi.  □

Teorema 4.7.4. Sia f: ℝ → ℝ continua e supponiamo che

lim x → a + f ( x ) ⋅ lim x → b − f ( x ) < 0

(supponendo anche che i due limiti esistano, finiti o infiniti). Allora esiste c ∈ (a,b) tale che f(c) = 0.

Dim. Come nella dimostrazione del t. precendente. Si hanno i seguenti casi:

  • entrambi i limiti sono infiniti. Supponiamo che lim x →a+ f(x) = −∞, se ∀X > 0, ∃δ > 0 tale che quando 0 < x − a < δ, si abbia f(x) < −X. Sia x1 = a + δ/2. Abbiamo così f(x1) < 0. Per valutare l'altro limite supponiamo che lim x →b− f(x) = +∞, ciò è verificato se ∀X > 0, ∃δ > 0 tale che quando −δ < x − b < 0, si abbia f(x) > X. Sia x2 = b + δ/2. Abbiamo così f(x1) > 0. Abbiamo trovato così un intervallo [x1,x2] per cui f(x1) ⋅ f(x2>) < 0, quindi per il T. degli zeri, ∃c ∈ [x1,x2] : f(c) = 0.

Gli altri casi si trattano analogamente.  □
