Estensione del concetto di limite: limiti all'infinito e limiti infiniti

Finora abbbiamo considerato solo limiti per x che tende a un dato punto x0 ∈ ℝ. In molte situazioni si vuole però studiare il comportamento di una fuznione quando x "tende a +∞" (diventa cioè arbitrariamente grande), oppure a −∞. Ad es., intuitivamente la funzione f(x) = 1/x, x > 0, diventa arbitrariamente piccola quando x è arbitrariamente grande; il tale caso si scriverà

lim x → + ∞ 1 x = 0

La Definizione 4.1, come già visto per le successioni, si estende ai casi ℓ = ±∞ e x0 = ±∞.

Definzione 4.3.1. Limiti all'infinito. Sia f: (a, +∞) → ℝ. Diremo che il numero reale ℓ è il limite di f per x che tende a +∞, e scriveremo

lim x → + ∞ f ( x ) = ℓ

se ∀ ε > 0, esiste un numero K = K(ε) > a tale che

|f(x) − ℓ| < ε,   ∀x >K

Analogamente, se f: (−∞, a) → ℝ, diremo che il numero reale ℓ è il limite di f per x che tende a −∞, e scriveremo

lim x → − ∞ f ( x ) = ℓ

se ∀ ε > 0, esiste un numero K = K(ε) < a tale che

|f(x) − ℓ| < ε,   ∀x <K.  □

Esempio 4.3.2. Riprendiamo la funzione f(x) = 1/x, x > 0, e mostriamo che limx → +∞ f(x) = 0. Fissato ε > 0, la disequazione |f(x) − ℓ| < ε, con (ℓ = 0) è soddisfatta per quei valori positivi di x tali che 1/x < ε, cioè tali che x > 1/ε. Basterà quindi scegliere K(ε) = 1/ε.  ■

Esempio 4.3.3. Dimostriamo che:

lim x → − ∞ e x = 0

Applicando la definizione, si tratta di provare che per ogni successione {xn} tale che xn → − ∞, si ha

lim n → ∞ e x n = 0

A sua volta, per definizione di limite di successione, questo significa provare che fissato ε > 0 qualsiasi, risulti

|exn| < ε definitivamente,

ossia (essendo l'esponenziale sempre positivo),

exn < ε definitivamente.

L'ultima disuguaglianza è equivalente a:

xn < log ε

Se ε è un numero piccolo (< 1), log ε sarà un numero negativo (grande in valore assoluto); poniamo K = −log ε > 0. Dobbiamo provare che, fissato questo numero K > 0, risulta xn < −K definitivamente; ma questo è proprio ciò che vale per ipotesi, perché xn → − ∞. Questo conclude la dimostrazione.  □

Esempio 4.3.5

lim x → x 0 + 1 x = + ∞ lim x → x 0 − 1 x = − ∞

come si verifiac facilmente. Si noti che, invece,

lim x → x 0 1 x non esiste!

Asintoto orizzontale

In termini poco rigorosi, un asistoto per una funzione f è una retta del piano a cui il grafico di f tende ad avvicinarsi indefinitamente. La formalizzazione di questa proprietà fa uso della nozione di limite. Si possono definire tre tipi di asintoti: verticale, orizzontale e obliquo.

Si dice che f ha un asintoto orizzontale di equazione y = ℓ (ℓ ∈ ℝ) per x → +∞ oppure per x → − ∞ quando esiste finito il limite

lim x → ± ∞ f ( x ) = ℓ

quando in corrispondenza ad un arbitrario ε >0, si può determinare un numero N > 0 : ∀|x| verificante la condizione |x| > N, si abbia |f(x) - ℓ| < ε.

Ogni situazione di limite finito all'infinito, quindi, corrisponde graficamente alla presenza di un asintoto orizzontale, ossia di una retta orizzontale a cui il grafico della funzione si avvicina sempre più (come nell'esempio precedente).

Limite per eccesso e per difetto

Come abbiamo visto per le successioni, quando una funzione ha limite finito ℓ, talvolta è possibile precisare se questo limite viene raggiunto per eccesso (ℓ+) o per difetto (ℓ−). Graficamente, questo significa che il grafico della funzione si avvicina lla quota y = ℓ dall'alto o dal basso. La definizione precisa èla seguente:

Definizione 4.5 (Limite per eccesso o per difetto) Se ℓ ∈ ℝ e c ∈ ℝ*, si dice che

lim x → c f ( x ) = ℓ + ( rispettivamente  ℓ − )

e in tal caso si dice che f(x) tende a ℓ per eccesso (per difetto) per x tendente a c, se per ogni successione {xn} di punti I diversi da c, tale che xn → c, si ha che f(xn) → ℓ+ (rispettivamente a ℓ-) per n → ∞. Ricordiamo che, come visto nel par. 1.1, affermare che f(xn) → ℓ+ signigica che f(xn) → ℓ e inoltre f(xn) ≥ ℓ definitivamente.  □

Ad es.,

lim x → − ∞ e x = 0 +

Un errore comune consiste nel pensare che il simbolo 0+ denoti un (misterioso) numero diverso da zero e "poco più grande di 0". Sbagliato: il limite della funzione è il "solito" numero zero che conosciamo; semplicemente, la scrittura 0+ aggiunge una informazione: la funzione tende a zero per eccesso, ossia i valore di f(x) tendono a zero mantenendosi non negativi.
Si osservi che non ogni limite finito è necessariamente assunto per eccesso o per difeto, come mostra la Fig.1. Questa funzione tende a 1 per x → +∞, ma non si può affermare che f(x) → 1+, né che f(x) → 1−.

Definizione 4.6 (Limite infinito all'infinito) Sia f una funzione definita in I = (a, +∞) (risp. in (−∞, a). Si definisce

lim x → + ∞ f ( x ) = + ∞

se per ogni M > 0 esiste un k > 0, tale che ∀x ∈ I si ha che:

x > k ⇒ f(x) > M

(risp., limx→−∞ f(x) = +∞ se x < − K ⇒ f(x) > M).
Si definisce inoltre

limx→−∞ f(x) = −∞

se ∀M >0 esiste k > 0 tale che ∀x ∈ si ha che

x > k ⇒ f(x) < −M □

Esempio 4.7 (Limite infinito all'infinito) Dimostriamo che

log x → + ∞ ⁡ log 1 / 2 ⁡ x = − ∞

Dobbiamo provare che per ogni successione {xn} tale che xn → +∞, si ha

log x → + ∞ ⁡ log 1 / 2 ⁡ x n = − ∞

A sua volta, questo significa provare che fissato M > 0 qualsiasi, risulti

log1/2 xn < −M definitivamente,

il che equivale a:

xn > (1/2)−M = 2M definitivamente

Ma per ipotesi xn → ∞, quindi fissata la quantità positiva 2M, certamente risulta xn > 2M definitivamente, e l'asserto è dimostrato.  ■

Asintoto obliquo

Nei casi in cui una funzione presenta limite infinito all'infinito, può accadere (ma non sempre accade) che esista una retta, obliqua, a cui il grafico della fuznione si avvicina indefinitamente. Si parla in tal baso di asintoto obliquo. Precisamente: si dice che una funzione f ha asintoto obliquo y = mx + q (m ≠ 0, q ∈ ℝ) per x → +∞ (o per x → − ∞) se

lim x → + ∞ ( − ∞ ) [ f ( x ) − ( m x + q ) ] = 0

Esempio. Sia f(x) = 2x + 1 + ex. Studiamo la funzione per x → − ∞. Si vede facilmente che f(x) → −∞; inoltre

lim x → − ∞ [ f ( x ) − ( 2 x + 1 ) ] = lim x → − ∞ e x = 0

Perciò, per definizione di asintoto obliquo, si riconosce che la retta y = 2x + 1 è asintoto obliquo per f, per x → −∞.

In casi meno elementari, anziché dover "indovinare" qual è l'asintoto obliquo è utile avere un criterio operativo per cercarlo. Vale in proposito la seguente:

Proposizione . La funzione f ammette asintoto obliquo per x → +∞ se e solo valogono le seguenti condizioni:

  1. Esiste finito

    lim x → + ∞ f ( x ) x = m ≠ 0

  2. Esiste finito

    limx → +∞ [f(x) − mx] = q

    dove m è il numero calcolato al punto 1. In tal caso l'asintoto è y = mx + q. Analogo criterio vale per x → −∞.

Studiamo la funzione f(x) = 3x + √x per x → +∞. Si vede subito che f(x) → +∞ per x → +∞. Vediamo se presenta un asintoto obliquo. Calcoliamo perciò (in base alla proposizione precedente):

lim x → + ∞ 3 x x x = lim x → + ∞ ( 3 + 1 x ) = 3 ≡ m

(come si verifica facilmente applicando la definizione di limite). Calcoliamo ora

lim x → + ∞ [ f ( x ) − m x ] = lim x → + ∞ [ ( 3 x + x ) − 3 x ] = lim x → + ∞ x = + ∞

Poiché questo limite è infinito (non esiste q), la funzione non ammette asintoto obliquo.  ■

Limite infinito al finito

Dimostriamo che:

lim x → 0 1 x 2 = + ∞

Dobbiamo provare che ∀{xn} tale che xn ⟶ 0 e xn ≠0 ∀n si ha

lim x → 0 1 x n 2 = + ∞

A sua volta, questo significa provare che fissato K > 0, risulti

1/xn2 > K definitivamente

il che equivale a:

|xn| < 1/√K definitivamente

Ma per ipotesi xn ⟶ 0, quindi fissata la quantità positiva ε = 1/√K, certamente risulta |xn| < ε definitivamente, e l'asserto è dimostrato. ■

Asintoto verticale

Si dice che f ha un asintoto verticale di equazione x = x0 (x0 ∈ ℝ) per x ⟶ x0 (oppure per x ⟶ x0+ o x ⟶ x0−) se

lim x → x 0 f ( x ) = ± ∞

(oppure questo accade per x ⟶ x0+ o x ⟶ x0−) se limx ⟶ x0+ f(x) = +∞ o −∞ (oppure questo accade per x ⟶ x0+ o x ⟶ x0−, rispettivamente).
Ogni situazione di limite infinito al finito, quindi, corrisponde graficamente alla presenza di un asintoto verticale, ossia di una retta verticale a cui il grafico della funzione si avvicina sempre più.

Esplicitamente, sia f una funzione definita alla destra (risp. sinistra) di x = a.

  • Diciamo che la funzione f ha limite +∞ per x tendente ad a da destra se ∀X > 0, ∃δ > 0 tale che quando 0 < x − a < δ, si abbia f(x) > X. Scriviamo

    lim x → a + f ( x ) = + ∞

  • Diciamo che la funzione f ha limite +∞ per x tendente ad a da sinistra se ∀X > 0, ∃δ > 0 tale che quando −δ < x − a < 0, si abbia f(x) > X Scriviamo.

    lim x → a − f ( x ) = + ∞

  • Diciamo che la funzione f ha limite −∞ per x tendente ad a da destra se ∀X > 0, ∃δ > 0 tale che quando 0 < x − a < δ, si abbia f(x) < −X. Scriviamo

    lim x → a + f ( x ) = − ∞

  • Diciamo che la funzione f ha limite −∞ per x tendente ad a da sinistra se ∀X > 0, ∃δ > 0 tale che quando −δ < x − a < 0, si abbia f(x) < −X. Scriviamo.

    lim x → a − f ( x ) = − ∞

    • In caso di esistenza dei limiti sopracitati, chiameremo x = a, asintoto verticlae del grafico di f.

Ad es, diciamo che

  • x = 0 è asintoto verticale per 1/x2 (per x ⟶ 0);

  • x = 0 è asintoto verticale per 1/x (per x ⟶ 0+ e per x ⟶ 0−);

  • x = 0 è asintoto verticale per log x (per x ⟶ 0+);

Limite finito al finito

Questo è, in un certo senso, il caso meno intuitivo dei quattro finora trattati; vediamo perché.

Esempio xx

Chiediamoci quanto vale:

lim x → 0 sin ⁡ x

Chiunque, probabilmente, risponderebbe: zero. Dopotutto, sin 0 = 0. Occorre rendersi conto, però, che anche se il risultato è proprio questo, la motivazione è errata. Infatti, la definizione di limite richiede che:

∀ {xn} : xn ≠ 0 ∀n e xn ⟶ 0 per n ⟶ ∞, si abbia sin xn ⟶ 0 per n ⟶ ∞

Quindi, per mostrare che limx ⟶ 0 sin x = 0 non si deve calcolare sin 0! Si può procedere così: dalla disuguaglianza elementare:

|sin x| ≤ |x|, valida ∀x ∈ ℝ

(si ragioni sulla circonferenza trigonometrica per rendersene conto), leggiamo che |sin xn| ≤ xn e quindi, per il teorema del confronto, se xn ⟶ 0 anche sin xn ⟶ 0. Pertanto

limx ⟶ 0 sin x = 0

Esempio xx. Sia

f ( x ) = { 1 if  x ≠ 0 , 0 if  x = 0

e si voglia calcolare

lim x → 0 f ( x )

Applicando la definizione di limite si vede subito che il limite vale 1. Infatti, se xn ⟶ 0 per n ⟶ ∞ ma xn ≠ 0 ∀n, risulta f(xn) = 1 ∀n, e qundi f(xn) ⟶ 1 per n → ∞. In questo caso, dunque (a differenza dell'esempio precedente), si ha:

lim x → 0 f ( x ) ≠ f ( 0 )

Non esistenza del limite

Il limite di una funzione può anche non esistere.

Esempio.

limx → ∞ sin x non esiste

Per dimostrarlo è sufficiente trovare due successioni {xn} e {yn} divergenti a +∞ tali che sin xn e sin yn tendano a due limiti diversi. Si può scegliere xn = nπ e yn = π/2 + 2nπ; in corrispondenza di tali successioni si ha sin xn = 0 e sin yn = 1, quindi la definizione di limite non è soddisfatta.

Si rifletta sul fatto che, in base alla definizione successionale di limite, dimostrare rigorosamente che una funzione non ha limite è molto più facile che dimostrare che una successione non ha limite. Ad es. non è così banale dimostrare che non esiste limn → ∞ sin n. Analogamente

Example 4.1.4

lim x → 0 sin ⁡ 1 x

non esiste. Indeed we can consider two sequences {xn} and {yn} converging to 0. Let xn = 1/(2πn) and yn = 1/(π/2 + 2nπ). We have f(xn) = 0 for each n, hence the limit of the sequence of functions is 0. However f(yn) = 1 for each n and this time the limit is one. Hence, two sequences that both tend to zero are mapped by the function into sequences that converge to different limits. Then the limit does not exits.

The definition of limit given requires to verify that for every sequence tending to x0. Cauchy gave a criterion for the continuity of a function at a point that does not involve convergence of sequences.

Theorem 4.1.8. The limit of f exists if and only if both the limits, the left hand and the right hand, exist and are equal.

Proof. Let limx ⟶ x0 f(x) = ℓ. Then for any ε > 0, there exists some δ > 0 such that

|f(x) − ℓ | < ε, when x0 − δ < x < x0 + δ,  x ≠ x0

it follows that

|f(x) − ℓ | < ε, when x0 − δ < x < x0  (4.1.9)
|f(x) − ℓ | < ε, when x0 < x < x0 + δ  (4.1.10)

From equation (4.1.9) and (4.1.10) we get that the RHS and LHL both exist and are equal to ℓ.
To prove the converse let RHS = LHL = ℓ. Then for any ε > 0, there exists some δ1 > 0 and δ2 > 0 such that

|f(x) − ℓ| < ε   when   x0 < x < x0 + δ1  (4.1.11)
|f(x) − ℓ| < ε   when   x0 − δ2 < x < x0  (4.1.12)

Let δ = min(δ1, δ2). Then

δ ≤ δ1 and δ ≤ δ2  ⇒  x0 + δ ≤ x0 + δ1 and x0 − δ ≥ x0 − δ2  (4.1.13)

From equations (4.1.11), (4.1.12) and (4.1.13), we get

|f(x) − ℓ| < ε   when   x0 − δ < x < x0 + δ, x ≠ x0

Hence,

lim x → x 0 f ( x ) = ℓ  □

Example 4.1.14. The function sign of x

defined as

sgn  x := { − 1 if  x < 0 , 0 if  x = 0 , 1 if  x > 0.

admits no limt at x = 0 (i.e. is not continous at 0), since sgn has left hand limit equal to −1 at x = 0 and right hand limit equal to +1.  ■

Example 4.1.15. Evaluate

lim x → 1 x 2 − 1 x − 1

Let us evaluate the left hand and the right hand limits. When x ⟶ 1−, put x = 1 − h, h >0.
As x ⟶ 1−, h ⟶ 0+, so that

lim x → 1 − x 2 − 1 x − 1 = lim h → 0 + ( 1 − h ) 2 − 1 − h = lim h → 0 + − h ( 2 − h ) − h = lim h → 0 + ( 2 − h ) = 2

For the right hand limit, x ⟶ 1+, put x = 1 + h, h > 0.

lim x → 1 + x 2 − 1 x − 1 = lim h → 0 + ( 1 + h ) 2 − 1 h = lim h → 0 + ( 2 + h ) = 2

Since both, the left hand and right hand limits exist and are equal, the given limit of the given function exists and is equal to 2.  ■

Example 4.1.16. Evaluate

lim x → 0 e 1 / x e 1 / x + 1

When x ⟶ 0+, 1/x ⟶ ∞ and e1/x ⟶ 0

lim x → 0 + e 1 / x e 1 / x + 1 = lim x → 0 + 1 1 + e − 1 / x = 1

As x ⟶ 0−, 1/x ⟶ − ∞ and e1/x ⟶ 0.

lim x → 0 − e 1 / x e 1 / x + 1 = 0 1 = 0

So that the left hand limit is not equal to the right hand limit, the original limit does not exist.  ■

Theorem 4.1.17 (The sign permanence theorem). If f is a continuous function at x = x0 and if f(x0) > 0, there there exists a neighborhood of x0 : (x0 − δ, x0 + δ) for δ > 0 such that f(x) > 0. If f(x0) < 0, there is a neighborhood of x0 in which f(x) is negative.

Proof. By definition of continuity of f at x0, for any ε > 0, there exists some δ > 0 such that

|f(x) − f(x0)| < δ   when  x0 − δ < x < x0 + δ

if f(x0) > 0, chose ε = f(x0)/2. Then

f(x0) − ε < f(x) < f(x0) + ε
0 < f(x0)/2 < f(x) < 3f(x0)+ ε

for |x − x0| < δ. The other statement can be proved in a similar manner.  □

Esempio 4.7 (Limite infinito all'infinito) Dimostriamo che

log x → + ∞ ⁡ log 1 / 2 ⁡ x = − ∞

Dobbiamo provare che per ogni successione {xn} tale che xn → +∞, si ha

log x → + ∞ ⁡ log 1 / 2 ⁡ x n = − ∞

A sua volta, questo significa provare che fissato M > 0 qualsiasi, risulti

log1/2 xn < −M definitivamente,

il che equivale a:

xn > (1/2)−M = 2M definitivamente

Ma per ipotesi xn → ∞, quindi fissata la quantità positiva 2M, certamente risulta xn > 2M definitivamente, e l'asserto è dimostrato.  ■

Asintoto obliquo

Nei casi in cui una funzione presenta limite infinito all'infinito, può accadere (ma non sempre accade) che esista una retta, obliqua, a cui il grafico della fuznione si avvicina indefinitamente. Si parla in tal baso di asintoto obliquo. Precisamente: si dice che una funzione f ha asintoto obliquo y = mx + q (m ≠ 0, q ∈ ℝ) per x → +∞ (o per x → − ∞) se

lim x → + ∞ ( − ∞ ) [ f ( x ) − ( m x + q ) ] = 0

Esempio. Sia f(x) = 2x + 1 + ex. Studiamo la funzione per x → − ∞. Si vede facilmente che f(x) → −∞; inoltre

lim x → − ∞ [ f ( x ) − ( 2 x + 1 ) ] = lim x → − ∞ e x = 0

Perciò, per definizione di asintoto obliquo, si riconosce che la retta y = 2x + 1 è asintoto obliquo per f, per x → −∞.

In casi meno elementari, anziché dover "indovinare" qual è l'asintoto obliquo è utile avere un criterio operativo per cercarlo. Vale in proposito la seguente:

Proposizione . La funzione f ammette asintoto obliquo per x → +∞ se e solo valogono le seguenti condizioni:

  1. Esiste finito

    lim x → + ∞ f ( x ) x = m ≠ 0

  2. Esiste finito

    limx → +∞ [f(x) − mx] = q

    dove m è il numero calcolato al punto 1. In tal caso l'asintoto è y = mx + q. Analogo criterio vale per x → −∞.

Studiamo la funzione f(x) = 3x + √x per x → +∞. Si vede subito che f(x) → +∞ per x → +∞. Vediamo se presenta un asintoto obliquo. Calcoliamo perciò (in base alla proposizione precedente):

lim x → + ∞ 3 x x x = lim x → + ∞ ( 3 + 1 x ) = 3 ≡ m

(come si verifica facilmente applicando la definizione di limite). Calcoliamo ora

lim x → + ∞ [ f ( x ) − m x ] = lim x → + ∞ [ ( 3 x + x ) − 3 x ] = lim x → + ∞ x = + ∞

Poiché questo limite è infinito (non esiste q), la funzione non ammette asintoto obliquo.  ■

Asintoto verticale

Si dice che f ha un asintoto verticale di equazione x = x0 (x0 ∈ ℝ) per x ⟶ x0 (oppure per x ⟶ x0+ o x ⟶ x0−) se

lim x → x 0 f ( x ) = ± ∞

(oppure questo accade per x ⟶ x0+ o x ⟶ x0−) se limx ⟶ x0+ f(x) = +∞ o −∞ (oppure questo accade per x ⟶ x0+ o x ⟶ x0−, rispettivamente).
Ogni situazione di limite infinito al finito, quindi, corrisponde graficamente alla presenza di un asintoto verticale, ossia di una retta verticale a cui il grafico della funzione si avvicina sempre più.

Esplicitamente, sia f una funzione definita alla destra (risp. sinistra) di x = a.

  • Diciamo che la funzione f ha limite +∞ per x tendente ad a da destra se ∀X > 0, ∃δ > 0 tale che quando 0 < x − a < δ, si abbia f(x) > X. Scriviamo

    lim x → a + f ( x ) = + ∞

  • Diciamo che la funzione f ha limite +∞ per x tendente ad a da sinistra se ∀X > 0, ∃δ > 0 tale che quando −δ < x − a < 0, si abbia f(x) > X Scriviamo.

    lim x → a − f ( x ) = + ∞

  • Diciamo che la funzione f ha limite −∞ per x tendente ad a da destra se ∀X > 0, ∃δ > 0 tale che quando 0 < x − a < δ, si abbia f(x) < −X. Scriviamo

    lim x → a + f ( x ) = − ∞

  • Diciamo che la funzione f ha limite −∞ per x tendente ad a da sinistra se ∀X > 0, ∃δ > 0 tale che quando −δ < x − a < 0, si abbia f(x) < −X. Scriviamo.

    lim x → a − f ( x ) = − ∞

    • In caso di esistenza dei limiti sopracitati, chiameremo x = a, asintoto verticlae del grafico di f.

Ad es, diciamo che

  • x = 0 è asintoto verticale per 1/x2 (per x ⟶ 0);

  • x = 0 è asintoto verticale per 1/x (per x ⟶ 0+ e per x ⟶ 0−);

  • x = 0 è asintoto verticale per log x (per x ⟶ 0+);

Non esistenza del limite

Il limite di una funzione può anche non esistere.

Esempio.

limx → ∞ sin x non esiste

Per dimostrarlo è sufficiente trovare due successioni {xn} e {yn} divergenti a +∞ tali che sin xn e sin yn tendano a due limiti diversi. Si può scegliere xn = nπ e yn = π/2 + 2nπ; in corrispondenza di tali successioni si ha sin xn = 0 e sin yn = 1, quindi la definizione di limite non è soddisfatta.

Si rifletta sul fatto che, in base alla definizione successionale di limite, dimostrare rigorosamente che una funzione non ha limite è molto più facile che dimostrare che una successione non ha limite. Ad es. non è così banale dimostrare che non esiste limn → ∞ sin n. Analogamente

|f(x) − ℓ | < ε, when x0 < x < x0 + δ  (4.1.10)

Example 4.1.15. Evaluate

lim x → 1 x 2 − 1 x − 1

Let us evaluate the left hand and the right hand limits. When x ⟶ 1−, put x = 1 − h, h >0.
As x ⟶ 1−, h ⟶ 0+, so that

lim x → 1 − x 2 − 1 x − 1 = lim h → 0 + ( 1 − h ) 2 − 1 − h = lim h → 0 + − h ( 2 − h ) − h = lim h → 0 + ( 2 − h ) = 2

For the right hand limit, x ⟶ 1+, put x = 1 + h, h > 0.

lim x → 1 + x 2 − 1 x − 1 = lim h → 0 + ( 1 + h ) 2 − 1 h = lim h → 0 + ( 2 + h ) = 2

Since both, the left hand and right hand limits exist and are equal, the given limit of the given function exists and is equal to 2.  ■
