Confronti e stime asintotiche

I limiti notevoli visti nel precedente paragrafo sono espressi in modo forse più significativo usando il simbolo di equivalenza asintotica ~, già introdotto per le successioni.

Definizione 4.5.1 Si dice che due funzioni f,g sono asintotiche per x → c se

lim x → c f ( x ) g ( x ) = 1

e si scrive f ~ g per x → c.

Il simbolo di asintotico gode di tutte le proprietà giò enunciate per le successioni. I limiti notevoli precedenti si possono allora riscrivere come segue:

sin x ~ x,   1 − cos x ~ x2/2,   ex − 1 ~ x
log (1 + x) ~ x,   (1 + x)α − 1 ~ αx
per x → 0

tali relazioni si possono esprimere dicendo ad es. che sin x, log (1 + x) e ex − 1, in prima approssimazione o al primo ordine, si comportano come x, per x → 0.
Tali relazioni possono essere generalizzate: Se ε(x) + una funzione che tende a zero (cioè un infinitesimo), si può scrivere

sin ε(x) ~ ε(x),
1 − cos x ~ ε(x)2/2
eε(x) − 1 ~ ε(x)
log (1 + ε(x)) ~ ε(x)
(1 + ε(x))α − 1 ~ αε(x)

Esempio 4.5.2

lim x → 0 log ⁡ ( 1 + 2 x ) sin ⁡ 3 x = 2 3

Il limite dà una forma di indeterminazione [0/0]. Tuttavia, per x → 0, log (1 + 2x) ~ 2x, sin 3x ~ 3x, quindi

log ⁡ ( 1 + 2 x ) sin ⁡ 3 x ∼ 2 x 3 x = 2 3

Esempio 4.5.3

lim x → 1 ( x − 1 ) 2 e 3 ( x − 1 ) 2 − 1 = 1 3

Il limite dà una forma di indeterminaizone [0/0]. Tuttavia, e3(x − 1)2 − 1 ~ 3(x − 1)2 per x → 1 (la funzione ε(x) = 3(x − 1)2 è infinitesima per x → 1), perciò

( x − 1 ) 2 e 3 ( x − 1 ) 2 − 1 ∼ ( x − 1 ) 2 3 ( x − 1 ) 2 = 1 3

Esempio 4.5.4

lim x → ∞ x 3 + 2 x 2 + 1 − x 3 = 2 3

È è una forma di indeterminazione del tipo [+∞ − ∞]. Tuttavia:

x 3 + 2 x 2 + 1 − x 3 = x ( 1 + ( 2 x + 1 x 3 ) − 1 3 ) ∼ x [ 1 3 ( 2 x + 1 x 3 ) ] → 2 3

Abbiamo usato la stima

1 + ϵ ( x ) 3 − 1 ∼ 1 3 ϵ ( x ) con  ϵ ( x ) = ( 2 x + 1 x 3 )

Stime asintotiche e grafici

Le stime astintotiche non servono solo per calcolare limiti, ma anche per tracciare il grafico qualitativo di una funzione nell'intorno di un certo punto, oppure nell'interno di ±∞.

Esempio 4.5.5. Sia

f ( x ) = x 3 + x 2

Altri importanti limiti riguardano il comportamento all'infinito di potenze, esponenziali e logaritmi, analoghi a quelli validi per le successioni:

Teorema 4.5.6. (Gerarchia degli infiniti) Consideriamo le tre famiglie di funzioni:

(loga x)α   xβ,   bx   con αβ > 0, a,b > 1

Allora, per x → +∞, ognuna è un infinito di ordine inferiore rispetto a quella che le sta a destra. Esplicitamente:

lim x → + ∞ ( log a ⁡ x ) α x β = 0 , lim x → + ∞ x β b x = 0

il che si può esprimere a parole dicendo che qualunque potenza (positiva) di x prevale su qualunque potenza di log x, e qualunque esponenziale (base > 1) di x prevale su qualunque potenza di x.

La dimostrazione del teorema sarà data più avanti con gli strumenti del calcolo differenziale.
