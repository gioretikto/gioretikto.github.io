Funzioni reali di variabile reale

È già stato introdotto il concetto di funzione. Ci occuperemo ora delle funzioni reali di variabile reale, ossia le funzioni per cui sia la variabile di "ingresso" che quella di "uscita" sono numeri reali. La funzione ha dunque per dominio, D, un sottoinsieme di ℝ e per codominio ℝ; l'immagine di f sarà anch'essa un sottoinsieme di ℝ:

f: D → ℝ con D ⊆ ℝ
f: x ↦ f(x)

L'uscita corrispondente a x si chiama immagine di x; l'insieme delle possibili uscite di D tramite f si chiama anch'esso immagine e si indica con f(D) o Im f. Si noti che nella scrittura f: A → B, il codominio B può essere più grande dell'immagine f(A) (ossia in generale f(A) ⊆ B). Ad es., se f ha valori reali, solitamente si scrive f: A → ℝ senza precisare quale sia l'effettiva immagine di f.

Le funzioni più comuni, come vedremo, hanno come dominio e come immagine un intervallo (che eventualmente è tutto ℝ) o l'unione di un numero finito di intervalli.
La dipendenza di f(x) da x si visualizza efficacemente disegnando il grafico di f, ossia l'insieme dei punti del piano di coordinate (x,f(x), con x ∈ D.

Funzioni limitate

Se il grafico di una funzione f: D → ℝ è contenuto nel semipiano inferiore delimiatato da una retta parallela all'asse delle ascisse, per es. di equazione y = M, la funzione si dice limitata superiormente. Analiticamente significa che

f(x) ≤ M ∀x ∈ D

Analogamente, f si dirà limitata inferiormente se il suo grafico è contenuto nel semipiano superiore

f(x) ≥ m ∀x ∈ D

Una funzione si dirà limitata se è limitata sia inferiormente che superiormente. Il grafico di una fuznione limitata è contenuto in una striscia orizzontale del piano xy.
Per esempio

  • x ↦ x3, x ℝ, non è limitata né inferiormente, né superiormente;

  • x ↦ x2, x ℝ, è limitata inferiormente, infatti x2 ≥ 0, ∀x ∈ ℝ;

  • x ↦ 1/(1 + x2), x ℝ, è limitata poiché, 0 < 1/(1 + x2) ≤ 1, ∀x ∈ ℝ;

Equivalentemente, si può dire che una fuznione è limtiata superiormente (limita inferiormente, limitata) se, rispettivamente, la sua immagine è un sottoinsieme di ℝ limitato superiormente (limitato inferiormente, limitato).

Funzioni simmetriche

I grafici di alcune funzioni presentano particolari proprietà di simmetria. Per es. vi sono funzioni il cui grafico è simmetrico rispetto all'asse delle ordinate. Queste funzioni si chiamano funzioni pari, hanno un dominio simmetrico rispetto a x = 0 (tipicamente un intervallo del tipo (-a,a)) e sono caratterizzate dalla relazione

f(−x) = f(x)

che esprime l'uguaglianza delle ordinate corrispondenti ai punti x e −x, simmetrici rispetto a x = 0.
Funzioni che hanno il grafico simmetrico rispetto all'origine si chiamano dispari; anch'esse hanno dominio simmetrico ripetto a x = 0 e sono caratterizzate dalla relazione

f(−x) = −f(x)

Per es., la funzione x ↦ x2 mentre x ↦ x3 è dispari. Più in generale, le potenze a esponente intero sono funzioni pari (dispari) se l'esponente è pari (dispari).
Notiamo invece che una funzione non può, invece, avere il grafico simmetrico rispetto all'asse x: una curva con tale simmetria non sarebbe il grafico di una funzione, venendo meno l'univocità della corrispondenza.

Funzioni monòtone

Una funzione si dice monòtona crescente (o non decrescente) se per ogni coppia di punti x1,x2 nel dominio di f si ha:

x1 > x2  ⇒  f(x1) ≥ f(x2)  (2.2.1)

Se nella (2.2.1) vale il maggiore anziché il maggiore o uguale si dice che f è strettamente crescente.
Se invece nella (2.2.1) si sostituisce il "≥" con "≤" f si dice decrescente (o non crescente); se poi la (2.2.1) vale con "<" anziché con "&le" f si dirà strettamente decrescente.
In altri termini: f è crecente (strettamente crescente) se, all'aumentare di x, l'ordinata corrispondente sul grafico non diminuisce (aumenta); f è descrescente (strettamente decrescente) se, all'aumentare di x, l'ordinata corrispondente non aumenta (diminuisce).

Per es., x ↦ x3 è strettamente crescente; la funzione x ↦ k è sia non decrescente sia non crescente. Tutte queste funzioni (crescenti o decrescenti, strettamente o nono) si dicono monotone.
