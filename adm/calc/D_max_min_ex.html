    • lim x → 0 x − sin ⁡ x x 2 sin ⁡ x Identify the numerator and denominator: $f(x) = x - \sin x$ $g(x) = x^2 \sin x$ Find the derivative of the numerator, $f'(x)$: The derivative of $x$ is 1. The derivative of $\sin x$ is $\cos x$. So, $f'(x) = 1 - \cos x$. Find the derivative of the denominator, $g'(x)$: This requires the product rule because $g(x)$ is a product of two functions, $x^2$ and $\sin x$. The product rule states that the derivative of $u(x)v(x)$ is $u'(x)v(x) + u(x)v'(x)$. Let $u(x) = x^2$, so $u'(x) = 2x$. Let $v(x) = \sin x$, so $v'(x) = \cos x$. Applying the product rule: $g'(x) = (2x)(\sin x) + (x^2)(\cos x) = 2x \sin x + x^2 \cos x$. Apply L'Hôpital's Rule: Now, our limit problem becomes: $\lim_{x \to 0} \frac{1 - \cos x}{2x \sin x + x^2 \cos x}$ Check for indeterminate form again: Let's plug in $x=0$ into the new expression: Numerator: $1 - \cos(0) = 1 - 1 = 0$ Denominator: $2(0) \sin(0) + 0^2 \cos(0) = 0 \times 0 + 0 \times 1 = 0$ We still have the indeterminate form $\frac{0}{0}$! This means we need to apply L'Hôpital's Rule again. Apply L'Hôpital's Rule a second time: New numerator: $f_2(x) = 1 - \cos x$ $f_2'(x) = 0 - (-\sin x) = \sin x$ New denominator: $g_2(x) = 2x \sin x + x^2 \cos x$ We need to use the product rule again for both terms in the denominator. Derivative of $2x \sin x$: $(2)(\sin x) + (2x)(\cos x) = 2 \sin x + 2x \cos x$ Derivative of $x^2 \cos x$: $(2x)(\cos x) + (x^2)(-\sin x) = 2x \cos x - x^2 \sin x$ So, $g_2'(x) = (2 \sin x + 2x \cos x) + (2x \cos x - x^2 \sin x) = 2 \sin x + 4x \cos x - x^2 \sin x$ Now, our limit problem is: $\lim_{x \to 0} \frac{\sin x}{2 \sin x + 4x \cos x - x^2 \sin x}$ Check for indeterminate form again: Let's plug in $x=0$: Numerator: $\sin(0) = 0$ Denominator: $2 \sin(0) + 4(0) \cos(0) - 0^2 \sin(0) = 2(0) + 0(1) - 0(0) = 0$ Still $\frac{0}{0}$! We need to apply L'Hôpital's Rule a third time. Apply L'Hôpital's Rule a third time: New numerator: $f_3(x) = \sin x$ $f_3'(x) = \cos x$ New denominator: $g_3(x) = 2 \sin x + 4x \cos x - x^2 \sin x$ Derivative of $2 \sin x$: $2 \cos x$ Derivative of $4x \cos x$: $(4)(\cos x) + (4x)(-\sin x) = 4 \cos x - 4x \sin x$ Derivative of $-x^2 \sin x$: $(-2x)(\sin x) + (-x^2)(\cos x) = -2x \sin x - x^2 \cos x$ So, $g_3'(x) = (2 \cos x) + (4 \cos x - 4x \sin x) + (-2x \sin x - x^2 \cos x) = 6 \cos x - 6x \sin x - x^2 \cos x$ Now, our limit problem is: $\lim_{x \to 0} \frac{\cos x}{6 \cos x - 6x \sin x - x^2 \cos x}$ Check for indeterminate form (hopefully for the last time!): Let's plug in $x=0$: Numerator: $\cos(0) = 1$ Denominator: $6 \cos(0) - 6(0) \sin(0) - 0^2 \cos(0) = 6(1) - 0(0) - 0(1) = 6 - 0 - 0 = 6$ We have $\frac{1}{6}$. This is a finite number, so the limit exists and is equal to this value! The Solution: $\lim_{x \to 0} \frac{x - \sin x}{x^2 \sin x} = \frac{1}{6}$

    V = x2 ⋅ h

    dV/dx = (108 − 3x2)/4

    V = 62 ⋅ 3 = 36 ⋅ 6 = 108 cm3
