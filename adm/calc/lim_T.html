Teoremi sui limiti

Esaminiamo ora le proprietà dei limiti e i principali metodi per il loro calcolo effettivo.
Anzitutto possiamo enunciare i teoremi sui limiti di funzioni che discendono immediatamente dai corrispondenti teoremi sui limiti di successioni.
Utilizzeremo qui la nozione di "proprietà vera definitivamente per x → x0". Nei prossimi enunciati x0 e ℓ saranno punti di ℝ*, salvo avviso contrario.

Teorema 4.3.1 (Del confronto) Siano date tre funzioni f,g,h: A → ℝ e sia x0 un punto di accumulazioen di A. Supponiamo che esista δ0 > 0 tale che

f(x) ≤ h(x) ≤ g(x)   ∀x ∈ [(x0 − δ0, x0 +δ0)] ∩ A, x ≠ x0

(ossia si ha f(x) ≤ h(x) ≤ g(x) definitivamente per x → x0).
Per x → x0, f(x) → ℓ e g(x) → ℓ. Allora anche h(x) → ℓ per x → x0.

Dim. Sia ε > 0. Poiché limx → x0 f(x) = ℓ, per definizione di limite esiste un numero δ1 = δ1(ε) > 0 tale che

ℓ − ε < f(x) < ℓ + ε   ∀x ∈ [(x0 − δ1, x0 +δ1)] ∩ A, x ≠ x0

Analogamente, poiché limx → x0 h(x) = ℓ, esiste un numero δ2 = δ1(ε) > 0 tale che

ℓ − ε < h(x) < ℓ + ε   ∀x ∈ [(x0 − δ2, x0 +δ2)] ∩ A, x ≠ x0

Preso quindi δ = δ(ε) il più piccolo fra δ,δ1,δ2, si avrà che

ℓ − ε < f(x) ≤ g(x) ≤ h(x) < ℓ + ε   ∀x ∈ [(x0 − δ, x0 +δ)] ∩ A, x ≠ x0

Dall'arbitrarietà di ε segue la tesi.  □

Teorema 4.3.2 Corollario Se:

  1. Per x → x0, g(x) → 0;

  2. |h(x)| ≤ g(x) definitivamente per x → x0;

allora anche h(x) → 0 per x → x0.

Corollario 4.3.3 Se f(x) → 0 per x → 0 e g(x) è limitata definitivamente per x → x0, allora f(x) g(x) → 0 per x → x0.

Teorema 4.3.4 (Di permanenza del segno 1a forma) Se per x → x0, f(x) → ℓ > 0, allora f(x) > 0 definitivamente per x → x0.

Teorema 4.3.5 (Di permanenza del segno 2a forma) Se per x → x0, f(x) → ℓ ≥ 0, definitivamente per x → x0, allora ℓ ≥ 0.

Teorema 4.3.6 (Di permanenza del segno per funzioni continue) Se per f è continua in x0 e f(x0) > 0 definitivamente per x → x0.

Teorema 4.3.7 (Algebra dei limiti, caso dei limiti finiti) Se per x → x0, f(x) → ℓ1 e g(x) → ℓ2 (ℓ1, ℓ2 ∈ ℝ), allora per x → x0 si ha:

  1. f(x) ± g(x) → ℓ1 ± ℓ2;

  2. f(x) g(x) → ℓ1ℓ2;

  3. f(x)/g(x) → ℓ1/ℓ2 (purché ℓ2 ≠ 0, g(x) ≠ 0 definitivamente per x → x0).

Teorema 4.3.8 (Aritmetizzazione parziale di ∞) Valgono per i limiti di funzioni gli stessi risultati di "aritmetizzazione parziale di ∞" che valgolo per i limiti di successioni.

Dim. Mostriamo in che modo i teoremi sui limiti di successioni già dimostrati e la definizione successionale di limite di funzione implicano questi teoremi sui limiti di funzioni. Come si vedrà, l'idea è semplice e applicata in modo ripetitivio.

  1. T. Confronto. Sia xn una qualsiasi successione tale che xn ≠ x0 ∀n e xn → x0 per n → ∞. Vogliamo provare che h(xn) → ℓ per n → ∞. D'altro canto sappiamo che

    f(xn) ≤ h(xn) ≤ g(xn) definitivamente, e
    f(xn) → ℓ e g(xn) → ℓ

    (dato che per ipotesi f(x) → ℓ e g(x) → ℓ e questo significa che per xn → x0 si ha f(xn) → ℓ e g(xn) → ℓ). Ma allora, per il T. del confronto applicato alla successione f(xn), h(xn. g(xn), si conclude che h(xn) → ℓ, e il teorema è provato.
    Dal teorema del confronto seguono i due corollari, esattamente come nel caso delle successioni.

  2. (T. Permanenza del segno) Sia xn una qualsiasi successione tale che xn ≠ x0, per ogni n e xn → x0; allora f(xn) → ℓ > 0 (per l'ipotesi), quindi per il T. permanenza del segno per successioni (applicato alla successione f(xn)) si conclude che f(xn) > 0 definitivamente. Poiché questo vale per ogni successione xn → x0, si conclude che f(x) > 0 definitivamente per x → x0.
    Da questo teorema si deduce il teorema nella sua seconda forma (come nel caso delle successioni), e anche il T. di permanenza del segno per funzioni continue. Infatti: se f è continua in x0, allora f(x0) = limx → x0 f(x), quindi l'ipotesi f(x0) > 0 significa che limx → x0 f(x) > 0, e questo per il T. di permanenza del segno (1a forma) implica che f(x) > 0 definitivamente per x → c.

  3. T. sull'algebra dei limiti. Sia xn una qualsiasi successione tale che xn ≠ x0 per ogni n e xn → x0; allora per l'ipotesi si ha che f(xn) → ℓ1 e g(xn) → ℓ2; dal teorema sull'algebra dei limiti per usccessioni si conclude che f(xn) ± g(xn) → ℓ1 ± ℓ2, e quindi f(x) ± g(x) → ℓ1 ± ℓ2. In modo perfettamente analogo si dimostrano i punti 2 e 3 del teorema, ed il teorema di aritmetizzazione parziale di ∞.

Dal teorema dell'algebra dei limiti segue un analogo risultato per quanto riguarda le funzioni continue:

Teorema 4.3.8 (Algebra delle funzioni continue) Siano f,g due funzioni definite almeno in un intorno di x0 ∈ ℝ, e continue in x0. Allora:

  1. f(x) ± g(x) è continua in x0;

  2. f(x)g(x) è continua in x0;

  3. f(x)/g(x) è continua in x0, purché g(x0) ≠ 0;

Dim. Proviamo ad es. la 3, essendo le altre analoghe (e più semplici). Per ipotesi sappiamo che f e g sono continue in x0, ossia:

f(x) → f(x0) e g(x) → g(x0) per x → x0.

Inoltre g(x0) ≠ 0 e quindi, per il T. di permanenza del segno per funzioni continue, g(x) ≠ 0 definitivamente per x → x0. Allora per il T. sull'algebra dei limiti si conclude che

f ( x ) g ( x ) → f ( x 0 ) g ( x 0 ) per  x → x 0

ossia f/g è continua in x0.

Vale anche il prossimo importante:

Teorema 4.3.9 (Di continuità delle funzioni elementari) Le seguenti funzioni funzioni elementari sono continue in tutti i punti del proprio insieme di definizione:

  1. Potenze a esponente intero, razionale o reale;

  2. Funzioni esponenziali;

  3. Funzioni logaritmiche;

  4. Funzioni trigonometriche elementari (sin x, cos x).

Vedremo in seguito come, combinando tra loro in molti modi questi "componenti elementari" continui si ottengano ancora funzioni continue. Avremo quindi modo, molto spesso di sapere a priori che una funzione è continua nel suo insieme di definizione (ossia senza applicare caso per caso la definizione di continuità). In questi casi, potremo allora sapere a priori che il limite di f(x) per x → x0 si può effettivamente calcolare valutando semplicemente f(x0).

Dim.. Non dimostriamo il teorema in tutte le sue affermazioni. Ci limitiamo, a titolo d'esempio, a provare la continuità in tutto ℝ delle funzioni sin x, cos x.

Sappiamo già che sin x è continuo (es. xx in x = 0. Da questo segue che anche cos x è continuo in x = 0. Infatti, uno sguardo alla circonferenza trigonometrica mostra che, se x è un angolo nel primo quadrante, sin x + cos x ≥ 1 (in quanto 1 è l'ipotenusa di un triangolo rettangolo di cateti sin x e cos x). Ne segue che

0 ≤ 1 - cos x ≤ sin x per x ∈ (0, π/2), e quindi
0 ≤ 1 - cos x ≤ |sin x| per x ∈ (-π/2, π/2)

Alla per il teorema dle confronto 1 - cos x → 0 per x → 0, quindi cos x → 1 per x → 0, e perciò cos x è continua in x = 0.
Per provare ora la continuità di sin x in un genertico punto x0 consideriamo la catena di relazioni:

|sin (x0 + h) - sin x0| = |sin x0 cos h + cos x0 sin h - sin x0| ≤ |sin x0| |cos h -1| + |cos x0| |sin h|.

Ora |sin h| e |cos h -1| tendono a zero per h → 0, per quanto appena dimostrato, mentre |sin x0| e |cos x0| sono costanti, dunque sin (x0 + h) - sin x0 → 0, ossia sin (x0 + h) → sin x0 per h → 0, e sin x è continua in x0. Una analoga catena di disuguaglianza dimostra la continuità di cos x.

Esempio. Per il teorema 4.3.8 possiamo affermare ad es. che:

  1. Le funzioni potenza a esponente intero sono continue (in tutto ℝ), in quanto f(x) = x è ovviamente continua, e f(x) = xn è il prodotto di n funzioni continue.

  2. I polinomi sono funzioni continue, in quanto ottenuti sommando funzioni del tipo cxn, continue per il punto precedente;

  3. Le funzioni razionali (cioè i quozienti di polinomi) sono funzioni continue, tranne ne ipunti in cui si annulla il denominatore (che è un polinomio, qundi si annulla in un numero finito di punti);

  4. Avendo dimostrato che sin x, cos x sono continue in tutto ℝ, deduciamo che le funzioni tg x, cotg x continue nel loro insieme di definizione.

Un'altra operazione che "conserva" i limiti e la continuità e la composizione di funzioni:

Teorema 4.3.9 (Di cambio di variabile nel limite) Siano f,g due funzioni per cui è ben definita la composizione f ∘ g, almeno definitivamente per x → x0 (x0 ∈ ℝ*) e supponiamo che:

  1. g(x) → t0 per x → x0;

  2. ∃ limt → t0 f(t) = ℓ ∈ ℝ*;

  3. g(x) ≠ t0 definitivamente per x → x0.

Allora esiste

limx → x0 f(g(x)) = limt → t0 f(t)

I punti x0, t0 ∈ ℝ*. L'ipotesi iii) non è necessaria nel acso in cui f sia continua in t0, oppure (ovviamente) nel caso in cui t0 = ±.

Dim. Basta osservare che se xn è una qualsiasi successione tale che xn ≠ x0 ∀n e xn → x0 per n → ∞, allora g(xn) → t0 per n → ∞ (per l'ipotesi i) e g(xn) ≠ t0 definitivamente (per l'ipotesi iii). Perciò

f(g(xn)) → ℓ

(per l'ipotesi ii), e questo prova la tesi. Se t0 = ±∞ la condizione g(x) ≠ ± ∞ è ovviamente verificata, mentre se f è continua in t0, ℓ = f(t0), perciò nel caso risultasse g(xn) = t0 per qualche n, si avrebbe

f(g(xn)) = f(g(t0)) = ℓ

quindi la convergenza f(g(xn)) → ℓ sarebbe comunque garantita. □

Teorema 4.3.10 (Di continuità della funzione composta) Siano: g una funzione definita almeno in un intorno di t0 = g(x0) e continua in t0, allora f ∘ g è definita almeno in un intorno di x0 ed è continua in x0.

Dim. Poiché g è continua in x0,

limx → x0 g(x) = g(x0) = t0

allora per il T. del cambio di variabile nel limite, si ha:

limx → x0 f(g(x)) = limx → t0 f(t) = poiché f è continua in t0 = f(t0) = f(g(x0))

e la tesi è dimostrata.  □

Vogliamo ora trarre qualche conclusione dai vari teoremi sulle f continue visti finora. Sappiamo che

  • le f elementari dell'Analisi matematica sono continue nel loro insieme di definizione;

  • somma, prodotto, quoziente di funzioni continue danno funzioni continue (dove il denominatore non si annulla);

  • composizione di funzioni continue dà una funzione continua.

Ne segue che tutte le funzioni che si possono ottenere con somma, prodotto, quoziente e composizioni da f elementari sono continue nel loro insieme di definizione.
Risultano pertanto continue funzioni come

e−x2   √sin x   loga(1 + (tg x)2), ...

Quindi, per es.

limx → π √sin x = √sin π = 0   limx → 1 e−x2 = e−1 = 1/e
limx → 0 (1 + (tg x)2) = loga (1 + (tg 0)2) = loga 1 = 0

poiché, per le funzioni continue, se x0 è un punto del dominio, il limite si calcola semplicemente sostituendo x0 nell'espressione analitica della funzione.
Mostriamo ora vari esempi di applicazioni dei teoremi di questo paragrafo.

Esempio 4.3.11 (Limiti di polinomi) Per calcolare il limite di un polinomio per x → ±∞ basta calcolare il limite del termine di grado massimo. Infatti se il polinomio è P(x) = a0 xn + a1xn −1 + ... + an, a0 ≠ 0, si ha:

P ( x ) = a 0 x n { 1 + a 1 a 0 x + a 2 a 0 x 2 + … + a n x n }

Tutti i termini dopo 1, in parentesi graffa, tendono a 0 per x → ± ∞ e quindi limx → ± ∞ P(x) = limx → ± ∞ a0xn.
Per es.

lim x → ∞ ( 3 x 2 − x 2 + 5 ) = lim x → ∞ 3 x 3 = + ∞ lim x → − ∞ ( − 2 x 3 + x + 152 ) = lim x → − ∞ − 2 x 3 = + ∞

Esempio 4.3.12 (Limiti di funzioni razionali). Sia ora f(x) = Pn/Qm(x) dove Pn e Qm sono, rispettivamente, polinomi di grado n e m dati data:

Pn(x) = a0xn + a1xn − 1 + ... + an
Qm(x) = b0xm + b1xnm − 1 + ... + bm

Per calcolare il limite di Pn(x)/Qn(x) per x → ±∞ basta calcolare li limite del rapporto dei monomi di grado massimo, cioè:

limx → ±∞ Pn(x)/Qm = imx → ±∞ a0xn/b0xm

Infatti,

limx → ±∞ Pn(x)/Qn(x) = limx → ±∞ a0 {1 + termini che tendono a 0}/b0 {1 + termini che tendono a 0} = limx → ±∞ a0xn/b0xm

Per es.:

lim x → ∞ 3 x 3 + 1 x 2 + 2 = − ∞ , lim x → ∞ 3 x 3 + 1 x 4 − 10 x + 1 = 0 , 6 x 5 + 7 x 2 7 x 5 − 4 = 6 7

Esempio 4.3.13 Una situazione simile analoga si ha nel caso del limite per x → 0 di certi quozienti:

lim x → 0 x 2 / 3 x 2 + 3 x x + x

È una sorta di "quoziente di polinomi" (per la verità, sono potenze a esponente razionale, non intero): per x → 0, tutte le potenze scritte tendono a 0, e si ha una forma di indeterminazione [0/0]. Questa volta, i temrini preponderanti sono quelli di esponente minimo, perché x → 0:

x 2 / 3 x 2 + 3 x 3 + x = x 2 / 3 3 x 1 / 3 ( 1 + x 2 / 3 3 + x 5 / 3 3 ) = x 1 / 3 3 ( 1 + x 2 / 3 3 + x 5 / 3 3 ) → 0

poiché x1/3 e 3(1 + x2/3/3 + x5/3/3) → 3.  ■

Esempio 4.3.14

lim x → 0 x sin ⁡ 1 x = 0

Infatti, poiché |sin 1/x| ≤ 1, si ha

|x sin 1/x| ≤ |x|

da cui, per il corollario del T. del confronto, x sin 1/x → 0. La conclusione è interessante perché il fattore sin 1/x, da sola, non ammette limite, quindi non si sarebbe potuto applicare il T. sul limite del prodotto. ■

lim x → 0 + x − 1 sin ⁡ x = − ∞

poiché (x − 1) → −1 e sin x → 0+. Abbiamo applicato il T. sull'aritmetizzazione parziale di ∞.

Esempio 4.3.17.

lim x → ∞ ( x + sin ⁡ x 2 x + cos ⁡ x ) = 1 2

Infatti:

x + sin ⁡ x 2 x + cos ⁡ x = x ( 1 + sin ⁡ x x ) 2 x ( 1 + cos ⁡ x 2 x ) = ( 1 + sin ⁡ x x ) 2 ( 1 + cos ⁡ x 2 x )

Ora per il corollario del T. del confronto,

sin x/x → 0 e cos x/2x → 0, per x → +∞.

Allora per il T. sull'algebra dei limiti,

( 1 + sin ⁡ x x ) 2 x ( 1 + cos ⁡ x 2 x ) → ( 1 + 0 ) 2 ( 1 + 0 ) = 1 2
