Soluzione di alcuni problemi di massimo e minimo

Uno degli usi più proficui del calcolo differenziale consiste nella ricerca dei massimi e minimi (in breve, nell'ottimizzazione) di una funzione. Ecco alcuni problemi di ottimizzazione in ambiti diversi.

Legge della riflessione

Vogliamo ricavare la legge sugli angoli di riflessione dal principio di Fermat. A tale scopo consideriamo, in un mezzo omogeneo, un raggio luminoso riflesso da uno specchio curvo. Immaginiamo che la sezione dello specchio coincida con il grafico di una funzione y = f(x), derivabile. Possiamo sempre scegliere il sistema di riferimento in modo che il punto di riflessione sia l'origine (0,0) e che l'asse x sia diretto come la tangente allo specchio nell'origine. Di conseguenza abbiamo

f(0) = 0,   f'(0) = 0

Con riferimento alla figura sopra, P è un punto "sul raggio incidente" e Q è un punto "sul raggio riflesso". Il punto A è un altro punto sullo specchio. Quello che dobbiamo mostrare è che, poiché il tempo impiegato dal raggio per andare da P a Q è minimo (principio di Fermat: Fermat's principle states that the path taken by a ray between two given points is the path that can be traveled in the least time), deve risultare î = r̂.
Ora, la velocità della luce è costante in un mezzo omogeneo e non varia dopo la riflessione e, pertanto, minimizzare il tempo di percorrenza da P a Q equivale a minimizzare il percorso S = PA + AQ.
Siano (x0, y0) le coordinate di P(x1, y1) quelle di Q e (x,f(x)) quelle di A.
Si ha:

S ( x ) = ( x − x 0 ) 2 + ( f ( x ) − y 0 ) 2 + ( x − x 1 ) 2 + ( f ( x ) − y 1 ) 2

La funzione S = S(x) deve avere un minimo in x = 0, ascissa del punto di flessione e perciò deve essere S'(0) = 0.
Essendo

S ′ ( x ) = ( x − x 0 ) + [ f ( x ) − y 0 ] ⋅ f ′ ( x ) ( x − x 0 ) 2 + ( f ( x ) − y 0 ) 2 + ( x − x 1 ) + [ f ( x ) − y 1 ] ⋅ f ′ ( x ) ( x − x 1 ) 2 + ( f ( x ) − y 1 ) 2

S'(0) = 0 equivale a (ricordando che f(0) = 0,   f'(0) = 0)

− x 0 x 0 2 + y 0 2 + − x 1 x 1 2 + y 1 2 = 0  (5.11.1)

Ma

x 0 2 + y 0 2 = P O ― e − x 0 x 0 2 + y 0 2 = sin ⁡ i ^

mentre

x 1 2 + y 1 2 = Q O ― e x 1 x 1 2 + y 1 2 = sin ⁡ r ^

Da (5.11.1) si deduce

sin î = sin r̂

che implica (essendo 0 < î, r̂ < π/2), î = r̂.  ■

Legge della rifrazione

Quando un raggio luminoso passa da un mezzo omogeneo a un altro (aria → acqua, ad es.) la sua velocità cambia. Supponiamo che i due mezzi siano separati da una superficie piana e che la velocità (scalare) nei due mezzi sia v1 nel primo e v2 nel secondo.
Nel passare da un mezzo all'altro, un raggio luminoso cambia direzione di propagazione (viene cioè rifratto) ma raggio incidente e raggio rifratto giacciono in un piano verticale alla superficie di separazione.
Fissiamo il sistema di riferimento in modo che l'asse x sia sulla superficie di separazione e che il punto di rifrazione sia l'origine.
Sia P = (x0, y0) un punto sul raggio incidente, Q = (x1, y1) un punto sul raggio rifratto e A = (x, 0) un altro punto sull'asse x.
In base al principio di Fermat, il percorso POQ, scelto dal raggio luminoso, è quello che minimizza il tempo di percorrenza da P a Q.

Ora, per un qualunque percorso PAQ il tempo T = T(x) di percorrenza è dato da (tempo = spazio/velocità)

T ( x ) = P A ― v 1 + A Q ― v 2 = ( x − x 0 ) 2 + y 0 2 v 1 + ( x − x 1 ) 2 + y 1 2 v 2

e quindi il principio di Fermat implica T'(0) = 0.

Poiché

T ( x ) = x − x 0 v 1 ( x − x 0 ) 2 + y 0 2 + x − x 1 v 2 ( x − x 1 ) 2 + y 1 2

l'equazione T'(0) = 0 equivale a

1 v 1 − x 0 x 0 2 + y 0 2 − 1 v 2 x 1 x 1 2 + y 1 2 = 0  (5.11.2)

Osservando che

− x 0 x 0 2 + y 0 2 = sin ⁡ i ^ e x 1 x 1 2 + y 1 2 = sin ⁡ r ^

Dalla (5.11.2) si deduce

sin ⁡ i ^ sin ⁡ r ^ = v 1 v 2

che si chiama legge di Snell della rifrazione.

Efficienza della produzione

Consideriamo ora il problema dell'efficienza della produzione di un dato bene, il cui costo medio (per unità di prodotto) sia c(p)/p. Per ottenre il massimo di efficienza occorre minimizzare il costo medio e perciò, supponendo la funzione c = c(p) derivabile dovrà essere

d d p c ( p ) p = c ′ ( p ) ⋅ p − c ( p ) p 2 = 0

Se p0 è punto di minimo dovrà quindi verificarsi che

c ′ ( p 0 ) = c ( p 0 ) p 0  (5.11.3)

ossia: costo marginale = costo medio. La (5.11.3) si può riscrivere come p0c'(p0)/c(p0) = 1 e significa che l'elasticità di c nel punti p0 è uguale a 1.

Se p0 verifica la (5.11.3), la retta OQ, di coefficiente angolare c(p0)/p0, coincide con la retta tangente al grafico di c = c(p) nel punto Q, di coefficiente angolare c'(p0).
Queste condiderazioni indicano che non sempre esiste un punto p0 che soddisfa (5.11.3) e che, se esiste, può non essere unico (v. fig. b).

Passaggio di un tubo attraverso un condott

Consideriamo il problema del passaggio di un tubo di lunghezza 4 m, con sezione trascurabile, attraverso il cunicolo in Fig. e scegliamo il riferimento cartesiano come in Fig.

Riflettendo sul problema, si noterà subito che il passaggio è possibile se la lunghezxza del tubo è minore della minima lunghezza dei possibili segmenti APB.
L'equazione della retta AP è

y = (x − a)/(2 − a)

Se a > 2, la retta interseca il semiasse positivo delle y nel punto B = (0, a/(a − 2). La lunghezza del segmento AB, da minimizzare, è data allora da:

L ( a ) = a 2 + ( a a − 2 ) 2

Poiché la radice quadrata è una funzione crescente basterà minimizzare il suo argomento

L 0 ( a ) = a 2 + ( a a − 2 ) 2 a > 2

Si ha

L ′ ( a ) = 2 a ( 1 − 2 ( a − 2 ) 3 )

L'(a) = 0 per (a − 2)2 = 2 cioè a = ∛2 + 2, che è punto di minimo come facilmente si verifica. La lunghezza minimia è quindi

L(∛2 + 2) = sqrt(5 + 3 ∛4 + 3 ∛16) ≃ 4.16

Essendo 4.16 > 4, il tubo può passare.   ■

Trovare il massimo o il minimo di una funzione è anche un modo per dimostrare disuguaglianze, altra tipica appliczione del calcolo differenziale.

Trovare la più piccola costante cn > 0 per cui è versa la seguente disuguaglianza:

(a + b)n ≤ cn (an + bn)  ∀a,b > 0

(dove n è un intero positivo fissato).

Anzitutto osserviamo che, per a,b > 0, è sempre

(a + b)n (a + b)n

(basta sviluppare la potenza di (a + b)); si chiede di dimostrare che vale anche una disuguaglianza di segno opposto, per una opportuna costante cn. Si osservi che la costante più piccola per cui questo è vero è la costante migliore, cioè quella per cui la disuguaglianza dà l'informazione più precisa. Anzitutto riduciamo il problema ad un problema in una sola variabile anziché due: basta raccogliere bn ad ambo i membri e semplificare:

( a b + 1 ) ≤ ( ( a b ) n + 1 )

e porre t = a/b:

(1 + t)n ≤ cn(tn + 1)

Sia ora

f ( t ) = ( t + 1 ) n t n + 1

La disuguaglianze da dimostrare è f(t) ≤ cn per ogni t > 0, e la minima costante per cui questo è vero è il massimo di f in (0, ∞). Ci siamo ricondotti ad un consueto problema di massimizzazione di una funzione di una variabile. Calcoliamo

f ′ ( t ) = n ( t + 1 ) n − 1 − n t n − 1 ( t + 1 ) n ( t n + 1 ) 2 = n ( t + 1 ) n − 1 ( t n + 1 ) 2 ( 1 − t n − 1 ) ≥ 0 per  t ≤ 1

Quindi t = 1 è il punto di massimo, e il massimo è

cn = f(1) = 2n − 1
