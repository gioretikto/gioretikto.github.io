Teorema di Cauchy

Passiamo ora ad un altro importante teorema del calcolo differenziale, che generalizza il Teorema di Lagrange.
If we apply the Mean value theorem to each of two functions f and g continuous on [a,b] and differentiable on (a, b), we can conclude that there are c1, c2 ∈ (a, b) such that

f ′ ( c 1 ) = f ( b ) − f ( a ) b − a and g ′ ( c 2 ) = g ( b ) − g ( a ) b − a

This gives

f'(c1) [g(b) − g(a)] = g'(c2) [f(b) − f(a)]

However it happens to exist one c ∈ (a,b) wich works for both function, as the next theorem precises.

Theorem 5.12.1 (Cauchy's Mean Value Theorem) Siano f,g : [a,b] → ℝ due funzioni continue nell'intervallo chiuso [a,b e derivabili in (a, b), con g'(x) ≠ 0, ∀ x ∈ (a,b). Allora g(a) ≠ g(b) ed esiste un punto c ∈ (a,b) tale che

f ( b ) − f ( a ) g ( b ) − g ( a ) = f ′ ( c ) g ′ ( c )  (5.12.2)

Proof. Se g(a) = g(b) il risultato seguirebbe applicando il T. di Rolle's a g. Assumiamo quindi g(a) ≠ g(b). Utilizziamo la stessa funzione ausiliaria usata nella dimostrazione del T. di Lagrange

h(x) = [f(b) − f(a)]g(x) − [g(b − g(a)]f(x)

Osserviamo che h è continua in [a,b] ed è derivabile in (a,b), in quanto differenza di funzioni che hanno queste prorpeità. Inoltre

h(a) = [f(b) − f(a)]g(a) − [g(b − g(a)]f(a) = f(b)g(a) − g(b)f(a)

h(b) = [f(b) − f(a)]g(b) − [g(b − g(a)]f(b) = −f(a)g(b) − g(a)f(b)

quindi h(a) = h(b). Possiamo dunque applicare il T. di Rolle alla funzione h ottenendo che esiste un punto c ∈ (a,b) tale che h'(c) = 0. Ma

h'(c) = [f(b) − f(a)]g'(c) − [g(b − g(a)]f'(c)

che equivale alla (5.12.2).  □
