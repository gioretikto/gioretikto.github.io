Logaritmi

Abbiamo visto che è possibile definire l'elevamento a potenza anche con numeri reali qualsiasi: se b > 0 (detto base) ed x ∈ ℝ (detto esponente) è definito il numero reale bx. Così come la sottrazione è l'operazione inversa della somma (si risolve in x l'equazione a + x = b), la divisione è l'operazione inversa della motliplicazione (si risolve x l'equazione ax = b), qual è l'operazione inversa dell'elevamento a potenza? Dal momento che l'operazione di elevamento a potenza non è commutativa (per es. 23 ≠ 32), ci sono due operazioni inverse possibili: si risolve nella base x l'equazione xs = c oppure si risolve nell'esponente x l'equazione bx = c. Nel primo caso basta elevare alla potenza s−1 entrambi i membri dell'equazione, ed ottenere x = c1/s. Nel secondo caso? Non si può fare in altro modo che introdurre il logaritmo in base b: si dice che x è il logaritmo in base b di c, dove b, e si indica

bx = c   x = logb c.

Dato che b > 0 e b ≠ 1, risulta anche c > 0 e qundi esiste solo il logaritmo dei numeri positivi. vediamo come dalle proprietà dell'elevamento a potenza discedono le proprietà dei logaritmi. Da b0 = 1 segue che in ogni base

logb = 1

da bx + y = bxby segue che

logb xy = logb x + logb y

e così via.

logb x/y = logb x − logb y

logb xb = blogb x  ∀b ∈ ℝ

logb x = 1/logx b = -log1/a x   (x ≠ 1)

Cambiamento di base:

log b ⁡ x = log a ⁡ x log a ⁡ b ∀ b > 0 , b ≠ 1

Vediamo qualche esempio sull'utilizzo della formula del cambiamento di base, che renderà tutto immediatamente semplice.

Esempio Dobbiamo calcolare il valore numerico di log27 81.

Poiché 27 e 81 sono entrambi potenze del 3, esprimiamo log27 81 in base 3:

log27 81 = log3 81/ log3 27 = 4/3 ■

Una delle caratteristiche fondamentali dei logaritmi è dunque di trasformare prodotti in somme. Era questa è una proprietà preziosissima ai tempi in cui tutte le operazioni andavano fatte a mano ed i complicati calcoli astronomici imponevano l'esecuzione di moltissime operazioni fra numeri con molti decimali. Il matematico scozzese John Napier (1550-1817) introdusse per primo i logaritmi ed impiegò vent'anni per calcolarne le prime tavole, pubblicate nel 1614 (logaritmo è una parola di origine greca che signigica "rapporto numerico"), Ora, le basi più utilizzate per i logaritmi sono 10, 2 (dato che queste sono le basi più utilizzate per la notazione posizionale) ed il numero e, che viene considerato la base naturale dei logaritmi. Il logaritmo in base e di un numero c biene anche detto logaritmo naturale di c e indicato con il simbolo ln c (a volte anche log c - quando non si indica la base del logaritmo si intende che sottintesa dal contesto: 10 oppure e solitamente).
