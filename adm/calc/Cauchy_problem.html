Cauchy problem

Consider an equation of the form

F (t, y, y') = 0  11.2.1

as for example the search of the primitive of a continuous function f over I, is equivalent to solve the first order differential equation

y'(t) = f (t)

which has infinite solutions of the form

y(t) = ∫f(t) dt + c,  c ∈ ℝ

As for example equation 11.1.6, which has infinite solutions dependent on the arbitrary constant c. The set of all solution of a linear first order differential equation, depends on a parameter c: t ⟼ φ(t,c); such a set is known as general integral of the equation. An initial value problem (i.v.p.) involves ﬁnding a solution of a differential equation that satisﬁes prescribed initial conditions. The condition

y (t0) = y0 11.8

allows to select a particular solution. A Particular Solution of a differential equation is a solution obtained from the General Solution by assigning specific values to the arbitrary constants. Thus a solution ȳ of a differential equation is particular if it contains no arbitrary constants. The problem of solving equations 11.8 and 11.7 is known as initial value first order system, more frequently called Cauchy problem.
