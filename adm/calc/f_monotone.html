Funzioni monotone su un intervallo

Ci occupiamo ora di funzioni monotone su un intervallo (e non necessariamente continue). Il prossimo teorema si può vedere come una estensione alle funzioni del T. di monotonia per le successioni. Questo teorema condivide con i teoremi sulle funzioni continue su un intervallo, che abbimao discusso nel paragrafo precedente, due caratteristiche importanti:

  • la sua dimostrazione si basa sull'assioma di continuità di ℝ;

  • il teorema assume come ipotesi una proprietà globale della funzione (ossia la sua monotonia su un intervallo).

Teorema 4.9.1 (Di Monotonia). Sia f: (a,b) → ℝ una funzione monotona. Allora ∀c ∈ (a,b) esistono finiti i limiti destro e sinistro, per x → c; ai due estremi a,b esistono i limiti destro (in a) e sinistro (in b), eventualmente infiniti.

Dim. Questo teorema non sarà dimostrato utilizzando il T. di monotonia per le successioni, ma piuttosto ripetendo una dimostrazione analoga.
Per fissare le idee, supponiamo f crescente in (a,b). Sia c ∈ (a,b) e sia

s = sup {f(x) : x ∈ (a,c)}

Notiamo che s esiste finito per la proprietà dell'estremo superiore, in quanto f(c) è un maggiorante dell'insieme {f(x) : x ∈ (a,c)}. Proviamo che

lim x → c − f ( x ) = s

Sia dunque {xn} una qualsiasi successione in (a,c) tale che xn → c, e proviamo che f(xn) → s, ossia per ogni ε > 0 risulta definitivamente

s − ε < f(xn) < s + ε

La disuguaglianza è ovvia in quanto f(xn) < s per definizione di s, perché xn ∈ (a,c). Per provare la prima, osserviamo che essendo s − ε minore di s, cioè del minimo maggiorante di {f(x) : x ∈ (a,c)}, non è un maggiorante di tale insieme, perciò esiste un punto x0 ∈ (a,c) tale che f(x0) ≥ s − ε. Poiché f è crescente, ne segue che

f(x) ≥ s − ε  x ∈ (x0, c)

D'altro canto per ipotesi x < x ∀n e xn → x per n → ∞, perciò xn ∈ (x0, c) definitivamente.
Ma allora

f(x) ≥ s − ε definitivamente,

che è quanto occorreva provare.
Analogamente si può dimostrare che esiste

lim x → c + f ( x ) = inf { f ( x ) : x ∈ ( c , b ) }

Per quanto riguarda i limiti ai due estremi dell'intervallo, se si vuole provare ad es. che esiste

lim x → b − f ( x )

si definirà ancora

s = sup {f(x): x ∈ (a,b)}

tuttavia in questo caso s potrebbe anche essere +∞ (non possiamo affermare che f(b) sia un maggiorante dell'insieme, perché in b la funzione non è definita). Nel caso in cui s < ∞ si può ripetere la dimostrazione precedente; se invece s = ∞, il ragionamento si modifica come segue. Dall'ipotesi

sup {f(x): x ∈ (a,b)} = +∞

segue che per ogni K > 0 esiste x0 ∈ (a,b) tale che f(x0) > K; per la monotonia di f, allora,

f(x) > K ∀x ∈ (x0, b),

allora presa una successione xn → b, si ha che xn ∈ (x0, b) definitivamente, e quindi

f(xn) > K definitivamente,

perciò

lim x → b − f ( x ) = + ∞  □

Una conseguenza del T. di monotonia è che se una funzione è monotona in un intervallo (a,b), i suoi eventuali punti di discontinuità in (a,b) sono necessariamente discontinuità a salto, ad eccezione degli estremi a,b, in cui può aversi anche un asintoto verticale.
