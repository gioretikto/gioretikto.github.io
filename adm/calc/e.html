Il numero e

We are not always able to express the limit explicitly (i.e. the real number itself), in fact some real numbers are defined as limits of sequences. For instance consider

a n = ( 1 + 1 n ) n

We shall show now that {an} is increasing and bounded above. Therefore it is convergent. The real number which is the limit of this sequence is denoted by e and called the Euler number.

Proposition 3.4.1. The sequence {an} = (1 + 1/n)n is strictly increasing.

a n = ( 1 + 1 n ) n = ∑ k = 0 n ( n k ) 1 n k = ∑ k = 0 n 1 k ! n ( n − 1 ) … ( n − k + 1 ) n k = ∑ k = 0 n 1 k ! n n n − 1 n … n − k + 1 n = ∑ k = 0 n 1 k ! ⋅ 1 ⋅ ( 1 − 1 n ) … ( 1 − k − 1 n )   3.3.1

similarly

a n + 1 = ∑ k = 0 n + 1 1 k ! ⋅ 1 ⋅ ( 1 − 1 n + 1 ) … ( 1 − k − 1 n + 1 )   3.3.2

We know that

( 1 − 1 n ) < ( 1 − 1 n + 1 ) , … , ( 1 − k − 1 n ) < ( 1 − k − 1 n + 1 )

so each summand of 3.3.1 is smaller than the corresponding term in (3.3.2). The latter sum, moreover, contains an additional positive summand labelled by k = n + 1. Therefore an < an+1 for each n.  □

Teorema 3.4.2. La successione

a n = ( 1 + 1 n ) n

è convergente

Si osservi che questa successione presenta una forma di indecisione 1∞.

Dim. Si proverà che la successione è monotona crescente

an+1 > an

e limitata:

2 ≤ an < 4

e perciò è convergente, per il Teorema di monotonia.
Per provare che an è monotona crescente, studiamo, per n ≥ 2, il rapporto:

a n a n − 1 = ( 1 + 1 n ) n ( 1 + 1 n − 1 ) n − 1 = ( n + 1 n ) n ( n n − 1 ) n − 1 = = ( n + 1 n n n − 1 ) n ⋅ 1 ( n n − 1 ) − 1 = ( n 2 − 1 n 2 ) n ⋅ 1 ( n n − 1 ) = = ( 1 − 1 n 2 ) n 1 − 1 n ≥ 1 − n ⋅ 1 n 2 1 − 1 n = 1

dove nell'ultima disuguaglianza scritta si è applicata la disuguaglianza di Bernoulli.

(1 + x)n ≥ 1 + nx

con x = -1/n2. Questo mostra che an/(an - 1) ≥ 1, ossia an ≥ an-1, ossia an ≥ an - 1, e la successione è monotona crescente. In particolare, essendo a1 = 2, segue an ≥ 2 ∀n ≥ 1.
Consideriamo ora la successione

b n = ( 1 + 1 n ) n + 1

Si noti che

b n = a n ⋅ ( 1 + 1 n )

perciò bn > an. Con calcoli simili a quelli appena svolti (lasciamo i dettagli al lettore), si mostra che bn ≤ bn-1. Poiché b1 = 4, risulta quindi

an < bn ≤ b1 = 4, ∀n ≥ 1

e an è limitata. □

Il limite della successione an appena studiata è un numero (irrazionale) molto importante in matematica, per varie ragioni che vedremo; esso viene indicato con la lettera e (numero di Nepero) e la sua rappresentazione decimale inizia così:

2.7182818284...

Dunque, per definizione

e = lim n → ∞ ( 1 + 1 n ) n

Un problema finanziario

Ripetiamo il ragionamento effettuato nell'esempio 1.2.5, riguardo le progressioni geometriche e supponiamo di possedere un capitale iniziale C0 (per es, 1 milione di €) e di investirlo al tasso di interesse annuale t (cioè con una rendita di t milioni all'anno).
Se l'interesse viene pagato annualmente, dopo un anno il capitale posseduto sarà 1 + t.
Se l'interese viene calcolato mensilmente avremo:

  • dopo il primo mese un capitale pari a 1 + t/12;

  • dopo il secondo mese, pari a 1 + t/12 + t/12(1 + t/12) = (1 + t/12)2;

  • dopo il terzo mese, pari a (1 + t/12)2t/12 + (1 + t/12)2 = (1 + t/12)3;

  • alla fine dell'anno avremo un capitale pari a (1 + t/12)12.

Se l'interesse viene calcolato ogni n-esimo di anno, avremo alla fine un capitale pari a

(1 + t/n)n

Per t = 1 (rendita del 100%) otteniamo esattamente la successione che definisce e.
Far tendere n a ∞ significa calcolare l'interesse dopo frazioni di anno sempre più piccole fino ad arrivare a calcolarlo con continuità.

Si può dimostrare che risulta anche:

e = lim n → ∞ ( 1 − 1 n ) − n

e dalle due definizione date per e, si può provare il seguente

Teorema 3.4.3. Sia {an} una successione divergente (a +∞ o -∞). Allora esiste

lim n → ∞ ( 1 + 1 a n ) a n = e

Dim. Detta [an] la parte intera di an, per definizione si ha [an] ≤ an ≤ [an] + 1, ∀n ∈ ℕ e quindi

( 1 + 1 [ a n ] + 1 ) [ a n ] ≤ ( 1 + 1 a n ) a n ≤ ( 1 + 1 [ a n ] ) [ a n ] + 1

Essendo (1 + 1/n)n → e, [an] ∈ ℕ e [an] → ∞ avremo che

( 1 + 1 [ a n ] ) [ a n ] → e

Infatti per ogni ε > 0 sia N ∈ ℕ tale che |(1 + 1/n)n − e| < ε, ∀n ≥ N e poiché [an] ∈ ℕ, [an] → ∞, per ν ∈ ℕ tale che [an] ≥ N ∀n ≥ ν, allora risulta |(1 + 1/[an])[an] − e| < ε, ∀ n ≥ ν.
Ne segue che

lim n → ∞ ( 1 + 1 [ a n ] + 1 ) [ a n ] = lim n → ∞ ( 1 + 1 [ a n ] ) [ a n ] + 1 = e

Dalla precedente disuguaglianza e dal Teoremaa del confronto segue la tesi.  □

Quest'ultimo teorema si rivela estremamente utile nel calcolo di limiti che coinvolgono la forma di indeterminazione 1∞.

Esempio 3.4.4. Calcoliamo

lim n → ∞ ( n 3 + n ) 5 n

Si tratta di una forma di indeterminazione [1∞]. Scriviamo:

lim n → ∞ ( n 3 + n ) n = 1 ( 1 + 3 n ) 5 n + 1 = 1 [ ( 1 + 3 n ) n 3 ] 3 ( 5 n + 1 ) n ≡ 1 [ ( 1 + 1 a n ) a n ] b n

dove abbiamo moltiplicato e diviso l'espressione originaria per 1/n; con an = n/3. Per il teorema precedente, la successione entro parentesi quadra tende ad e, mentre l'esponente

bn = 3(5n + 1)/n → 15

perciò, il limite cercato è 1/e15.  ■
