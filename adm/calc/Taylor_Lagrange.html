Formula di Taylor-MacLaurin con resto secondo Lagrange

Nella formula di Taylor con resto secondo Peano, l'informazione che abbiamo sull'errore commesso nell'approssimare f col suo polinomio di Taylor-MacLauring è di tipo dinamico: al tendere a zero dell'incremento (x − x0), sappiamo che il resto tende a zero più rapidamente di (x − x0)n. Questa informazione, come abbiamo visto, è utile ad es. nel calcolo dei limiti. Per un valore fissato dell'incremento (x − x0), però, la formula di Taylor con resto secondo Peano non dice nulla sull'entità dell'errore commesso. In varie questioni di calcolo approssimato è essenziale invece stimare l'errore che si commettte approssimando una funzione col suo polinomio di Taylor-MacLaurin, quando l'incremento (x − x0) ha un valore fissato, o un valore che non supera una soglia fissata.
Ad es., sappiamo che il polinomio di Taylor-MacLaurin al 3° ordine di ex è

T3 (x) = 1 + x + x2/2 + x3/6

Se volessimo utilizzare questo fatto per calcolare un valore approssimato di √e, ossia pensassimo di approssimare

e1/2 con T3 (1/2) = 79/48

come potremmo stimare a priori (ossia senza conoscenere già il valore vero di &radi;e) di quanto stiamo sbagliando?
A questo tipo di problema risponde il prossimo teorema, che dà un modo alternativo di quantificare l'errore di approssimazione commesso:

Teorema 5.20.1 (Formula di Taylor all'ordine n con resto secondo Lagrange). Sia f : [a,b] ⟶ ℝ derivabile n + 1 volte in [a,b], e sia x0 ∈ [a,b]. Allora esiste un punto c compreso tra x0 e x tale che

f ( x ) = T n , x 0 ( x ) + f ( n + 1 ) ( c ) ( n + 1 ) ! ( x − x 0 ) n + 1  (5.20.2)

La formula precedente si dice "formula di Taylor (o di MacLaurin se x0 = 0) di ordine n, con resto secondo Lagrange".

La formula ha la struttura:

funzione da approssimare = polinomio approssimante + errore di approssimazione

dove l'errore di approssimazione è il termine f(n+1)(c)/(n +1)! (x − x0)n+1, detto resto secondo Lagrange.
Per n = 0, la formula di Taylo scritta sopra coincide col Teorema di Lagrange.
Si noti che il punto c dipende da x0,x e n, ed è compreso tra x0 e x. Se si riesce a dimostrare che |f(n + 1)| ≤ M per ogni t compreso tra x0 e x, allora la formula di Taylor con resto secondo Lagrange dice che

| f ( x ) − T n , x 0 ( x ) | ≤ M ( n + 1 ) ! | x − x 05.20.2 | n + 1

che è appunto una stima dell'errore di approssimazione commesso.

Esempio 5.20.3 Torniamo al problema di approssimare √e mediante la formula di Taylor. Ponendo, nella (5.20.2), f(x) = ex, x0 = 0, x = 1/2, n = 1/3, otteniamo

√e = T3(1/2) + ec/4! (1/2)4

Poiché per 0 ≤ c ≤ 1/2 è ec ≤ √e ≤ √3, si ottiene

|√e − T3(1/2)| ≤ √2/(24 ⋅ 4!) ≃ 0.0045

Questo significa che approssimando &radi;e con il valore 79/48 si commette un errore non superiore a 0.0045.  ■

Example 4.5. The 3rd degree Taylor approximation for sin x with x0 = 0 is sin x ≈ x − x3/3!. This gives an estimate for the value of sin (1) which is

sin 1 ≈ 1 − 13/3!
≈ 1 − 0.1666.. = 0.8333...

But how accurate is this? The remainder formula tells us that the error R3(1) is given by

R3(1) = (sin x)(4)(c) (1)4/4!

for some c with 0 ≤ c ≤ 1. We don't know what this c is, but amazingly enough, we don't need to know what it is in order to bound R3(1). Sine the fourth derivative (sin x)(4) is equal to sin x, and the values of sin x are always beteween −1 and 1, we know that |(sin x)(4)(c)| ≤ 1. So

|R3(1)| ≤ 1/4! = 1/24 ≤ 0.042

The error is no large than this number, no matter what the value of c might be. The remainder formula tells us that the true value of sin (1) lies between (0.8333... - 0.042) and (0.8333... + 0.042). Just to verify this, note that the actual value of sin (1) is 0.8414709848079..., which is within this range.  ■
