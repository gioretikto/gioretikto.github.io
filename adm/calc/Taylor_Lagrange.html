Formula di Taylor-MacLaurin con resto secondo Lagrange

Nella formula di Taylor con resto secondo Peano, l'informazione che abbiamo sull'errore commesso nell'approssimare f col suo polinomio di Taylor-MacLaurin è di tipo dinamico: al tendere a zero dell'incremento (x − x0), sappiamo che il resto tende a zero più rapidamente di (x − x0)n. Questa informazione, come abbiamo visto, è utile ad es. nel calcolo dei limiti. Per un valore fissato dell'incremento (x − x0), però, la formula di Taylor con resto secondo Peano non dice nulla sull'entità dell'errore commesso. In varie questioni di calcolo approssimato è essenziale invece stimare l'errore che si commettte approssimando una funzione col suo polinomio di Taylor-MacLaurin, quando l'incremento (x − x0) ha un valore fissato, o un valore che non supera una soglia fissata.
Ad es., sappiamo che il polinomio di Taylor-MacLaurin al 3° ordine di ex è

T3(x) = 1 + x + x2/2 + x3/6

Se volessimo utilizzare questo fatto per calcolare un valore approssimato di √e, ossia pensassimo di approssimare

e1/2 con T3 (1/2) = 79/48

come potremmo stimare a priori (ossia senza conoscenere già il valore vero di &radi;e) di quanto stiamo sbagliando?
A questo tipo di problema risponde il prossimo teorema, che dà un modo alternativo di quantificare l'errore di approssimazione commesso:

Teorema 5.20.1 (Formula di Taylor all'ordine n con resto secondo Lagrange). Sia f : [a,b] ⟶ ℝ derivabile n + 1 volte in [a,b], e sia x0 ∈ [a,b]. Allora esiste un punto c compreso tra x0 e x tale che

f ( x ) = T n , x 0 ( x ) + f ( n + 1 ) ( c ) ( n + 1 ) ! ( x − x 0 ) n + 1  (5.20.2)

La formula precedente si dice "formula di Taylor (o di MacLaurin se x0 = 0) di ordine n, con resto secondo Lagrange".

La formula ha la struttura:

f(x) = Tn(x) + Rn(x)
funzione da approssimare = polinomio approssimante + errore di approssimazione

dove l'errore di approssimazione è il termine f(n+1)(c)/(n +1)! (x − x0)n+1, detto resto secondo Lagrange.
Per n = 0, la formula di Taylo scritta sopra coincide col Teorema di Lagrange.
Si noti che il punto c dipende da x0,x e n, ed è compreso tra x0 e x. Se si riesce a dimostrare che |f(n + 1)| ≤ M per ogni t compreso tra x0 e x, allora la formula di Taylor con resto secondo Lagrange dice che

| f ( x ) − T n , x 0 ( x ) | ≤ M ( n + 1 ) ! | x − x 05.20.2 | n + 1

che è appunto una stima dell'errore di approssimazione commesso.

Esempio 5.20.3 Torniamo al problema di approssimare √e mediante la formula di Taylor. Ponendo, nella (5.20.2), f(x) = ex, x0 = 0, x = 1/2, n = 1/3, otteniamo

√e = T3(1/2) + ec/4! (1/2)4

Poiché per 0 ≤ c ≤ 1/2 è ec ≤ √e ≤ √3, si ottiene

|√e − T3(1/2)| ≤ √2/(24 ⋅ 4!) ≃ 0.0045

Questo significa che approssimando &radi;e con il valore 79/48 si commette un errore non superiore a 0.0045.  ■

Dimostrazione del Teorema 5.20.1 Proviamo la (5.20.2) nel caso n = 1. Ponendo per comodità x0 = a, x = b l'enunciato diviede: se f: [a,b] ⟶ ℝ è derivabile 2 volte in [a,b], allora esiste un punto c ∈ [a,b] tale che

f(b) = f(a) + f'(a) (b − a) + f''(c)/2(b − a)2

Poniamo f(b) − {f(a) + f'(a) (b − a)} = k(b − a)2 e cerchiamo di determinare la forma di k.
Consideriamo la funzione

g(x) = f(b) − f(x) − f'(x)(b − x) − k(b − x)2

e aplichiamo ad essa il T. del valor medio. Poiché g(b) = g(a) = 0 (la seconda uguaglianza segue dalla definizione di k) si trova che ∃c ∈ (a,b) tale che

0 = [g(b) − g(a)]/(b − a) = g'(c)

Ma g′(x) = f'(x) − f''(x)(b − x) + 2k(b − x) = (b − x) = (b − x)[2k − f''(x)] e quindi g'(c) = 0 implica, essendo c ≠ b:

k = f′′(c)/2

Con lo stesso metodo si può dimostrare la (5.20.2) per n qualsiasi. Definiamo:

g ( x ) = f ( b ) − ∑ j = 0 n f ( j ) ( x ) ( b − x ) j j ! − k ( b − x ) n + 1

con k definito implicitamente dall'identità

f(b) − Tn,a(b) = k(b − a)n+1  (5.20.3)

Ora si procede così:

  1. Si verifica che g(b) = g(a) = 0

    g(b) = f(b) - f(b) = 0
    g(a) = f(b) − Tn,a(b) − k(b − a)n +1 = 0 per definizione di k.

  2. Si applica il T. di Lagrange a g in [a,b], e si mostra che l'affermazione "∃c ∈ (a,b) tale che g′(c) = 0 è esattamente la (5.20.2). Cominiamo a calcolare g′:

    g ′ ( x ) = − ∑ j = 1 n f ( j + 1 ) ( x ) ( b − x ) j j ! + ∑ j = 1 n f ( j ) ( x ) ( b − x ) j − 1 ( j − 1 ) ! + k ( n + 1 ) ( b − x ) n = − ∑ j = 0 n f ( j + 1 ) ( x ) ( b − x ) j j ! + ∑ j = 0 n − 1 f ( j + 1 ) ( x ) ( b − x ) j j ! + k ( n + 1 ) ( b − x ) n = − f ( n + 1 ) ( x ) ( b − x ) n n ! + k ( n + 1 ) ( b − x ) n

    (dove nel secondo passaggio abbiamo eseguito una traslazione di indici nella sommatoria).
    g′(c) = 0 significa:

    − f ( n + 1 ) ( c ) ( b − c ) n n ! + k ( n + 1 ) ( b − c ) n = 0

    ossia

    k = f ( n + 1 ) ( c ) ( n + 1 ) !

    che insierita nella (5.20.3) dà la tesi.

Example 5.20.4. The 3rd degree Taylor approximation for sin x with x0 = 0 is sin x ≈ x − x3/3!. This gives an estimate for the value of sin (1) which is

sin 1 ≈ 1 − 13/3!
≈ 1 − 0.1666.. = 0.8333...

But how accurate is this? The remainder formula tells us that the error R3(1) is given by

R3(1) = (sin x)(4)(c) (1)4/4!

for some c with 0 ≤ c ≤ 1. We don't know what this c is, but amazingly enough, we don't need to know what it is in order to bound R3(1). Sine the fourth derivative (sin x)(4) is equal to sin x, and the values of sin x are always beteween −1 and 1, we know that |(sin x)(4)(c)| ≤ 1. So

|R3(1)| ≤ 1/4! = 1/24 ≤ 0.042

The error is no large than this number, no matter what the value of c might be. The remainder formula tells us that the true value of sin (1) lies between (0.8333... - 0.042) and (0.8333... + 0.042). Just to verify this, note that the actual value of sin (1) is 0.8414709848079..., which is within this range.  ■

Formula di Taylor e convessità

Consideriamo la formula (5.20.2) con n = 1:

f(x) = f(x0) + f'(x)(x − x0) + f′′(c)/2 (x − x0)2

e supponiamo che in ogni punto di (a,b) si abbia f′′(x) ≥ 0, ovvero f si convessa in (a,b). Allora si ha f′′(c)/2 (x − x0)2 ≥ 0 e perciò si può scrivere, per ogni coppia di punti x0,x ∈ (a,b):

f(x) ≥ f(x0) + f′(x)(x − x0)  (5.20.5)

La (5.20.5) significa geometricamente che il grafico di f(x) si mantiene in tutto (a,b) sopra il grafico della sua retta tangente in x0 (e questo vale per ogni scelta del punto x0 ∈ (a,b)). Abbiamo dunque dimostrato che se una funzione (due volte derivabile) è convessa, è anche "convessa per tangenti" (v. T. 5.15.7).
Se invece f è concava, ossia f′′(x) ≤ 0 in tutto (a,b), la disuguaglianza (5.20.5) vale con "≤" anziché ≥: il grafico di f(x) si mantiene in tutto (a,b) sotto il grafico della sua retta tangente in x0.
