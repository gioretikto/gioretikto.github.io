Il concetto di limite

L'operazione di limite si può estendere dalle successioni alle funzioni. potremo così precisare il comportamento di una funzione quando la variabile indipendente si muove vicino a un determinato punto oppure diventa molto grande (in valore assoluto).
Deve essere possibile trovare punti nel dominio della funzione che siano arbitrariamente vicini al target x0. Se x0 è un punto di accumulazione del dominio, allora ci saranno infiniti altri punti del dominio nelle sue vicinanze.

Definizione 4.1 (Punto di accumulazione). Sia A ⊆ ℝ. Diremo che x0 ∈ ℝ è un punto di accumulazione di A se ogni intorno di x0 contiene punti di A distinti da x0 stesso, cioè ∀δ > 0 si ha [(x0 − δ, x0 + δ) \ {x0}] ∩ A ≠ 0. □

Esempio 4.2 Consideriamo l'insieme

A = {1/n : n ∈ ℕ, n ≥ 1}

Allora 0 è un punto di accumulazione di A pur non essendo un punto di A, in quanto a un suo qualsiasi intorno Ir(0) = (−r,r) appartengono tutti gli infiniti elementi 1/n dell'insieme A per cui è 1/n < r. Ogni altro punto x ≠ 0 (v. fig. per es., x = 1/3) non è di accumulazione, dato che si può sempre determinare un suo intorno che non contenga (altri) punti di A  ■

Come altro esempio, se A = (a,b) \ x0 con x0 ∈ [a,b], allora tutti i punti dell'intervallo chiuso [a,b] sono di accumulazione per A; in particolare x0, pur non appartenendo ad A, è di accumulazione per A.

Definizione 4.3 − Limite Sia data f: A ⊆ ℝ ⟶ ℝ e sia x0 ∈ ℝ un punto di accumulazione di A. Diremo che il numero reale ℓ è il limite di f per x che tende a x0, e scriveremo

lim x → x 0 f ( x ) = ℓ

se per ogni ε > 0 esiste un numero δ > 0 (dipendente da ε) tale che

|f(x) − ℓ| < ε,   ∀x ∈ [(x0 − δ, x0 + δ) \ {x0}] ∩ A ■

La funzione f può anche non essere definita nel punto x0 e anche se lo fosse, il valore f(x0) non gioca alcun ruolo nella definizione di limite. Inoltre, essendo x0 di accumulazione di A, l'insieme [(x0 − δ, x0 + δ) \ {x0}] ∩ A che compare nella Definizione è non vuoto per ogni δ > 0.
Geometricamente f(x) ⟶ ℓ per x ⟶ x0 quando, comunque scelto un intorno V = (ℓ − ε, ℓ + ε) di ℓ, esiste un intorno U = (x0 − δ, x0 + δ) di x0 tale che ogni punto x ∈ U \ {x0} viene mandato in un punto f(x) ∈ V (v. Figura).

Consideriamo come caso tipico un intervallo I, un punto c ∈ I e una funzione f a valori reali, definita in I, salvo al più nel punto c. L'intervallo I può essere limitato o illimitato, chiuso o aperto; il punto c può essere interno all'intervallo oppure uno dei suoi estremi (eventualmente +∞ o −∞).
Prendiamo ora una qualunque successioni di punti xn, nell'intervallo I e diversi da c, che tenda a c, per n → +∞.
In corrispondenza alla successione di ingressi xn, consideriamo la successione delle uscite f(xn).
Se, qualunque sia la successione scelta, si ha che f(xn) tende al limite ℓ (finito o infinito) si dice che il limite di f(x) per x che tende a c è ℓ e si scrive

lim x → c f ( x ) = ℓ

(che si legge "il limite per x tendente a c di f(x) è uguale a ℓ" oppure "f(x) tende a ℓ per x tendente a c".

Definizione 4.2 (Successione di Limite) Si dice che

lim x → c f ( x ) = ℓ (dove c, ℓ ∈ ℝ*) se per ogni successione {xn} di punti di I diversi da c, tale che xn → c, si ha che f(xn) → c per n → ∞.  □

La definizione appena data di limite di funzione di dice definizione successionale di limite, in quanto riconduce il concetto di limite di funzione a quello di limite di successione. Non è l'unica definizione possibile, come vedremo, e a prima vista può sembrare una definizione poco operativa. Tuttavia, come dimostranno sia il successivo sviluppo della teoria, sia gli esemi che faremo di calcolo di limiti, il fatto di avera già sviluppato le basi del calcolo dei limiti per le successioni, renderà molto vantaggiosa questa definizione, proprio dal punto di vista operativo. Ad es., possiamo subito affermare che vale il:

Infatti, se esistessero due limiti ℓ1, ℓ2 diversi tra loro, presa una qualsiasi successione xn → c si avrebbe

f(xn) → ℓ1 e f(xn) → ℓ2

dunque la successione f(xn) avrebbe due limiti distinti, assurdo.
Osserviamo ora un poì più da vicino le varie situazioni che si possono verificare nella definizione di limite di funzione. È subito chirao che la casistica sarà più ricca e completa rispetto al caso delle successioni, perché mentre per una successione an il limite si calcola necessariamente per n → +∞, per una funzione f(x) il limite si può calcolare per x → c dove c può essere qualsiasi elemento di ℝ*.

Parleremo di

  • limite finito se ℓ ∈ ℝ, e di limite infinito se ℓ = ±∞;

  • limite al finito se c ∈ ℝ, e all'infinito se c = ±∞.

La Definizione 4.1, come già visto per le successioni, si estende ai casi ℓ = ±∞ e x0 = ±∞.

Esempio 4.4 (Limite finito all'infinito). Dimostriamo che:

lim x → − ∞ e x = 0

Applicando la definizione, si tratta di provare che per ogni successione {xn} tale che xn → − ∞, si ha

lim n → ∞ e x n = 0

A sua volta, per definizione di limite di successione, questo significa provare che fissato ε > 0 qualsiasi, risulti

|exn| < ε definitivamente,

ossia (essendo l'esponenziale sempre positivo),

exn < ε definitivamente.

L'ultima disuguaglianza è equivalente a:

xn < log ε

Se ε è un numero piccolo (< 1), log ε sarà un numero negativo (grande in valore assoluto); poniamo K = −log ε > 0. Dobbiamo provare che, fissato questo numero K > 0, risulta xn < −K definitivamente; ma questo è proprio ciò che vale per ipotesi, perché xn → − ∞. Questo conclude la dimostrazione.  □

Asintoto orizzontale

Si dice che f ha un asintoto orizzontale di equazione y = ℓ (ℓ ∈ ℝ) per x → +∞ oppure per x → − ∞ se

lim x → ± ∞ f ( x ) = ℓ

Ogni situazione di limite finito all'infinito, quindi, corrisponde graficamente alla presenza di un asintoto orizzontale, ossia di una retta orizzontale a cui il grafico della funzione si avvicina sempre più (come nell'esempio precedente).

Limite per eccesso e per difetto

Come abbiamo visto per le successioni, quando una funzione ha limite finito ℓ, talvolta è possibile precisare se questo limite viene raggiunto per eccesso (ℓ+) o per difetto (ℓ−). Graficamente, questo significa che il grafico della funzione si avvicina lla quota y = ℓ dall'alto o dal basso. La definizione precisa èla seguente:

Definizione 4.5 (Limite per eccesso o per difetto) Se ℓ ∈ ℝ e c ∈ ℝ*, si dice che

lim x → c f ( x ) = ℓ + ( rispettivamente  ℓ − )

e in tal caso si dice che f(x) tende a ℓ per eccesso (per difetto) per x tendente a c, se per ogni successione {xn} di punti I diversi da c, tale che xn → c, si ha che f(xn) → ℓ+ (rispettivamente a ℓ-) per n → ∞. Ricordiamo che, come visto nel par. 1.1, affermare che f(xn) → ℓ+ signigica che f(xn) → ℓ e inoltre f(xn) ≥ ℓ definitivamente.  □

Ad es.,

lim x → − ∞ e x = 0 +

Un errore comune consiste nel pensare che il simbolo 0+ denoti un (misterioso) numero diverso da zero e "poco più grande di 0". Sbagliato: il limite della funzione è il "solito" numero zero che conosciamo; semplicemente, la scrittura 0+ aggiunge una informazione: la funzione tende a zero per eccesso, ossia i valore di f(x) tendono a zero mantenendosi non negativi.
Si osservi che non ogni limite finito è necessariamente assunto per eccesso o per difeto, come mostra la Fig.1. Questa funzione tende a 1 per x → +∞, ma non si può affermare che f(x) → 1+, né che f(x) → 1−.

Definizione 4.6 (Limite infinito all'infinito) Sia f una funzione definita in I = (a, +∞) (risp. in (−∞, a). Si definisce

lim x → + ∞ f ( x ) = + ∞

se per ogni M > 0 esiste un k > 0, tale che ∀x ∈ I si ha che:

x > k ⇒ f(x) > M

(risp., limx→−∞ f(x) = +∞ se x < − K ⇒ f(x) > M).
Si definisce inoltre

limx→−∞ f(x) = −∞

se ∀M >0 esiste k > 0 tale che ∀x ∈ si ha che

x > k ⇒ f(x) < −M □

Esempio 4.7 (Limite infinito all'infinito) Dimostriamo che

log x → + ∞ ⁡ log 1 / 2 ⁡ x = − ∞

Dobbiamo provare che per ogni successione {xn} tale che xn → +∞, si ha

log x → + ∞ ⁡ log 1 / 2 ⁡ x n = − ∞

A sua volta, questo significa provare che fissato M > 0 qualsiasi, risulti

log1/2 xn < −M definitivamente,

il che equivale a:

xn > (1/2)−M = 2M definitivamente

Ma per ipotesi xn → ∞, quindi fissata la quantità positiva 2M, certamente risulta xn > 2M definitivamente, e l'asserto è dimostrato.  ■

Asintoto obliquo

Nei casi in cui una funzione presenta limite infinito all'infinito, può accadere (ma non sempre accade) che esista una retta, obliqua, a cui il grafico della fuznione si avvicina indefinitamente. Si parla in tal baso di asintoto obliquo. Precisamente: si dice che una funzione f ha asintoto obliquo y = mx + q (m ≠ 0, q ∈ ℝ) per x → +∞ (o per x → − ∞) se

lim x → + ∞ ( − ∞ ) [ f ( x ) − ( m x + q ) ] = 0

Esempio. Sia f(x) = 2x + 1 + ex. Studiamo la funzione per x → − ∞. Si vede facilmente che f(x) → −∞; inoltre

lim x → − ∞ [ f ( x ) − ( 2 x + 1 ) ] = lim x → − ∞ e x = 0

Perciò, per definizione di asintoto obliquo, si riconosce che la retta y = 2x + 1 è asintoto obliquo per f, per x → −∞.

In casi meno elementari, anziché dover "indovinare" qual è l'asintoto obliquo è utile avere un criterio operativo per cercarlo. Vale in proposito la seguente:

Proposizione . La funzione f ammette asintoto obliquo per x → +∞ se e solo valogono le seguenti condizioni:

  1. Esiste finito

    lim x → + ∞ f ( x ) x = m ≠ 0

  2. Esiste finito

    limx → +∞ [f(x) − mx] = q

    dove m è il numero calcolato al punto 1. In tal caso l'asintoto è y = mx + q. Analogo criterio vale per x → −∞.

Studiamo la funzione f(x) = 3x + &rad;x per x → +&nfin;. Si vede subito che f(x) → +∞ per x → +∞. Vediamo se presenta un asintoto obliquo. Calcoliamo perciò (in base alla proposizione precedente):

lim x → + ∞ 3 x x x = lim x → + ∞ ( 3 + 1 x ) = 3 ≡ m

(come si verifica facilmente applicando la definizione di limite). Calcoliamo ora

lim x → + ∞ [ f ( x ) − m x ] = lim x → + ∞ [ ( 3 x + x ) − 3 x ] = lim x → + ∞ x = + ∞

Poiché questo limite è infinito (non esiste q), la funzione non ammette asintoto obliquo.  ■

Limite destro e sinistro

Talvolta una funzione si comporta diversamente (dal punto di vista del suo limite) a seconda che x si avvicini a x0 da destra o da sinistra. Ad es., per x ⟶ 0 la funzione 1/x diventa sempre più grande in valore assoluto, ma con un segno diverso a seconda che sia x ≷ 0. Per descrivere questo tipo di situazioni, si introduce il concetto di limite destro e sinistro:

Definizione (limite destro e sinistro) Se x0 ∈ ℝ e ℓ ∈ ℝ*, si dice che

lim x → x 0 + f ( x ) = ℓ ( rispettivamente  , lim x → x 0 − f ( x ) = ℓ )

e in tal caso si dice che il limite destro (sinistro) di f per x tendente a x0 è ℓ, se per ogni successione {xn} di punti di I tali che xn ⟶ x0+ (x0−) per n ⟶ ∞ e xn ≠ x0 ∀n, si ha che f(xn) ⟶ x0 per n ⟶ ∞. □

Ad es.,

lim x → x 0 + 1 x = + ∞ lim x → x 0 − 1 x = − ∞

come si verifiac facilmente. Si noti che, invece,

lim x → x 0 1 x non esiste!

Più in generale:

il limite limx→x0 f(x) = ℓ, esiste se e solo se esistono, e sono entrambi uguali a ℓ, i limiti destro e sinistro; tuttavia, può accadere che i limiti destro e sinistrono esistano ma siano diversi tra loro, oppure uno solo dei due esista: in questi casi limx→x0 f(x) non esiste.

Se f è definita in (a,b), il limite per x → a (rispettivamente x → b) è automaticamente un limite destro (rispettivamente sinistro).

Asintoto verticale

Si dice che f ha un asintoto verticale di equazione x = x0 (x0 ∈ ℝ) per x ⟶ x0 (oppure per x ⟶ x0+ o x ⟶ x0−) se

lim x → x 0 f ( x ) = ± ∞

(oppure questo accade per x ⟶ x0+ o x ⟶ x0−) se limx ⟶ x0+ f(x) = +∞ o −∞ (oppure questo accade per x ⟶ x0+ o x ⟶ x0−, rispettivamente).
Ogni situazione di limite infinito al finito, quindi, corrisponde graficamente alla presenza di un asintoto verticale, ossia di una retta verticale a cui il grafico della funzione si avvicina sempre più. Ad es, diciamo che

  • x = 0 è asintoto verticale per 1/x2 (per x ⟶ 0);

  • x = 0 è asintoto verticale per 1/x (per x ⟶ 0+ e per x ⟶ 0−);

  • x = 0 è asintoto verticale per log x (per x ⟶ 0+);

The following is the definition of limit of a function due to Heinrich Eduard Heine.

Definition 4.1.2. (Limit of a function; Heine's Definition) Let D ⊆ ℝ and c ∈ ℝ be an accumulation point of D. Also, let f: D → ℝ be a function. Let {xn} be any sequence that converges to the point c. Now consider the sequence {f(xn)}. If {f(xn)} converges to a real number ℓ, for every sequence {xn} that converges to c, we say that the function f has a limit ℓ. This is written as

lim x → c f ( x ) = ℓ

The great advantage of the previous definition is that the theory of limits of sequences carries over easily to limtis of functions. In particular the limit of is unique because is unique that for sequences.

One of the most important applications of the limit concept is that of continuity. At a point c, the function f can be befined or undefined. If it is defined and if we find that the limit at this point is equal to the defined value, that is, if

f ( c ) = lim x → c f ( x )

thne we say that the function f is continuos at c. We say that f is continuous if an open interval (a,b) is it is continuous at each point of the interval.

Definition 4.1.3 (Continuity of Functions). Let D ⊆ ℝ and consider the function f: D → ℝ and a point c ∈ D. We say that f is continuous at c if

for any sequence {xn} in D such that xn → c  ⇒   f(xn) → f(c).

If f is not continuous at x0, we say that f is discontinuous at x0. In case f is continuous at every x0 ∈ D, we say that f is continuous on D.  □

Thus, we consider a sequence {xn} from the domain of f, which tends to c but does not take the value c. The corresponding sequence {f(xn)} may or may not have a limit as n ⟶ ∞. If not, we say that f is discontinuous at c.

The definition of limit given requires to verify that for every sequence tending to x0. Cauchy gave a criterion for the continuity of a function at a point that does not involve convergence of sequences.

Proposition 4.1.5. (Cauchy's definition of limit) Let D ⊆ ℝ and let f: D → ℝ be a function. A number ℓ is the limit of a function f as x tends to x0 if and only if for any ε > 0 there is a δ > 0 (dependent on ε) such that

x ∈ D and |x − x0| < δ  ⇒  |f(x) − ℓ| < ε  (4.1.6)

Proof. Let ε > 0 and assume there exists δ > 0 such that the implication 4.1.6 holds. Let xn ⟶ x0 and xn ≠ x0, then there exists N ∈ ℕ such that 0 < |x − x0| < δ, and consequently |f(xn) − ℓ| < ε, for n > N. This prove f(xn) ⟶ ℓ and hence limx ⟶ x0 f(x) = ℓ.
Conversely assume f has limit ℓ but that it not true that for every ε > 0 there exists δ > 0, there is is x ∈ D satisfying 4.1.6. Then there is a sequence {xn} in D such that |xn − x0| < 1/n, but |f(xn) − f(x0)| ≥ ε for all N ∈ ℕ. But then xn ⟶ x0 and f(xn) ↛ f(x0). This contradicts the continuity of f at x0.  □

To have a limit, the function has to be defined on both sides of x0, but it doesn't matter what the function is at x = x0 itself; it need not even be defined there. For instance even though the function f(x) = sin x/x is not defined at x = 0, its limit as x ⟶ 0 exists and equals 1.

In the definition of the limit of a function f as x approaches x0 whether from right or from left, we have considered the values of f in the deleted neighbourhood x0. If we consider the behaviour of f for those values of x greater than x0, we say, that x approaches x0 from the right or from above. We can give the following definition for this concept.

Definition 4.1.6. (Right hand limit (RHL)) We say that a function f tends to ℓ as x tends to x0 from the right if for each ε > 0, there exists a δ > 0 such that

|f(x) − ℓ| < ε   when   x0 < x < x0 + δ

In symbol,

lim x → x 0 + f ( x ) = ℓ

Definition 4.1.7. (Left hand limit (LHL)) We say that a function f tends to ℓ as x tends to x0 from the left if for each ε > 0, there exists a δ > 0 such that

|f(x) − ℓ| < ε   when   x0 − δ < x < x0

In symbol,

lim x → x 0 − f ( x ) = ℓ

Clearly the limit of f exists, then both the left hand and right hand limits exist and each is equal to ℓ.

Theorem 4.1.8. The limit of f exists if and only if both the limits, the left hand and the right hand, exist and are equal.

Proof. Let limx ⟶ x0 f(x) = ℓ. Then for any ε > 0, there exists some δ > 0 such that

|f(x) − ℓ | < ε, when x0 − δ < x < x0 + δ,  x ≠ x0

it follows that

|f(x) − ℓ | < ε, when x0 − δ < x < x0  (4.1.9)
|f(x) − ℓ | < ε, when x0 < x < x0 + δ  (4.1.10)

From equation (4.1.9) and (4.1.10) we get that the RHS and LHL both exist and are equal to ℓ.

To prove the converse let RHS = LHL = ℓ. Then for any ε > 0, there exists some δ1 > 0 and δ2 > 0 such that

|f(x) − ℓ| < ε   when   x0 < x < x0 + δ1  (4.1.11)
|f(x) − ℓ| < ε   when   x0 − δ2 < x < x0  (4.1.12)

Let δ = min(δ1, δ2). Then

δ ≤ δ1 and δ ≤ δ2  ⇒  x0 + δ ≤ x0 + δ1 and x0 − δ ≥ x0 − δ2  (4.1.13)

From equations (4.1.11), (4.1.12) and (4.1.13), we get

|f(x) − ℓ| < ε   when   x0 − δ < x < x0 + δ, x ≠ x0

Hence,

lim x → x 0 f ( x ) = ℓ  □

As x ⟶ 1−, h ⟶ 0+, so that

lim x → 1 − x 2 − 1 x − 1 = lim h → 0 + ( 1 − h ) 2 − 1 − h = lim h → 0 + − h ( 2 − h ) − h = lim h → 0 + ( 2 − h ) = 2

For the right hand limit, x ⟶ 1+, put x = 1 + h, h > 0.

lim x → 1 + x 2 − 1 x − 1 = lim h → 0 + ( 1 + h ) 2 − 1 h = lim h → 0 + ( 2 + h ) = 2

Since both, the left hand and right hand limits exist and are equal, the given limit of the given function exists and is equal to 2.  ■
