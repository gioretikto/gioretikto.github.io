Il concetto di limite

L'operazione di limite si può estendere dalle successioni alle funzioni. potremo così precisare il comportamento di una funzione quando la variabile indipendente si muove vicino a un determinato punto oppure diventa molto grande (in valore assoluto).
Deve essere possibile trovare punti nel dominio della funzione che siano arbitrariamente vicini al target x0. Se x0 è un punto di accumulazione del dominio, allora ci saranno infiniti altri punti del dominio nelle sue vicinanze.

Definizione topologica di limite

Definizione 4.1.1 − Limite (definizione ε-δ o di Cauchy) Sia data f: A ⊆ ℝ ⟶ ℝ e sia x0 ∈ ℝ un punto di accumulazione di A. Diremo che il numero reale ℓ è il limite di f per x che tende a x0, e scriveremo

lim x → x 0 f ( x ) = ℓ

se per ogni ε > 0 esiste un numero δ > 0 (dipendente da ε) tale che

|f(x) − ℓ| < ε,   ∀x ∈ [(x0 − δ, x0 + δ) \ {x0}] ∩ A (4.1.2)■

La funzione f può anche non essere definita nel punto x0 e anche se lo fosse, il valore f(x0) non gioca alcun ruolo nella definizione di limite. Inoltre, essendo x0 di accumulazione di A, l'insieme [(x0 − δ, x0 + δ) \ {x0}] ∩ A che compare nella Definizione è non vuoto per ogni δ > 0.
Geometricamente f(x) ⟶ ℓ per x ⟶ x0 quando, comunque scelto un intorno V = (ℓ − ε, ℓ + ε) di ℓ, esiste un intorno U = (x0 − δ, x0 + δ) di x0 tale che ogni punto x ∈ U \ {x0} viene mandato in un punto f(x) ∈ V (v. Figura).

Esempio 4.1.3 Sia f(x) = 3x + 7, x ∈ ℝ; dimostriamo che limx → 2 f(x) = 13. Dobbiamo mostrare che è verigicata la definizione limite con x0 = 2 e ℓ = 13.
Fissato ε > 0 dobbiamo determinare l'insieme delle soluzioni della disequazione |3x + 7 −13| < ε, che è equivalente a: −ε < 3x −6 < ε. A conti fatti, si verifica che queste due disequaizoni sono contemporaneamente soddisfatte se e solo se x ∈ (2 −ε/3, 2 + ε/3), che è un intorno del punto x0 = 2 di raggio δ(ε) = ε/3. Quindi, quale che sia ε > 0, sia ha che la disequazione |f(x) − 13| < ε è soddisfatta per ogni x ∈ (2 −δ(ε), 2 + δ(ε)), e quseto completa la verifia richiesta. □

Esempio 4.1.4. Dimostriamo che

lim x → 0 ( 1 − x 2 2 ) = 1

Fissato ε > 1, la disuguaglianza |(1 − x2/2) − 1| < ε è soddisfatta quando x2 < 2ε, cioè quando |x| < √2ε. Basta quindi scegliere δ(ε) = √2ε nella definizione di limite.  □

Esempio 4.1.5 (Non esistenza del limite). Sia f(x) = |x|/x, x ≠ 0; vogliamo dimostrare che non eisiste il limite di f(x) per x che tende a 0. Dobbiamo dunque far vedere che, comunque si scelga un numero reale ℓ ∈ ℝ come "candidato" limite, non è possibile fare in modo che siano verificate le proprietà richiede nella Definizione 4.1.1 di limite. In altri termini, dobbiamo mostrare che per ogni scelta di ℓ, esiste un numero ε > 0 con la seguente proprietà: comunque si scelga δ > 0, esistono dei punti x ∈ (−δ, δ), x ≠ 0, tali che |f(x) − ℓ| ≥ ε (in questo modo la proprietà (4.1.2) non è verificata). Nel nostro caso basta scegliere ε = 1/2 e procedere a una verifica grafica di questo fatto: è infatti chiaro che, quale che sia il valore di ℓ, o i punti x a sinistra dell'origine o quelli a destra (o entrambi) sono tali che |x|/x − ℓ| ≥ 1/2 (v. figura). Ad es., se ℓ = 1 la disequeazione |x|/x − ℓ| ≥ 1/2 è verificata da tutti i punti x < 0 ma non è verificata dai punti x > 0.  □

Definizione 4.1.5 (Definitivamente) Diremo che una funzione f ha una certa proprietà definitivamente per x → x0 se esiste un intorno I di x0 tale che la proprietà vale per f(x) ∀x ∈ I, x ≠ x0.  □

Definizione successionale di limite (o di Heine)

The following is the definition of limit of a function due to Heinrich Eduard Heine (1821-1881) – German mathematician.

Definition 4.1.2. (Limit of a function; Heine's Definition) Let D ⊆ ℝ and x0 ∈ ℝ be an accumulation point of D. Also, let f: D → ℝ be a function. Let {xn} be any sequence that converges to the point x0. Now consider the sequence {f(xn)}. If {f(xn)} converges to a real number ℓ, for every sequence {xn} that converges to x0, we say that the function f has a limit ℓ. This is written as

lim x → x 0 f ( x ) = ℓ

The great advantage of the previous definition is that the theory of limits of sequences carries over easily to limtis of functions. In particular the limit of is unique because is unique that for sequences.

Heine's definition of a limit function defined in a certan domain D ⊂ ℝn at a given point x0 ∈ ℝn that is the cluster point for D, has then the following form

lim x → x 0 f ( x ) = ℓ ⟺ [ ∀ { x n } ( x n ∈ D ∧ x n ≠ x 0 , n = 1 , 2 , 3. . . ) ∧ lim n → ∞ x n = x 0 ⇒ lim n → ∞ f ( x n ) = ℓ ]

Note. The definition of the limit after Heine is conveniently applied when we have to prove that a function f has no limit. For it is sufficient to show that there are two sequences {xn} and {x'n} such that limn ⟶ ∞ xn = limn ⟶ ∞ x'n = A but the corresponding sequences {f(xn)} and {f(x'n)} do not have identical limits.  ■

Non è l'unica definizione possibile, come vedremo, e a prima vista può sembrare una definizione poco operativa. Tuttavia, come dimostranno sia il successivo sviluppo della teoria, sia gli esemi che faremo di calcolo di limiti, il fatto di avera già sviluppato le basi del calcolo dei limiti per le successioni, renderà molto vantaggiosa questa definizione, proprio dal punto di vista operativo. Teorema 4.3 (Di unicità del Limite) Se esiste limc → x0 = ℓ, tale limite ℓ è unico.

Infatti, se esistessero due limiti ℓ1, ℓ2 diversi tra loro, presa una qualsiasi successione xn → c si avrebbe

f(xn) → ℓ1 e f(xn) → ℓ2

dunque la successione f(xn) avrebbe due limiti distinti, assurdo.
Osserviamo ora un poì più da vicino le varie situazioni che si possono verificare nella definizione di limite di funzione. È subito chirao che la casistica sarà più ricca e completa rispetto al caso delle successioni, perché mentre per una successione an il limite si calcola necessariamente per n → +∞, per una funzione f(x) il limite si può calcolare per x → c dove c può essere qualsiasi elemento di ℝ*.

Parleremo di

  • limite finito se ℓ ∈ ℝ, e di limite infinito se ℓ = ±∞;

  • limite al finito se x0 ∈ ℝ, e all'infinito se x0 = ±∞.
