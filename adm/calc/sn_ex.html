Esercizi sulle serie

  1. Dare una stima asintotica del termine generale delle seguenti serie a termini positivi. Dedurne il carattere della serie (convergente o divergente).

    ∑ k = 0 ∞ n + 3 2 n 3 + 2 n + 2 ∑ k = 0 ∞ n 3 n 2 + n + 1

    Soluzione.

    1. (n + 3)/(2n3 + 2n + 2) ∼ n/2n3 = 1/2n2.
      Dato che la serie Σ1/2n2 è una serie p-armonica con p = 2, che è convergente, la serie originale è convergente;

    2. n1/3 /(n2 + n + 1) ∼ n1/3/n2 = 1/n5/3. La serie Σ 1/n5/3 è una serie p-armonica con p = 5/3, che è convergente. Quindi, la serie originale è convergente.

  2. Stabilire se le seguenti serie a termini positivi convergono o divergono, precisando il criterio utilizzato:

    1. ∑ n = 1 ∞ n ! n n

      usiamo il criterio del rapporto:

      a n + 1 a n = ( n + 1 ) ! ( n + 1 ) n + 1 ⋅ n n n ! = ( n + 1 ) n n ( n + 1 ) n + 1 = n n ( n + 1 ) n = ( n n + 1 ) n

      Ora, vediamo il comportamento di [n/(n+1)]n per n → ∞

      ( n n + 1 ) n = ( 1 − 1 n + 1 ) n .

      Utilizzando il limite noto

      lim n → ∞ ( 1 − 1 n + 1 ) n = 1 e

      Poiché 1/e ≈ 0.3679, che è minore di 1, possiamo concludere che la serie originale converge.   ■

    2. ∑ n = 1 ∞ n 1 / n n !

      Usiamo il criterio del rapporto e calcoliamo il rapporto tra i termini successivi:

      a n + 1 a n = ( n + 1 ) 1 / ( n + 1 ) ( n + 1 ) ! ⋅ n ! n 1 / n = ( n + 1 ) 1 / ( n + 1 ) ( n + 1 ) n 1 / n = ( n + 1 ) 1 / ( n + 1 ) ( n + 1 ) ⋅ 1 n 1 / n

      Poiché (n+1)1/(n+1) → 1 e n1/n → 1 per n → ∞, possiamo riscrivere il limite come 1/(n + 1); Poiché il limite è 0, che è minore di 1 la serie originale converge.  ■

    3. ∑ n = 0 ∞ 2 n e 2 n

      si può riscrivere come

      Σ (2/e2)n

      che è una serie geometrica con primo termine a = 1 (il termine corrispondente a n = 0) e ragione q = 2/e2.
      Sappiamo che la serie converge se |q| < 1, e la somma è data data

      S = 1/(1 − r) = 1/(1 − 2/e2) = 1/(e2−2)/e2 = e2/(e2 − 2). ■

    4. ∑ n = 1 ∞ log ⁡ ( n + 2 n + 4 )

      Possiamo riscrivere il termine generale come

      log(n + 2)/(n + 4) = log (n + 2) − log(n + 4)

      La serie diviene quindi

      Σ log (n + 2) − log(n + 4)
