Fourier Series

Our first motivation for the development of the Fourier series is to express a periodic function in a manner which is both concise and easy to manipulate. An important difference between Fourier series and Taylor series is that Fourier series can be used to represent and approximate non-continuous functions as well as continuous ones, whereas Taylor series are applicable only to infinitely differentiable functions. Taylor series are constructed from the expansion of the function around a point<7i>, which means that Taylor series are in fact a local description of a function. Adding more terms toa Taylor series therefore only augments the local accuracy of the function, which is equivalent to noting that a Taylor series converges pointwise to the function of interest for a large enough domain.
The mathematical function that spring imeediately to mind as building blocks for any periodic function are the trigonometric ones sine and cosine. A great advantage of the trigonometric functions is that they are very easy to integrate and to differentiate. Jean Baptiste Joseph Fourier (1768–1830), showed that all periodic functions, could be expressd as a combination of such terms.
If f is periodic function of period 2π, then we attempt to represent f in the form of an infinite series

f ( t ) = a 0 + ∑ n = 1 n ( a k cos ⁡ n t + b k sin ⁡ n t ) (1)

Each term of the series has period 2π, so if the series coverges, its sum will be a function of period 2π.

We define the trigonometric polynomial of order n, as the function:

P n ( x ) := a 0 + ∑ k = 1 n ( a k cos ⁡ k x + b k sin ⁡ k x )

where ak, bk are given real or complex numbers. It is a regular function, infinitely differentiable, with period 2π (or submultiples of 2π).

Eq. 1 has two implications, Fourier synthesis and Fourier analysis; Fourier synthesis is something that you already know about from your study of vibrations and waves: A complicated vibration can be created by adding up simple harmonic motions of various frequncies, i.e. by adding up sine waves. The opposite of synthesis is analysis: one begins with a complex wave and discovers what the sine waves are that make it. Just as a chemist analyzes a material to discover how much of what elements are present, the acoustician analyzes a complex tone to discover how much of what sine frequencies are present.

There are two questions to be answered:

  1. What values must a0, an, bn have?

  2. If the appropriate values are assigned to the coefficients, does the series actually represent the given function f?

On the face of it, the first question is tremendously difficult, for Fourier series represents an equation in an infinite number of unknowns. But a reasonable answer can be found easily by using the orthogonality relations shown in Table 1.

∫ − π π sin ⁡ ( n x ) d x = 0 ∫ − π π cos ⁡ ( n x ) d x = { 0 , n ≠ 0 2 π , n = 0 ∫ − π π sin ⁡ ( n x ) cos ⁡ ( m x ) d x = 0 ∫ − π π sin ⁡ ( n x ) sin ⁡ ( m x ) d x = { 0 , n ≠ m π , n = m ∫ − π π cos ⁡ ( n x ) cos ⁡ ( m x ) d x = { 0 , n ≠ m π , n = m ≠ 0

We may summarize these relations by saying: The definite integral (over the interval −π to π ) of the product of any two different functions from the series in Eq. 1 is zero.
The fundamental idea is that if the equality proposed in Eq.1 is to be a real equality, then both sides must give the same result after the same operation. The orthogonality relations then suggest operations that simplify the righthand side of Eq. (1). Namely, we multiply both sides of the proposed equation by one of the functions that appears there and integrate from −π to π . (We must assume that the integration of the series can be carried out term by term. This is sometimes difficult to justify, but we do it nonetheless.)
Multiplying both sides of Eq. (1) by the constant 1 (= cos(0x)) and integrating from −π to π , we find

∫ − π π f ( x ) d x = ∫ − π π a 0 d x + ∑ n = 1 ∞ ∫ − π π ( a n cos ⁡ ( n x ) + b n sin ⁡ n x ) d x

Each of the terms in the integrated series is zero, so the right-hand side of this equation reduces to 2π · a0 , giving

a 0 = 1 2 π ∫ − π π f ( x ) d x

(In words, a0 is the mean value of f(x) over one period.)
Now multiplying each side of Eq. (1) by sin(mx), where m is a fixed integer, and integrating from −π to π , we find

∫ − π π f ( x ) sin ⁡ ( m x ) d x = ∫ − π π a 0 sin ⁡ ( m x ) d x + ∑ n = 1 ∞ ∫ − π π a n cos ⁡ ( n x ) sin ⁡ ( m x ) d x + ∑ n = 1 ∞ ∫ − π π b n sin ⁡ ( n x ) sin ⁡ ( n x ) sin ⁡ ( m x ) d x

All terms containing a0 or an disappear, according to the orthogonality relations. Furthermore, of all those containing a bn, the only one that is not zero is the one in which n = m. (Notice that n is a summation index and runs through all the integers 1, 2, . . . . We chose m to be a fixed integer, so n = m once.) We now have the formula

b m = 1 π ∫ − π π f ( x ) sin ⁡ ( m x ) d x

By multiplying both sides of Eq. (1) by cos(mx) (m is a fixed integer) and integrating, we also find

a m = 1 π ∫ − π π f ( x ) cos ⁡ ( m x ) d x

When the coefficients are chosen this way, the right-hand side of Eq. (1) is called the Fourier series of f . The a’s and b’s are called Fourier coefficients.
