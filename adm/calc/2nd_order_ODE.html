Second order linear differential equations

A second order differential equation is said linear if it is of the form

a2(t)y′′ + a1(t)y′ + a0(t)y = g(t)  11.5.1

where the coefficients ai and g(t) are functions defined and continous in an interval I. If g is null, the equation is said homogeneous, otherwise non-homogeneous. If the functions ai are constants, the euqation with constant coefficients (g can instead depend on t), otherwise it is said equation with variable coefficients.
The reason why the equation is said linear is the following. If we defnote by Ly(t) the left hand side of equation 11.5.1, we note that the operator

L: C2(I) ⟶ C0(I)
L: y(t) ⟼ Ly(t)

is a linear operator between these space of functions. Indeed, in the hypothesis of continuity of the coefficients, if u ∈ C2(I) we have that Lu ∈ C0(I) ; moreover, if y1,y2 ∈ C2(I) and λ1, λ2 ∈ ℝ, then

L(λ1y1 + λ2y2) = λ1 Ly1 + λ2Ly2

We shall study the importance of the linearity of L, when determining the structure of the general integral of the equation 11.5.1

Consider f(x) = |x|3 in ℝ. Then f'(x) = 3x|x| and f''(x) = 6|x|. Clearly, f''(x) = 6|x| isn't differentiable (at 0).

Example 1.

y'' + 5y = x

is second order, linear, non homogeneous and with constant coefficients.

y' + x2 y = ex

is first order, linear, non homogeneous.

yy'' + y' = 0

is non linear, second order, homogeneous.

Example 2. The simple second order equation is

y''(t) = 0

It is equivalent to y'(t) = c1 and thus y(t) = c1t + c2 with c1 and c2 arbitraty constant. So the only solutions of this equation are all the first degree polynomials . ■

We shall see that the general integral of a second order linear differential equation depends on two arbitrary parameters. To select a particular solutions we must assign two condition of the type:

{ y ( t 0 ) = y 0 y ′ ( t 0 ) = y 1  11.5.2

These two conditions along with equation 11.5.1 represent the Cauchy problem.

Example 3 (A mass on a spring). Consider a mass m attached to the end of a spring, lying on a horizontal surface. Let y(t) be the position of the point on the horizontal surface with respect to the equilibrium position. Then y satisfies Hooke's Law

my'' = −ky

with k ≥ 0, the spring constant. The equation can be rewritten in the form

y'' + ω2y = 0

with ω2 = k/m, and it is known as harmonic oscillator equation. It is a linear second order equations with consant coefficients.
If on the point mass an external force would act (dependent only on time), the equation would be rewritten as

y'' +ω2y = f(t)

(where f represents the force by mass unit), that is it wouldn't be homogeneous anymore.
In the presence of friction proportional to speed, the equation would be

y'' +hy' + ω2y = 0

with h positive constant. All these examples are particular cases of equation 11.5.1. A particular solution is obtained by assigning the initial speed and position of the point mass at an instant t0. For this reason we often refer to conditions 11.5.2 as initial conditions. ■

If the coefficient a2(t) in equation 11.5.1 is never zero, we can divide both sides of the equation by this term, obtaining the canonical (or normal, or standard) form, that has the coefficient of the leading derivative equal to 1

y'' + a(t) y' + b(ty = f(t)

The following result holds for second order linear differential equations:

Theorem 11.5.3. If a,b,f are continuous functions in the interval [αβ], t0 ∈ [αβ], y0,y1 ∈ ℝ, the Cauchy problem

{ y ″ + a ( t ) y ′ + b ( t ) y = f ( t ) y ( t 0 ) = y 0 y ′ ( t 0 ) = y 1

has a unique solution y(t) ∈ C2([α, β].
