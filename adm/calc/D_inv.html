Derivata di una funzione inversa

Teorema 5.7.2 f:(a,b) → ℝ continua e invertibile in (a,b) e f−1 la sua inversa, definita in f(a,b). Supponiamo inoltre che esista f'(x0) ≠ 0 per un certo x0 ∈ (a,b).
Allora f−1 è derivabile in y0 = f(x0) e

( f − 1 ) ′ ( y 0 ) = 1 f ′ ( x 0 )  (5.7.3)

Dim. Poniamo g = f−1 e sia

f(x0) = y0;   g(y0) = x0;
f(x0) = y0 + k;   g(y0 + k) = x0 + h

Consideriamo il rapporto incrementale di g in y0:

g ( y 0 + k ) − g ( y 0 ) k = h f ( x 0 + k ) − f ( x 0 )

Se k ≠ 0, f(x0 + h) − f(x0) ≠ 0 e quindi anche h ≠ 0; dunque l'ultimo quoziente si può riscrivere anche nella forma

1 f ( x 0 + k ) − f ( x 0 ) h

Inoltre, per k → 0 si ha g(y0 + k) → g(y0) perché g è continua, essendo l'inversa di una funzione continua su un intervallo (v. Teorema 4.10.1); d'altro canto h = g (y0 + k) − g(y0), quindi per k → 0 anche h → 0, e per ipotesi

1 f ( x 0 + k ) − f ( x 0 ) h → 1 f ′ ( x 0 )

quindi esiste

lim k → 0 g ( y 0 + k ) − g ( y 0 ) k = 1 f ′ ( x 0 )

Osserviamo che, assumendo la derivabilità di f−1, la (5.7.3) segue subito dall'identità g(f(x)) = x e dalla regola della catena:

g'(f(x)) ⋅ f'(x) = 1

da cui, se f'(x) ≠ 0, la (5.7.3). La dimostrazione che abbiamo dato, tuttavia, è necessaria per dedurre la derivabilità di g dalle nostre ipotesi su f.  □

La (5.7.3) ha un semplice significato geometrico, ricordando che i grafici di f e g = f−1 sono simmetrici rispetto alla bisettrice y = x.

Con la notazione di Leibniz, posto y = f(x), x = g(y), la (5.7.3) si scrive nella forma

dx/dy = 1/dy/dx

Si faccia attenzione al fatto che nella formula di derivazione della funzione inversa, le derivate f' e g' sono calcolate in due punti diversi: è questa la principale attenzione da avere nell'applicazione di questo teorema, come mostreranno i prossimi esempi.

Esempio 5.7.4 Derivata dell'arcotangente, dell'arcosenso e dell'arcocoseno.
Poniamo y = tg x, x = arctg y con x ∈ (−π/2, π/2), y ∈ ℝ

d x d y = 1 d y d x = 1 1 + ( tan ⁡ x ) 2 = 1 1 + y 2

Poniamo ora y = sin x, x = arcsin y, con x ∈ [−π/2, π/2], y ∈ [−1, 1]. Poiché per quei valori di x si ha cos x = sqrt[1 − (sin x)2]:

d x d y = 1 cos ⁡ x = 1 1 − y 2

Analogamente, se y = cos x, x = arcosin y con x ∈ [0, π], y ∈ [−1,1],

d x d y = 1 − sin ⁡ x = 1 − 1 − y 2

Si osservi che le funzioni arcsin x, arccos x, pur essendo definite e continue in [-1,1], non sono derivabili agli estremi dell'intervallo: precisamente, presentano in questi punti tangente verticale.  ■

L'utilità del teorema di derivazione della funzione inversa consiste nel fatto che permette di calcolare la derivata di g anche in situazioni in cui g non si sa scrivere esplicitamente:

Esempio 5.7.5. Sia f(x) = x + ex. La funzione è strettamente crescente in tutto ℝ, dunque invertibile; sia g la sua inversa. Calcoliamo, ad es., g'(y0) per y0 = f(0) = 1. Si ha:

f'(x) = 1 + ex; f'(0) = 2 ≠ 0; g'(1) = 1/f'(0) = 1/2.  ■
