Radici

Teorema 2.6. Sia y ∈ ℝ, y > 0 e n un intero ≥ 1. Esiste un unico numero reale positivo x tale che xn = y.

Dim. Una semplice dimostrazione di questo teorema potrà essere data più avanti, utilizzando le proprietà delle funzioni continue.

Tale numero si chiama radice n-esima aritmetica di y e si indica con y1/n. Si noti che la radice n-esima aritmetica è non negativa, per es. 41/2 = 2, 91/2 = 3 e più generale √x2 = |x|.

Vediamo come si può costruire la rappresentazione decimale della radice n-esima. Cerchiamo l'allineamento decimale di √2; questo numero, non essendo razionale, sarà rappresentato da un allineamento infinito (non periodico). Si procede così: si costruisce una classe di numeri razionali della forma

0 < a0,
a0, a1
a0, a1a2
a0, a1a2a3
.... a0, a1a2a3an

con la regola che: ognuno di questi numeri è il più grande tra quelli con lo stesso numero di decimali dopo la virgola il cui quadrato è minore di 2:

1   12   22 = 4
1.4   (1.4)2 = 1.96   (1.5)2 = 2.25
1.41   (1.41)2 = 1.9881   (1.42)2 = 2.0164
..

Questo insieme di numeri (chiamiamolo E-) è limitato superiormente (ognuno è < 2); per la proprietà R4 esso possiede estremo superiore; il numero √2 si definisce precisamente come E-.
Si sarebbe potuto anche costruire una classe di numeri E+ come la precedente con la regola che ognu di essi sia il più piccolo tra quelli con lo stesso numero di decimali dopo la virgola il cui quadrato è > 2; abremmo ottenuto

2   1.5   1.42 ...

Questo insieme E+ è limitato inferiormente (ogni element è > 1), perciò possiede estremo inferiore; si dimostra che inf E+ = sup E-. I numeri della classe E- approssimano √2 per difetto, quelli della classe E+ per eccesso.

Potenze

Il teorema 2.6 permette di definire la radice aritmetica n-esima di un qualunque numero reale positivo a come l'unica soluzione reale positiva dell'equazione xn = a. Più in generale, la potenza di base a ∈ ℝ, a > 0, ed esponente razionale m/n ∈ ℚ, con m,n ∈ ℤ, n ≥ 1, MCD(m,n) = 1, è il numero reale definito da

a m / n = ( a m ) 1 / n = a m n

A determinate condizioni è possibile definire una potenza, anche quando la base è negativa se l'esponente è intero (am con m ∈ ℤ), oppure se l'esponente è razinale, m/n, purché n non sia pari. Infatti, ad es., la seguente potenza esiste

( − 3 ) 2 / 5 = ( − 3 ) 2 5 = 9 5

mentre, la potenza

( − 3 ) 3 / 2 = ( − 3 ) 3 3 = − 27

Osservazione. Una potenza è un'esprezzione xα in cui la base è variabile e l'esponente è fisso, mentre un'esponenziale è un'espressione ax in cui la base è fissa e l'esponente è variabile. Le loro proprietà principali sono:

a0  ∀ a ≠ 0;
1c = 1, ∀c
ac >0 ∀c,
ac ≤ 1 se a ≤ 1 e c > 0
ac+d = ac ⋅ ad
(ab)c = ac ⋅ bc
(ab)c = abc
se c < d  ⇒  ac ≷ ad se a ≷ 1
0 < a ≤ b  ⇒  ac ≥ bc, ∀c >0
