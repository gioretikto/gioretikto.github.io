Il teorema di de l'Hôpital

Una notevole applicazione del calcolo differenziale si ha nel calcolo dei limiti che si presentano nelle forme di indecisione 0/0 and ∞/∞.
L'Hospital's Rule is named after a French nobleman, the Marquis de l'Hospital (1661–1704) a student of Johann Bernoulli, a marquis for whom the latter wrote the first textbook of analysis in the years 1691-1692. The portion of this textbook devoted to differential calculus was published in slightly altered form by l'Hôpital under his own name in the "Analyse des infiniments petits" in 1696, the first text on differential calculus. Thus "l'Hôpital's rule" is really due to Johann Bernoulli. L'Hôpital actually considered only the following special case in which two functions f, g are coninuos on [a,b], differentiable on (a,b) and both infinitesimal for x ⟶ a and showed that

Teorema 5.12.1 (Di de l'Hôpital) Siano f,g: (a,b) \ {x0} → ℝ dove x0 ∈ (a,b). Supponiamo che

  1. lim x → x 0 f ( x ) = 0 , lim x → x 0 g ( x ) = 0

  2. f e g siano derivabili ∀x ∈ (a,b), x ≠ x0.

  3. g'(x) ≠ 0 ∀x ∈ (a,b), x ≠ x0;

  4. esista, finito o ±∞, lim x → x 0 f ′ ( x ) g ′ ( x ) = ℓ

Allora esiste anche il limite lim x → x 0 f ( x ) g ( x ) ed è uguale a ℓ.

La medesima conclusione vale anche se l'ipotesi (a) è sostituita da

(a') limx → x0 g(x) = ±∞.

Dim.. Grazie all'ipotesi (a) le funzioni f e g possono essere estese con continuità anche nel punto x0 ponendo

f(x0) = 0, g(x0) = 0

Supponiamo inizialmente che il limite in d) sia finito, cioè ℓ ∈ ℝ. Da d) e dalla definizione di limite si ha che, preso ad arbitrio ε, esiste un numero δ > 0 tale che

| f ′ ( t ) g ′ ( t ) − ℓ | < ϵ , ∀ t ∈ ( x 0 − δ , x 0 + δ ) , t ≠ x 0  (5.12.2)

Sia ora x ∈ (x0 − δ, x0 + δ), x ≠ x0. Assumiamo ad es. che x0 < x < x0 + δ (il caso x0 − δ < x &/lt; x0 si tratta in maniera analoga). Applichiamo il T. di Cauchy alle funzioni f e g nell'intervallo [x0, x]: esiste c ∈ (x0, x) tale che

f ′ ( c ) g ′ ( c ) = f ( x ) − f ( x 0 ) g ( x ) − g ( x 0 ) = f ( x ) g ( x )

Nell'ultimo passaggio abbiamo usato il fatto che f(x0) = g(x0) = 0; si osservi inoltre che come conseguenza del T. di Cauchy, sia che g(x) ≠ x0 quando x ≠ x0. Poiché c ∈ (x0, x) ⊂ (x0, x0 + δ), da (5.12.2) con t = x otteniamo che

| f ( x ) g ( x ) − ℓ | = | f ′ ( c ) g ′ ( c ) − ℓ | < ϵ

Poiché questa relazione vale ∀x ∈ (x0 − δ, x0 + δ), x ≠ x0, dall'arbitrarietà di ε segue la tesi.

ii) Consideriamo ora il caso di ℓ = +∞ (il caso ℓ = − ∞ si tratta in maniera analoga). Da d) e dalla definzione di limite si ha che, fissato ad arbitrio M > 0 esiste un numero δ > 0 tale che

f ( x ) g ( x ) = f ′ ( c ) g ′ ( c ) > M

La medesima disuguaglianza si ha anche quando x ∈ (x0 − δ, x0), quindi per l'arbitrarietà di M possiamo concludere che limx → x0 f(x)/g(x) = +∞.  □

Facciamo qualche osservazione

  1. Il T. rimane valido anche se f e g sono definite su (x0, b) e i limiti sono sostituiti dal limite destro in x0, oppure se f e g sono definite su (a, x0) e i limiti sono sostituiti dal limite sinistro in x0;

    f = { cos ⁡ 2 x − cos ⁡ x x 2 , if  x ∈ ( 0 , ∞ ) − 2 , if  x ∉ ( 0 , ∞ ) .

    Applicando il T. di de L’Hôpital

    limx ⟶ 0+ f(x) = limx ⟶ 0+(−2sen 2x + sen x)/(2x)

    Applicando, una seconda volta il T. di de L’Hôpital

    limx ⟶ 0+ f(x) = limx ⟶ 0+(−4cos 2x + cos x)/(2) = −3/2. ■

  2. il T. rimane valido anche per limiti all'infinito, cioè se si considerano limiti per x → ± ∞ anziché per x → x0;

  3. Il T. fornisce solo una condizione sufficiente per l'esistenza del limite limx → x0 f(x)/g(x). In altri termini, quest'ultimo limite può esiste anche se non esiste il limite limx → x0 f'(x)/g'(x). Si considerino ad es. le funzioni f(x) = x2, g(x) = x, con x0 = 0. Allora

    lim x → 0 f ( x ) g ( x ) = lim x → 0 x sin ⁡ 1 x = 0

    D'altra parte, se x ≠ 0,

    f ′ ( x ) g ′ ( x ) = 2 x sin ⁡ 1 x − cos ⁡ 1 x

    e questa funzione non ammette limite per x → 0 (il primo termine tende a zero, mentre non esiste il limite del secondo).

  4. Il T. si usa per quozienti, non per prodotti!. Tuttaiva può essere usato anche per trattare forme di indeterminazione del tipo 0 ⋅ ∞. Si voglia dimostrare, ad es., che

    lim x → 0 + x log ⁡ x = 0

    Qui si presenta una forma di indeterminazione del tipo 0 ⋅ (−∞). Tuttavia, se questo limite viene riscritto come

    lim x → 0 + log ⁡ x ( 1 / x )

    adesso la forma di indeterminazione è del tipo ∞/∞. È facile verificare che le ipotesi a'), b), c) sono verificate nella semiretta (0,+∞) con x0 = 0. Rimande da vedere se è valida anche l'ipotesi d), cioè se esiste il limite del rapporto delle derivate delle funzioni log x e 1/x:

    lim x → 0 + 1 / x ( − 1 / x 2 ) = lim x → 0 + ( − x ) = 0

    Poiché questo limite esiste e vale 0, possiamo concludere che anche il limite iniziale esisa e valga 0.

  5. Possono essere trattae anche altre forme di indeterminazione. Ad es., si voglia dimostrare che

    lim x → 0 + x x = 1

    In questo caso, la forma di indetermianzione è del tipo 00. Tuttavia, passando in forma esponenziale e utilizzando il limite già calcolato nel passaggio precedente, si ha

    lim x → 0 + x x = e x log ⁡ x = e 0 = 1

  6. Il T. si usa per quozienti che siano effettive forme di indeterminazione!. Un errore assai frequente consiste nell'utilizzare il T. quando non si è in presenza di una forma di indeterminazione come ad es.

    lim x → 0 x 2 + sin ⁡ x = 0

    in quanto il numeratore tende a 0 mentre il denominatore tende a 2; tuttavia, se si applica il T. di de l'Hôpital, il limite del rapporto delle derivate vale

    lim x → 0 1 cos ⁡ x = 1

    che non coincide col limite precedente!

  7. Non è detto che utilizzsare il T. di de l'Hôpital sia il mezzo più veloce per calcolare un limite. Si consideri ad es. il limite

    lim x → 0 e x 7 cos ⁡ ( x 2 − 2 x ) sin ⁡ x x log ⁡ ( e + 3 x cos ⁡ x

    Si può verificare facilmente che questo limite si presenta in forma indeterminata del tipo 0/0. È però evidente che cercare di calcolare il limite delle derivate (o anche solo calcolare le derivate) è un'operazione molto laboriosa. D'altra parte questo limite si calcola facilmente osservando che, quan do x → 0.

    e x 7 → 1 , cos ⁡ ( x 2 − 2 x ) → 1 , sin ⁡ x x → 1 , log ⁡ ( e + 3 x cos ⁡ x ) → log ⁡ e = 1

    Quindi per il T. 4.3.7 si ha

    lim x → 0 e x 7 cos ⁡ ( x 2 − 2 x ) sin ⁡ x x log ⁡ ( e + 3 x cos ⁡ x = 1

Un'altra applicazione tipica del T. di de L'Hopital è il calcolo di limiti che coinvolgono formule di indeterminazione non risuolubili mediante i limiti notevoli (ossia, al primo ordine).

Esempio 5.12.4 Si voglia calcolare

lim x → ∞ x − sin ⁡ x x + sin ⁡ x = [ + ∞ + ∞ ]

Il valore del limite si può stabilire immediatamente:

x − sin ⁡ x x + sin ⁡ x ∼ x x = 1

Supponiamo di volerlo calcolare col teorema di de L'Hospital:

lim x → + ∞ ( x − sin ⁡ x ) ′ ( x + sin ⁡ x ) ′ = lim x → + ∞ = 1 − cos ⁡ x 1 + cos ⁡ x

che non esiste. Se concludessimo che il limite di partenza non esiste, diremmo il falso (quel limite vale 1). Il punto è che se limx → x0 non esiste (né finito né infinito), cade una delle ipotesi del teorema, e in base ad esso non si può semplicemente concludere nulla. ■
