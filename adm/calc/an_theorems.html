Fundamental theorems on sequences

Trattereo ora i teoremi basilari sul calcolo dei limiti. Le dimostrazioni si basano sulla deifnizione di limite, sull'uso di disuguaglianze, e sull'uso di proprietà definitivamente vere. In particolare, questi toeremi illustrano la relazione tra l'operazione di limite e le strutture algebriche e d'ordine presenti in ℝ.
Cominciamo ad esaminare le proprietà dell'operazione di limite rispetto alle operazioni algebriche. Quando il limite esiste finito, si dimostra un risultato semplice e naturale: l'operazione di limite commuta con queste operazioni, cioè:

Teorema 3.3.1 (Algebra dei limiti) Se an → a e bn → b allora

  1. an ± bn → a ± b

  2. anbn → ab

  3. an/bn → a/b (bn,b≠ 0)

  4. anbn → ab

Dim.

  1. Proviamo ad es. che:

    an → a,  bn → b  ⇒  an + bn → a + b

    Fissiamo ε > 0 e consideriamo:

    |(an + bn)| − (a + b)| = |(an − a) + (bn + b)| ≤ (an − a)| + |(bn − b)|

    (per la disuguaglianza triangolare). Poiché per ipotesi an → a e bn → b, si ha che

    |an − a| < ε e |bn − b| < ε definitivamente

    perciò concludiamo che

    |(an + bn)| − (a + b)| < 2ε definitivamente,

    quindi an + bn → a + b.

  2. Fissiamo ε >0. Usando ancora la disuguaglanza triangolare si ha:

    |anbn| = |an(bn − b) + b (an − a)| ≤ |an (bn − b)| + |bn(an − a)| = |an| |bn − b| + |b||an − a| ≤ |an| |bn − b| + |b||an − a|

    Poiché an ⟶ a, |an − a| < ε definitivamente; inoltre |an| < |a| + ε definitivamente; poiché bn ⟶ b, |bn −b| < ε definitivamente. Quindi:

    |anbn − ab| < (|a| + ε)ε + |b|ε < ε ⋅ cost

    definitivamente. Per l'arbitrarietà di ε segue la tesi.

Teorema 3.3.2. (Permanenza del segno) Se an → a e a > 0 (a < 0) allora asn > 0, definitivamente (ann < 0 definitivamente).

Dim. Fissato ε > 0, per definizione di limite abbiamo che

|an − a| < ε definitivamente

che riscriviamo nella forma:

a − ε < an < a + ε definitivamente

Poiché a > 0, possiamo scegliere ε >0 in modo che sia anche a − ε > 0, allora la disuguaglianza a − ε < an mostra che an > 0 definitivamente. In modo analogo si dimostra il caso a < 0.  □

Corollario 3.3.3. Se an → a e an ≥ 0 definitivamente, allora a ≥ 0.

Dim. Per assurdo, se a < a allora dal Teorema della permanenza del segno avremo che esiste ν ∈ ℕ tale che an < 0 per ogni n ≤ ν, in contraddizione con an ≥ 0 ∀n ∈ ℕ.  □

Corollario 3.3.4. Se an → a, bn → b e an ≥ bn definitivamente, allora a ≥ b.

Dim. Basta applicare il precedente corollario alla successione cn = an − bn.  □

Teorema 3.3.5 (Del confronto) Se an ≤ bn ≤ cn definitivamente e

an → ℓ, cn → ℓ ∈ ℝ

allora anche bn → ℓ.

Dim.. Fissiamo ε > 0. Allora definitivamente si ha

ℓ − ε < an < ℓ + ε; ℓ − ε; < cn < ℓ + ε;

da cui segue (definitivamente)

ℓ − ε < an ≤ bn ≤ cn < ℓ + ε;

e quindi, definitivamente

ℓ − < bn < ε < ℓ + ε

Dunque bn →ℓ.  □

Theorem 3.3.6 (Uniqueness of the limit). If {an} is a convergent sequence with limit ℓ, there are no other limits.

Proof. Suppose there exist two limits ℓ1 and ℓ2 for the same sequence. By using triangular inequality:

|ℓ1 − ℓ2| = |ℓ1 − an + an − ℓ2| ≤ |ℓ1 − an| + |an − ℓ2| < 2ε

Since ε is a real positive number (can be chosen to be arbitrarily small), this inequality can only hold if ℓ1 = ℓ2.  □

Il precedente risultato viene anche detto Teorema dei due carabinieri in quanto le successioni {an} e {cn} possono essere paragonate a due carabinieri che accompagnano il delinquente {bn} in galera.

Corollario

  1. Se |bn| ≤ cn definitivamente e cn → 0, allora anche bn → 0;

  2. Se cn → 0 e bn (ma non necessariamente convergente), allora cnbn → 0. Detto a parole: il prodotto di una successione infinitesima e una limitata è infinitesima.

Dim.

  1. Sappiamo che definitivamente è −cn ≤ bn ≤ cn; d'altro canto se cn → 0 anche −cn → 0, quindi per il toerema precedente (con an = −cn e ℓ = 0) si ha che bn → 0.

  2. Se bn è limitata, ossia |bn| ≤ K per un certo K > 0 e per ogni n, possiamo scrivere

    |bncn| ≤ K|cn|

    Poiché cn → 0, anche K|cn| → 0, e per il punto 1 si conclude che bncn → 0. □

Example 3.3.7. To illustrate the squeeze theorem, we find the limit of the following sequence (sin n)/n as n → ∞. Recall that the sine function has range [−1, +1] so

−1 ≤ sin n ≤ 1

for every value of n; when we divide these inequalities by n, we obtain

−1/n ≤ (sin n)/n ≤ 1/n

The flanking sequences are −1∕n and 1∕n, and both of them converge to 0 as n → ∞. Therefore, {an} converges to 0 too.  ■

Esempio. Limiti che si presentano nella forma di rapporto di due espressioni, ognuna costituita dalla somma di potenze di n:

n 5 / 2 − 3 n + 7 n 3 + n − 3 n 2

Si mette in evidenza a numeratore come a denominatore la potenza maggiore

n 5 / 2 ( 1 − 3 n 3 / 2 + 7 n 5 / 2 ) n 3 ( 1 + 1 n 5 / 2 − 3 n )

Ora per il teorema sull'algebra dei limiti e sapendo che potenze negative di n tendono a zero possiamo affermare che:

1 − 3 n 3 / 2 + 7 n 5 / 2 → 1 ; 1 + 1 n 5 / 2 − 3 n → 1

Pertanto la successioni tra parantesi tende a 1; d'altro canto 1/√n → 0, perciò la successione di partenza tende a zero. ■

Esempio 3.3.8. La successione sin n/n è il prodotto di 1/n (convergente) e sin n (irregolare); quindi il teorema sull'algebra dei limiti non è applicabile. Tuttavia è applicabile l'ultimo corollario: la successione 1/n è infinitesima mentre |sin n| ≤ 1, perciò sin n/n → 0. ■

Fin qui abbiamo visto teoremi che operano su coppie di successioni entrambe convergenti o comunque limitate. Consideriamo ora il caso in cui i limiti sono +∞ o −∞. Supponiamo, per es., che an → a e bn → +∞; allora è facile (e intuitivo) vedere che an + bn → +∞. Abbrevieremo questa scrittura così: a + ∞ = ∞. Ragionando in maniera analoga possiamo compendiare le regole per il limite della somma (o differenza) di due successioni delle quali una o entrambe sono divergenti con le scritture seguenti:

a + ∞ = ∞
a − ∞ = −∞
∞ + ∞ = ∞
−∞ − ∞ = −∞

Analogamente per il prodotto (o il rapporto) abbiamo le regole seguenti; il segno di ∞ va determinato con la ususale regolae dei segni:

a ⋅ ∞ = ∞
a/0 = ∞
a/∞ = 0

Esplicitamente, la seconda regola scritta significa ad es. quanto segue:

an → a > 0 e bn → 0+, allora an/bn → ∞

In modo analogo si procede, in base alla regola dei segni, se a < 0 e bn → 0-; è coomunque necessario, per applicare questo teorema, sapere che bn tende a zero per eccesso o per difetto.

Le precedenti regole prendono il nome di "aritmetizzazione parziale del simboli di infinito".

Dim.. Proviamo ad es. che:

an → ∞, bn → b ∈ ℝ, ⇒ bn/an → 0

Fissiamo ε. Poiché an → +∞, definitivamente si ha

an > 1/ε

Poiché bn → b, definitivamente si ha

|bn| < |b| + ε

Ne segue definitivamente

|bn/an| < ε(|b| + ε) < ε ⋅ cost

Per l'arbitrarietà di ε, segue la tesi. □

Si sarà notata l'assenza delle regole relative a quattro operazioni, e cioè:

+∞ −∞,   0 ⋅ ∞, 0/0, ∞/∞

Queste espressioni si chiamano forme di indecisione, poiché nessuna regola può essere stabilita a priori per determinare il risultato, come vedremo negli esempi sotto illustrati.
Le regole sopra elencate (e la mancanza di regole per le forme di indeterminazione) confermano la natura particolare dei "punti" +∞ e −∞, che non possono essere considerati "numeri" poiché non rispettano le proprietà delle operazioni dell'anello.

Esempio 3.3.9. sqrt(n + 1) − sqrt(n − 1): forma di indecisione del tipo +∞ e −∞.
Moltiplicando e dividendo per sqrt(n + 1) + sqrt(n − 1) si trova

… = ( n + 1 − n − 1 ) ( n + 1 + n − 1 ) n + 1 + n − 1 = 2 n + 1 + n − 1 → 0

Esempio 3.3.10. Limiti di successioni che si presentano nella forma anbn si trattano più facilmente considerando la successione dei logaritmi (per fissare le idee, in base 10)

log10 anbn = bn log10 an

Si mostra che se questa successione converge a ℓ, diverge a +∞, −∞, o è indeterminata, la successione anbn, rispettivamente, converge a 10ℓ, +∞, 0, è indeterminata.
Si può così osservare che le espressioni

1±∞   o0   (+∞)0

sono altrettante forme di indecisione, corrispondenti (passando al logaritmo) alla forma di indecisione 0 ⋅ ∞.
