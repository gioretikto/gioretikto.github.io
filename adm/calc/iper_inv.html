Le funzioni iperboliche inverse

La funzione y = Sh x è definita e strettamente crescente su tutto ℝ, dunqe invertibile. Per trovare la funzione inversa poniamo

x = Sh y = (ey − e−y)/2

e risolviamo rispetto ad y. Moltiplicando ambo i membri per ey l'equazione si può riscrivere come:

e2y − 2xey − 1 = 0

che è un'equazione di 2° nell'incognita ey. Ricaviamo

e y = x ± − x 2 + 1

Essendo ey >0, scartiamo la soluzione negativa. Quella positiva è

y = log ⁡ ( x + x 2 + 1 )

Questa è la funzione inversa di Sh x, nota come settore seno iperbolico, SettSh x, definita per ogni x reale.
Con passaggi analoghi si ricava la funzione inversa del coseno iperbolico Ch x, strettamente crescente per x ≥0, descrescente per x ≤0. Non è quindi invertibile su tutto ℝ, ma la sua restrizione ax ≥ 0, lo è. Vogliamo determinare l'espressione analitica della funzione inversa di questa restrizione.
Con calcoli analoghi ai precedenti, scriviamo

x = Ch y = (ey − e−y)/2

risolvendo rispetto a y:

ey − 2xey + 1 = 0

e y = x ± x 2 − 1

In questo caso entrambe le soluzioni x = ± sqrt(x2 −1) sono positivi; ricordiamo però che stiamo ragionando solo per y ≥ 0, che equivale a scegliere il segno +. Pertanto passando al logaritmo:

y = log ⁡ ( x + x 2 − 1 )

che è nota come settore coseno iperbolico, SettCh x (anche arsinh, for area sinus hyperbolicus), è definita per x &ge ;1. Si osservi che, mentre ad es., l'equazione

Sh x = 3  unica soluzione   x = SettSh 3 = log 3 + √10

l'equazione

Ch x = 3  due soluzioni   x = ±SettCh 3 = log 3 + √8

Le funzioni iverse del seno e coseno iperbolico risultano utili nell'integrazione delle funzioni irrazionali.

Example 1. Let f(x)= sin(x) + cos(x). Then

f(x + 2π) = sin(x + 2π) + cos(x + 2π) = sin(x) + cos(x) = f(x)

Le funzioni iperboliche inverse saranno utilizzate nell'integrazione delle funzioni irrazionali.
