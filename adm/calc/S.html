Sommatorie e coefficienti binomiali

Definizione 1.2.1 (Il simbolo di sommatoria) Siano a1, a2, ..., an, n numeri reali. La loro somma

a1 + a2 + ... + an

si può indicare in forma compatta col simbolo di sommatoria:

∑ i = 1 n a i

che si legge: "sommatoria per i da 1 n di ai". Il simbolo i si dice indice di sommatoria.  □

Il simbolo di sommatoria è dunque una pura e semplice stenografia, che tuttavia risulta molto utile quando i termini ai sono definiti esplicitamente in funzione dell'indice i, ad es:

∑ i = 1 10 1 i = 1 + 1 2 + 1 3 + … + + 1 10 progressione geometrica per cui il primo termine vale 2 e la ragione vale ¾. ∑ i = 1 n i 2 = 3 2 + 4 2 + 5 2 + … + n 2

Per usare agilmente il simbolo di sommatoria occorre capire bene l'utilizzo dell'indice di sommatoria. Anzitutto, l'indice di sommatoria è un indice muto. Questo vuol dire che se si sostituisce i con j, con k o qualunque altro indice (in tutte le sue occorrenze) il senso dell'espressioen non cambia:

∑ i = 1 n i 2 = ∑ i = 1 n j 2

Invece,

∑ i = 1 n i 2 ≠ ∑ i = 1 m i 2

in quanto i due simboli indicano la somma, rispettivamente, dei primi n oppure dei primi m quadrati: se n ≠ m il risultato sarà diverso.
Le seguenti proprietà formali delle sommatorie sono facilmente comprendibvili se si pena a ciò che affermano in termini di somme scritte per esteso.

Proposizione 1.2.2 (Proprietà formali delle sommatorie).

  1. Prodotto per una costante (proprietà distributiva):

    ∑ k = 1 n ( c ⋅ a k ) = c ⋅ ∑ k = 1 n a k

  2. Sommatoria con termine costante:

    ∑ k = 1 n c = c ⋅ n = c ⋅ ( numero di addendi della sommatoria )
  3. Somma di sommatorie:

    ∑ k = 1 n a k + ∑ k = 1 n b k = ∑ k = 1 n ( a k + b k )

  4. Scomposizione:

    ∑ k = 1 n + m a k = ∑ k = 1 n a k + ∑ k = n + 1 n + m a k

  5. Traslazione di indici:

    ∑ k = 1 n a k = ∑ k = 1 + m n + m a k − m

  6. Riflessione di indici:

    ∑ k = 1 n a k = ∑ k = 1 n a n − k + 1 = ∑ k = 0 n − 1 a n − k

La dimostrazione di queste proprietà è un esercizio di trascrizione: basta, cioè, scrivere per esteso che cosa indcia ciascuna sommatoria ed eseguire qualche passaggio elementare.

Progressione geometriche

Una progressione geometrica è una successione di numeri reali tali che il rapporto tra due termini consecutivi della successione è costante.

a, aq, aq2, aq3, ..., aqn

dove a è detto primo termine, e q la costante detta ragione della progressione: se la ragione è positiva e maggiore di 1, la successione è crescente; se la ragione è compresa tra zero e 1 (1 escluso), la succesione è decrescente; se la ragione è uguale a 1 la successione è costante (tutti i suoi termini sono uguali); se la ragione è negativa la successione è oscillante (i suoi termini sono alternativamente positivi e negativi).
Simbolicamente, indicando con an il termine n-esimo della successione e con q la sua ragione, possiamo scrivere:

a1
a2 = a1q
a3 = a2q = a1qq = a1q2
a4 = a3q = a1q2q = a1q3
...
an = an−1 q = a1qn-2q = a1qn-1

dove l’ultima espressione:

an = a1qn-1 (1.2.3)

fornisce una relazione generale per calcolare il termine n-esimo, an di una progressione geometrica di cui si conosce il primo termine a1 e la ragione q.

Ad esempio

Progressione Ragione
1,3,9,27,81, ... 3
16/27, −8/9, 4/3, −2, ... −3/2
x, x2, x3, x4 ... x

Esempio 1.2.4 Calcolare il quinto termine di una progressione geometrica per cui il primo termine vale 2 e la ragione vale ¾.

Applicando la formula precedente, abbiamo subito:

a5 = a1q4 = 2 x (3/4) = 2 x 81/256 = 81/128 ■

Esempio 1.2.5 Calcoliamo l’interesse composto di un capitale C0 depositato in banca per n anni con un interesse percentuale annuo uguale a i.
Alla fine di ogni anno abbiamo la seguente situazione:

Anno Capitale
1 C0 + i C0 = C0(1+i) = C1
2 C1 + i C1 = C0(1+i) + iC0(1+i) = C0(1+i)(1+i) = C0(1+i)2 = C2
3 C2 + iC2 = C0(1+i)2 + iC0(1+i)2 = C0(1+i)2 (1+i) = C0(1+i)3 = C3
... ...
n C0(1+i)n = Cn

Possiamo allora notare che

  • la successione:

    C0, C1, C2, C3,..., Cn

    è una progressione geometrica il cui primo termine è C0 e la cui ragione è 1 + i;

  • il capitale Cn alla fine dell’n-simo anno di deposito è dato da

    Cn = C0 (1 + i)n

    Allora se C0 = 1 ML (un milione di lire) e i = 8%, dopo 10 anni il capitale sarà uguale a:

    C 10 = 1 × ( 1 + 8 100 ) 10 ML = ( 108 100 ) 10 ≈ 2.158 ML

    cioè quasi raddoppiato (ma certamente svalutato!). ■

La relazione (1.2.3) può essere utilizzata per calcolare uno qualunque degli elementi presenti a partire dagli altri. Così possiamo scrivere anche le relazioni:

a1 = an/qn − 1 (1.2.6)

q = a n a 1 n − 1  (1.2.4)

n = logq(an/a1) + 1 (1.2.7)

Esempio 1.2.5 Calcolare la ragione di una progressione geometrica di cui si conosce il primo termine uguale a 4 ed il quinto termine uguale a 128.

Applicando la formula (1.2.4) precedente, abbiamo subito:

q = (128/4)1/4 = 321/4 = 2(2)1/4. ■

Esempio 1.2.6 Di una progressione geometrica si sa che il primo termine vale 72, che il suo termine n-simo vale 1 e che la ragione vale 8/27. Calcolare n.

Applicando la formula (1.2.4) precedente, abbiamo subito:

n = log 1 / 3 ⁡ ( 72 8 27 ) + 1 = log 1 / 3 ⁡ ( 243 ) + 1 = log 1 / 3 ⁡ ( 1 / 5 ) − 5 + 1 = 5 + 1 = 6

Somma di una progressione geometrica

Calcoliamo la somma Sn dei primi n termini di una progressione geometrica. Possiamo scrivere:

Sn = a1 + a1q + a1q2 + ... + a1qn − 1
qSn = a1q + a1q2 + a1q3 + ... + a1qn − 1 + a1qn
e, sottraendo membro a membro:
Sn = qSn = a1 − a1qn ⇒ Sn (1 − q) = a1 (1 − qn) ⇒
infine,

S n = a 1 1 − q n 1 − q

Usiamo le proprietà del simbolo di sommatoria per provare il seguente caso particolare della precedente:

Proposizione 1.2.7 Per la somma dei primi termini della progressione geometrica di ragione q ≠ 1 (e a = 1), vale la formula:

∑ k = 0 n q k = 1 − q n + 1 1 − q  (1.2.4)

Se q = 1 la sommatoria scritta vale n + 1, ovviamente.
