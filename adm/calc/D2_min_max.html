Studio del massimo e del minimo di una funzione medinate la derivata seconda

Sia f(x) una funzione la cui derivata si annulla nel punto x0 cioè f'(x0) = 0. Supponiamo inoltre che la derivata seconda f''(x) esista e sia continua in un intorno del punto x0.
Possiamo allora enunciare il seguente teorema.

Torema 5.17.1. Sia f'(x0) = 0; allora la funzione ha un massimo nel punto x = x0 se f''(x0) < 0 e un minimo se f'(x0) > 0.

Dim. Dimostraimo innanzitutto la prima parte del teorema. Siano

f'(x0) = 0  e  f''(x0) < 0

Essendo f''(x) per ipotesi continua in un certo intorno del punto x = x0, si può evidentemente trovare un intorno abbastanza piccolo contentente il punto x0, in ogni punto del quale la derivata seconda f''(x) è negativa.
Ma f''(x) è la derivata prima di f'(x), quindi dalla condizione f''(x) < 0 segue che la funzione f'(x) è decrescemnte nell'intorno contenente il punto x = x0. Siccome f'(x0) = 0, abbiamo in questo intorno f'(x) > 0 per x < x0 e f'(x) < 0 per x > x0, cioè la derivata f'(x) cambia il segno da più a meno passando per il punto x0. Ciò significa precisamente che la funzione f(x) ha un massimo nel punto x0. In modo analogo si dimostra il caso del minimo. □

Se nel punto critrico f(x0) = 0, la funzione può ammettere in questo punto un massimo o un minimo oppure non avervi nessun estremo. In tal caso lo studio della funzione deve essere condotto col primo metodo.
Lo studio degli estremi mediante la derivata seconda può essere schematizzato nella tabella seguente:

f'(x0) f''(x0) Natura del punto critico
0 − Massimo
0 + Minimo
0 0 Non determinato

Esempio 5.17.2. Determinare i massimi ed i minimi della funzione

y = 2 sen x + cos 2x

Soluzione. Essendo la funzione periodica (di periodo 2π), è sufficiente studiare la variazione della funzione in [0,2π].

  1. Calcoliamo la derivata

    y' = 2 cos x − 2 sen 2x = 2(cos x − 2 sen x ⋅ cos x) = 2 cos x (1 − 2 sen x)

  2. Troviamo i valori critici dell'argomento

    2 cos x − 2 sen 2x = 2(cos x − 2 sen x) = 0

    x1 = π/6;   x2 = π/2;   x3 = 5π/6;   x5 = 3π/2.

  3. Calcoliamo la derivata seconda

    y'' = −2sen x −4 cos 2x

  4. Esaminiamo la natura di ciascun punto critico

    (y'')x1 = π/6 = −2 ⋅(1/2) − 4(1/2) = −3 <0

    Dunque, nel punto x1 = π/6 abbiamo un massimo

    (y)x1 = π/6 = 2 ⋅(1/2) + (1/2) = 3/2

    D'altra parte

    (y'')x2 = π/2 = −2 ⋅1 + 4 ⋅ 1 = 2 >0

    Nel punto x3 = 5π/6 abbiamo

    (y'')x3 = 5π/6 = 2 ⋅(1/2) − 4 ⋅ (1/2) = −3 <0

    Quindi la funzione ha un massimo nel punto x3 = 5π/6;

    (y)x3 = 5π/6 = 2 ⋅(1/2) + 1/2 = 3/2

    Infine

    (y'')x4 = 3π/2 = −2 ⋅(−1) −1 = −3 >0

    Quindi abbiamo un minimo nel punto x4 = 3π/2.

    (y)x4 = 3π/2 = −2 ⋅(−1) −1 = −3

Mostraiamo con alcuni esempi che, nei casi in cui si abbia f'(x) = 0 e f''(x) = 0 in un certo punto x0, la funzione f(x) può avere in questo punto un massimo, un minimo oppure non avere né l'uno né l'altro.

Esempio 5.17.3. Determiniamo i massimi ed i minimi della funzione

y = 1 −x4

Soluzione.

  1. Troviamo i punti critici.

    y' = −4x3,  −4x3 = 0,  x = 0

  2. Determinare il segno della derivata seconda nel punto x = 0

    y'' = −12x2,   (y'')x = 0 = 0

    Ciò vuole dire che non possiamo in questo caso determinare la natura del punto critico esaminato servendoci del segno della derivata seconda.

  3. Studiamo allora la natura del punto critico applicando il primo metodo

    (y')x < 0 > 0, (y')x >0 < 0

Esempio 5.17.3. Determiniamo i massimi e i minimi della funzione

y = x6

Soluzione. Attenendoci al secondo metodo troviamo

  1. y' = 6x5 = 0, x = 0;

  2. y'' = 30x4, (y'')x = 0 = 0.

Il secondo metodo non ci permette dunque di determinare la natura edei punti critici. Applicando il primo metodo troviamo

(y'')x < 0 < 0  (y'')x > 0 > 0

Quindi la funzione ha un minimo nel punto x = 0. ■

Esempio 5.17.3. Determiniamo i massimi e i minimi della funzione

y = (x − 1)3

Soluzione. Secondo metodo:

  1. y' = 3(x−1)2 = 0, x = 1;

  2. y'' = 6(x −1),   (y'')x = 1 = 0.

si vede dunque che il secondo metodo è inefficace, e dobbiamo quindi ricorrere al primo metodo:

(y')x < 1 > 0,  (y')x >1 > 0

Pertanto la funzione non ha né massimo, né minimo nel punto x = 1. (v. fig.)  ■
