Stime asintotiche e grafici

Le stime astintotiche non servono solo per calcolare limiti, ma anche per tracciare il grafico qualitativo di una funzione nell'intorno di un certo punto, oppure nell'interno di ±∞.

Esempio 4.6.1. Sia

f ( x ) = x 3 + x 2

Si può ragionare così: la funzione è definita e continua su tutto ℝ; per x → ± ∞, f(x) ~ x2. Dunque f(x) → +∞; inoltre il suo grafico sarà simile, per x grande in valore assoluto, a quello di x2. La funzione inoltre si annulla in x = 0 e, per x → 0, f(x) ~ ∛x; dunque il suo grafico sarà simile, in un intorno di = 0, a quello di ∛x: in particolare, avrà tangente verticale nell'origine. Spesso l'andamento di una funzione nell'intorno di un punto (ad es. il fatto che abbia tangente verticale o orizzontale) è prevedibile in base a un'opportuna stima asintotica, che consente di tracciare il grafico qualitativo di f (nell'intorno del punto) per confronto con quello di una funzione nota (ad es., una potenza a esponente razionale). Analoghe stime sono utili per x → ±∞.

Esempio. Uno dei problemi fisici all'origine della teoria quantistica è quello della radiazione del corpo nero. La legge proposta nel 1900 da Planck per la distribuzione spettrale della radiazione emessa da un corpo nerl è:

f ( λ ) = 8 π h c λ − 5 e h c / λ k T − 1

Qui f(λ) è la densità di energia raggiante corrispondente alla lunghezza d'onda λ, quando la temperatura del corpo è T; h è la costante di Planck, k la costanten di Boltzmann e c la velocità della luce. Questa legge è in accordo con i dati sperimentali, mentre la legge che si può dedurre usando la fisica classica (legge di Rayleigh-Jeans) è

f(λ) = 8πkTλ−4

che è in accordo coi dati sperimentali solo per λ grande, mentre per λ piccolo è in profondo disaccordo: tende a +∞, mentre la curva sperimentale tende a zero!
Studiamo la curva di Planck con opportune stime asintotiche.
Per λ → +∞,

ehc/λkT − 1 ~ hc/λkT

e

8 π h c λ − 5 e h c / λ k T − 1 ∼ 8 π h c λ − 5 h c / λ k T = 8 π k T λ − 4

ossia si trova la legge di Rayleigh-Jeans, in prima approssimazione; in particolare, la densità di energia raggiante tende a zero come λ−4.
Per λ → 0+

8 π h c λ − 5 e h c / λ k T − 1 ∼ 8 π h c λ − 5 e h c / λ k T → 0 +

(l'eseponenziale a denominatore tende a infinito più rapidamente di λ−5 a numeratore). Quindi la curva di Planck è in accordo coi dati sperimentali anche per λ piccolo. Poiché la funzione èe sempre positiva, il suo grafico, in base a queste informazioni, sarà del tipo:

Crescita di una funzione all'infinito

Supponiamo di voler tracciare il gradico di una fuznione f che, per x → +∞ (o −∞) tende a +∞ (o −∞). Per descrivere la velocità con cui la funzione tende all'infinito, sono utili le seguenti nozioni: diremo che, per x → +∞),

limx → ∞ f(x)/x f ha crescita
±∞ sopralineare
lineare m (finito e diverso da 0)
0 sottolineare

(Analoghe definizione si danno per x → −∞). Solo nel caso in cui una funzione ha crescita lineare, è possibile che ammetta asintoto obliquo. (v. par. 3 per il calcolo di tale asintoto).
Esempi di funzioni con cresci sopralineare per x → +∞ sono gli esponenziali ax e le potenze xa con a > 1; esempi di funzioni con crescita sottolineare per x → +∞ sono i logaritmi loga x e le potenze xa con 0 < a < 1.
