Series of functions

Let I ⊆ ℝ and let the functions

fn: I ⟶ ℝ  for n = 1,2,3

be befined on I. For every x ∈ I we can consider the sequence of real numbers {fn(x)} and from this define the series

∑ n = 1 ∞ f n ( x )

that for every x ∈ I, can be convergent, divergent or irregular. Esprimeremo questo fatyto dicendo che la serie di funzioni converge o è irregolare in un certo punto x ∈ I. Particolarmente interessante è il caso in cui la serie converge in tutti i punti dell'intervallo (o tutti i punti tranne un numero finito); in tal caso la somma della serie è una nuova funzione, f(x), definita in I (o I meno un numero finito di punti).
Abbiamo già incontrato una classe importante di serie di funzioni: le serie di potenze,

∑ n = 0 ∞ a n x n

in cui le funzioni fn sono del tipo particolare anxn. Ad esempio:

  • la serie geometrica:

    ∑ n = 0 ∞ x n = 1 1 − x for − 1 < x < 1

  • la serie esponenziale,

    ∑ n = 0 ∞ x n n ! = e x ∀ x ∈ R

    e più in generale la serie di Taylor di una funzione derivabile infinite volte.

Incotreremo tra breve un'altra classe, altrettanto importante, di serie di funzioni, le serie di Fourier. Introduciamo prima, qualche risultato di carattere generale sulle serie di funzioni che ci permetterà anche di comprendere le principali proprietà delle serie di potenze.
Un problema tipico delle serie di funzioni è: quali sono le proprietà che la somma f(x) della serie "eredita" dai termini fn(x)? Ad esempio:
Se le fn sono continue, anche f lo è?
Se le fn sono derivabili anche f lo è? Si può calcolare f' calcolando derivando la serie termine a termine", cioè estendendo alla "somma infinita" la linearità della derivata? Questo significa scambiare le operazioni di somma e di derivazione.

[ ∑ n = 1 ∞ f n ( x ) ] ′ = ∑ n = 1 ∞ f n ′ ( x )

Analogamente, si può calcolare l'integrale di f "integrando termine a termine", cioè estendendo alla "somma infinita" la linearità dell'integrale? Ossia scambiare le operazioni di somma e di integrazione.

∫ a b ( ∑ n = 1 ∞ f n ( x ) ) d x = ∑ n = 1 ∞ ( ∫ a b f n ( x ) d x )

Non possiamo aspettarci che la sola convergenza della serie ∑∞n=1 fn(x), senza altre ipotesi, garantisca la risposta affermatica alle domande precedenti. Esiste però un modo abbastanza semplice per "rafforzare" la convergenza in modo che tutto funzioni bene. Lo esprimeremo attraverso la definizione di convergenza assoluta.

Definition 9.2.1 (Convergenza assoluta). Siano fn: [a,b] ⟶ ℝ, n = 1, 2, 3, .. Diremo che la serie ∑∞n=1 fn(x), converge totalmente se:

  1. |fn| ≤ an ∀x ∈ [a,b], n ∈ ℕ.

  2. ∑∞n=0 an converge.

Vale allora il:

Teorema 9.2.2 Siano fn: [a,b] ⟶ ℝ, n = 1, 2, 3, .., e supponiamo che la serie ∑∞n=1 fn(x), converga totalmente. Allora:

  1. La serie ∑∞n=0 fn(x), converge (assolutamente) in [a,b] a una funzione f(x), e

    | f ( x ) | = | ∑ n = 0 ∞ f n ( x ) | ≤ ∑ n = 0 ∞ a n

  2. Se le fn sono continue in [a,b], anche f è cotinua in [a,b];

  3. Se le fn sono integrabili, anche f è integrabile, e vale l'identità

    ∫ a b f ( x ) d x = ∫ a b ( ∑ n = 1 ∞ f n ( x ) ) d x = ∑ n = 1 ∞ ( ∫ a b f n ( x ) d x )

    4. Supponiamo inoltre che le fn siano funzioni derivabili in [a,b], e che anche la serie ∑∞n=1 f'n(x) converga totalmente. Allora anche f è deribvabile in [a,b], e f' si può calcolare derivando termine a termine:

    f ( x ) = [ ∑ n = 1 ∞ f n ( x ) ] ′ = ∑ n = 1 ∞ f n ′ ( x )

Example 8.3.1 Consider the series ∑∞n=1 fn(x) where,

fn(x) = nx/(1 + n2 x2)

Clearly, ∑∞n=1 fn(x) is a series of continuous functions fn(x) on ℝ and since its nth partial sum is sn(x) = nx /(1 + n2x2) from exercise 1 it follows that this series converges pointwise to f(x) = 0 on ℝ, but the convergence is not uniform.

A useful criterion to check the uniform convergence of a series of functions.

Theorem (Weierstrass M-test). Let {fn} be a sequence of real- or complex-valued functions on E. Suppose there is a sequence {Mn} of nonnegative real numbers such that the following conditions hold:

  1. |fn(x)| ≤ Mn ∀x ∈ E, ∀n ∈ ℕ;

  2. the series Σn=0∞ Mn is convergent.

Then the series Σn=0∞ fn(x) converges uniformly in E.

Proof. For (fixed) x ∈ E, the comparison test and (ii) show that the series Σn=1∞ fn(x) is (absolutely) convergent. So we can define a function f on E by f(x) = Σn=1∞ fn(x).
