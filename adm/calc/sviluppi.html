Limiti notevoli e sviluppi

Abbiamo visto che il processo di linearizzazione permette di approssimare una funzione derivabile, localmente, mediante la sua retta tangente, ossia una funzione lineare. D'altro canto, anche i "limiti notevoli" a suo tempo incontrati permettevano di scrivere simili risultati di approssimazione, per certe funzioni particolari. Ad es., per la funzione y = sin x, il processo di linearizzazione, in x = 0, porta a scrivere

sin x = x + o(x)

mentre a suo tempo abbiamo visto che

sin x ~ x per x → 0

Che relazione c'è tra queste due affermazioni? Sono in effetti equivalente, come mostra il prossimo fatto generale:

Teorema 5.18.1 (Relazione tra "o piccolo" e "asintotico") Vale la seguente equivalenza:

Per x → x0, f(x) ~ g(x) se e solo se f(x) = g(x) + o(g(x))

Dim. Le due implicazioni contenute nell'affermazione precedente, si dimostrano usando le definizioni.

  1. Definizione di asintotico: f(x) ~ g(x) significa che limx → x0 f(x)/g(x) = 1;

  2. Definizione di o piccolo: f(x) = o(g(x)) significa che limx → x0 f(x)/g(x) = 0.

Analizziamo entrambne le direzioni.

(⇒) If f(x) ~ g(x), then f(x) = g(x) + o(g(x)). Dalla definizione di asintotico:

f(x) ~ g(x) significa che limx → x0 f(x)/g(x) = 1

Goal: Show f(x) = g(x) + o(g(x)), which means

f ( x ) − g ( x ) g ( x ) → 0 as x → x 0

Infatti

f ( x ) − g ( x ) g ( x ) = f ( x ) g ( x ) − 1

Since limx → x0 f(x)/g(x) = 1, it follows that:

lim x → x 0 ( f ( x ) g ( x ) − 1 ) = 0

which implies

f ( x ) − g ( x ) g ( x ) → 0

or equivalently,

f(x) = g(x) + o(g(x))

(⇐) If f(x) = g(x) + o(g(x)), then f(x) ~ g(x). Assumendo:

f ( x ) = g ( x ) + o ( g ( x ) ) ⇒ f ( x ) − g ( x ) g ( x ) → 0

cioè

f ( x ) g ( x ) = 1 + f ( x ) − g ( x ) g ( x ) → 1 + 0 = 1

Quindi

lim x → x 0 f ( x ) g ( x ) = 1

which is exactly the definition of f(x) ~ g(x). □
