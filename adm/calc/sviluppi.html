Limiti notevoli e sviluppi

Abbiamo visto che il processo di linearizzazione permette di approssimare una funzione derivabile, localmente, mediante la sua retta tangente, ossia una funzione lineare. D'altro canto, anche i "limiti notevoli" a suo tempo incontrati permettevano di scrivere simili risultati di approssimazione, per certe funzioni particolari. Ad es., per la funzione y = sin x, il processo di linearizzazione, in x = 0, porta a scrivere

sin x = x + o(x)

mentre a suo tempo abbiamo visto che

sin x ~ x per x → 0

Che relazione c'è tra queste due affermazioni? Sono in effetti equivalente, come mostra il prossimo fatto generale:

Teorema 5.18.1 (Relazione tra "o piccolo" e "asintotico") Vale la seguente equivalenza:

Per x → x0, f(x) ~ g(x) se e solo se f(x) = g(x) + o(g(x))

Dim. Le due implicazioni contenute nell'affermazione precedente, si dimostrano usando le definizioni.

  1. Definizione di asintotico: f(x) ~ g(x) significa che limx → x0 f(x)/g(x) = 1;

  2. Definizione di o piccolo: f(x) = o(g(x)) significa che limx → x0 f(x)/g(x) = 0.

Analizziamo entrambne le direzioni.

(⇒) If f(x) ~ g(x), then f(x) = g(x) + o(g(x)). Dalla definizione di asintotico:

f(x) ~ g(x) significa che limx → x0 f(x)/g(x) = 1

Goal: Mostrare che f(x) = g(x) + o(g(x)); ciò significa

f ( x ) − g ( x ) g ( x ) → 0 as x → x 0

Infatti

f ( x ) − g ( x ) g ( x ) = f ( x ) g ( x ) − 1

Dato che limx → x0 f(x)/g(x) = 1, segue che:

lim x → x 0 ( f ( x ) g ( x ) − 1 ) = 0

il ché implica

f ( x ) − g ( x ) g ( x ) → 0

o equivalentemente,

f(x) = g(x) + o(g(x))

(⇐) If f(x) = g(x) + o(g(x)), then f(x) ~ g(x). Assumendo:

f ( x ) = g ( x ) + o ( g ( x ) ) ⇒ f ( x ) − g ( x ) g ( x ) → 0

cioè

f ( x ) g ( x ) = 1 + f ( x ) − g ( x ) g ( x ) → 1 + 0 = 1

Quindi

lim x → x 0 f ( x ) g ( x ) = 1

che è esattamente la definizione di f(x) ~ g(x). □

Per dimostrare la b) basta notare che limx → x0 f(x)/x e limx → x0 −f(x)/x sono entrambi o-piccoli di x.  □

Questo permette di rileggere i limiti notevoli che abbiamo finora espresso con stime asintotiche, mediante uguaglianze che coinvolgono "o piccolo". Ad es.,

per x ⟶ 0
sin x ~ x, quindi sin x = x + o(x);
1 − cos x ~ x2/2, quindi 1 − cos x = x2/2 + o(x2).

Riflettiamo sull'ultima: l'uguaglianza 1 − cos x = x2/2 + o(x2) si può riscrivere nella forma:

cos x = 1 − x2/2 + o(x2)

(questo è ovvio: abbiamo sostato dei termini da un membro a un altro di un'uguaglianza). Notiamo, invece, che la relazione asintotica 1 − cos x ~ x2/2 non dice la stessa cosa della relazione cos x ~ 1 − x2/2. Infatti, per x ⟶ 0, 1 − x2/2 ~ 1, perciò la stima cos x ~ 1 − x2/2 contiene la stessa informazione che dire semplicemente "cos x ⟶ 1", ed è altrettanto vera che la stima cos x ~ 1 + 50x14! Questa osservazione mostra un vantaggio del simbolo "o piccolo" rispetto a quello di asintotico: un'uguaglianza si può scrivere in vari modi, è più facile da usare senza errori, rispetto ad una stima asintotica.
Per esercizio, riscriviamo mediante il simbolo di "o piccolo" i limiti notevoli, ex, log (1 + x), xα √(1 + x):

ex = 1 + x + o(x)
log (1 + x) = x − x2/2 + o(x2)

Proprietà del simbolo di "o piccolo"

Ci si trova ad eseguire calcoli che coinvolgono o(xn) (ad es. quando si opera su sviluppi di Taylor-MacLaurin con resto secondo Peano). Per procedere correttamente è utile comprendere alcune proprietà del simbolo di "o piccolo".

Notiamo che ad es., per x ⟶ 0, x2 = o(x); x3 = o(x); x3 = o(x2). Il simbolo di o(..) non denota quindi una particolare funzione, ma qualsiasi funzione abbia la proprietà espressa dalla definizione. Da questo fatto seguono alcune proprietà della relazione di "o piccolo" che possono apparire "strane". Ne illustriamo alcune su esempi che si incontrano frequentemente:

  1. o(x) − o(x) = o(x) (non fa zero!);

  2. (−3x2) = o(x2),   − o(x) = o(x)  (coefficienti numerici moltiplicativi sono irrilevanti).

    3. Dimostriamo a): consideriamo due funzioni f(x) e g(x) tali che entrambe siano o(x) per x → x0. Questo implica che:

    f(x) = o(x) e g(x) = o(x)

    Per definizione, ciò implica che:

    limx → x0 f(x)/x = 0 e limx → x0 g(x)/x = 0

    Ora, consideriamo la differenza f(x) − g(x). Vogliamo dimostrare che anche questa differenza è o(x).

    f(x) − g(x) = o(x) − o(x)

    Per verificare che f(x) − g(x) sia o(x), osserviamo il limite:

    limx → x0 [f(x) − g(x)]/x

    ​

    Poiché f(x) e g(x) sono entrambe o(x), possiamo scrivere:

    limx → x0 [f(x) − g(x)]/x = limx → x0 f(x)/x − g(x)/x

    Dato che entrambi i limiti individuali sono zero, la differenza dei limiti è anch'essa zero:

    limx → x0 [f(x)/x − g(x)/x] = 0 − 0 = 0

    Quindi, f(x) − g(x) = o(x), il che dimostra che: o(x) −o(x) = o(x).   ■

    Per x ⟶ 0, o(x) + o(x2) = o(x) (l'errore più grossolano, cioè o(x), ingloba quello più fine).
    Analogamente si avrebbe:

    per x ⟶ +∞, o(x) + o(x2) = o(x2)

    Il simbolo di "o piccolo" si comporta al seguente modo con somme e prodotti (verifiarlo in base alla definizione):

    f ⋅ o(g) = o(f ⋅ g), ad esempio:

    x ⋅ o(x2) = o(x3)   o(x3)/x = o(x2)   o(x2)/x2 = o(1)

    (ricordiamo che o(1) indica semplicemente una funzione che tende a zero per x ⟶ x0, il punto considerato). Ed anche:

    o(f) ⋅ o(g) = o(f ⋅ g), ad esempio:

    x ⋅ o(x2) = o(x3)
