Linear differential equations

In many applications the respose of a system to a disturbance from the environment is linearly proportioanl to the disturbance. For example, if we have a load applied on a linaerly elastic structure, then doubling the magnitude of the load will lead to twice as large values of stresses, strains and displacementes in the interior of the structure. Linear systems are describe by linear differential equations of the form

an(t)y(n) + an−1(t)y(n−1) + ... + a1 y' + a0y = b(t)  15.1.1

Their linearity implies the existence of a linear differential operator L[ ], defined as

L = a 0 ( t ) + a 1 ( t ) d d t + … + a n ( t ) d n d t n

The left hand side of equation 1.5.1 can be written as

L[y] = a0(x)y + a1(x)y' + a2(x)y'' + ... + an(x)y(n) = b(x)

So the linear differential equation can be written as

L y(x) = b(x)

Where L: Cn(I) ⟶ C0(I) is a linear operator between function spaces.

The linearity of L, means that

L(αy1 + βy2) = αLy1 + βLy2   (L.1)

con y1,y2 ∈ Cn(I) and α,β; ∈ ℝ.

The kernel of a linear differential operator is its kernel as a linear mapping, that is the vector space of the solutions of the (homogeneous) differential equation Ly = 0.

A direct consequence of the linearity of differential equations is the satisfaction of the so-called superposition principle (or linearity principle), which says that the sum of two input gives rise to the sum of the outputs and that we describe in mathematical terms with the following proposition.

Proposition 11.5.2 (Superposition principle)

  1. The general solution of a linear homogeneous ordinary differential equation of order n is a linear combination of n linearly independent solutions y1,y2, ..., yn with n arbitrary constants c1, c2, ..., cn. That is

    y ( x ) = ∑ k = 1 n c k y k ( x )

    represents general solution, which depends on exactly n arbitraty constants.

  2. (General solution to the inhomogeneous equation) Let y* be a particular solution to the inhomogenous linear equation L[y*] = f. Then the general solution to L[y] = f is given by y = y* + y0, where y0 is the general solution to the corresponding homogeneous equation, L[y0] = 0.

    Proof.

    1. From linearity and the fact that L(y1) = 0, L(y2) = 0, ..., L(yn) = 0. Any linear combination satisfies

      L(c1y1 + c2y2 + L(yn) = c1L(y1) + c2L(y2) + ... + cn L(yn) = 0

      That is every linear combination c1y1 + c2y2 + .... + cnyn is still a solution of the homogeneous equation. The set of solution of the homogeneous equation is a subspace of Cn(I), and it is in particular a vector space.

    2. Let us first show that y = y* + y0 is also a solution whenever L[u] = 0. By linearity,

      L[y] = L[y* + y0] = L[y*] + L[y0] = f + 0 = f

      To show that every solution to the inhomogeneous equation can be expressed in this manner, let y2 be the general solution of the inhomogeneous equation: L[y2] = f, by linearity we have

      L[y2 − y*] = L[y2] + L[y*] = f − f = 0

      That is y2 − y* is solution to the homogeneous equation, that is y2 − y* = y*0 for a particular solution y*0 of the homogeneous equation. □

    Another important property of the homogeneous equation is the following.

    Theorem 11.6.2 The vector space of the solutions of a second order linear differential equation has dimension 2.

    Proof. The proof is based on Theorem 11.5.3 (existence and unicity of the solution of the Cauchy problem for the equation in Canonical form).
    Let y1, y2 solutions of the equation

    Ly ≡ y'' + a(t) y' + b(t) y = 0

    in the interval I, and let t0 ∈ I such that

    { L y 1 = 0 y 1 ( t 0 ) = 1 y 1 ′ ( t 0 ) = 0 { L y 2 = 0 y 2 ( t 0 ) = 0 y 2 ′ ( t 0 ) = 1

    This is an arbitrary choice made to find a 2-element basis of the vector space of solutions.
