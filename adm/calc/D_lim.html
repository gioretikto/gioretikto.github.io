Limite della derivata e derivabilità

Esempio 5.13.1 Supponiamo di voler studiare la derivabilità o meno di una funzione del tipo:

f(x) = x |log x|

La funzione è definita e continua per x > 0, ed è certamente derivabile per x ≠ 1, mentre in x = 1, annulandosi l'argomento del modulo, sospsettiamo la presenza di un punto angoloso.
Poiché

f ( x ) = { x log ⁡ x per  x ≥ 1 − x log ⁡ x per  0 < x ≤ 1

possiamo calcolare

f ′ ( x ) = { log ⁡ x + 1 per  x > 1 − ( log ⁡ x + 1 ) per  0 < x < 1

Rimane da studiare la derivabilità di f in x = 1. È facile osservare che

lim x → 1 + f ′ ( x ) = lim x → 1 + ( log ⁡ x + 1 ) = 1

mentre

lim x → 1 − f ′ ( x ) = lim x → 1 − ( − log ⁡ x − 1 ) = − 1

Possiamo usare queste informazioni per affermare che la derivata destra e sinistra di f, in x = 1, valgono 1, −1, rispettivamente? (Se è così, x = 1 è un punto angoloso). In altre parole, è vero ad es. che

lim x → 1 + f ′ ( x ) = f + ′ ( 1 )

ossia il limite destro della derivata per x → 1 è uguale alla derivata destra in x = 1?

Si osservi che la cosa non è ovvia perché, per definizione, la derivatra destra è il limite destro del rapporto incrementale:

f + ′ ( 1 ) = lim h → 0 + f ( 1 + h ) − f ( 1 ) h

che però è molto più scomodo da calcolare, in pratica, rispetto al limite della derivata. Il T. di Lagrange permette di rispondere affermativamente a questa domanda:

Esempio 5.13.2. Sia f : [a,b) → ℝ, continua in a, derivabile in (a,b), ed esista (finito o infinito)

limx → a+ f'(x) = m ∈ ℝ*

Allora esiste

f+(a) = m

Detto in prole: se la funzione è continua in a ed esiste il limite destro della derivata, allora esiste la derivata destra, e coincide con quel limite. Un analogo enunciato vale per la derivata sinistra, e quindi per la derivata.

Dim. Sia h > 0. Applicando il T. di Lagrange a f sull'intervallo [a,a+h], otteniamo che esiste th ∈ [a,a+h] tale che

f ( a + h ) − f ( a ) f ( h ) = f ′ ( t h )

Per h > 0+ si ha che th → a+, quindi per ipotesi f'(th) → m. Di conseguenza esiste

f + ′ ( a ) = lim h → 0 + f ( a + h ) − f ( a ) f ( h ) = m  □

Nell'esempio precedente, f è continua in x = 1, quindi il teorema è applicabile e permette di concludere che

f'+(1) = 1, f−(1) = − 1, per cui x = 1 è un punto angoloso.

Si considerino anche i prossimi esempi.

Esempio 5.13.3. Sia

f(x) = x2 log |x|

definita per x ≠ 0. Sappiamo però che esiste

lim x → 0 x 2 log ⁡ | x | = 0

perciò possiamo prolungare f per continuità definendo f(0) = 0, ed f risulta così continua anche in 0. Chiediamoci se f è anche derivabile in 0. Cominciamo a calcolare, per x ≠ 0,

f'(x) = 2x log |x| + x

Ora calcoliamo

lim x → 0 ( 2 x log ⁡ | x | + x ) = 0

Perciò per il teorema precedente esiste f'(0) = 0, in particolare f è derivabile in x = 0.  ■

Esempio 5.13.4. Sia

f(x) = arctg 1/x

Calcoliamo

f ′ ( x ) = 1 1 + 1 x 2 ⋅ ( 1 x 2 ) = − 1 x 2 + 1

Osserviamo che

lim x → 0 f ′ ( x ) = lim x → 0 − 1 x 2 + 1 = − 1

Si può concludere allora che esiste f'(x) = −1? No! In questo caso viene meno l'ipotesi di continuiutà di f in 0: si osservi che f non è definita per x = 0, e

lim x → 0 f ( x ) = ± π 2

dunque x = 0 è un punto di discontinuità a salto, ed f non è derivabile in x = 0. Si osservi dunque che l'ipotesi di continuità, nel teorema precedente, è essenziale.  ■

Esempio 5.13.4 Sia

f(x) = x2 sin (1/x)

La funzione non è definita in x = 0, ma si può prolungare per continuità ponendo f(0) (infatti, f(x) → 0 per x → 0). Dunque ora f è continua in 0; per x ≠ 0, inoltre, f è certamente derivabile e si ha:

f ( x ) = 2 x sin ⁡ x ( 1 x ) + x 2 cos ⁡ ( 1 x ) ⋅ ( − 1 x 2 ) = 2 x sin ⁡ x ( 1 x ) − cos ⁡ ( 1 x )

Osserviamo ora che limx → 0 f'(x) non esiste, né finito né infinito. Possiamo concludere che non esiste f'(0)? No! Si noti che il teorema non permette questa conclusione, ed in effetti, in questo caso, calcolando f'(0) mediante la definizione si ha:

f ′ ( 0 ) = lim h → 0 f ( h ) − 0 h = lim h → 0 sin ⁡ ( 1 x ) = 0

perciò in realtà f è derivabile e f'(0) = 0. Quando il limite della derivata non esiste (né finito, né infinito), il teorema non applicabile; in particolare non è detto che non esista la derivata nel punto considerato: occorre calcolarla dalla definizione.  ■
