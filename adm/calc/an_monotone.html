Successioni monotone

Definizione 3.3.1. Una successione {an} si dirà:

  • monotona crescente se an ≤ an+1; strettamente crescente se an < an+1 ∀n;

  • monotona decrescente an ≥ an+1; strettamente decrescente se an > an+1 ∀n. □

Per es, la successione {n2} è monotona strettamente crescente, la successione {1/n} è monotona strettamente decrescente, la successione {(−1)n} non è monotona; ogni uccesione costante è monotona (crescente o decrescente, non strettamente).
Riguardo all'operazione di limite, queste successioni hanno un'importanza particolare; infatti esse non sono mai irregolari, ma sono convergenti oppure divergenti a seconda che siano limitate oppure no. Il risultato fondamentale è il seguente:

Teorema 3.3.2 (Di monotonia) Sia {an} una successione monotona crescente e superiormente limitata.
Allora {an} è convergente e il suo limite è uguale a sup {an: n ∈ ℕ}.
Analogamente, se {an} è una successione monotona decrescente e inferiormente limitata, allora {an} convergente e il suo limite è uguale a inf{an: n ∈ ℕ}. □

Ricordiamo che il simbolo sup {an: n ∈ ℕ} denota l'estremo superiore dell'insieme dei valori an assunti dalla successione.

Dim. Consideriamo l'insieme dei valori assunti dalla successione, {an: n ∈ ℕ}. Poiché la successione è limitata superiormente, questo insieme è limitato superiormente, quindi (per la proprietà dell'estremo superiore di cui gode ℝ) esiste il suo estremo superiore,

Λ = sup {an: n ∈ ℕ}, Λ ∈ ℝ

Proviamo ora che

lim n → ∞ 3.3.2 a n = Λ

Occorre mostrare che per ogni ε > 0 si ha

Λ − ε < an < Λ + ε

La seconda disuguaglianza è ovvia: ∀n è an < Λ (quindi an < Λ + ε), perchè Λ è l'estremo superiore degli an, quindi in particolare è un maggiorante dell'insieme dei valori an. Per provare la prima disuguaglianza, consideriamo il numero Λ − ε. Ricordinamo che per definizione di estremo superiore, Λ è il minimo dei maggioranti dell'insieme {an: n ∈ ℕ}. Perciò, essendo Λ − ε < Λ, certamente Λ − ε non è un maggiorante dell'insieme {an: n ∈ ℕ}.
Questo significa che esiste un n0 per cui

an0 > Λ − ε

D'altro canto la successione è monotona crescente, perciò ∀n > n0 risulta an ≥ an0.
Abbiamo quindi provato che

an ≥ an0 > Λ − ε

per ogni n ≥ n0, ossia definitivamente. Quindi limn → ∞ = Λ. □

Questo teorema è una conseguenza dell'assioma di continuità dei numeri reali e pertanto vale se l'ambiente considerato è ℝ. Ad es., non è vero che una successione crescente e limitata di numeri razinali ammette sempre limite razionale, cioè in ℚ.

Esempio 3.3.3. Sia {an} la successione così definita:

a0
a1 = 0.1
a2 = 0.1011
a3 = 0.10110111
a4 = 0.1011011101111
....

e così via. (Al passo n si aggiunge al numero decimale ottenuto al passo precedente una crifra zero seguita da n cifre uguali a 1).
La successione {an} è evidentemente crescente, e superiormente limitata (ad es., an ≤). In ℝ, la successione converge al numero sup {an: n ∈ ℕ}, che dopo la virgola presenta un allineamente decimale illimitato e non periodico ci cifre (una cifra 1, una cifra 0, due cifre 1, una cifra 0, tre cifre i, una cifra 0, e così via all'infinito). Quindi il limite della successione è un numero irrazionale. Quest'esempio mostra che nell'insieme ℚ il teorema di monotonia è falso.  ■

Vedremo in seguitio che il teorema di monotonia sarà utilizzato per dimostrare importanti proprietà di funzioni continue. Questo teorema quindi costituisce una delle motivazioni per cui è uitle studiare l'analisi matematica nell'ambiente dei numeri reali, anziché in quello dei numeri razionali.
Il teorema di monotonia si può completare con il prossimo enunciato, che considera successioni limitate o illimitate:

Corollario 3.3.2. Sia {an} una successione monotona crescente. Allora esiste

limn→ ∞ an = sup {an : n ∈ ℕ}

Esplicitamente: se {an} è superiormente limitata, allora converge (e il suo limite è uguale all'estremo superiore dei suoi valori, che in questo aso è un numero reale); se invece {an} è superiormente illimitata, allora an tende a ∞ (che in questo caso è pari all'estremo superiore dei suoi valori).

SI può anche dire, sinteticamente: una successione monotona, converge o diverge (non può essere irregolae).

Dim. Se {an} è superiormente limitata, l'enunciato è contenuto nel teorema di monotonia. Se invece {an} è superiormente illimitata, questo significa che per ogni K > 0 esiste un n0 > K. D'altro canto la successione è crescente, perciò per ogni n > n0 si ha an ≥ an0 > K. Abbiamo quindi provato che per ogni K > 0 an > K definitivamente.
Questo significa che an → +∞. □

Esempio 3.3.3 (Successione geometrica). Si consideri la progressione geometrica di ragione q, {qn}:

1, q, q2, q3, qn,...

Se q > 1 la successione è monotona crescente, illimitata superiormente.
Se q = 1 la successione è costante. Se 0 < q < 1, la successione è monotona decrescente; è facile mostrare che tende a zero. Se q è negativo la successione non è più monotona. Lo studente verifichi le seguenti affermazioni:
