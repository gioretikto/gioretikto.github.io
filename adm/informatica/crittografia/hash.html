Password Hashing

Il Password hashing è una pratica di sicurezza seguita per immagazzinare le passwords delgi utenti nei database. Senza hashing, ogni password sarebbe memorizzata nel database in chiaro testo e in caso di furto sarebbe molto compromettente.

Tramite l'hashing algorithm applicato alle passwords prima del salvataggio nel database, diventa implausibile per un hacker determinare la password originale, ma si è sempre in grado di comparare l'hash alla password originale.

Hashing algorithms come MD5, SHA1 e SHA256 sono progettati per essere veloci ed efficienti. With modern techniques and computer equipment, it has become trivial to "brute force" the output of these algorithms, in order to determine the original input.
