Objects

The main building block in JavaScript is an object. An object is a collection of elements; each element can either be a property or a method. A property holds a single data value, and a method will perform some action thati is defined as a function. Another way of stating this is that a property is simply a variable (note the var keyword that is defined on an object. Likewise, as methos is a function that is defined on an object. The easises way to create an object is by specifying the members through a set of name/value pairs like this:


var myObject = {
    color: "Blue",
    count: 4
    };

You can also define methods using this notation, like this:

var myObject = {
    color: "Blue",
    count: 4,
    printf: function () {
        console.log("Quantity: " +  this.count + ", Color: " + this.color);
        }
    };

Creating an object this way is sometimes referred to as object literal notation. The object, named myObject contains two properties, color and count and one function, log(). You can then access the properties and function using hte dot notatin like this:

window.alert(myObject.color);
myObject.log();

Another way to create an object, is to use an object constructor. This uses the new keyword and invokes the default constructor. The following code would create the exact same object:

var redObject = new Object();
redObject. color = "Red";
redObject.counbt = 5;
redObject.printf = function(){
    console.log("Quantity: " +  this.count + ", Color: " + this.color);
}

If you need a second instance to count your green items, for example, just repeat the code like this:

var greenObject = new Object();
greenObject. color = "Red";
greenObject.counbt = 5;
greenObject.printf = function(){
    console.log("Quantity: " +  this.count + ", Color: " + this.color);
}

You can then use objects in your code, for example:

if (greenObject.count > redObject.count) {
    console.log("You have more green items than red items");
}
