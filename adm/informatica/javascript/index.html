Javascript

The naming was just a marketing ploy to help the new scripting language benefit from the popularity of the Java programming language. JavaScript gained new power when the HTML elements of the web page got a more formal, structured definition in what is called the Document Object Model, or DOM. The DOM makes it relatively easy to add a new paragraph or focus on a piece of text and change it.
