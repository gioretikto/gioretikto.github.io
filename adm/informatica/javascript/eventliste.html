Event Listener

If you click insise the inner div element, you'll see a massage printed inside the console. The addEventListener() method is available on all HTML element and has two required parameters: the name of the event and a reference to the function that will be called when the event occurs.

<!DOCTYPE html>
<html lang="en"> 
  <head> 
    <meta charset="utf-8" />
    <title>Prova</title>
  </head> 

  <body> 
     <div id="div2"><p>Clicca qui<p> </div>
     
         
         <script> 
            function someAction(){
                console.log("Bene funziona");
            }
            
            var div = document.getElementById("div2");
            div.addEventListener("click", someAction);
         </script>
  </body> 
</html>

This is another example, of usage of addEventListener() method used to play an audio file by clicking on a button:


<script> 
var butt = document.querySelector('button');
butt.addEventListener("click", playfile);
 function playfile(){
       var audio = document.querySelector("audio");
       audio.play();
 }
</script>

<p><b>Werden</b> noun = il divenire</p>

<div class="arrow-down" data-target="#Werden"></div>

    <li><p>große Dinge sind im Werden</p></li>
</ul>

</div>

con il seguente codice javascript

<script>
document.querySelectorAll('.arrow-down').forEach(el => {
	el.nextElementSibling.style.display = 'none';
  el.addEventListener('click', () => {
    const sibling = el.nextElementSibling;
    const {
      display
    } = sibling.style;
    sibling.style.display = display === 'none' ? '' : 'none';
  })
});
</script>
