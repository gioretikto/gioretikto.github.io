Spaces

Aggiungi gli spazi tramite:

    Type “&nbsp” to add a single space.
    Type “&ensp” to add 2 spaces.
    Type “&emsp” to add 4 spaces.
    Use the non-breaking space (nbsp) 4 times to insert a tab.
    Use “br” to add a line break.
