Connessione a MySQL

Per cercare contenuti nel database, collegarsi al database:


mysql -u root - p

In tutti i sistemi operativi per connetersi al server MySQL si utilizza il seguente comando:


mysql -h hostname -u username -p

Selezionare un database:

MariaDB [(none)]> USE mydb;

Utilizzare l'espressione:

select * from latino where lemma LIKE '%abactor%';
