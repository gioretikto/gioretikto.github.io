Creare nuovi utenti

Per mostrare tutti gli account presenti nel sistema utilizzare:

mysql> select distinct User from mysql.user;

Per mostrare i privilegi di un utente:

mysql> SHOW GRANTS FOR 'user'@'localhost';

Per creare un nuovo account per un utente, effettuiamo dapprima il login come root:


mysql -h localhost -u root -p




mysql> CREATE USER 'gino'@'localhost' IDENTIFIED BY 'some_pass';
mysql> GRANT SELECT, INSERT, UPDATE ON 'secure_login'.* TO 'gino'@'localhost';

Mostrare i databases

MariaDB [(none)]>SHOW DATABASES;

Come noterete i comandi MySQL terminano con un ;.

Per creare un database si utilizza il comando:

MariaDB [(none)]>CREATE db;

Adesso comunichiamo a MySQL di voler utilizzare un dato database con il comando:

MariaDB [(none)]> USE db;

Se qualcosa va storto MySQL fornisce degli avvertimenti per mostrarli usare il comando:

MariaDB [(none)]> SHOW WARNINGS;
