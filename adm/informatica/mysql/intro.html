Introduzione a MySQL

Avviare prima di tutto il database tramite il comando

# systemctl start mysql

Se è il primo log in che si effettua, non sarà stata impostata nessuna password, e per loggarsi come root sarà necessario il comando:


mysql -u root

per impostare la password:

SET PASSWORD FOR 'root'@'localhost' = PASSWORD('new_password');

Provare a loggarsi adesso con phpMyAdmin http://localhost/phpMyAdmin/index.php

In tutti i sistemi operativi per connetersi al server MySQL si utilizza il seguente comando:


mysql -h hostname -u username -p

L'argomento -p chiede al programma il prompt per la vostra password. Sarà necessario sostituire hostname con il nome host o indirizzo IP del computer sul quale il server MySQL è presente, ad esempio -h localhost oppure -h 127.0.0.1.

Una volta connessi al server MySQL, si presenterà il client che vi porterà al prompt di comando MySQL:

MariaDB [(none)]>

Mostrare e creare databases

MariaDB [(none)]>SHOW DATABASES;

Come noterete i comandi MySQL terminano con un ;.

Per creare un database si utilizza il comando:

MariaDB [(none)]>CREATE database mydb;

Adesso comunichiamo a MySQL di voler utilizzare un dato database con il comando:

MariaDB [(none)]> USE db;

Se qualcosa va storto MySQL fornisce degli avvertimenti per mostrarli usare il comando:

MariaDB [(none)]> SHOW WARNINGS;
