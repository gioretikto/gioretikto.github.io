Tabelle

Un databse è composto da una o più tabelle, ciascuna contenente un elenco di oggetti. Una volta creato il database passiamo alla creazione di una tabella con il seguente comando:

mysql> CREATE TABLE table_name(
-> column1_name colum1_type colum_1_details,
-> column2_name colum2_type colum_2_details,
-> ...
->);

Importare tabelle da file sql

mysql -u username -p password db_name < file.sql

E' importante che nel file sql sia ha presente:

drop table table_name;

Copiare contenuto di una colonna

Per copiare il contenuto di column2 in column1, basta il comando:

UPDATE table_name
SET column1 = column2;
