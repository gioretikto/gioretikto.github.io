Local Area Network (LAN)

A Local Area Network (LAN) is a collection of devices connected together in one physical location, such as a building, office, or home. Hosts are conncted with the switch in the same netweork and can communicate with each other. A switch is a small hardware device used in the same building that joins the hosts to communicate with each other within one LAN. Technically, network switchd operate Data Link Layer. It can take a decision for the route of information. There is no need for a router to transmit the information between the hosts in the same network.

Network switches appear nearly identical to network hubs, but a switch generally contains more intelligence (and a slightly higher price tag) than a hub. Unlike hubs, network switches are capable of inspecting data packets as they are received, determining the source and destination device of each packer, and forwarding them appropiately. By delivering messages only to the connected device intended, a network switch conserves network and offers generally better performance than a hub.

One LAN can be connected to other LANs over any distance via telephone line and radio waves.
A WAN is a communications network that spans a large geographic area such as across cities, states, or countries. A WAN is used to communicate outside the organization, city and country.

A router derives its name from the fact that it can route data it receives from one network onto another. Routers are used to link the LAN to the WAN. A router is an intelligent device that is used to transmit information far away and can decide the shortest path to go to the destination.

A router is a layer three device. The network is identified by the router. Router A at the left-hand side is connected to Router B. Hosts are connected in the LAN with a switch and the switch is connected with the router. When Host 1 desires to connect to Host 4, Router A will route the information to Router B which will send this information to the host 4.

Routers

Router—A network device that forwards data by logical address and can also be used to segment large networks into smaller subnetworks. This logical addressing scheme is maintained by the IP protocol at the Internet layer. The logical address is called the IP address. So Routers operate at the Network layer (layer 3) of the OSI model.
Fig.1 describes communitation between a host on Network 1 and one on Network 2. The host at network 1 sends the message by using an application layer that is the user layer. The communication is moved to the transport layer. The information is used to break into small parts called segments. The port number and header are attached to segments. Segments are changed into an IP address at the netweork layer that is also called a packet. Packets are change to frame ath the data link layer. Error detection and flow control are the responsibilities of the data link layer. The frame in an envelope that has to be transmitted. The frame at the physical layer is converted into bits. Bits cannot be transmitted to the receiver. Bits are converted to signals and then transmitted to the receiver. The receiver repeats the same process in down to up approach. The message is seen by the user on the device.
