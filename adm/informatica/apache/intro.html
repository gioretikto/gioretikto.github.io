Introduzione a MySQL

Apache è il web server più utilizzato, sviluppato dalla Apache Software Foundation (http://www.apache.org/).

In questa sezione installeremo e configureremo Apache2, operazione che necessiterà i privilegi di root.

Si può avviare Apache automaticamente durante il boot o manualmente a seguito dell'avviamento del sistema.

To make sure that Apache is automatically started during boot in the targets multi-user.target and graphical.target, eseguire il seguente comando:

root #  systemctl enable apache2
>

Se all'avvio si ottiene il messaggio Access forbidden!, assicurarsi che ci sia la pagina index.html in htdocs/.

Se non si riesce a visualizzare la pagina index.php, assicurarsi che il modulo php7 sia installato e caricato, con a2enmod -l:

Per caricare il modulo usare:

a2enmod php7
