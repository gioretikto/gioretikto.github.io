Introduzione a php

Se tutto è andato a buon fine tramite il seguente codice sarà possibile visualizzare informazioni riguardandi i file di installazione di php:

<?php
php_info();
?>

Il file php.ini è un file di configurazione dell'interprete php lo si trova in /etc/php7/apache2/php.ini. Se si vogliono attivare le notifiche riguardanti i parse errrors, allora bisogna impostare:

display_errors = on
