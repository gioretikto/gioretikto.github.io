Il comando sort

Prima di procedere con le modifiche di un file bisogna accertarsi che questi sia nello Unix format con:

dos2unix pwd passwords.txt

Per ordinare una wordlist utilizzare il comando:

sort -o pwd pwd

Per eliminare righe doppie da una wordlist, utilizzare il comando uniq:

sort pwd | uniq >> prova

Per sostituire un carattere con un altro all'interno di una worlist si utilizza il comand tr:

<password.txt tr : $'\n' > newfile

Quest'ultimo comando sostituisce il carattere ":" con un carattere di newline '\n'.

Per eliminare dalla wordlist le parole con meno di 8 caratteri:

awk 'length($0)>8' newfile > newfile2

Per sottrarre il contenuto di una wordlist a quello di una seconda:

grep -vxF -f pwd passwords.txt >> out

che sottrare il contenuto di pwd a quello di passwords.txt.

Per eliminare tutte le righe che iniziano per "S" e per "e":

sed -e '/^[Se]/d' -e '/^$/d' myspace > space

So to delete all empty lines from a file called /tmp/data.txt, enter:

sed '/^$/d' /tmp/data.txt > /tmp/output.txt
