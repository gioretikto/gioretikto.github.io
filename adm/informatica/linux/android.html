Android File transfer

mtpfs

Installare go-mtpfs, poi

sudo go-mtpfs -allow-other ~/mnt

Al termine

sudo umount ~/mnt

Installare jmtpfs, which is a command line utility to be used like "mount" would and then execute

sudo jmtpfs /home/cadmio/media/

per smontare il dispositivo

umount /home/cadmio/media/

in caso la dir media sia vuota navigare con shell root.
