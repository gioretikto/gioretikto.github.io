apt

There are multiple ways to install packages in Debian of which the most common and easiest method is to use the set of default package management tools called apt which is an acronym for 'advanced packaging tool', the tool maintais a local database of software which are available on debian linux software repositories.

apt install

This is used for installing packages, upon issuing this command the apt package manager will provide a list of all dependencies which need to be satisfied for the given package along with the space required for installation ask for confirmation, if a confirmation is given by the user the package manager will go ahead and install the package along with all the dependencies required for the package.

Per evitare di installare la documentazione quando si installa un pacchetto usare l'opzione --no-install-recommends:

sudo apt install xfce4-screenshooter --no-install-recommends

apt search

Search option in apt is used for finding packages by name or type, this option provides a list of packages available within the repository matching the search criteria, this is particularly helpful when the exact package names arent' available. For example if we are interested in a program for taking screenshots, we could use

apt search screenshot



apt remove 



The remove option is used to remove the package matching the name provided, upon issuing this command the apt package manager will ask the user for confirmation for package removal, if a confirmation is given by the user the package manager will proceed to remove the package from the system.



apt pruge



purge, means to completely wipe away a package, including its configuration files.  Use «apt purge $package»



dpkg



Per mostrare i file inclusi in un pacchetto installato e loro collorazione usare dpkg -L package. Esempio



dpkg -L libglew-dev



apt-dev
