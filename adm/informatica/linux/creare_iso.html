Creare immagini ISO

Per creare copie identiche di un CD, DVD, o Blue ray o del contenuto di una directory all'interno di una immagine ISO si possono usare semplici comandi linux.

Create an ISO image from optical media

In questo esempio copieremo il contenuto di un CD/DVD drive (/dev/cdrom) all'interno di una immagine ISO. All'interno di una shell, eseguire il seguente comando:

dd if=/dev/cdrom of=/directory/example.iso

Notazioni utilizzate:

  • dd è il programma usato per convertire e copiare il file.

  • if definisce il file di input.

  • of definisce il file di output.

  • iso è la risultante immagine ISO.

Volendo caricare il file ISO su una pendrive USB. To prepare the USB stick, you will need a system where GNU/Linux is already running and where USB is supported. With current GNU/Linux systems the USB stick should be automatically recognized when you insert it. If it is not you should check that the usb-storage kernel module is loaded. When the USB stick is inserted, it will be mapped to a device named /dev/sdX, where the “X” is a letter in the range a-z.

sudo dd if=/Downloads/debian-12.0.0-amd64-netinst.iso of=/dev/sdb

Fai attenzione a non aggiungere i numeri della partizione come /dev/sdb1.

Creare una immagine ISO partendo da files in una directory

To create an ISO image from files within a directory is just as simple. State an output directory and name of the ISO to create, along with a source directory. For example: 

mkisofs -o /home/linuxlookup/example.iso /source/directory/

Convertire un'immagine mdf ad ISO

Basta utilizzare il programma mdf2iso, tramite il seguente comando

mdf2iso image.mdf image.iso

Copiare l'ISO su una pendrive

# umount /dev/sdX
# dd if=/path/to/downloaded.iso of=/dev/sdX bs=4M status=progress && sync
