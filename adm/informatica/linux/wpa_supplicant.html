WPA Supplicant

wpa_supplicant is a cross-platform supplicant (An entity at one end of a point-to-point LAN segment that seeks to be authenticated by an authenticator attached to the other end of that link) with support for WEP, WPA and WPA2 (IEEE 802.11i). It is suitable for desktops, laptops and embedded systems. It is the IEEE 802.1X/WPA component that is used in the client stations. It implements key negotiation with a WPA authenticator and it controls the roaming and IEEE 802.11 authentication/association of the wireless driver.

Install with

sudo apt-get install wpasupplicant

Configuration

Creare il file wpa_supplicant.conf in /etc/wpa_supplicant con il seguente contenuto

network={
         ssid="TP-LINK_2.4GHz_4F7005_2GEXT"
         psk="46781314"
}

or

network={
         ssid="TP-LINK_834432"
         psk="8QBNQU5HHAZ26"
}

Utilizzare il seguente script per collegarsi

ip link set wlxf4f26d13b2bd down
macchanger -m 0a:be:d9:23:28:8e wlxf4f26d13b2bd
ip link set wlxf4f26d13b2bd up
wpa_supplicant -B -i wlxf4f26d13b2bd -c /etc/wpa_supplicant/wpa_supplicant.conf -Dnl80211,wext
dhclient wlxf4f26d13b2bd
