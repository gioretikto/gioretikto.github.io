Raster Images

Il formato di file raster, insieme a quello vettoriale, è uno dei due principali tipi di file utilizzati per le immagini digitali. Uno dei suoi punti di forza è la capacità di acquisire grafiche dettagliate e fotografie di alta qualità scattate con fotocamere digitali. Esplora le caratteristiche principali delle immagini raster, i loro usi più comuni e le differenze rispetto alle immagini vettoriali.

Esistono due tipologie principali di file immagine digitali: raster (also known as bitmap images) e vettoriali. Foto digitali e grafiche dettagliate sono entrambe in formato raster. Tra i file raster più comuni troviamo le immagini JPEG, PNG e GIF.
Tuttavia, dal momento che contengono un numero fisso di pixel, le immagini raster possono diventare distorte o sfocate se vengono ridimensionate per riempire uno spazio più grande o più piccolo. Ecco perché le immagini vettoriali sono più comunemente utilizzate per illustrazioni e loghi: i vettori sono costituiti da formule matematiche e possono essere ingranditi o rimpiccioliti all’infinito. Ciò significa che, rispetto ai pixel, non perdono la loro risoluzione quando vengono ridimensionati.

A raster image is simply a 2D rectangular array of pixels. Each pixel can be exactly one color, usually stored as three numbers, for red, green and blue.

Rasters are also prevalen in input devices for images. A digital camera contain an image sensor comprising a grid of light-sensitive pixels, each of which records the color and intensity of light falling on it.
A desktop scanner contains a linear array of pixels that is swept across the page being scanned, making many measuraments per second to produce grid of pixels. But we don't always want to display an image this way. Even in television, the display rarely has the same number of pixels as the image being displayed.
