rand()

The stdlib.h header file provides a rand() function that generates a psuedo-random positive integer when called. This will, by default, return a number in the range from 0 to a large value RAND_MAX (at least 32,767) defined in the headr <stdlib.h>. A specific range can be set by using the modulus operator to specify a maximum upper limit. For example, to specify a range from 0 to 8 with the expression rand() % 9. We use the modulus operator in a way that we get a random number between 0 and 9. To set a minimum lower limit for the range its value can be added to the result of the expression. For example, to specify a range from 1 to 9 with the expression 1 + rand() % 9.

#include <stdio.h>
#include <stdlib.h>
int main()
{
	srand(time (NULL));
	printf("Random number between 0-99 : %d\n", rand()%100);
}

If floating random numbers between 0 and 1.0 are desired instead of between 0 and RAND_MAX, the following program is used

#include <stdio.h>
#include <stdlib.h>
int main()
{
	int i;
	for(i=0; i <10;i++)
		printf("%f\n", 1.0*rand()/RAND_MAX);
	return 0;
}

The value of rand()/RAND_MAX alone returns 0 as both rand() and RAND_MAX are integers and the result is truncated integer.

The same numbers are output again and again each time the program is run.

The numbers generated by the rand() function are not truly random as the function successively generates the same sequence of numbers each time a program using rand() is executed. In order to generate a different sequence of numbers the "seed" that starts the sequence must be specified. By default the initial see value is 1 but this can be changed by specifying an alternative integer argument to the srand() function. For example, srand(8). This will cause rand() to generate a different sequence of numbers to those generated using the default seed value but the sequence will still be repeated each time a program using rand() is executed. 

#include <stdio.h>
#include <stdlib.h>
int main()
{
	int i;
	printf("Enter see integer = ");
	scanf("%d", &i);
	srand(i);
	printf("%d\n", rand());
	return 0;
}

To generate different random sequences upon successive execution the argument specified to srand() must be something other than a static integer. The common solution is to seed the rand() function with the current time in elapsed seconds with srand(time(NULL)). The function time(), with the argument NULL return the elapsed time since 00:00:00 GMT, January 1, 1970, measured in seconds.

Now the sequence of numbers generated by rand will be different each time a program using rand() is executed.
