Gestione degli errori

Per gestire gli errori esiste un secondo flusso in uscita, detto stderr (da standard error).

#include <stdio.h>

    FILE *fp;
    if( (fp = fopen(argv[1], "r")) == NULL ){
        fprintf(stderr, "%s: non riesco ad aprire %s\n",argv[0],argv[1]);
        exit(1);
    }
