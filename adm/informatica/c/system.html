Executing a Shell Command: system()

The system() function allows the calling program to exercute an arbitrary shell command. 

#include <stdlib.h>
int system(const char *command);

The system() function creates a child process that invokes a shell to execute command. Here is an example fo a call to system():

#include <stdio.h>
#include <stdlib.h>

int main() {

	system("ls");
	
	return 0;
}

If command is a NULL pointer, the system() returns a nonzero value if a shell is available, and 0 if not shell is avaiable. So a shell might not be available if system() is not running on a UNIX system.
