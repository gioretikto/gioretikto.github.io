Binary Number System

We now look at binary numbers. Binary numbers follow very similar rules to decimal numbers. In fact the only real change is that the role of the the number 10 in writing a decimal number is replaced by the number 2. Consider the binary number 10112; it is a sum of multiples of powers of 2

10112 = (1 x 23) + (0 x 22) + (1 x 21) + (1 x 20)

Convert binary to decimal

to convert the binary number to decimal simply express the powers of 2 in decimal:

10112 = (1 x 23) + (0 x 22) + (1 x 21) + (1 x 20)
 = (1 x 8) + (0 x 4) + (1 x 2) + (1 x 1) = 11

Convert decimal to binary

Converting a decimal number into binary is essentially just the reverse of the process for converting binaty into decimal. For a small number such as 15 we can easily determine the powers of 2, the decimal number can be decomposed to

1510 = 8 + 4 + 2 + 1 =
23 + 22 + 21 + 20
1 x 23 + 1 x 22 + 1 x 21 + 1 x 20 = 11112

Real numbers to binary

A binary fraction number like 11.1012 is represented as

11.1012 = 1 ⋅ 21 + 1 ⋅ 20 + 1 ⋅ 2−1 + 1 ⋅ 2−3

Example 1. If we want to convert a real number such as 85.5, first convert the integer part (85) and then the fractional part (.5):

85 = 26 + 24 + 22 + 20 = 1010101

1/2 = 2−1

So 85.510 = 1010101.1 ■

Example 2. Convert the decimal number 5.625 to binary.

Solution. 5 is 101 in binary. For the fraction 0.625 notice that is clearly 5/8 so (22 + 20)/23 by multiplying numerator and denominator it by 23 you get 25 + 23 which should be the number 101000. So 101.101. ■
