Vettori di puntatori a funzione

I puntatori a funzioni sono utilizzati tipicamente nei cosiddetti sistemi guidati da menu. In questi sistemi si richiede all'utente di selezionare un'opzione da un menu (per esempio, da 1 a 3). Ogni opzione sarà servita da una funzione differente. I puntatori che fanno riferimento a ognuna di quelle funzioni sono immagazzinati in un vettore di puntatori a funzione. La scelta dell'utente sarà utilizzata come indice del vettore per rilevare il puntatore che sarà utilizzato per invocare la funzione.

Il prossimo programma fornisce un esempio generico dei meccanismi per la dichiarazione e l'utilizzo di un vettore di puntatori a funzioni. Sono state definite tre funzioni, function1, function2, function3, ognuna delle quali riceverà un argomento intero e non restituirà alcun valore. I puntatori che fanno riferimento a queste tre funzioni sono stati immaggazzinati nel vettore f, dichiarato come segue:

void (*f[3]) (int) = { function1, function2, function3};

La dichiarazione va letta cominciando dall'insieme di parentesi più a sinistra , "f è vettore di 3 puntatori a funzioni che ricevono come argomento un int e che restituiscono void. Il vettore è stato inizializzato con i nomi delle tre funzioni. Il valore, compreso tra 0 e 2, chje l'utente immetterà sarà utilizzato come indice per il vettore di puntatori a funzioni. L'invocazione della funzione sarà quindi effettuata nel modo seguente:

(*f[choice]) (choice);

In questa chiamata f[choice] selezionerà il puntatore immagazzinato nella posizione choice del vettore. Una volta risolto il riferimento del puntatore, la funzione sarà invocata e choice gli sarà fornito come argomento. Ogni funzione visualizzarà il valore dell'argomento ricevuto e il proprio nome di funzione per indicare che sarà stata richiamata correttamente.

#include <stdio.h>
#define SIZE 10

void function1 (int a);
void function2 (int a);
void function3 (int a);

int main()
{
	/*inizializza il vettori di 3 puntatori a funzioni */
	
	void (*f[3]) (int) = {function1, function2, function3};
	int choice;

	printf("Enter a number between 0 and 2, 3 to end:");
	scanf("%d", &choice);
	
	while ( choice >=0 && choice < 3) {
	
		(*f[choice]) (choice);
		printf("Enter a number between 0 and 2, 3 to end:");
		scanf("%d", &choice);
	}
	
	return 0;
}

void function1 (int a) {
	printf("You enetered %d so function1 was called\n\n", a);
}

void function2 (int a) {
	printf("You enetered %d so function2 was called\n\n", a);
}

void function3 (int a) {
	printf("You enetered %d so function3 was called\n\n", a);
}
