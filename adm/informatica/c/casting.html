Type Casting in C

Typecasting allows us to convert one data type into other. In C language, we use cast operator for typecasting which is denoted by (type).

Syntax:

(type)value

Without type Casting

int f = 9/4;
printf("f: %d\n",f);
//output 2

With type Casting

int f = (float)9/4;
printf("f: %f\n",f);
//output 2.250000

Another example is the following

int x;
for(x=97; x<=122; x++)
{
    printf("%c", (char)x);   /*Explicit casting from int to char*/
}

void * pointer

The void* is a generic pointer in the sense that it can point to a variable of any type. Any pointer can be cast to void

The void* pointer is guaranteed to be large enough to hold a pointer to any type of object, except a function type. A pointer to a variable of any type can be converted to a void* pointer and back again without losing any information. You can define a pointer of type void, but not a variable

Before you can use a void pointer, you must cast it to the proper type. Note that you don't need to cast a pointer to assing a value to it or to compare it with NULL.

#include <stdio.h>

//function prototype
void printString(void *ptr);

int main()
{
	char *str="Hi, there!";
	
	printString(str);
	
	return 0;
}

//function definition
void printString(void *ptr)
{
	printf("str: %s\n",ptr);
}
