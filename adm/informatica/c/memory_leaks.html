Memory Leaks

When dynamically allocated memory is no longer needed, it should be freed so that i can be reused again later. If the allocated memory is not freed, it causes a memory leak. Sometimes, a memory leak occur if we've lost the address of allocated memory. For example,

float f;

float *p = malloc (sizeof(float));

p = &f;

From the code above, note that the dynamically allocated memory's address has been lost since we've reassigned the new address to the pointer p before we will free the old one.
