ncurses

After installing the ncurses library, let's explore some of the important functions that are a part of this library:

  • initscr(): initializes the screen. It sets up the memory, and clears the command window screen.

  • refresh(): refreshes the screen.

  • getch(): will detect the user's touch, and will return the ASCII number for that particular key. The ASCII number is then stored in an integer variable, which is later used for comparison purposes.

  • printw(): print string values inside the command window.

  • keypad(): If the keypad function is set to trye, we can also take the user's input from the function keys, as well as the arrow keys.

  • break(). exit the program if the program is running in a lopp;

Otherwise your terminal might behave strangely after the program quits. endwin frees the memory taken by the curses sub-system and its data structures and puts the terminal in normal mode. This function must be called after you are done with curses mode.

int main(){

	initscr();		/*initializes and clear the screen */
	int keypressed = getch();

	}

	return 0;
}
