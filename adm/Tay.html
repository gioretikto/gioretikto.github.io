Taylor

A zero-order Taylor expansion, also known as a constant approximation, essentially represents the function's value at a specific point, according to Expii. For a function f(x), the zero-order Taylor expansion at x=a is simply f(a).

Se arrestiamo lo sviluppo di f(x) al primo ordine, otteniamo sen x = x + o(x), ovvero (sen x)/x = 1 + o(1). Otteniamo così, come caso particolare dello sviluppo di MacLaurin (arrestato al 1° ordine), il limite notevole

limx ⟶ 0 sen x/x = 1

Analogamenteo, dallo sviluppo di f(x) = log (1 + x) e di f(x) = ex, si ottengono altri limiti, tradizionalmente presentanti come limiti "notevoli".

Con Taylor al secon'ordine col resto secondo Lagrange si dia una maggiorazione del massimo errore che si commette approssimando (1+x)^1/2 con 1 + x/2, quando 0 <= x <= 1/2, e si precisi se l'errore commesso è per eccesso o per difetto.
Il massimo errore comesso è in realtà per x = 1/2: si calcoli questo errore. Il risultato è coerente con la stima effettuta?

Lo sviluppo al 2° ordine con resto secondo Lagrange è:

1 + x = 1 + x 2 − x 2 8 + R 3

dove il resto di terzo ordine, secondo Lagrange, è dato da:

R 3 ( x ) = f ( 3 ) ( ξ ) 3 ! x 3

f'(x) = 1/2⋅(1+x)−1/2
f′′(x) = −1/4⋅(1+x)−3/2
f′′′(x) = 3/8(1+x)−5/2

Quindi, il resto si scrive come:

R 3 ( x ) = 3 8 ( 1 + ξ ) 5 / 2 ⋅ x 3 6

L'errore massimo è dato:

E max = | − x 2 8 | + | R 3 |

Per massimizzare Emax occorre scegliere x = 1/2 e ξ = 0. Si ha quindi

(1/2)2/8 = 1/32 = 0.03125

e

R 3 max = 3 8 ( 1 + 0 ) 5 / 2 ⋅ ( 1 / 2 ) 3 6 = 3 8 ⋅ 1 48 = 3 384 ≈ 0.0078 .

Quindi, l'errore totale massimo è:

Emax ⁡≈ 0.03125+0.0078 ≈ 0.039

Il termine di errore principale −x2/8 è negativo, il che indica che l'approssimazione iniziasle è per eccesso. ■

    Calcolare i seguenti limiti utilizzando opportunamente, se necessario, gli sviluppi di Taylor.

    1. lim x → + ∞ log ⁡ 1 + x 2 x 4

      Solutione. Per x → +∞, possiamo approssimare:

      log (1 + x2) ≈ log x2 = 2 log x ≈ log x

      Pertanto, il limite si riduce a:

      limx → +∞ log x/x1/4

      A questo punto, osserviamo che la funzione log⁡ x cresce molto più lentamente di x1/4, e dunque il rapporto tende a zero. Oppure possiamo applicare De L'Hôpital

      limx → +∞ (1/x)/[(1/4)x−3/4] = limx → +∞ (4/x) ⋅ (x3/4) = 4limx → +∞ x−1/4 = 0 ■

    2. lim x → + ∞ x α ( x 2 + 2 x + 3 − x − 1 )

      Soluzione. Possiamo riscrivere il termine sotto radice come

      x 2 + 2 x + 3 = x 2 ( 1 + 2 x + 3 x 2 )

      Poiché

      √(1 + y) = 1 + y/2 + o(y)

      Possiamo usare tale sviluppo e riscrivere come

      x 2 + 2 x + 3 = x 1 + 2 x + 3 x 2 = x ( 1 + 1 x + 3 2 x 2 + o ( x ) ) = x + 1 + 3 2 x + o ( x )

      Quindi sottraendo x+1, abbiamo: 3/(2x) + o(x). Ora consideriamo il limite dato:

      lim x → + ∞ x α ⋅ 3 2 x

      La convergenza del limite dipende dal valore di α:

      • Se α−1 < 0 , allora xα−1 → 0 xα - 1 ⟶ 0 e il limite è zero;

      • Se α−1 > 0, allora xα−1 → +∞ xα - 1 ⟶ +∞ e il limite è infinito;

      • Se α = 1, il limite è costante e vale 3/2. ■
