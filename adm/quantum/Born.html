The statistical interpretation

Born's statistical interpretation of the wave function, says that |Ψ(x,t)|2 gives the probability of finding the particle at a point r, at time t - or more precisely,

∫ba |Ψ(x)|2 dx = probability of finding the particle between a and b, at time t.

The wave function itself is complex, but |Ψ(x)|2 = Ψ*(x)Ψ(x) is real and nonnegative - as a probability, of course, must be.

Probability is the area under the graph of |Ψ(x)|2. For the wave function in Fig.1, you would be quite likely to find the particle in the vicinity of point A, where |Ψ|2 is large, and relatively unlikely to find it near point B.

The statistical interpretation introduces a king of indeterminacy into quantum mechanics, for even if you know everything the theory has to tell you about the particle (to wit: its wave function), still you cannot predict with certainty the outcome of a simple experiment to measure it position - all quantum mechanics has to offer is statistical information about the possible results. A particle simply does nto have a precise position prior to mearurement. Space localization ca be achieved by measuring position with infinite precision so that the indeterminacy Δx vanishes. What if I meade a second measurement, immediately after the first? A repeated measurement (on the same particle) must return the same value. Indeed, it would be tough to prove that the particle was really found at C in the first instance, if this coul not be confirmed by immediate repetition of the measurement. Evidently the first measurement radically alters the wave function, so that is is now sharply peaked about C (Fig.2). We say that the wave function collapses, upon measurement, to spike at the point C (it soon spreads out again, in accordance with the Schrödinger equation, so the second measurement must be made quickly).

Normalization

Because the particle must be found somewhere between x = -∞ and x = + ∞, the wave function must obey the normalization condition.

∫ba |Ψ(x)|2 dx = 1 (1.6.1)

Without this, the statistical interpretation would be nonsense. However, this requirement should disturb you: After all, the wave function is supposed to be determined by the Schrödinger equation - we can't go imposing an extraneous condition on Ψ without checking that the two are consistent. Well, a glance at Schrödinger equation reveals that if Ψ(x,t) is a solution, so too is AΨ(x,t), where A is any (complex) constant. What we must do then, is pick this undetermined multiplicative factor so as to ensure that 1.6.1 is satisfied. This process is called normalizing the wave function. For some solutions to the Schrödinger equation the integral is infinite; in that case no multiplicative factor is going to make it 1. The same goes for the trivial solution Ψ = 0. Such non-normalizable solutions cannot represent particles, and must be rejected. Physically realizable states correspond to the square-integrable solutions to Schrödinger equation.

We shall show now that if the wave function is normalized at time t = 0, it will stay normalized as time goes on. The Schrödinger equation automatically preserves the normalization of the wave function - without this crucial feature the Schrödinger equation would be incompatible with the statistical interpretation and the whole theory would crumble. To prove it, begin with

d d t ∫ − ∞ + ∞ | Ψ ( x , t ) | 2 d x = ∫ − ∞ + ∞ ∂ ∂ t | Ψ ( x , t ) | 2 d x (1.6.2)

Note that the integral is a function only of t, so the total derivative (d/dt) in the first expression was used, but the integrand is a function of x as well as t, so it's a partial derivative (∂/∂t) in the second one. By the product rule,

∂ ∂ t | Ψ | 2 = ∂ ∂ t ( Ψ ∗ Ψ ) = Ψ ∗ ∂ Ψ ∂ t + ∂ Ψ ∗ ∂ t Ψ

Now the Schrödinger equation says,

∂ Ψ ∂ t = i ℏ 2 m ∂ 2 Ψ ∂ x 2 − i ℏ V Ψ

and hence also (taking the comple conjugate)

∂ Ψ ∗ ∂ t = − i ℏ 2 m ∂ 2 Ψ ∗ ∂ x 2 + i ℏ V Ψ ∗

so

∂ ∂ t | Ψ | 2 = i ℏ 2 m ( Ψ ∗ ∂ 2 Ψ ∂ x 2 − ∂ 2 Ψ ∗ ∂ x 2 Ψ ) = ∂ ∂ x [ i ℏ 2 m ( Ψ ∗ ∂ Ψ ∂ x − ∂ Ψ ∗ ∂ x Ψ ) ]

The integral in Equation 1.6.2 can now be evaluated explicitly:

d d t ∫ − ∞ + ∞ | Ψ ( x , t ) | 2 d x = i ℏ 2 m ( Ψ ∗ ∂ Ψ ∂ x − ∂ Ψ ∗ ∂ x Ψ ) | − ∞ + ∞

But Ψ(x,t) must go to zero as x goes to (±) infinity - otherwise the wave function would not be normalizable. It follows that

d d t ∫ − ∞ + ∞ | Ψ ( x , t ) | 2 d x = 0

and hence that the integral is constant (independent of time); if Ψ is normalized at t = 0, it stays normalized for all future time.
