Classi e oggetti

I linguaggi di programmazione orientati agli oggetti utilizzano classi e oggetti. Ogni programma Java ha almeno una classe, quindi tutti i programmi Java sono orientati agli oggetti.

Una classe è un insieme di variabili e funzioni. Le variabili di una classe sono note come data members, le funzioni delle classi sono note come metodi. Gli oggetti sono istanze delle classi. Ogni oggetto creato avrà le proprie copie dei campi definiti nella classe, noti come instance variables. Le class variables sono invece data members, variabili condivise tra più oggetti.

Si può pensare alle classi, come progetti per la creazione i oggetti. Si pensi alla classe come al progetto per la costruzione di una casa e alla casa come l'oggetto che deriva dal progetto.
