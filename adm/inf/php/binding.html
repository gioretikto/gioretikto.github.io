Binding parameters

Per associare ad un parametro una variabile si usa il seguente codice:


<?php
/* Execute a prepared statement by binding PHP variables */
$calories = 150;
$colour = 'red';
$sth = $dbh->prepare('SELECT name, colour, calories
    FROM fruit
    WHERE colour = ?');
$sth->bindParam(1, $colour);
$sth->execute();
?>
