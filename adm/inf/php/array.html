Array

Un array Ã¨ un insieme di variabili accomunate da uno stesso nome. Ad esempio possiamo creare un array contenente i nomi di una classe di studenti:

$classe = array('Gino', 'Mario', 'Mirco', 'Ciro', 'Anna');

Per visualizzare il valore di un membro dell'array utilizziamo gli indici:

echo $classe[3]; // Visualizza il nome Ciro

Ogni volta che si aggegna un valore dell'array, la prima locazione vuota viene utilizzata, quindi si possono assegnare i valori dell'array anche in questo modo:

<?php
$classe[] = "Gino;
$classe[] = "Mario";
$classe[] = "Mirco";
$classe[] = "Ciro";
echo $classe[2];
?>
