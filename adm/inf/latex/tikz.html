TikZ

Drawing straight lines

Straight lines are given by coordinates separated by a double minus,

\draw (1,0) -- (0,0) -- (0,1);

To draw a line you do something like

\begin{tikzpicture}
\draw (0,0) --(1,2);
\end{tikzpicture}

Changing the thickness of lines

Other decorations include changing the thickness:

\begin{tikzpicture}
\draw [ultra thick] (0,1) -- (2,1);
\draw [thick] (0,0.5) -- (2,0.5);
\draw [thin] (0,0) -- (2,0);
\end{tikzpicture}

The result is a L shape graphic.

A connected path can be closed using the "--cycle" operation: 

\draw (1,0) -- (0,0) -- (0,1) -- cycle;

2D Regions

To represent a region of the plane parametrised by functions, keep in mind that a function 2x −x3 as to be entered as 2*x−x3.

There are different methods to place axis descriptions. One of them is to provide coordinates relativeto the axis’ rectangle such that(0,0) is the lower left corner and(1,1)is the upper right corner – this isvery useful for figure titles or legends. Coordinates of this type, i.e. without unit like (0,0) or (1.03,1), are called axis description cs (the cs stands for “coordinate system”).

\documentclass[preview, convert={size=424x334,outext=.png}]{standalone}
\usepackage{pgfplots}
\usepgfplotslibrary{fillbetween}
\pgfplotsset{compat=1.17}
\begin{document}
        \begin{tikzpicture}
            \begin{axis}[ 
                xmax=1.1,
                axis lines=center,
                xlabel=$x$,
                xlabel style={at={(axis description cs:1.01,0.0)},anchor=north},
                ylabel = $y$
                ]
                \addplot [name path=A,domain=0:1, smooth] {2*x-x^3} node[pos=.7,left] {$f$};
                \addplot [name path=B,domain=0:1, smooth] {x-x^2} node[pos=.9,left] {$g$};
                \draw [thick] (1,0) -- (1,1);
                \addplot[gray!30] fill between[of=A and B];
            \end{axis}
        \end{tikzpicture}
\end{document}

Plot parametric equations

The parameter must be expressed in deg and not radians.

\documentclass[preview, convert={size=424x334,outext=.png}]{standalone}
\usepackage{pgfplots}
\usepgfplotslibrary{fillbetween}
\pgfplotsset{compat=1.17}
\begin{document}

\begin{tikzpicture}
            \begin{axis} [xlabel = x, ylabel = y, xmax=3.1, xmin = -3, ymax=5, ymin = -1.6, axis lines=center]
	\addplot [domain=0:pi,smooth, samples=300, line width=0.7pt]({2*cos(deg(x))}, {3*sin(deg(x))});
\end{axis}
\end{tikzpicture}
\end{document}
