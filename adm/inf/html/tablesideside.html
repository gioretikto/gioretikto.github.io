Allineare due tabelle una accanto all'altra

Il codice CSS è il seguente:


<style>

table.smalltable {
width:17.500em; /*280px */
margin:5px;
border-collapse: collapse;
display:inline-block;
}

table.smalltable td {
font-size:100%;
padding: 5px;
text-align:center;
vertical-align:top; }

</style>
