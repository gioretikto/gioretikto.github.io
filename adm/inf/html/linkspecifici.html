Collegamenti a parti specifiche di un file HTML

Per creare collegamenti a specifiche parti di una pagina HTML basta porre un identificatore nel paragrafo da collegare:

<div id="permanenza" >

ed indicare nel collegamento il nome dell'id:

<a href="http://www.esempio.com/unacertapagina.html#permanenza">Click here</a>
