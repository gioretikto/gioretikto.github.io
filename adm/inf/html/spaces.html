Different ways to add Breaks and Spaces in HTML

&nbsp; Entity: The &nbsp; entity represents a non-breaking space, preventing the browser from collapsing consecutive spaces.

If you wanted to display the following:

three   spaces.

You could insert 3 &nbsp; entities instead of using spaces like so:

three &nbsp;&nbsp;&nbsp; spaces.

  • Type “&nbsp;” to add a single space.

  • Type “&nsp;” to add 2 spaces.

  • Type “&msp;” to add 4 spaces.

  • Use the non-breaking space (&nbsp;) 4 times to insert a tab.

Sometimes, HTML might break up words that are supposed to be together into another line – For example, if the author was referring to the name Pepsi Cola and he wanted to be certain that the Cola part didn't break to the next line on the web page he would type Pepsi Cola in the source code.

The <pre> tag defines preformatted text and preserves both spaces and line breaks.
