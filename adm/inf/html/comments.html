Including JavaScript in your web Page

JavaScript programs are passed to the browser along with page content. We can either include JavaScript statements directly into the HTML file,

<script>

	.... Javascript statemets are written here ...
</script>

or write the JavaScript code into a separate file and use the <script> tag to include that file by name using the src attribute:

<script> src="mycode.js"></script>

You're not limited to using a single script element. You can have as many of them on your page as you need.

Comments

Si possono includere commenti nella pagina che non verranno visualizzati dal browser:

<!-- This is a comment -->

JavaScript can also use the HTML comment syntax for single-line comments. However, this is not commonly used in JavaScript programs. JavaScript comments start with two slash characters // and extend to the end of the line. JavaScript also supports multiline comments enclodes in /* ... */, like this:

//this is a comment

/* this
is a
comments */
