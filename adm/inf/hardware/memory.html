Memory

The other fundamental component of a computer, according to the scheme described by Von Neumann, is the memory. Memories are realized by semiconductors devices and are idealized as bit tables whose columns and rows are the memory cells; The smallest unit of memory,called a cell, can be used to store a single bit of data: that is, a logic 0 or a logic 1. Usually, the cells in a memory array are accessed in groups, and each group is called a word. The cells in a word are located in the same horizontal row in the array. A word can contain 8, 16, 32, or even more bits, which depends on the system requirements.

A read cycle begin with a word-line being chose. The word-line will enable all cells in the chosen row. If the memory has 2k rows and 2m columns,m then it has 2(k + m) cells. There will be 2k word-lines. These word-loines have to be enables only one at a time. For example, if there are 16 word-lines and we need to enable, the 6th, then we only need to supply the word "0110", to indicate that line number 6 needs to be activated.
The operation can be performed by a decoder that accepts k-bits and provides 2k outputs bits. The decoder operates by enabling the word-line indicated by the k-bit word-line indicate by the k-bit word and disabling the rest of the lines. This is the function of the row decoder.
