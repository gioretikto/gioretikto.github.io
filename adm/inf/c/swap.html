Reordering arrays

Suppose we've an array of 8 elements and we want to move the last element in the 3rd position. This is how the algorithm works

Initial state:
1, 2, 3, 4, 5, 6, 7  positions
a, b, c, d, e, f, g  contents

To move the value in position 7 to position 3:

Swap 7 and 6
1, 2, 3, 4, 5, 6, 7
a, b, c, d, e, g, f

Swap 6 and 5
1, 2, 3, 4, 5, 6, 7
a, b, c, d, g, e, f

Swap 5 and 4
1, 2, 3, 4, 5, 6, 7
a, b, c, g, d, e, f

Swap 4 and 3
1, 2, 3, 4, 5, 6, 7
a, b, g, c, d, e, f

Done.

This is the code for swapping the elements

#include <stdio.h>

void swap(int *list, int a, int b)
{
	int tmp = list[a];

	list[a] = list[b];

	list[b] = tmp;
}

int main()
{
	int a[8] = {1,2,3,4,5,6,7,8};

	int i = 0;
	int start_page = 6;
	int dest_page = 2; /* destination page */

	printf("The array at the beginning is\n");

	for(i = 0; i < 8; i++)
		printf("%d\t", a[i]);

	if (start_page > dest_page)
		for(i = start_page; i > dest_page; i--) { swap(a, i - 1, i); }

	else
		for(i = start_page; i < dest_page; i++) { swap(a, i + 1, i); }

	printf("\nThe array after reordering is\n");

	for(i = 0; i < 8; i++)
		printf("%d\t", a[i]);

	return 0;
}

Bubblesort

There exist complex algorithms for sorting arrays. We discuss a rather elementary and inefficient sorting algorithm called BubbleSort. The basic idea of this algorithm is to let greater elements "bubble" (as bubbles rise in a liquid, hence the algoritm name) to the right-hand side of the array. The sorting process is as follows

  • First, compare two elements in the neighbor. If they are not in the proper order, exchange them (that is swap them). Therefore, the greatest elements "bubbles" to the most right. Thus n numbers require n âˆ’ 1 comparisons;

  • each number is examines at most n times.

#include 

void bubbleSort(int *list, int size)
{
	int i,j,tmp;

	for (i = 0; i < size - 1; i++)
	{
		for (j = size - 1; j > i; j--)
		{
			if(list[j-1] > list[j])
			{
				tmp = list[j-1];
				list[j-1] = list[j];
				list[j] = tmp;
			}
		}
	}
}

int main()
{
	const int size = 8;

	int a[] = {8,2,7,4,5,6,3,1};

	int i = 0;

	printf("The array at the beginning is\n");

	for(i = 0; i < size; i++)
		printf("%d\t", a[i]);

	bubbleSort(a, size);

	printf("\nThe array after sorting is\n");

	for(i = 0; i < size; i++)
		printf("%d\t", a[i]);

	return 0;
}
