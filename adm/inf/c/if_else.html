L'istruzione if

Mole istruzioni C si basano su un'espressione condizionale che determina l'azione che deve essere eseguita. Un'espressione condizionale può fornire un risultato vero o falso (rispettivamente true e false). In C il valore vero corrispondene a qualsiasi valore diverso da zero, inclusi i numeri negativi. Il valore falso corrisponde allo 0.

Vi sono due istruzioni di selezione if e switch. In alcue circostanze in alternativa si può usare l'operatore ?.
La forma generale dell'istruzione if è la seguente:

if (espressione)
	istruzione;
else
	istruzione;

dove istruzione può essere una singola istruzione, un blocco di istruzioni o un'istruzione nulla. La clausola else è opzionale. Se espressione fornisce un risultato vero (diverso da zero), viene eseguita l'istruzione o blocco relativo alla parte if; in caso contrario verrà eseguita, l'isutrzione (o il blocco) else.

L'istruzione condizionale che controlla if deve produrre un risultato scalare. Uno scalare può essere un intero, un carattere, un puntatore o un numero in virgola mobile. In C99 anche _Bool è uno scalare e pertanto può essere utilizzato in un'istruzione if.

Raramente si usa un numero in virgola mobile per controllare un'istruzione condizionale in quanto questo rallenta molto il tempo di esecuzione (poiché per eseguire un'operazione in virgola mobile occorrono molte più istruzioni di codice macchina rispetto alla corrispondente operazione eseguita su un intero o un carattere.

Costrutto if-else-if

Quando viene trovata una condizione vera, viene eseguita l'istruzione associata e la parte rimanente del costrutto viene ignorata.

#include <stdio.h>

int main(int argc, char *argv[])
{
	if (0)
		printf("Ciao mondo\n");
	else if (-1)
		printf("print -1\n");
	else if (2)
		printf("print 2\n");
	else
		printf("Basta");
	
	return 0;
	
}
