The int ungetc(int c, FILE *fp) Function

The ungetc() function takes a single character and shoves it back onto an input stream. It is the opposite of the getc() function, which reads a single character from an input stream. Also, ungetc() is an input function, not an output function. Syntax:

int ungetc(int c, FILE *stream)

riporta il carattere c di nuovo sul file fp, e restituisce c oppure EOF in caso di errore. Per ogni file è garantito l'accantonamento di un solo carattere; ungetc può essere usata con tutte le le funzioni per la gestione dei dati in ingresso, come scanf, getc o getchar.
