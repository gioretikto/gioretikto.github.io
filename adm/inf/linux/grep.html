Il comando grep

Il comando grep mostra le linee che combaciano con un pattern. Può essere utilizzato per trovare del testo in un file and search a directory structure of files recursively. It also supports showing the context of a match by showing lines before and after the result and has support for regular expressions in pattern matching.

Mostrare il numero della linea del match

Per mostrare la linea del risultato del matching basta aggiungere l'opzione -n. Questo comando mostrerà il match e la linea in cui questo occorre nel file.

grep 'computer' -n /usr/share/dict/myfile
    1545

Per cercare una stessa parola all'interno di più file, come ad esempio la parola 'Template' in file sorgente .c:

grep -n Template *.c

Questa opzione è molto utitle volendo modificare un file ad esempio con vim and subito alla linea del file che si vuole modificare.

vim +1545 /usr/share/dict/myfile

How to print lines before and after a match

To print lines before and after a match the -A and -B options can be used. Both expect a number and will print this number of lines.

grep -B 2 -A 2 'computer' /usr/share/dict/words
    computativeness
    compute
    computer
    computist
    computus

How to use extended regular expressions when searching

To use extended regular expressions use the -e option. The following line matches lines that do not contain the words ‘foo’ or ‘bar’.

grep -v -e 'foo' -e 'bar'

Subtraction of file1 from file2

Per sottrarre il contenuto di una wordlist a quello di una seconda:

grep -vxF -f pwd passwords.txt >> out
