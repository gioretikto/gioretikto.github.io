Il comando chattr

Il comando chattr (Change Attribute) modifica certi attributi di file e directory in modo da evitare l'eventuale modificazione e cancellazione di questti anche da parte dell'utente root.

I file dei filesystem "ext" (ext2, ext3, etc.) supportano tutti questi attributi evidenziabili con il comando lsattr.

Ad esempio per evitare la scrittura del file /etc/resolv.conf utilizziamo il comandi


chattr +i /etc/resolv.conf

come possiamo vedere con lsattr, l'attributo immutable Ã¨ stato impostato:

# lsattr /etc/resolv.conf
----i----------- /etc/resolv.conf

per rimuovere l'attributo

chattr -i /etc/resolv.conf

The operator can be used

  • +: which adds selected attributes to attribute list

  • -: which removes selected attributes from attribute list

  • =: which forces selected attributes only.

      • Some of available attributes are the following.

      • a: can be opened in append mode only.

      • A: do not update atime (file access time).

      • c: automatically compressed when written to disk.

      • C: turn off copy-on-write.

      • i: set immutable.

      • s: securely deleted with automatic zeroing.
