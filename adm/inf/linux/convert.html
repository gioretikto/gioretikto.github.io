convert

When you deal with images on a regular basis, it's often useful to convert them to different image formats. You can use graphical tools such as the GIMP to open files and save them into different formats, but if you deal with a lot of images you might find that process cumbersome. The ImageMagick tool convert solves this problem by providing a command-line interface to image conversion. You can install it with

sudo apt-get install imagemagick

With convert you can change any ImageMagick-supported image format to any other ImageMagick-supported image format.

convert images/*.png -resize 90x131 *.png

To generate a pdf file out of two jpg images:

convert file1.jpg file2.jpg -resample 50% -compress jpeg out.pdf
