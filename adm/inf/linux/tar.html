Comando Tar

Creare archivi e comprimere intere directory utilizzando una shell in Linux/UNIX è davvero semplice, ed è molto utile per effettuare il backup dei files. A questo scopo si utilizza il comando GNU tar, il quale esegue i seguenti passaggi

  • Crea un archivio

  • Comprime l'archivio

Il comando tar viene utilizzato con la seguente sintassi:

tar -cvzf output.tar.gz file.pdf

Dove

  • z : Comprime l'archivio usando il programma gzip

  • c : Crea l'archivio

  • v : Verbose i.e mostra i file archiviati durante l'esecuzione

  • f : Nome dell'archivio/file :-f has to be last in a string of single-lettre switches and has to be followed immediately by the name of the file you're creating or untarring.

Ad esempio avendo una directory chiamata /home/mario/video volendo comprimere la directory si utilizza il comando tar nel seguente modo

$ tar zcvf raccolta_video.tar.gz /home/mario/video

Con questo comando si creerà un archivio compresso chiamato raccolta_video.tar.gz nella directory corrente.

Volendo estrarre i file dall'archivio si utilizza il seguente comando

$ tar zxvf raccolta_video.tar.gz

L'opzione -x: serve per estrarre i files.

Se si vuole creare un tarball del contenuto di una directory senza includere la directory stessa si utilizza -C - change to directory DIR

tar -c Music -zcvf music.tar.gz

Volendo estrarre i file in una particolare directory ad esempio all'interno di /tmp si utilizza il seguente comando:

$ tar zxvf prog-1-jan-2005.tar.gz  -C /tmp

Opzioni avanzate

Se si vuole escludere una directory dall'archiviazione si può utilizzare il comando tar con l'opzione --exclude nel seguente modo:

tar --exclude='phpMyAdmin' -zcvf webservernew.tar.gz -C /srv/www htdocs

or to exclude file based on extension

tar zcvf litos2.2.tar.gz --exclude='*.o' litos4/

L'opzione -C, --directory=DIR, cambia la directory a DIR.

Questo comando creerà un archivio contenente tutti i files e directory di /srv/www/htdocs/ ad eccezzione della directory chiamata phpMyAdmin.
