List of IRC commands:

Change Nick

/nick NewNickname.

Register your IRC nick:

/msg NickServ REGISTER password youremail@example.com

Identificare il nickname

/msg NickServ identify username password

To formally register the channel, and set yourself as Founder.

/msg chanserv register #CHANNEL_NAME

To give yourself operator status

/msg chanserv op #cansi

To give a user OP status

/mode #channel +o user

to enable the automatic op +O - Enables automatic op.

/msg ChanServ flags #channel thenick +O

To give voice to user

/msg chanserv flags #cansi username +V

Impostare il Topic: Inside the new channel, assign a topic to it by typing

/topic Welcome in cansi! a channel about....

if there are no user in the channel the topic will expire to avoid this:

/msg ChanServ SET #foo KEEPTOPIC ON
