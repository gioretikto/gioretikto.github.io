Create a png

With the standalone class you can produce PDF files and with the convert option turn the produced PDF into a graphics file automatically (using external software, which requires the -shell-escape compiler option). Per la conversione si utilizza imagemagick

sudo apt-get install imagemagick
With the command

pdflatex --shell-escape formula.tex

Nel preambolo del codice sorgente si utilliza

\documentclass[preview, border=1.5 pt, convert={size=800x800,outext=.png}]{standalone}
