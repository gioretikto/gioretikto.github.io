Some techinique of the calculus of variations

Consider first the problem in an essentially one-dimensional form: We have a function f(y, ẏ, x) defined on a path y = y(x) between two values x1 and x2. We wish to tind a particular y(x) such that the line integral J of the function f between x1 and x2

J = ∫x1x2 f(y, ẏ, x) dx

has a stationary value, relative to paths differing infinitesimaly from the correct function y(x). We put the problem in a form that enables us to use the familiar apparatus of the differential calculus for finding stationaty points of a function. Since J must have a stationary value fro the correct relative to any neighboring path, the variation must be zero relative to some particular set neighboring paths labeled by an infinitesimal parameter α. Such a set of paths might be denoted by y(x,α) with y(x,0) the correct path. For example, if we select any function η(x) that vanishes at x = x1 and x = x2, then a possible set of varied paths is given by

y(x,α) = y(x,0) + αη(x)   1.1.1

For simplicity, it is assumed that both the correct path y(x) and the auxiliary function η(x) are well-behaved functions − continuous and nonsingular between x1 and x2, with continuous first and second derivatives in the same interval. For any such paremetric family of curves, J is also a function of α:

J ( α ) = ∫ x 1 x 2 f ( y ( x , α ) , y ˙ ( x , α ) , x ) d x

and the condition for obtaining a stationary point is the familiar one that

(dJ / dα)α = 0 = 0   1.1.2

By the usual methods of differentiating under the integral sign, we find that

d J d α = ∫ x 1 x 2 ( ∂ f ∂ y ∂ y ∂ α + ∂ f ∂ y ˙ ∂ y ˙ ∂ α ) d x  1.1.3

Consider the second of these integrals:

∫ x 1 x 2 ∂ f ∂ y ˙ y ˙ ∂ α d x = ∫ x 1 x 2 ∂ f ∂ y ˙ ∂ 2 y ∂ x ∂ α d x

Integrating by parts, the integral becomes

∫ x 1 x 2 ∂ f ∂ y ˙ ∂ 2 y ∂ x ∂ α d x = ∂ f ∂ y ˙ ∂ y ∂ x | x 1 x 2 − ∫ x 1 x 2 d d x ( ∂ f ∂ y ˙ ) ∂ y ∂ α d x  1.1.4

The conditions on all the varied curves are that they pass through the points (x1, y1), (x2, y2), and hence the partial derivative of y with respect to α at x1 and x2 must vanish. Therfore, the first term of (1.1.3) vanishes and Eq. (1.1.4) reduces to

d J d α = ∫ x 1 x 2 ( ∂ f ∂ y − d d x ∂ f ∂ y ˙ ) ∂ y ∂ α d x

The condition for a statioary value, Eq. 1.1.2, is therefore equivalent to the equation

∫ x 1 x 2 ( ∂ f ∂ y − d d x ∂ f ∂ y ˙ ) ( ∂ y ∂ α ) 0 d x = 0   1.2.5

Now, the partial derivative of y with respect to α occuring in Eq. 1.2.5 is a function of x that is arbitrary except for continuity and end point conditions. For example, for the particular parametric family of varied paths given by Eq. 1.1.1, is the arbitrary function η(x). We can therefore apply to Eq. 1.2.5 the Fundamental lemma of calculus of variations, which says that if

∫ x 1 x 2 M ( x ) η ( x ) = 0

for all arbitrary functions η(x) continuous through the second derivative, then M(x) must identically vanish in the interval (x1, x2). From Eq. 1.2.5 and the fundamental lemma, it therefore follows that J can have a stationary value only if

∂ f ∂ y − d d x ( ∂ f ∂ y ˙ ) = 0
