Compactness

Compactness is a generalization of being bounded and closed in Euclidean space.

Like connectivity, compactness is one of the most important concepts in topology and analysis. One reason for its importance is that a continuous function whose domain is a compact set behaves very well. Indeed, you have already encountered compactness in calculus in the extreme value theorem that says that a continuous real-valued function on a closed and bounded interval attains both its maximum and minimum values; since all of the theory of the Riemann integral rests on this theorem, it is one of the most important theorems underlying the calculus. It relates to compactness because a closed and bounded interval is an example of a compact subset of ℝ.
In this chapter, we will investigate compactness in general. At first you may find the study of compactness more difficult than connectivity, but after you get used to the somewhat strange definition, you should find it a fascinating subject.

Definition 5.1.1 Let A be a subset of a topological space X. An open cover of A is a collection {Ui}i∈I of open subsets of X so that A lies in the union of the elements of {Ui}i∈I, i.e.

A ⊆ ⋃ i ∈ I U i

A subcover of {Ui}i∈I is a collection {U'i}i∈I ⊆ {Ui}i∈I so that A lies in the union of the elements of {U'i}i∈I, i.e.

A ⊆ ⋃ i ∈ I U i ′

A cover if finite if there are only finitely many sets in the family (that is, if the index set is finite).  □

Definition (Compactness of Subsets). Let A be a subset of a topological space (X, τ). Then A is said to be compact if every open cover of A has a finite subcover. □

Example. The interval (0,1) in Euclidean line is not compact. Consider the open intervals Un = (1,/1/n), n = 1,2,.. The family {un: n ∈ ℕ} is an open cover for (0,1) since for every a ∈ (0,1) there exists a positive integer k such that k > 1/a (Archimedean property of integers), and therefore a ∈ Uk. However. no finite subfamily of this family can cover (0,1) since the union of any such subfamily Un1, ..., Un, is equal to (1/m,1), where m = max {n1, ..., ns}.

Heine-Borel Theorem. Any closed interval [a,b] ⊂ ℝ is compact.

Proof. Let G = {Gi:i ∈ I} be an open cover of [a,b]. Let C denote the set of all points c ∈ [a,b] satisfying the condition that [a,c] is covered by a finite subcollection of open sets of G. Observe that a ∈ C and hence C ≠ ∅. Further, if c ∈ C and z ∈ [a,c], then [a,z] is covered by a finite sub-collection of open sets of G, so z ∈ C, and hence the entire interval [a,c] ⊂ C. Thus the set C is an interval of the form [a,x]. We claim that x = b. Assume, to the contraty, that x ∉ b, then clearly x < b. Since x ∈ Gi for some i and each Gi is open, there is an interval (x − ε, x + ε) ⊂ Gi. The fact that [a,x] is covered by a finite number of open sets in G implies that [a, x + ε/2] is also covered by a finite number of open sets in G. We have thus a contradictions and this completes the proof.

Not only is compactness a topological property (preserved by homeomorphisms), it is preserved by any continuous function.

Theorem. The continuous image of a compact set is compact. That is, if f: X → Y is a continuous function and A ⊆ X is compact, then f(A) is compact in Y.

Proof. Suppose f: X → Y is a continuous function, A ⊆ X is compact, and C is an open cover of f(A). Then fsup>−1(C) = {f−1(V): V ∈ C} is an open cover of A, so there exists a finite subcover {f−1(V1, ..., f−1(Vn)} covering A. Now {V1, ..., Vn} is a finite subcover from C covering f(A).
