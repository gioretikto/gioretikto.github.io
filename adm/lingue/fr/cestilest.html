Quand utiliser «c'est» et «il est» en français

C'est: Formes et utlisations

  1. Présenter ou identifier une personne ou une chose

  2. Annocer l'arrivée d'une personne

  3. Parler d'une date

  4. Préciser un lieu ou un moment

  5. Exprimer une opinion ou faire un commentaire sur une chose

    6. Pour Présenter ou identifier une personne ou une chose, il y a trois constructions possible:

    1. C'est + déterminant (article défini, indéfini, partitif, possesif, adjectif /demostratif/numeral) + nom comune:

      • C'est mon ami

      • C'est le cahier de sophie

      • C'est du poisson

      • C'est trois euros

      • C'est cette fille

    2. C'est + pronom tonique (moi, toi, lui)

      • C'est toi qui m'as appelé?

    3. C'est + nom propre

      • C'est Marie

      • Ce son Paul et Pierre

    Annocer l'arrivée d'une personne on utilise: C'est + pronom tonique ou encore C'est + nom propre:

    • C'est mois! Je suis là

    • C'est Marie, elle vient d'arriver!

    Pour Parler d'une date: C'est + jour/mois/année:

    • Aujourd'hui, c'est mercredi

    • C'est en 2015 que j'ai commencé à trvailler

    Faites attention, pour parle d'une date, vous devez obligatoirment utiliser la forme "C'est", au singulier "Il est" Jamais la forme au pluriel

    Pour Préciser un lieu ou un moment: C'est + adverbe de lieu/ de temps:

    • Les inscriptions, c'est ici

    • C'est demain qu'il arrive

    Exprimer une opinion ou faire un commentaire sur une chose vous allez utilise la structure: C'est + adjectif masulin singuilier/adverb:

    • Je regarde souvent TV5 Monde, c'est intéressant

    Il est

    Il est est utilizé pour:

    1. Décrire ou caractériser une chose ou une personne.

    2. Indiquer l'heure

    3. Faire référence à une chose ou à une personne déjà citée

    4. Exprimer une opinion impersonelle

    Il est + adjectif (de profession, de nationalité, de religion ...) singulier:

    • Il est gentil

    • Il est médicin

    • Elles son gentilles

    Pour indiquer l'heure:

    • Il est huit heures

    • Il est tard

    • Il est tôt

    Faites attention, quand vous indiquze l'heure, vous devez obligatoirment utiliser la forme au singulier "Il est" Jamais la forme au pluriel.

    Faire référence à une chose ou à une personne déjà citée: Il est + adjectis ou Il est + adverbe

    • Paul, il est génial; Le pronom "il" renvoi ici à Paul, cité précédemment

    • Où est ton sac? Il est là-bas

    Pour Exprimer une opinion impersonelle, vous allez utiliser la structure: Il est + adjectif + de + infinitif.

    • Il est important de travailler

    Ici, le français remplacent très souvent "Il est" par "C'est" à l'oral. Cette construction est incorrecte.

    • Une chose est claire .. - E' chiara una cosa ...
