user [yze] verbe transitif 1. [consommer] consumare 2. [abîmer, affaiblir] logorare (le travail l'a usé - il lavoro l'ha logorato).

user [yze] verbe intransitif user de (servirsi di, fare uso di)

s'user verbe pronominal 1. [s'abîmer, se fatiguer] logorarsi 2. [s'affaiblir] affievolirsi

utiliser v.tr. = utilizzare, adoperare, impiegare; sfruttare: un espace bien utilisé, uno spazio ben sfruttato; l’art d’— les restes, l’arte di utilizzare gli avanzi; — un procédé d’avant-garde, adoperare un procedimento d’avanguardia; — une machine, impiegare una macchina | (inform.)non utilisé, inattivo.
