The Euclidean Topology

In general for any totally ordered set X, the interval (a,b) {x ∈ X | a < x < b}, along with the intervals (a,∞) and (−∞,b), define a topologu called the order topology.

The usual linear ordering of ℝ generates an order topology in which open sets are unions of open intervals (a,b). This topology coincides with the metric space topology (or the Euclidean topology) over ℝ. In metric spaces, an open set is defined as a set which contains an open ball around each of its points. Every metric space defines a topology, called a metric topology.
