Linear Transformations

Linear functions are the simplest functions defined on vector spaces.

Definition 2.1.1 - Let V and W be vector spaces over the field ùïÇ. A linear transformation T: V ‚Üí W is a function (map, transformation), which satisfies the following two conditions:

  1. T(v1 + v2) = T(v1) + T(v2),  for every v1, v2 ‚àà V

  2. T(λv) = λT(v),   for every v ‚àà V and scalar λ ‚àà ùïÇ.

We will denote the collection of all linear maps from V to W by ùìõ(V, W). ‚ñ°

Solution. We need to show that the two properties of Definition 2.1.1 hold:

  1. Tr(A+B) = a1,1 + b1,1 + a2,2 + ¬∑¬∑¬∑ + an,n + bn,n = tr(A) + tr(B);

  2. tr(cA) = ca1,1 + ¬∑¬∑¬∑ + can,n = ctr(A).

It follows that the trace is a linear transformation. ‚ñ†

Example 2.1.3. Define T : Mn,n ‚ü∂ Mn,n by T(A) = AT. Show that T is a linear transformation.

Solution. We check that, for A and B in Mnn and scalars c,

T(A + B) = (A + B)T = AT + BT = T(A) + T(B)

and

T(cA) = (cA)T = cAT = cT(A)

Therefore, T is a linear transformation. ‚ñ†

Example 2.1.4 The derivative is a linear transformation, that is, the function D: D ‚Üí F defined by D(f) = f' is a linear transformation.

We need to show that the two properties of Definition 2.1.1 hold. That is, we need to show that (f + g)' = f' + g' and (cf)' = cf' for all f, g ‚àà D and c ‚àà ‚Ñù. We have already proved these properties studying the algebra of derivatives, so we do not prove them again here.  ‚ñ†

We noted that multiplication by an m x n matrix defines a transformation from Fn into Fm. Matrix transformations are linear transformations since the properties in Definition 2.1.1 are simply restatement of the linearity properties for matrix multiplication. The matrix representation theorem says that every linear transformation from Fn into Fm is a matrix transformation.
Just like every vector in a finite-dimensional vector space can be associated with a vector in Fn, every linear transformation between vector spaces can be associated with a matrix in Mm,n(F):

Theorem 2.1.5. (Matrix Representation Theorem) Let V be an n-dimensional vector space over F and W be m-dimensional vector space over F. Let B = {u1, u2, ..., un} and B' = {w1, w2, ..., wm} be ordered bases for V and W respectively. Let T: V ‚ü∂ W be any linear transformation. Then there is a unique matrix A such that T(x) = Ax for all X ∈ V.

Proof. Let x ∈ V then x = x1 u1 + x2 u2 + ... + xnun, where uj is the jth element of the basis of V.
Applying the additivity and scalar properties of T, we obtain

T(x) = x1 T(u1) + x2 T(u2) + ...+ xn T(un)

The vectors T(uj) ‚àà W, can be written as linear combination of the vectors of the basis B' of W:

T(u1) = A11 w1 + A21 w2 + ... + Am1 wm
T(u3) = A12 w1 + A22 w2 + ... + Am2 wm
...
T(un) = A1n w1 + A2n w2 + ... + Amn wm

Clearly these mn scalar Aij, 1 ≤ i ≤ m, 1 ≤ j ≤ n determine T completely.

The m x n matrix A = (Aij) whose j-th column is the coordinate matrix of T(uj) relative to ordered basis B' is called the matrix of T relative to the pair of ordered basis B and B'. It is denoted by [T]B,B'. Thus

[ T ] B , B ′ = ( A 11 A 12 … A 1 n A 21 A 22 … A 2 n ⋮ ⋮ ⋮ ⋮ A m 1 A m 2 … A m n )

Then we can express the linear transformation as T(x) = [T]B,B [x]B = Ax, since

T ( x ) = T ( ∑ j = 1 n x j u j ) = ∑ j = 1 n x j T ( u j ) = ∑ j = 1 n x j ∑ i = 1 m A i j w i = ∑ j = 1 n ∑ i = 1 m A i j x j w i = ∑ i = 1 m ( ∑ j = 1 n A i j x j ) w i

from which is evident that the scalar components of L(x), with respect to the basis B' are given by

y i = ∑ j = 1 n A i j x i j

or in matrix form as

( y 1 y 2 ⋮ y m ) = A ( x 1 x 2 ⋮ x n )

Conversely suppose that A = (Aij) be any given m x n matrix. Define T: V ‚ü∂ W by

T ( ∑ j = 1 n x j u j ) = ∑ i = 1 n ( ∑ j = 1 n A i j x j ) w i

Then T, is a linear transformation such that the properties in definition 2.1.1 hold:

  1. Let a ∈ F then

    ( a T ) ( x ) = ∑ i = 1 n ( a ( ∑ j = 1 n A i j x j ) ) w i

    Hence

    [aT]B' = aTB,B' [x]B

  2. Let S be any other linear transformation then

    [(T + S)(x)]B' = ([TB,B'] + [SB,B']) [x]B

This completely proves the theorem.  ‚ñ°
