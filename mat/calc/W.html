Weierstrass' theorem

Teorema 4.8.7. Let f: ℝ → ℝ continuos, f(x) ≥ 0 ∀x ∈ ℝ and suppose that f(x) → 0 for x → ±∞. Then f has maximum in ℝ.

Proof. First notice that if f = 0 then the maximum is trivially zero. Let x0 a point s.t. f(x0) > 0; since limx → +∞ f(x) = 0, ∃K > 0 : f(x) < f(x0) ∀|x| > K. By Weiestrass theorem there exists an interval [−K, K] in which f assumes the global maximum, so that f(x0) is the global maximum.  □

Exercise 4.8.8. By introducing a different version of the previous theorem, prove that

f(x) = x3e−x2
