Weierstrass' theorem

Teorema 4.8.6.. Let f : [a,b] → ℝ continuos. Then f attains maximum and minimum in [a,b], i.e.: ∃ xm,xM ∈ [a,b] : f(xm) ≤ f(x) ≤ f(xM) ∀x ∈ [a,b].

Teorema 4.8.7. Let f: ℝ → ℝ continuos, f(x) ≥ 0 ∀x ∈ ℝ and suppose that f(x) → 0 for x → ±∞. Then f has maximum in ℝ.

Proof. First notice that if f = 0 then the maximum is trivially zero. Let x0 a point s.t. f(x0) > 0; since limx → +∞ f(x) = 0, ∃K > 0 : f(x) < f(x0) ∀|x| > K. By Weiestrass theorem and definition of x0, f assumes in [−K, K], the global maximum f(x0).  □

Exercise 4.8.8. By introducing a different version of the previous theorem, prove that

f(x) = x3e−x2

has maximum and minimum in ℝ.
