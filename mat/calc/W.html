Teorema di Weierstrass

Teorema 4.8.7. Let f: ℝ → ℝ continuos, f(x) ≥ 0 ∀x ∈ ℝ e supponiamo che f(x) → 0 per x → ±∞. Allora f ha massimo in ℝ.

Proof. Let x0 a point s.t. f(x0) > 0; since limx → +∞ f(x) = 0, ∃K > 0 : f(x) < f(x0) ∀|x| > K. By Weiestrass theorem there exists an interval [−K, K] in which f assumes relative maximum and minimum. Such a maximum can also be an absolute one. Notice that since limx → ± ∞ f(x) = 0, f is surely monotone decreasing starting from two points x1 > 0 and x2 < 0 on. By repeating the initial step, taking points x: x1 > x > x2, it is possible to find an interval [−KM, KM] containing an xM : f(xM) is the global maximum of f.  □

Exercise 4.8.8. By introducing a different versione of the previous theorem, prove that

f(x) = x3e−x2

has maximum and minimum in ℝ.
