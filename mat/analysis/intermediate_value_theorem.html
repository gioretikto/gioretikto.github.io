The Intermediate Value Theorem

Theorem 5.7. If the f is continuous on a closed interval [a,b] and if f(a) and f(b) have opposite signs, then there exists a number x0 between a and b such that f(x0) = 0. .

Proof: Assume (with no loss of generality) that f(a) < 0 < f(b). Let

S = {x ∈ [a,b]: f(x) > 0}

this set is not empty, since a ∈ S, and is bounded above by b, therefore by the completeness property of ℝ, it has a least upper bound, x0 = sup S. Since a ≤ x0 < b, follows that x0 ∈ [a,b]. We shall show that f(x0) = 0.
If it was f(x0) > 0 owning to Theorem 4.1.17 (permanennce of sign), it would be f(x) > 0 ∀x ∈ (x0 − δ, x0 + δ) and in this neighborhood of x0 we could find x ∈ S, but this contradicts the hypothesis x0 = sup S. If it was f(x0) < 0 for the same reason, f(x) < 0 ∀x ∈ (x0 − σ, x0 + σ) and in this neighborhood of x0 we could find x < x0 such that x = sup S but this also contradicts the initial hypothesis x0 = sup S, then f(x0) = 0.  □
