Intermediate value theorem

The Intermediate Value Theorem states that a continuous function takes on every intermediate value between the function values f(a) and f(b)

Theorem 5.8 (Weierstrass theorem (extreme value theorem)). Let f continuous on the closed and bounded interval [a,b], then f assumes all values between f(a) and f(b).

Proof. From the Weierstrass' theorem on extreme values, f has a maximum, M and minimum m.

m = min f,   M = max f

We define the auxiliary function g(x) = f(x) − y0 which results continuous in [m,M] with y0 ∈ (m,M), moreover g(m) < 0 and g(M) > 0; The hypothesis of the Intermediate Zero Theorem hold, thus ∃x0 ∈ (m,M) ⊂ [a,b]: g(x0) = 0 ⇒ f(x0) = y0; i.e. ∃x0 ∈ (a,b) such that f(x0) = y0. □
