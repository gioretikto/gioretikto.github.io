The Extreme Value theorem

We now show that if a function defined on a closed and bounded set is continuous, then it is necessarily bounded.

Theorem 4.3.2 (The Boundedness theorem). Let f: [a, b] → ℝ, a continuous function. Then f is bounded and attins it bounds, so that the range of f is again a closed bounded interval.

Proof. We prove this by contradiction. If the set f([a,b]) were unbounded above, then there would exist a sequence {xn} in [a, b] such that f(xn) ⟶ ∞. Since {xn} ∈ [a, b], it is a bounded sequence. The Bolzano–Weierstrass theorem asserts that there exists a convergent subsequence {xnk} ⟶ c. Since {xnk} ∈ [a, b] and since f is continuous on [a,b], namely at c, it must be the case that f(xnk) ⟶ f(c). But {f(xnk)} is a subsequence of {f(xn)} by Theorem 2.3.2 we know that they must converge to the same limit

f ( c ) = lim k → ∞ f ( x n k ) = lim n → ∞ f ( x n ) = sup  f ( [ a , b ] )

If f(xn) ⟶ ∞, it would be f(xnk) ⟶ ∞ as well. This is a contradiction, since f(xnk) ⟶ f(x) ≠ ∞. So we cannot have sup f([a,b]) = +∞. If f([a,b]) is unbounded below, a similar argument is used.  □

So if f is continuous on a closed boundedn interval I then the image of I under f is also closed and bounded interval. For this is crucial that the interval I is closed: for example, the function f(x) = 1/x is continuous on the interval (0,1), but takes arbitrarly large values as x ⟶ 0. The function g(x) = x is bounded on (0,1), but it does not have a maximum or minimum on the interval (0,1) − it approaches 0 and 1 arbitrarly closely, but never attaints either, as its value lie strictly between 0 and 1.
A continuous function defined on a bounded and closed interval is not just bounded but it attains a maximum and a minimum as well.
The following theorem is attributed to the great German mathematician Karl Weierstrass (1815–1897).
