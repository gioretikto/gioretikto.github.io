Infinite Sequences

A sequence of real numbers is a function whose domain is the set of positive integers and range a set of real numbers.

f : ℕ → ℝ

A sequence of complex numbers is defined similarly, with ℝ replaced by ℂ. A sequence can be thought of as a list of numbers written in a definite order:

a0, a1 , a2 , a3, ... , an, ...

The number a1 is called the first term, a1 is the second term, and in general an is the nth term. The sequence {a0, a1, ..., an} is generally indicated as {an}. For example consider the sequence

{an} = {(−1)n} = {−1, 1, −1, 1, −1, ...}<

{an} = {n2} = {0, 1, 4, 9, 16, ..}

{an} = {1/n} = {1, 1/2, 1/3, 1/4, ..}

{an} = {5} = {5, 5, 5} (constant sequence)

Its nth term is an = (−1)n and the range is {−1, 1}. Some sequences can be defined by giving a formula for the nth term.

an> = n/(n + 1),    {1/2, 2/3, 3/4, 4/5, ..., i>n/(n + 1)}

Definition 2.1.1. A sequence {an} is said

  • Bounded below if there exists a real number m such that   an ≥ m,   ∀n ∈ ℕ;

  • Bounded above if there exists a real number M such that   an ≤ M,   ∀n ∈ ℕ;

  • Bounded. A sequence which is bounded both above and below. Thus a sequence {an} is said to be bounded if there exist two real numbers m and M such that m ≤ an ≤ M ∀ n ∈ ℕ. □

For example {(−1)n)} is bounded by the two numbers {−1, 1}; {1/n} is bounded: 1 is the upper limit and 0 is a lower bound; {n2} is bounded below; {(−2)n)} is not bounded.

The most important question about a sequence is whether it converges. We define this notion as follows.

Convergence of Sequences

Definition 2.1.2 - (Limit of a sequence) A sequence {an} is said to converge to a real number ℓ, called the limit of {an}, if for every ε > there exists a natural number n0 such that:

|an − ℓ| < ε,   ∀n ≥ n0

If the sequence {an} converges to the limit ℓ, then we write

limn ⟶ ∞ an = ℓ.  □

note that the inequality |an − ℓ| < ε corresponds to the chain of inequalities:

ℓ − ε < an < ℓ + ε

from this relation is clear that: every convergent sequence is as well bounded.

Example 2.1.3 Let an = 1/n, n = 1, 2, ... . Then the sequence converges to 0. For let ε > 0. Choose N to be the next integer after 1/ε (we use the Archimedean property, if r,ε are positive real numbers, then there is a positive integer n sucht that nε > r). If n > N then

|an − 0| = |an| = 1/n < 1/N < ε

proving the claim.  ■

When discussing the convergence of a sequence, we often find it inconvenient to deal with the definition of convergence as given. For this definition makes reference to the number to which the sequence is supposed to converge, and we often do not know what the limit is in advance. Would it not be useful to be able to decide whether a series converges without knowing to what limit it converges?

Definition 2.1.4. (Cauchy Sequence) Let {an} be a sequence of real numbers. We say that the sequence satisfies the Cauchy criterion — or more briefly, that the sequence is Cauchy — if, for each ε > 0, there is an integer N > 0 such that if j, k > N then |aj − ak| < ε  □

Theorem 2.1.5. (Cauchy’s Criterion) A sequence is convergent only if it is Cauchy.

Proof.. If the sequence has limit ℓ, then by definition, by fixing an ε > 0, there is an integer N such that |an − ℓ| < ε for n ≥ N. Thus if n,m ≥ N

|am − an| = |am − ℓ + ℓ − an| ≤ |an − ℓ| + |ℓ − an| < 2ε.

since 2ε is an arbitrary positive number it follows that {an} is a Cauchy sequence.

For the converse we need to show that there is a value ℓ to which the sequence converges.
