The Intermediate Zero Theorem

Theorem 5.7. If the f is continuous on a closed interval [a,b] and if f(a) and f(b) have opposite signs, then there exists a number x0 between a and b such that f(x0) = 0. If f is also strictly monotonous, the zero is unique.

Proof: Assume (with no loss of generality) that f(a) < 0 < f(b). Let

S = {x ∈ [a,b]: f(x) < 0}

this set is not empty, since a ∈ S is in it. Moreover is bounded above by b, so it has a least upper bound, x0 in S

x0 = sup S

it must be f(x0) = 0; If it was f (x0) > 0 owning to Corollary 6.16 (permanennce of sign), it would exists an interval (x0 -δ, x0 + δ): f(x) > 0 ∀x ∈ (x0 -δ, x0 + δ), but this contradicts the hypothesis x0 = sup S. If it was f (x0) < 0 for the same reason, there would be an interval (x0 -σ, x0 + σ): f(x) < 0 ∀x ∈ (x0 -σ, x0 + σ), but this also contradicts the initial hypothesis x0 = sup S, then f(x0) = 0. □

Step I Let c be middle point of the interval [a,b]:

c = (a+b)/2

if f(c) = 0 = f (x0) we are lucky and the theorem is proved. If it is not the case we continue with the 2nd step:

Step II

  • If f(a)*f (c) < 0; then x0 is located in the left half of the interval x0 ∈ [a,c], cwe build up a new interval [a,b] with b = c

  • If f(b)*f (c) < 0; then x0 is located in the right half of the interval x0 ∈ [c,b]; we build up a new interval [a,b] with a = c

By considering the sequence {an} made by extrems a of each interval of each iteration and that {bn} made up by the extrems b, {an} will be increasing and {bn} decreasing. The lenght of the interval at the n-th step is equal to

bn − an = (a + b) / 2n

by taking the limit for n → ∞

limn → ∞ bn − an = limn → ∞(a + b) / 2n = 0

i.e.

an → ℓ1,   bn → ℓ2,   for   n → ∞ ⇒ ℓ1 = ℓ2

Given that at each step f(a) ⋅ f (b) ≤ 0, by the theorem on permanence of sign it must result ℓ1 = ℓ2 = 0.□
