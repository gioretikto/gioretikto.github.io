Hôpital's Rule

Differential calculus offers a way to calculate limits in the case of indeterminate forms of the type 0/0 and ∞/∞.
L'Hospital's Rule is named after a French nobleman, the Marquis de l'Hospital (1661–1704).

Theorem 5.7.1 (L'Hôpital's Rule). Let f and g be continuous on [a,b] and differentiable on (a,b), such that g'(x) ≠ 0 ∀ x ∈ (a,b). Suppose that

lim x → x 0 f ( x ) = 0 , lim x → x 0 g ( x ) = 0

Then

lim x → x 0 f ′ ( x ) g ′ ( x ) = ℓ ⇒ lim x → x 0 f ( x ) g ( x ) = ℓ

By Cauchy's Mean Value Theorem there is c between x and x0 such that

f ( x ) g ( x ) = f ( x ) − f ( x 0 ) g ( x ) − g ( x 0 ) = f ′ ( c ) g ′ ( c )

Now as x ⟶ x0, we must have also c ⟶ x0 because c is between x and x0. So if

lim x → x 0 f ′ ( x ) g ′ ( x ) = ℓ

then

lim x → x 0 f ′ ( c ) g ′ ( c ) = ℓ

Thefore

lim x → x 0 f ( x ) g ( x ) = ℓ

as desired.  □
