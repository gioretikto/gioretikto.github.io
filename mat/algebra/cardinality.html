Cantor's Theorem

By using the the idea of "power set" we are able to reate bigger infinities. For example if we take the set {1,2,3} (with cardinality 3) its power set contains the empty set, {1}, {2}, {3}, {1,2}, {1,3}, {2,3} and {1,2,3} and so has cardinality 8. The formula is that, if the original set has cardinality n, then its power set has size 2n. Amazingly, the same conclusion is true for infinite sets: the power set of an infinite set creates a new set of a bigger infinity.

Cantor's Theorem states that the cardinality of a set is strictly less than the cardinality of its power set.
