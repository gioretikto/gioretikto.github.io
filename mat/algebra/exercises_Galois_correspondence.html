Exercises on Galois correspondence theorem

  1. Calculate the Galois group of the following polynomials in ℚ[x]

    1. x3 + 2x2 + 5x + 10

    2. x4 − 7x2 + 10

    3. x5 + 2x4 − 5x3 − 10x2 + 6x + 12

    4. x3 − 1

    5. x3 − 7

    6. x4 − 5

Solutions

    1. x3 + 2x2 + 5x + 10 = (x2 + 5) (x + 2). The splitting field is K = ℚ(√5i). One automorphism is the identity and the other σ(√5i) = −√5i; So {id, σ} ≅ ℤ₂. 

    2. If we replace x2 = y we get y2 −7y + 10. We find the two roots y = 2 and y = 5, so (x2 − 2)(x2 − 5), and each of the factor is irreducible over ℚ. Hence the splitting field is ℚ(√2,√5). The four roots of thr original polynomial are √2, −√2, √5, −√5, and an automorhpsms of the splitting field has to send each thing in {√2, −√2} to something in there, similarly for {√5, −√5}. If f(a) = −a, then automatically f(−a) = a, because f(−a) = −f(a) = −(−a) = a. Further, since f is an automorphism, we must have f(a)2 = f(a2)

      σ1 = id
      σ2: σ2(√2) = −√2 σ2(√5) = −√5
      σ3: σ3(√2) = −√2 σ3(√5) = √5
      σ4: σ4(√2) = √2 σ4(√5) = −√5

      G(ℚ(√2,√5),ℚ) ≅ ℤ₂ x ℤ₂.  ■

    3. A factor is (x + 2). Dividing the polynomial by x + 2 yo find x4−5y2+6; by substitution of y = x2 I've found two roots +3;+2, so x5 + 2x4 − 5x3 − 10x2 + 6x + 12 = (x + 2) (x2 − 3) (x2 − 2). So the splitting field is ℚ(√2,√3). The four roots of thr original poly are −2, √2, −√2, √3, −√3. So we have G(ℚ(√2,√3),ℚ) with the following elements

      σ1 = id
      σ2: σ2(√2) = −√2 σ2(√3) = −√3
      σ3: σ3(√2) = −√2 σ3(√3) = √3
      σ4: σ4(√2) = √2 σ4(√3) = −√3

      G(ℚ(√2,√3),ℚ) ≅ ℤ₂ x ℤ₂.  ■

    4. ℚ(ζ) is the splitting field with ζ = ei2π/3 = cos 2π/3 + i sin 2π; cos value is −1/2, that sin value is √3/2., so ζ = (−1 + √3i)/2. The Galois group G(ℚ(ζ), ℚ) ≅ ℤ₂ since it contains the ideintity automorphism and the automorphism such that ζ ⟶ ζ2.  ■

    5. The roots are ∛7, ∛7ζ, ∛7ζ2. So the splitting field is ℚ(∛7, ζ).

      σ1 = id
      σ2: σ2(ζ) = ζ2 σ2(∛7) = ∛7
      σ3: σ2(ζ) = ζ σ2(∛7) = ζ ∛7
      σ4 = σ32: σ4(ζ) = ζ σ4(∛7) = ζ2 ∛7
      σ5 = σ2 ∘ σ3: σ5(ζ) = ζ2 σ5(∛7) = ζ2 ∛7
      σ6 = σ2 ∘ σ32: σ(ζ) = ζ2 σ6(∛7) = ζ ∛7

      So G(ℚ(∛7, ζ, ℚ) ≃ S3.  ■
