Divisibility in ℤ

A Fundamental Concept in ℤ is that divisibility.

Definition 2.1.5. Given two integers a,b is said that a divides b (or that a is a divisor of b), and we write a|b, if there is a c ∈ ℤ so that b = ac. If a does not divide b, we write a ∤ b.

For example 3|12, 4|-16 by 4 ∤ 2 e 0 ∤ a if a ≠ 0.

Definition 2.1.8. A common divisor of elements a and b in ℤ is an element c ∈ ℤ for which c|a and c|b.

Lemma 2.1.9. If c is a common divisor of a and b, then c divides any integer of the form sa + tb, with s and t in ℤ, in formula:

c|a and c|b ⇒ c|sa + tb ∀ s,t ∈ ℤ

Proof. c|a ⇒ a = ch per qualche h ∈ ℤ; c|b ⇒ b = ck per qualche k ∈ ℤ. Allora, per ogni s, t in ℤ, sa + tb = s(ch) + t(ck) = c(sh + tk), da cui c|sa + tb.□

Corollary - If c|a and c|a + b then c|b

Proof - If c is a common divisor of a and a+b, then for s =-1 and t = 1 we get that it divides (-1)*a + 1*(a + b) = b. □

Diamo alcune definizioni che saranno importanti nello studio degli anelli in generale, ma che a prima vista possono sembrare poco significatine in ℤ.

Definition 2.1.10. An element u ∈ ℤ which divides 1 is called unit (or invertivle element) of ℤ.

It is immediate to find that the only units of ℤ are 1 e -1.
