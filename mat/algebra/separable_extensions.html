Separable extensions

We now on we should consider only field of characeristic zero, i.e. extensions of the rational field. As we shall see "An algebraic field extension is Galois if and only if it is normal and separable" and this latter condition is always attained in zero characteristic field. We recall that a polynomial is separable if it has no multiple roots.

8.8.1 Definition. An algebraic extension K of a field F is called a separable extension if each element of K is separable over F, i.e. its minimal polynomial is separable.  □

8.8.2 Proposition. Let a and b two algebraic elements overa field F of characteristic zero. Then there exists an element α ∈ F(a,b) such that F(a,b) = F(a).

Proof. Let f(x) and g(x) in F[x] two minial polynomials of a and b respectively. Let K an extension in which both polynomials split, and let

a = a1,a2, ..., an   b = b1,b2, ..., an

the roots in K of f(x) and g(x) respectively. We know from Proposion 8.3.6 that over a field of char. 0, every irreducible polynomial is "separable" -- its roots in any splitting field are all distinct, hence since f(x) and g(x) are irreducible, all the ai (and bi) are distinct. For every i = 1, ..., n and every j = 2, ..., m, the equation in λ

ai + λbj = a + λb

admits only one root in K, given by

λ = (ai − a) /(b − bj)

Being F of char 0, it possesses infinite elements. We pick an element γ ∈ F which does not satisfy none of the previous n(m − 1) equations. Set α = a + γb. We shall prove that

F(a,b) = F(α)
