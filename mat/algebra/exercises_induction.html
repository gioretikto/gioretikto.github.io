Exercises on natural numbers

Exercise 2. Prove by induction that for every positive integer n, ∑nk=0 (4k + 1) = (2n + 1)(n + 1)

Solution. For n = 0, we have 1 = 1. We suppose the relation true for n−1 and we prove it for n.

∑nk=0 (4k + 1) = ∑n−1k=0 (4k + 1) + 4n +1 = (2n + 1)(n + 1) (2n + 1)(n + 1)[2(n−1) + 1] ⋅ n + 4n + 1 = (2n − 1) ⋅ n + 4n + 1 = 2n2 −n + 4n +1 = 2n2 + 3n +1 = (2n+1) ⋅ (n+1)

Exercise 3. Prove by induction that for every positive integer n

12 + 22 + 32 + ... + n2 = n(n+1)(2n+1)/6

Solution. For n = 1 is true. We suppose 12 + 22 + 32 + ... + (n − 1)2 = ((n−1)n(2(n−1) + 1))/6 and prove it for n.

12 + 22 + 32 + ... + n2 = ((n−1)n(2(n−1) + 1))/6 + n2 = n(n+1)(2n+1)/6.

Exercise 4. Prove by induction that for every positive integer n, the power set 𝓟(X) of a finite set X with n elements has 2n elements.

Solution. It's true for n = 0, you could use that as the base case, X has exact one (= 20) the empty subset. We suppose true that every set X with n−1 elements has 2n−1 subsets. The set {X} \ {x} has n−1 elements, hence it has 2n−1 subsets. To obtain the subsets of X we must add to each of these 2n−1 subsests the element x, obtaining another 2n−1 subsests thus in total 2n−1 + 2n−1 = 2n subsests.

But then, the students from 2 to n-1 belonging to different groups will have the same marks of the student of the first and second group. Hence all students have the same marks.
