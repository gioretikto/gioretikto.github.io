Divisibility in ℤ

A Fundamental concept in ℤ is divisibility.

Defintion 2.2.1. Let a,b in ℤ. We say that a divides b (or that a is a divisor of b), and we write a|b, if b = ac for some c ∈ ℤ. If a does not divide b, we write a ∤ b.

For example 3|12, 4|-16 by 4 ∤ 2 e 0 ∤ a if a ≠ 0.

Definition 2.2.2. In a commutative ring, an element a ≠ 0 is said zero divisor if there exists b ≠ 0 such that ab = 0.

Definition 2.2.3. An integral domain is a commutative ring without zero divisors.

Proposition 2.1.2 says that the ring ℤ is an integral domain.

Definition 2.2.4. It is said common divisor of elements a and b in ℤ an element c ∈ ℤ such that c|a and c|b.

Lemma 2.2.5. If c is a common divisor of a and b, then c divides any integer of the form sa + tb, with s and t in ℤ, in formula:

c|a and c|b   ⇒   c|sa + tb   ∀s,t ∈ ℤ

Proof. c|a ⇒ a = ch for some h ∈ ℤ; c|b ⇒ b = ck for some k ∈ ℤ. Thus, for each s, t in ℤ, sa + tb = s(ch) + t(ck) = c(sh + tk), consequently c|sa + tb.□

Corollary 2.2.6 - If c|a and c|a + b then c|b

Proof. If c is a common divisor of a and a + b, then for s = −1 and t = 1 we get that it divides (−1)*a + 1*(a + b) = b. □

We introduce now some definitions important in the study of rings, ma che a prima vista possono sembrare poco significatine in ℤ.

Definition 2.2.7. An element u ∈ ℤ which divides 1 is called unit (or invertible element) of ℤ.

It is immediate to find that the only units of ℤ are 1 and -1.

Definition 2.2.8. Two elemens a and b of ℤ such that a|b e b|a are said associates.

It follows immediately from the definition that two elements a and b are associates if and only if a = bu, with u is a unit. Thus, in ℤ two elements are associates if and only if they differ by the sign. The relation of being "associates" is an equivalence relation.

Definition 2.2.9. An element a ∈ ℤ which is nonzero, not a unit is irreducibile, if a = bc, with b,c in ℤ implies that either b or c is a unit.

Definition 2.2.10. A non-zero element p ∈ which is not a unit, is said prime, if p|ab with a,b in ℤ, implies either p|a or p|b.

Remarks

The definition just given of irriducible element corresponds to the common definition of prime number in ℤ, i.e.: positive integers without other divosors execpt of themselves and the unit. Here we're talking about irreducible elements and not of prime elements (or numbers). We'll see that the two notions prime element and irreducibile elementi are the same in ℤ, even though this is not the general case. At this point we are able to prove that a prime element is necessarily irreducibile.

Proposition 2.2.11. Any prime element in ℤ, is an irreducible element.

Proof. Let a a prime element in ℤ. To prove a is irreducibile, we have to show that from a = bc with b,c ∈ ℤ follows up that b or c are units. Let a = bc; in particular a | bc. Thus (since a is prime by hypothesis) either a | b or a | c, thus either b = ah or c = ak, with h,k ∈ ℤ; but a = bc, along with one of these identities imply that either b or c are units.□

We'll show that irreducible elements in ℤ are also prime in Proposition 2.19.
