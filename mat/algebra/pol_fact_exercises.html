Exercise on polynomials

Exercise 1. Decompose the following polynomial on ℂ, ℝ, ℚ, ℤ2

x5 + 2x4 -5x3 - 10x2 + 6x + 12

Solution. Thoughtfully split the expression at hand into groups, each group having two terms:

x4(x+2) -5x2(x+2) +6(x+2) = (x+2) (x4 − 5x2 + 6)

and it is fairly easy to see that (x4−5x2 + 6) = (x2−2)(x2−3). So the result on ℚ is

(x+2)(x2−2)(x2−3)

In ℂ, x2−a=(x–√a)(x+√a) thus (x+2)(x+√2)(x–√2)(x+√3)(x–√3).

In ℤ2

(x+2)(x2-2)(x2 − 3) ⟶ (x)(x2)(x2 − 3), since 2 = 0 mod 2, so

x3 (x2 +1).

x2+1 factors as: (x+1)(x+1).

Exercise 2. Decompose the following polynomial on ℂ, ℝ, ℚ, ℤ2

x5 + 2x4 −x3 − 2x2 − 2x − 4

Solution. Group as

x4(x+2) −x2(x+2) −2(x+2) = (x+2)(x4 −x2 −2).

In ℤ2 since 2 = 0 mod 2

x5+x3.

It is easy to see that (x4 − x2 -2) = (x2 + 1)(x2 − 2) so (x+2)(x+2)(x2 + 1)(x2 − 2).

on ℂ since x2 = −1, you just factorize the x²+1 into (x − i)(x + i) so (x+2)(x − i)(x + i)(x + √2)(x – √2).

Exercise 3. Decompose the following polynomial on ℂ, ℝ, ℚ, ℤ2

Solution. Group as

(x+3)(x4 −x2 −2) = (x+3)(x2 + 1)(x2 − 2) .
