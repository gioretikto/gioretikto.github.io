Exercises on splitting fields

  1. Find all irreducible polynomials p(x) of second degree in ℤ2[x]. For each one of these polynomial compute the field ℤ2/(p(x)).

Solutions

  1. Let I mean the ideal (x2 + x + 1) in ℤ2[x]. This is defined as ⟨x2 + x + 1⟩ = {p(x) (x2 + x + 1)| p(x) in Z2[x]}.

    Technically the four elements of the quotient ring ℤ2[x]/I are the cosets 0 + I, 1 + I, x + I, x + 1 + I. You can caculate (x + I)(x + 1 + I) = x2 + x + I, but this = 1 + I because is in I. (- is the same as + in this ring, since it has characteristic 2).
