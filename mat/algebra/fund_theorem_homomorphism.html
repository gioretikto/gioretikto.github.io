Fundamental theorem of ring homorpshims and isomorphism

6.4.0 Proposition (Fundamental theorem of ring homomorphism) Let R and R' two rings and let f: R ⟶ R' an homomorphsim between rings. Then there exists a unique isomorphim f* from R/Ker f to Imf

R/Ker f ≃ Imf

such that f* ∘ π = f, with π the canonical projection of R on R/Ker f.

Proof. If f* has to satisfy f = f* ∘ π, we must define it as follows

f* : R/Ker f ⟶ Im f

x + Ker f ⟼ f(x)

we now must check that

  1. f* is well-defined, i.e. if [x] = [y] ⇒ f(x) = f(y)

  2. f* is bijective.

  3. f* is an homomorphism.

  1. [x] = [y] ⇐⇒ x − y ∈ Ker f ⇐⇒ f(x − y) = 0 ⇐⇒ f(x) = f(y). Double implication prove at the same time that f* is well-defined and injective.

  2. It is left to prove that f* is surjective: An element f(x) ∈ Imf, has counter image x ∈ R. And an element [x] is mapped through f* to f(x), hence f* is surjective Imf* = Imf.

  3. f*([x] + [y]) = f*([x + y]) = f(x+y) = (being f an homomorphism) = f(x) + f(y) = f*([x]) + f*([y]). Analogously for the product.

It follows naturally from the definition of f* that f = f* ∘ π.

If there exists an homomorphism f of R onto R', then R' is called an isomorphic image of R.

The theorem states that every homomorphic image of a ring R is isomorphic to a quotient ring R/I for some ideal I. Thus if you know all the quotient rings of R, then you know all the possible homomorphic images of R. The ideal I measures how much information is lost in passing from the ring R to the homomorphic image R/I.

Moreover to prove that a quotienti ring R/i is isomorphic to a ring R' it is sufficient to find an epimorphism φ from R' to R such that I = Ker φ.

6.4.1 Proposition Let f an homomorphism between two rings R and R'. Then

  1. if A is a subring of R, f(A) is a subring of R'.

  2. if I is an ideal of R, f(I) is a ideal of f(R) (= Im f) (not necessarily of R').

  3. if A' is a subring of R', f−1(A') is a subring of R containing Ker f.

  4. if I' is a an ideal of R', f−1(I') is an ideal of R containing Ker f.

Note f−1(A') here means the preimage or "inverse image" i.e. the set {r ∈ R: f(r) ∈ A'}.

Proof.

  1. Given two elements f(a) and f(a̅) of f(A), we have f(a) − f(a̅) = f(a −a̅) ∈ f(A) and f(a)f(a̅) = f(aa̅) ∈ f(I) since a − a̅ ∈ A and aa̅ ∈ A.

  2. It is sufficient to prove that f(a)f(x) ∈ f(I) and f(x)f(a) ∈ f(I) because ax ∈ I. Analogously f(x)f(a) ∈ f(I).

  3. Let a,a̅ ∈ f−1(A'), such that f(a) ∈ A' and f(a̅) ∈ A'. Then f(a − a̅) = f(a) − f(a̅) ∈ A', and f(aa̅) = f(a)f(a̅) ∈ A' hence a − a̅ and aa̅ are in f−1(A'). The last element contains Ker f, since ∀k ∈ Ker f, f(k) = 0R', which is contained in every ring A'.

  4. Let x ∈ R and a ∈ f−1(I'). We must prove that xa and ax are in f−1(I'). Indeed, f(xa) = f(x)f(a) ∈ I' (since f(a) ∈ I'). Analogously f(ax) ∈ I'. In this case as well f−1(I') contains Ker f.□

6.4.2. Example. Let R = ℤ and R' = ℤ4 = ℤ/4ℤ, and f = π the canonical projection of ℤ on ℤ/4ℤ. Let A = 6ℤ. The elements of 6ℤ are distributed in only two classes of ℤ modulo 4ℤ. Considering the quotient modulo I = 4ℤ.

6Z + 4Z = {a + b | a in 6Z, b in 4Z}. For that, the mean the multiples of 6 in Z fall into only two of the four possible cosets of 4Z. In other words, out of the possible values 0,1,2,3 for what some n in Z can be mod 4, only two of those values come up when you look at just the n in 6Z.

π(A) = 6ℤ/4ℤ = [0, 2] = {I, 2+I}, because, a multiple of 6 divided by 4 can have as remainder either 0 or 2.

R/I = ℤ/4ℤ because every n in ℤ can be written uniquely as 4m + r where r is in {0,1,2,3} by the division algorithm. So R/I = {I, 1+I, 2+I, 3+I}.

6.4.3. Corollary. Let R a ring, I one of its two-sided ideal and π: R ⟶ R/I, the canonical projection. Then

  1. S subring (ideal) of R ⇒ π(S) = (S + I)/I is a subring (ideal) of R/I;

  2. S subring (ideal) of R/I ⇒ π−1(S') is a subring (ideal) of R containing I.

Proof.

  1. let S be a subring of R. π is a homomorphism between R and R/I, so by point 1 of 6.4.1, π(S) is a subring of R/I.

  2. let S be an ideal of R. π is a homomorphism between R and R/I, so by point 2 of 6.4.1, π(S) is an ideal of π(R). Furthermore, π defined as r ⟼ r + I is a surjection, so π(R) = R/I. Therefore, π(S) is an ideal of R/I.

On why (S + I)/I = π(S) and π(s) is an element of (S+I)/I, π(s) = s+I by definition. Anything in (S + I)/I is some coset x + I where x is in the set S + I, so x is some s + i (s ∈ S, i ∈ I), so x + I = s + i + I = s + I = pi(s). Conversely, take any π(s) in π(S). π(s) = s + I = s + i + I with i = 0 in I.□

6.4.4. Proposition. (The Correspondence Theorem) Let R a ring, I one of its ideal and π: R ⟶ R/I the canonical projection. Let

𝓛 = {subrings (ideals) of R containing I}

𝓛'= {subrings (ideals) of R/I}

Then

Ψ: 𝓛 ⟶ 𝓛'

  A ⟼ π(A)

it is a bijection between 𝓛 and 𝓛'.

Proof. In the hypothesis (A ⊇ I), we have π(A) = A/I. As stated in Corollary 6.4.3., Ψ maps a subring (ideal) of R containing I to a subring (ideal) of R/I, thus an element of 𝓛'. The application (by point b) of Corollary 6.4.3.) Ψ*: 𝓛' ⟶ 𝓛 defined as

A' ⟼ π−1(A')

is the inverse application of Ψ, and the following relation hold true:

  1. π−1(π(A)) = A

  2. π(π−1(A')) = A'

π−1(π(A)) ⊃ A and π(π−1(A')) ⊂ A' are general properties of any map of sets and for arbitrary subsets. The equality π(π−1(A')) = A' holds for any surjective map of sets. We need to prove the double inclusion i.e. π−1(π(A)) ⊂ A, which requires A ⊃ I. Let x ∈ π−1(π(A)). Then π(x) ∈ π(A), i.e. there exist an element y ∈ I such that π(y) = π(x). Since π is a homomorphism, π(x−y) = 0 and x − y ∈ I = Ker π. Since y ∈ I and A ⊃ I, this implies that x ∈ A □.

6.4.5. Example. Determine all ideals of the quotient ring Z/12ℤ .

The ring is now made of elements of the sort z + 12ℤ with z ∈ ℤ. We must find (z + 12ℤ)(i + 12ℤ) = zi + 12ℤ. Using last theorem, find all ideals of Z containing 12Z and then use the Ψ to create the correspondence with the ideals of Z/12Z. For example if we take even numbers 2Z this is an ideal of Z since an even number multiplied by an integer is again even: 12ℤ ⊂ 2ℤ. We have that mZ \subset nZ iff n | m, so these sets are

Z, 2Z, 3Z, 4Z, 6Z, 12Z

Then the ideals of Z/12Z are

Z/12Z, 2Z/12Z, 3Z/12Z, 4Z/12Z, 6Z/12Z, 12Z/12Z={0}

12Z/12Z: this set is nothing but {12Z}.

6.4.6. Theorem. (First Isomorphism theorem) Let R a ring, I one of its ideal and π the canonical projection. Let A a subring of R not necessarily containing I. Then

  1. π−1(π(A)) = A + I = {a + i | a ∈ A, i ∈ I}

  2. A ∩ I is an ideal of A and I is an ideal of A + I.

  3. A/(A ∩ I) ≃ (A + I)/I.

Proof.

  1. From corollary 6.4.3 π(A) = (A+I)/I so π−1((A+I)/I) = {(a+i) +j {a + i | a ∈ A, i ∈ I}.

  2. I is an additive subgroup, and i(a+i) ∈ I since both ai and ii are in I (because I is an ideal of R and a ∈ R. A ∩ I is an additive subgroup: if x,y ∈ A ∩ I then x and y are in A, so x + y is in A and x and y are in I, so x + y is in I that is A ∩ I is an additive subgroup. Moreover xa ∈ A since A is an ideal of R

  3. If π: R ⟶ R/I, let π|A the restriction of π to A that is

    π|A: A ⟶ (A+I)/I

    It is a ring homomorpshim whose kernel is

    Ker π|A = { x ∈ A | π|A (x) = I} = {x ∈ A | x + I = I} = A ∩ I}

    The result follows by virtue of the fundamental theorem of ring homomorphism.□
