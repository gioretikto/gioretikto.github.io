Functions

A mapping (or a function) f from a set A to a set B is a rule that for each element of A associates a uniquely determined element of B.

This is usually written:

f: A → B

if a ∈ A then the unique element b∈ B, which corresponds to a is denoted by f(a). We say that b = f(a) is an image of a, and a is a preimage or inverse image of b.

The set A is called domain of the mapping f, while the set of alla images of all elements a/i> is a subset of B called range of f which we denote by Im(f). The set B is usually called the codomain of f.

Im f = f(A) := {b ∈ B | b = f(a) for some a ∈ A}

Graph of a function

A function ca be thought of as a correspondence between sets A and B and in particular as a set of ordered pairs (a,b) where the first element a of the pair belongs to the domain A and the second element b to the codomain B.

Thus each correspondence is a subset of the cartesian product A x B.

The subset F of A x B (i.e. the relation F from A and B) given from

F = { (a,f(a)) | a ∈ A}

Note that not every correspondende can serve as the graph of a function, only a set of ordered pairs in which each element of the domaind has only one element associated with it in the range is the grap of a function.

Note also that a function is a (particular) relation, while not every relation is a function. If A = B we will say there is a binary relation between the elements of A.

Definition 1.3. A function f: X → Y is injective (or one-to-one) if, for all a and b in X, f(a) = f(b) implies that a = b. We say in this case that f is a one-to-one mapping for X to Y.

Definition 1.4. A function f: X → Y is surjective (or onto) if, for all y ∈ Y, there exists an x ∈ X such that f(x = y. In this case we say that f maps X onto Y and we write Im f = Y).

Definition 1.5.. A function f: X → Y is biunique when each y ∈ Y has one counterimage in X. In thica case f is both injective and surjective.

Well-defined functions

The condition that a function must be well defined means that f has a definition that assigns it a unique value

a = b   ⇒   f(a) = f(b)

Example 1.2 Let ℚ be the set of rational number. Let

f(x/y) = x + y

for any x/y in ℚ. Does this yield to a well-defined function f: ℚ → ℝ.

Solution. The answer in in the negative. For example not that 2/3 = 4/6, but

f(2/3) = 2 + 3 = 5 ≠ f(4/6) = 4 + 6 = 10

Example 1.3 Show that the function f: ℚ → ℚ defined by

f(x / y) = (x + y) / y

is well defined.

Solution. Let a/b = c/d ∈ ℚ. Then

f(a / b) = (a + b) / b

=a / b + b /a

=c / d + c /d

=(c + d)/d

=f(c/d)

so f is well defined.

Inverse Function

Le funzioni biettive (iniettive e suriettive) sono particoloari poichè mettono in relazione due insiemi aventi la stessa cardinalità. Ad ogni elemento del dominio corrisponde uno ed un solo elemento del codominio e viceversa a ciascun elemento del codominio corrisponde uno e solamente un elemento del dominio. Questa proprietà porta alla definizione di una nuova funzione: la funzione inversa.

Quando la funzione f: A → B è biunivoca allora è anche invertibile. Ossia esiste la funzione f -1: B → A tale che ad ogni y ∈ B fa corrispondere l'unico x ∈ A, tale funzione si chiama funzione inversa e si indica con f -1, si ha che:

f -1 (f(x)) = x,   ∀x ∈ A;   f (f -1 (y)) = y,   ∀y ∈ Y;

Sia K un sottoinsieme B, il sottoinsieme:

f-1 (K) = {x ∈ A: f(x) ∈ K}

è l'immagine inversa di K mediante f.
