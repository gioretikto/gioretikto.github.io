Composite Functions

Definition 1.5.1. Let f: X ⟶ Y and g: Y ⟶ Z be two functions. Then the function g ∘ f: X ⟶ Z given by

(g ∘ f) (x) = g(f(x))

is called the composition of the function f and g.  □

g ◦ f reads as g composed with f, or as f followed by g.

For example, consider the function f: ℕ ⟶ ℤ given by f(n) = n − 3, and the function g: ℤ ⟶ ℤ given by g(x)

(g ∘ f) (n) = g(f(n)) = g(n − 3) = (n − 2)2 = n2 −3n + 9.

Notice that though the codomain of g is the set ℤ its range is

Range(g) = {n2 | n ∈ ℤ} ⊂ ℕ.

so we can also define the composite f ∘ g, where

(f ∘ g)(n) = f(g(n)) = f(n2) = n2 − 3.

Notice how

f ∘ g ≠ g ∘ f

Esempio Let f(x) = x2 and g(x) = sin (x). The composite function g ∘ f is

h(x) = g[f(x)] = sin(x2)

Since g is well-defined over all ℝ, also h is. The function k = f ∘ g is also well defined by the relation

k(x) = f[g(x)] = [sin(x)]2

The composition operation is associative, that is if f, g, and h are functions respectevely from A to B, and from B to C from C to D, then it results also

h ∘ (g ∘ f) = (h ∘ g) ∘ f

Inverse Functions

Definition 1.5.2. Let X and Y be two nonempy sets, and f: X ⟶ Y be a function. We say g: Y ⟶ X is the inverse function of f if

g ∘ f = idX  and  f ∘ g = idY

In this case, we write g = f−1. We say a function f: X ⟶ Y is invertible if it has an inverse.  □

where idX is the identity function on the set X; that is, the function that leaves its argument unchanged.

As an example, consider the real-valued function of a real variable given by f(x) = 5x − 8. One can think of f as the function which multiplies its input by 5 then subtracts 8 from the result. To undo this, one adds 8 to the input, then divides the result by 5. Therefore, the inverse of f is the function f−1: ℝ → ℝ defined by f−1(y) = (y + 8)/ 5.
