Cyclotomic polynomials

Recall that the roots of unity for some integer n (i.e. the solutions of xn − 1 = 0) are the n complex numbers ζn,0, ζn,1, ..., ζn,n-1, where

ζn,k = cos 2πk/n + i sin2πk/n,   0 ≤ k < n

Definition 4.25. We define the order of the n-th roots of unity, ζ, the smallest integer m such that ζm = 1 and ζh ≠ 1 ∀ h = 1,...,m-1.

Lemma 4.26. Let ζ a root of unity such that for some t > 0, t ∈ ℕ, ζt = 1. Then the order m of ζ is a divisor of t.

Proof. If m divides t, then t = mc for some integer c, and since ζm = 1 by the definition of order, it readily follows from ζt = (ζm)c = 1.□

Definition 4.27. A n-th primitive root of unity is an n-th root of unity of order n.

We'll indicate by ζn the n-root of unity corresponding to k = 1, i.e. ζn = ζn,1 = cos (2π/n) + i sin(2π/n). It is a root of unity because for (h = 1, ..., n-1) by De Moivre formula

ζnh = cos (2hπ/n) + i sin(2hπ/n) = ζn,h

which is different from 1. Raising ζn to the power of h leads to all the other roots of unity.

Proposition 4.28. If ζ is a n-root of unity different from 1.

1 + ζ + ζ2 + ... + ζn−1 = 0

Proof. We have

(xn − 1) = (x − 1) (xn−1 + xn−2 + ... + x + 1)

letting x = ζ ≠ 1, yields to

(ζn−1 + ζn−2 + ... + ζ + 1) = 0.□

We've found a single n-th primitive root of unit, and now we want to determine them all. For this task we calculate the order of all n-th roots and isolate the ones which have n as order. We start by an arbitrary n-th primitive root of unit and we'll calculate the order of all its powers.

Proposition 4.29. Let ζ be a primitive n-th root of unity and let k be an integer, then ζk has order n/(n,k).

Proof. let δ = GCD(n,k), k = δk1 and n = δn1, then (n1,k1) = 1. We want to determine the order m = m(k) of ζk. We have

(ζk)n1 = (ζk1δ)n/δ = (ζn)k1 = 1

hence

m | n1   (4.30)

We have also

ζkm = (ζk)m = 1

being ζ a primitive n-th root of unity and hence of period n it follows that n | km ⇒ n1δ | n1 δm, which yields to n1 | k1m. Since (n,k1) = 1

n1 | m   (4.31)

Both (4.30) and (4.31) imply that m = n1 = n/δ.□

Corollary 4.32. The primitive n-th roots of unity are

ζn,k = cos 2πk/n + i sin2πk/n = ζnk

with 0 ≤ k < n and (n,k) = 1 and hence they are Φ(n), with Φ Euler's totient function.

Proof. Let ζ be a primitive n-th root of unity. If ζk is primitive then (ζk)n = 1. This implies that from

(ζk)n/δ = 1

ζk is primitive if and only if δ = 1. Moreover, if k ≥ n, ζn,k would be one of the roots with index less than k given the periodicity of sin and cos.□

Corollary 4.33. We've seen that the powers of ζn,1 are all the primitive n-th roots of the unit. This is true for any primitive n-th root of the unit, which can be used to generate all n-th roots of the unit.

Consider the polynomial

Φn(x) = (x − ζ1) (x - ζ2) ⋅⋅⋅ (x − ζφ(n)

This polynomial is known as n-th cyclotomic polynomial, because of their relation with the division of the circle; ζ1, ζ2, ..., ζn are the φ(n) primitive n-th roots of unit. It is evident that the degree of Φn is Φ(n). Any of the φ(n) primitive nth roots of unity generate all nth roots of unity (primtive or not) so the ζi that appear in the cyclotomic polynomial can be expressed as ζi for i such that 1 ≤ i < n.

Moreover for every divisor d of n there exists a n-th root of the unit having d as order (consider ζn/d and see exercise 3. Every n-th root of unity is then an d-th primitive root of the unity with d its order dividng n. We can group all n-th roots of the unit based on their order d and indicate with Φd(x) the d-th cyclotomic polynomial (whose roots are all the dth primitive roots of the unity).

Example 4.34 - Consider x4 − 1. There are four roots {1,-1,i,-i}; 1 and -1 are not primitive roots of the unity, because by definition of order even if 14 = 1 or -14 = 1 we have 12 = 1 and -12 = 1. Only i and −i are primitive because no power smaller than 4 gives im = 1, or (−i)m = 1. So that

Φ4 (x)= (x-i) (x+i) = x2 + 1

The polynomial xn − whose roots are all the n-th roots of the unit can be factorized as

xn − 1 = ∏d|n Φd(x)   (4.35)

this relation yields to

n = ∑d|n Φ(d)

which is a property of Euler's totient function and can be demostrated without resorting to cyclotomic polynomials.

Example 4.36 - When n = 2, the only primitive square root of unit is −1 (note that 1 has order 1, i.e. not primitive), so that Φ2(x) = x + 1. When n = 4, the primitive fourth roots of unity are i and -i, so that

Φ4 = (x−i) (x+i) = x2 + 1

Since Φ1 = x −1, we get the factorization

xn − 1 = Φ1(x) Φ2(x) Φ4(x) = (x −1) (x +1) (x2 +1)

Example 4.37 - Cyclotomic polynomials have the following properties:

  1. every Φn(x) is monic with integers coefficients;

  2. the degree Φn(x) is Φ(n);

  3. every Φn(x) is irreducile over ℚ;

Proof. a) We proceed by induction over n. for n = 1, Φ1(x) = x − 1 which is monic and with integers coefficients. We suppose the same to hold true for Φm(x) with m< n and we prove it for Φn(x). We have

Φn(x) = xn − 1 / ∏d|n,d≠ n Φd(x)

Dividing xn − 1 by ∏d|n,d≠ n Φd(x), we obtain a monic polynomial with integer coefficients.

b) It follows from the construction of Φn(x.)

c) We'll prove this property only for n = ph, with p prime number. As we have already seen

Φph(x) = 1 + xph−1 + (xh−1)p−1 = xph − 1 / xph− 1 − 1

To prove the irreducibility of Φph we canno apply direclty Eisenstein criterion. We make the substitution x = y +1 which yields to a neq polynomial in y

Ψ(y) = Φph (y+1) = 1 + (y + 1)ph−1 + ... + ((y+1)ph−1)p−1 = (y+1)ph − 1 / (y+1)ph− 1 − 1 = yph + pf(y) / yph−1 + pg(y)

We employed Newton binomial formula and grouped the addedends multiple of p in pf(y) and pg(y). We have then

[1 + (y + 1)ph−1 + ... + ((y+1)ph−1)p−1][yph−1 + pg(y)] = yph + pf(y)

by grouping the highest degree terms in each polynomial, we obtain

[(y + 1)ph−1 + ∑ αi yi] [yph−1 + pg(y)] = yph + pf(y)

doing the calculations and grouping the terms which are multiple of p yields

yph−1 ∑ αi yi + yph−1(p−1) ⋅ yph−1 = yph + pf̄(y)

in this form we are sure that all coefficients αi different from the leading coefficients (which is 1) of Ψ(y) are multiples of p. To apply Eisenstein criterion we only need to prove that p2 does not divide the constant term. This is true since theconstant term of Ψ(y) is obtained by putting y = 0, and hence it is equal to p times 1 i.e. p. We have proved trhat Ψ(y) is irreducible and thus Ψh(y) as well.

This result tells us that cyclotomic polynomials are irreducible over ℚ, and we can use them to find the factorization of xn-1. We have

xn − 1 = ∏d|n Φd(x)   (4.35)

From the cyclotomic polynomials we've written above it may seem that their coefficient are all either 0 ri ±1. This is not true as it can be shown that for polynomials with higher n (about 100) other coefficients start to appear.
