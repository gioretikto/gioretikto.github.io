Continuous Functions

Continuous functions represent key tools for comparing topological spaces and for identifying their topological properties. Suppose f is a function from one topological space X to another topological space Y. Our first attempt to define a continuos function could be to require that f takes each open set in X to another open set in Y, but looking at continuous functions from calculus as a guide we see that f(x) = x2 does not take open intervals around 0 to open sets in ℝ (e.g. take (−1, 1) then f( (−1, 1) ) = [0, 1). However, our experience with continuous functions on ℝ gives us some indication of what features we want for continuous functions between topological spaces; the ε-δ definition definition starts with an opens interval in the codomain with ε > then it finds an open interval in the domain (with δ > 0) that is f : ℝ → ℝ is continuous if and only if for each a ∈ ℝ and each interval (f(a) − ε, f(a) + ε), for ε > 0 there exists a δ > 0 such that f(x) ∈ (f(a) − ε, f(a) + ε) for all x ∈ (a − δ, a + δ). At a point x this means that f maps the ε-neighbhorood to the δ-neighbhorood. This definition gives us the motivation for our definition of continuous functions betwee topological spaces; The topological definition of a continuous function is in terms of its inverse function.

Definition 3.1.1 (Continuous Functions). A function f : (X,τX) → (Y,τY) is said to be continuous if and only if for every open set U in Y, f−1(U) is open in X. In other words, a function f is continuous if f−1 maps open sets into open sets.

To check continuity, you need to check all opens in the codomain, and see if thery are mapped by f−1 to open sets.

Continuity expressed in terms of open sets is more general and applied to functions defined on topological spaces; Definition 3.1.1 agrees with that introduced in calculus for metric spaces, however it does not require to introduce a metric.

Lemma 3.1.2. Let f : S → T be a function between two topological spaces. Then f is continuous if, and only if, f−1 (U) is closed whenever U is a closed subset of T.

Proof First let us assume that f is continuous, i.e., f−1 (V) is open whenever V is an open subset of T. If U ⊂ T is closed, then T − U is open, so f−1 (T − U) is open. However, f−1 (T − U) = S − f−1 (U), so the complement of f−1 (U) is open, i.e., f−1 (U) is closed.

For the converse, supposing f is continuous we have

f−1(U) is open for all open U  ⇐⇒  f−1(Uc) is open for all closed U  ⇐⇒  [f−1(Uc)]c is closed for all closed U.  □

Proposition 3.1.3. (Intermediate Value Theorem) Let (X,τ) and (Y,τ') be topological spaces and f : (X, τ) → (Y, τ') surjective and continuous. If (X, τ) is connected, then (Y, τ') is connected.

Proof. Suppose (Y,τ') is not connected. Then it has a clopen subset U such that U ≠ Ø and U ≠ Y . Then f−1 (U) is an open set, since f is continuous, and also a closed set, by Lemma 3.1.2; that is, f−1 (U) is a clopen subset of X. Now f−1 (U) ≠ Ø as f is surjective and U ≠ Ø. ..
