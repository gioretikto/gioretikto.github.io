The Euclidean Topology

Order Topology

Topologies can also be defined with respect to a total order. In general for any totally ordered set X, given a linear order ≤ o a set X, define the open intervals of X to be the subsets

(a,b) = {x ∈ X | a < x < b}
(−∞,a) = {x ∈ X | x < a}
(a,∞) = {x ∈ X | x > a}
(−∞,∞) = X

for a,b ∈ X, and define the open sets of X to be the unions of arbitrary families of open intervals. The family of sets so defined is a topology on X, called the order topology.

The usual linear ordering of ℝ generates an order topology in which open sets are unions of open intervals (a,b). This topology coincides with the metric space topology (or the Euclidean topology) over ℝ. In metric spaces, an open set is defined as a set which contains an open ball around each of its points. Every metric space defines a topology, called a metric topology.

Proposition 1.3.1. The set of open intervals (a,b) ⊂ ℝ constitutes a topology on ℝ, called the standard topology, τℝ, for −∞ ≤ a ≤ b ≤ ∞, for a,b ∈ ℝ.

Proof. Since one can write ℝ as (−∞, a) ∪ (a − 1, ∞) for any a ∈ ℝ, this allows all of ℝ to be τℝ. And if we let a = b, then the element (b,b) = ∅ is also in τℝ. The union of any collection of open intervals is certainly an open set also.
The intersection of two open intervals is either ∅ or an open interval.  □

It is important to note that while every open interval is an open set in ℝ, the converse is false. Not all open sets in ℝ are intervals. For example, the set (1, 4) ∪ (7, 9) is an open set in ℝ, but it is not an open interval.
