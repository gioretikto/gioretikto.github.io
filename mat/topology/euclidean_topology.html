The Euclidean Topology

Order Topology

Topologies can also be defined with respect to a total order.

Definition 1.2.1. Let X a totally ordered set, and ≤ linear order on X, define the open intervals of X to be the subsets

(a,b) = {x ∈ X | a < x < b}
(−∞,a) = {x ∈ X | x < a}
(a,∞) = {x ∈ X | x > a}
(−∞,∞) = X

for a,b ∈ X, and define the open sets of X to be the unions of arbitrary families of open intervals. The family of open sets so defined is a topology on X, called the order topology.

The usual linear ordering of ℝ generates an order topology on ℝ, known as Euclidean topology, in which open sets are unions of open intervals (a,b). This topology coincides with the metric space topology (or the Euclidean topology) over ℝ. In metric spaces, an open set is defined as a set which contains an open ball around each of its points. Every metric space defines a topology, called a metric topology.

Proposition 1.3.1. The set of open intervals (a,b) ⊂ ℝ is a basis, ℬ, for the euclidean topology of ℝ, τℝ. The “Euclidean topology” is a topology.

Proof. We have to verify the topology axioms in Definition 1.1.1. Since one can write ℝ as the interval (−∞, +∞), ℝ ∈ τℝ. Ø is an empty union of members of ℬ and so Ø ∈ τℝ.
To verify ii) and iii) in Definition 1.1.1 let

A = ⋃i ∈ I (ai, bi),   B = ⋃j ∈ J (cj, dj),  ...,  Z = ⋃k ∈ K (xk, yk)

a family of open sets; Then to prove ii), notice that

A ∪ B ∪ ... ∪ Z = (⋃i ∈ I (ai, bi)) ∪ (⋃j ∈ J (cj, dj)) ∪ (⋃k ∈ K (xi, yi)) = ⋃i ∈ I, j ∈ J, k ∈ K (ai, bi) ∪ (cj, dj) ∪ ... ∪ (xk, yk)

still a union of open intervals (and can be generalized to unions of infinite open sets). For iii) by distributive law the intersection of two open sets is

A ∩ B = (⋃i ∈ I (ai, bi)) ⋂ (⋃j ∈ J (cj, dj)) = ⋃i ∈ I, j ∈ J (ai, bi) ∩ (cj, dj)

is still a union of open intervals. Thus τℝ is indeed a topology on ℝ.  □

We call this topology the Euclidean topology or the standard topology or as well the usual topology on the real line.

It is important to note that while every open interval is an open set in ℝ, the converse is false. Not all open sets in ℝ are intervals. For example, the set (1, 4) ∪ (7, 9) is an open set in ℝ, but it is not an open interval.

So the collection of open intervals in ℝ is a basis for the Euclidean topology of ℝ. An open set in ℝ is obtained by forming an arbitrary union of open intervals, and a closed set in ℝ results from taking the complement in ℝ of such an open set. For example (0,2) and (0, ∞) are open in ℝ, whereas [0,2] and {0} are closed.
