Topology

Definition 1.1.1. Let X be a non-empty set. A collection τ of subsets of X, called open sets is said to be a topology on X if

  1. X, Ø ∈ τ;

  2. the union of any (finite or infinite) number of sets in τ belongs to τ;

  3. the intersection of any two sets in τ belongs to τ.

The pair (X, τ) is called a topological space. A subset A ⊂ X is called closed if its complement X − C is open.  □

Example 1.1.2. Let X = {a, b, c, d, e, f} and

τ = {X, Ø, {a}, {c, d}, {a, c, d}, {b, c, d, e, f}}.

Then τ is a topology on X as it satisfies conditions (i), (ii) and (iii) of Definitions 1.1.1. ■

Example 1.1.3. Let X = {a, b, c, d, e} and

τ = {X, Ø, {a}, {c, d}, {a, c, e}, {b, c, d}}.

Then τ is not a topology on X as the union {c, d} ∪ {a, c, e} = {a, c, d, e} of two members of τ does not belong to τ condition (ii) of Definitions 1.1.1. ■

Example 1.1.4. Let X = {a, b, c, d, e, f} and τ = {X, Ø, {a}, {f}, {a, f}, {b, c, d, e, f}}. Then τ is not a topology on X since the intersection {a, c, f} ∩ {b, c, d, e, f} = {c, f} of two sets in τ does not belong to τ; that is, τ does not have property (iii) of Definitions 1.1.1.  ■

Definition 1.1.5. Let X be any non-empty set and let τ be the collection of all subsets of X. Then τ is called the discrete topology on the set X. The topological space (X, τ) is called a discrete space.

Nota bene. Every subset of the discrete metric space is open as well as closed.

We note that τ in Definitions 1.1.5 does satisfy the conditions of Definitions 1.1.1 and so is indeed a topology.

Observe that the set X in Definitions 1.1.5 can be any non-empty set. So there is an infinite number of discrete spaces – one for each set X.

Definition 1.1.6. Let X be any non-empty set and let τ = {X, Ø}. Then τ is called the indiscrete topology and (X, τ) is said to be an indiscrete space.

Once again we have to check that τ satisfies the conditions of Definitions 1.1.1 and so is indeed a topology.
We observe again that the set X in Definitions 1.1.6 can be any non-empty set. So there is an infinite number of indiscrete spaces – one for each set X.

Proposition 1.1.7. Let (X, τ) be any topological space. Verify that the intersection of any finite number of members of τ is a member of τ.

By definition the intersection of any two open sets gives an open set, so by repeating that finitely many times you end with an open set. For example, let O1, O2, and O3 be open sets. O1 ∩ O2 is open, so (O1 ∩ O2) ∩ O3 is also open. If there is also an O4, then ((O1 ∩ O2) ∩ O3) ∩ O4 is open. This process can be carried on for any finite collection of open sets.

Proof. Let S = {n ∈ ℕ: the intersection of any n members of τ is a member of τ}. It is clear that 1 and 2 belong to S. Suppose n ≥ 2 and {1,2,...,n} ⊆ S. Let {Ui : i ∈ {1,2,...,n+1}} be a collection of members of τ. Since n ∈ S, ⋂ni=1 Ui ∈ τ. Therefore, since 2 ∈ S, ⋂n+1i=1 Ui = (⋂ni=1 Ui) ∩ Un+1 ∈ τ. Thus n + 1 ∈ S, and by the second principle of mathematical induction, S = ℕ.  □

Proposition 1.1.8. If (X, τ) be any topological space, then

  1. Ø and X are closed sets.

  2. The union of any finite number of closed sets is a closed set.

  3. The intersection of any (finite or infinite) number of closed sets is a closed set.

Proof. i)The fact that Ø and X are open sets is a trivial fact derived from definition 1.1.1. They are as well closed sets since the complement of X is Ø and the complement of Ø is X.  □

To prove that (ii) is true, let A1, A2, ... ,An be closed sets. We are required to prove that A1 ∪ A2 ∪ ··· ∪ An is a closed set. It suffices to show, by Definition 1.1.1, that X \ (A1 ∪ A2 ∪ ··· ∪ An) is an open set. As A1, A2, ... , An are closed sets, their complements X \ A1, A2, ... , An are open sets. But

X \ (A1 ∪ A2 ∪ ··· ∪ An) = (X \ A1) ∩ (X \ A2) ∩ ··· ∩ (X \ An).

As the right hand side of (1) is a finite intersection of open sets, it is an open set, by proposition 1.1.7. So the left hand side of (1) is an open set. Hence A1 ∪ A2 ∪ ··· ∪ An is a closed set, as required. So (ii) is true. The proof of (iii) is analogous to (ii). □

As we have seen despite the names, some open sets are also closed sets. Moreover, some sets are neither open sets nor closed sets. The names “open” and “closed” should not lead the reader into error. Indeed, if we consider Example 1.1.2 we see that

  1. The set {a} is both open and closed;

  2. The set {b, c} is neither open nor closed;

  3. The set {c, d} is open but not closed;

  4. The set {a, b, e, f} is closed but not open.

Definition 1.1.9. A subset A of a topological space (X, τ) is said to be clopen if it is both open and closed in (X, τ).

  • In every topological space (X, τ) both X and Ø are clopen.

  • In a discrete space all subsets of X are clopen.

  • In an indiscrete space the only clopen subsets are X and Ø.
