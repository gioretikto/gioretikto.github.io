Connectedness

Definition 1.5.1. A topological space (X,τ) is said to be connected if the only subsets that are both open and closed are X are X and Ø .

Example 1.5.2. Let X = {a, b, c, d, e} and τ = {X, Ø, {b}, {c, d}, {b, c, d}, {a, c, d, e}}, then (X, τ) is not connected as {a, c, d, e} is a clopen subset.  ■

Proposition 1.5.3. Let (X,τ) be any topological space; (X,τ) is not connected (or disconnected) if and only if it has proper non-empty disjoint open subsets A and B such that A ∪ B = X.

Proof. If X = A ∪ B with A and B open and disjoint, then X − A = B and so B is the complement of an open set and hence is also closed. Similarly, A is clopen.  □

So a topological space is said to be connected if it is impossible to partition it into the union of two non-empty disjoint open sets A and B.
