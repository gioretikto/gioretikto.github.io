Metric Spaces

A metric space is a topological space that includes the notion of a metric or distance between its elements. The notion of metric space was introduced in 1906 by Maurice Fréchet and developed and named by Felix Hausdorff in 1914.

Definition 2.1.1. Let X be a non-empty set and d a real-valued function defined on X × X such that for a, b ∈ X:

  1. d(a, b) > 0 and d(a, b) = 0 if and only if a = b;

  2. d(a, b) = d(a, b);

  3. d(a, c) ≤ d(a, b) + d(b, c), [the triangle inequality] for all a, b and c in X.

Then d is said to be a metric on X, (X, d) is called a metric space and d(a, b) is referred to as the distance between a and b.

Definition 2.1.2. The function d : ℝ × ℝ → ℝ given by

d(a, b) = |a − b|,  a, b ∈ ℝ

is a metric on the set ℝ since

  1. |a − b| > 0 , for all a and b in ℝ, and |a − b| = 0 if and only if a = b,

  2. |a − b| = |b − a|

  3. |a − c| ≤ |a − b| + |b − c|. (Deduce this from |x + y| ≤ |x| + |y|.)

We call d the euclidean metric on ℝ.  □
