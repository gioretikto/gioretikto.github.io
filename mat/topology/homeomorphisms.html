Homeomorphisms

We are often interested in recognzing when two sets are equivalent, i.e. as far as set theory is concerned, when there exists a bijective function which maps one set onto the other. Two groups are equivalent, known as isomorphic, if there exists a a homomorphism of one to the other which is one-to-one and onto. Two topological spaces are equivalent, known as homeomorphic, if there exists a homeomorphism of one onto the other. We begin by considering an example:

Example 1.6.1. Let

X = {a, b, c, d, e},
Y = {g, h, i, j, k},
τ = {X, Ø, {a}, {c, d}, {a, c, d}, {b, c, d, e}},
τ' = {Y, Ø, {g}, {i, j}, {g, i, j}, {h, i, j, k}}.

It is intuitive that (X, τ) is homeomorphic to (Y, τ'). The function f : X → Y defined by f (a) = g, f (b) = h, f(c) = i, f (d) = j, and f (e) = k, provides the equivalence.  ■.

Definition 1.6.2. Let (X,τ) and (Y,τ') be topological spaces. Then they are said to be homeomorphic if there exists a function f : X → Y which has the following properties:

  1. f is one-to-one (that is f (x1) = f (x2) implies x1 = x2);

  2. f is onto (that is, for any y ∈ Y there exists an x ∈ X such that f(x) = y);

  3. for each U ∈ τ', f−1(U) ∈ τ;

  4. for each V ∈ τ, f(V) ∈ τ'.

The map f is said to be a homeomorphism between (X, τ) and (Y, τ'). We write (X, τ) ≌ (Y, τ').

Definition 1.6.3. “≌” is an equivalence relation.

Proof. Indeed it is easily verified that it is an equivalence relation; that is,

  1. (X, τ) ≌ (X, τ) (Reflexive);

  2. (X, τ) ≌ (Y, τ') implies (Y, τ) ≌ (X, τ') (Symmetric);

  3. [Observe that if f : (X, τ) → (Y, τ') is a homeomorphism, then its inverse f−1 : (Y, τ') → (X, τ) is also a homeomorphism.]

  4. (X, τ) ≌ (Y, τ') and (Y, τ') ≌ (Z, τ'') implies (X, τ) ≌ (Z, τ'') (Transitive).  □

We shall show now that all open intervals in ℝ are homeomorphic. Length is certainly not a topological property. In particular, an open interval of finite length, such as (0, 1), is homeomorphic to one of infinite length, such as (−∞, 1). Indeed all open intervals are homeomorphic to ℝ.

Definition 1.6.4. Every two non-empty open intervals (a, b) and (c, d) of real numbers are homeomorphic.

Proof. It suffices to show that (a, b) is homeomorphic to (0, 1) and (c, d) is homeomorphic to (0, 1). But as a and b are arbitrary (except that a < b), if (a, b) is homeomorphic to (0, 1) then (c, d) is also homeomorphic to (0, 1). To prove that (a, b) is homeomorphic to (0, 1) it suffices to find a homeomorphism f : (0, 1) → (a, b). Let a,b ∈ ℝ with a < b and consider the function f: (0,1) ⟶ (a,b) given by f(x) = a (1 − x) + bx.
Clearly f : (0, 1) → (a, b) is one-to-one and onto. It is also clear from the diagram that the image under f of any open interval in (0, 1) is an open interval in (a, b); that is,

f (open interval in (0, 1) ) = an open interval in (a, b).
But every open set in (0, 1) is a union of open intervals in (0, 1) and so
f (open set in (0, 1)) = f (union of open intervals in (0, 1))
= union of open intervals in (a, b)
= open set in (a, b).

So condition (iv) of Definition 1.6.1 is satisfied. Similarly, we see that f−1 (open set in (a, b)) is an open set in (0, 1). So condition (iii) of Definition 1.6.1 is also satisfied. Hence f is a homeomorphism and (0, 1) ≌ (a, b) , for all a, b ∈ ℝ with a < b. From the above it immediately follows that (a, b) ≌ (c, d) , as required.  □

Example 1.6.5. Let A be the opne interval (0,1) and let B be the open interval (1, ∞) of all number greater than 1. There is a simpole homeomorphisms f from A to B, namely x ⟶ 1/x for 0 < x < 1. This mapping is continuous on (0,1). The inverse of f is also x ⟶ 1/x but now for x > 1. This example shows that the property of being bounded is not a topological property, for we have two homeomorphic sets, one of which is bounded, but the other is not.  ■

Example 1.6.6. The space ℝ is homeomorphic to the open interval (−1, 1) with the usual topology.

Proof. Define f: (−1,1) ⟶ ℝ by

f(x) = x/(1 − |x|)

We want to show that f is bijective. First we prove injectivity; Let f(x) = f(y), that is, x/(1−|x|) = y/(1−|y|). This can only be true if x and y have the same signs (or one of them is zero, but then the other one also must be zero). Multiplying by (1−|x|)(1−|y|) gives x(1−|y|)= y(1−|x|), i.e., x − x|y| = y − y|x|. But as x and y have the same sign, we have x|y| = y|x|. So, it follows that x = y.

For surjectivity, let z ∈ ℝ be arbitrary. We have to find some x such that f(x)= z, i.e., x/(1−|x|) = z. This is equivalent to x + z|x| = z. If z ≥0, then we try x ≥ 0 for which the equation is (1+z)x = z and so x = z/(1+z), which is in [0,1) and therefore indeed a solution. Let z < 0. Here, we try x < 0, for which the equation is (1−z)x = z with solution x = z/(1−z). This is indeed < 0 and >−1 (i.e., x ∈ (−1,0)). Hence, for each z ∈ ℝ we have found a solution of f(x) = z. So, f is bijective and indeed a homeomorphism.  ■
